User Interface Features

In-place Editable Parameters

The parameters for a trace log session can be modified directly from the Trace Log Session List Window as shown below.  All modifiable parameters are shown with dark text.  Grayed out items cannot be modified.  The number of parameters that can be modified changes with respect to whether the trace log session is running or not.

Modifying a value for a currently active trace log session causes an immediate update to the active trace log session.  Modifying values for a stopped trace log session affects the behavior of the trace log session upon re-start.


 

Dockable Windows

Each of the windows can be “undocked” from the main TraceView frame window by grabbing the tool-bar type “handle” on the left of the window, and dragging it to the desired location.  Such an “undocked” window is shown below:

Double clicking the gripper bar from the window will also cause the window to undock.  Double clicking the title bar of an undocked window will cause it to re-dock back to the main window.

Configurable Columns

The individual columns of any of the dockable windows in TraceView can be dynamically resized.  Columns can also be moved within the window.  Columns may be resized in the usual way, by selecting (with the left mouse button) the divider between two columns and dragging it to the new width.  To move a column, simply click and hold down the left mouse button on the column’s header and drag the column to its new position.

Pausing The Output

The output of the display windows can be paused by selecting a row, or moving the scroll handle on the right of the display.  Traces will continue to be added, but the display will stay on the current line.  Hitting ‘Enter’ will start the display showing the most current line again.

Eliminating Unwanted Output

It is possible to select or deselect the display of any column in any display except the “Group ID / Session Name” column in the Log Session List window which is always displayed. 

 


Right clicking on the column header of any column will bring up a menu for that display window which has all of the columns listed with the ability to check or uncheck the name to add or remove the column from the display.

Column Sorts

Left clicking the header of any column in the display output windows will cause the data in that window to be sorted by the data in the selected column.  Click the header once to sort ascending, click again to sort descending.
