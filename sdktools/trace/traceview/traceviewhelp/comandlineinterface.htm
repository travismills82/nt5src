Command-Line Interface

The TraceView utility is designed to be useful as a command line utility in addition to providing a useful graphical user interface.  TraceView is intended to operate as three command-line tools in one. 

  • A trace logger control tool as described in Logger Control Interface.
     
  • A trace event consumption tool as described in Trace Event Consumption Interface.
  • a trace format extraction tool for parsing program database files described in PDB Parsing Interface.
