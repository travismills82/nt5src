PDB Parsing Interface

The TraceView command-line interface uses the following command-line syntax:

            traceview [action] [options] | [-h | -help | -?]

The possible actions for the TraceView command-line interface are broken down into three categories:

  • Control
  • Consumption
  • Parsing

The Parsing actions are used to extract trace session control GUID, and trace event format GUID information from a program database (.PDB) file.  The following table describes the PDB parsing interface options:

 

 

Parsing Actions:

 

-parsepdb <pdbname>

Extract TMFs from PDB named pdbname

 

 

Parsing Options

 

-p

Specifies the path to create the tmf's, by default the current directory.
