Fjärrhjälp

Använda Fjärrhjälp för att hjälpa till

Ibland när man vill hjälpa någon lösa ett problem är det enklast att visa hur man gör. Fjärrhjälp är ett bekvämt sätt för dig att ansluta till en kompis dator (ni måste båda använda Microsoft Windows XP) så att ni tillsammans kan hitta en lösning.

Följande information gäller för användare som har fått en inbjudan till Fjärrhjälp.

När du har anslutit kommer du kunna se din kompis datorskärm och chatta med varandra i realtid om det ni båda ser. Om din kompis ger tillåtelse kan du till och med använda din mus och ditt tangentbord för att arbeta tillsammans med din kompis på dennes dator. Du bör besvara inbjudan innan den förfallotid som anges inträffat.

Varning! Acceptera bara inbjudningar från personer som du känner och litar på. E-postmeddelanden kan innehålla virus och annat skadligt innehåll.

   Så här kan du acceptera denna inbjudan
 
  1. Öppna den fil som är bifogad i e-postmeddelandet. Kontrollera att filtypen är MsRCincident. Om du tillfrågas om du vill öppna eller spara filen väljer du att öppna den.
  2. Klicka på Ja för att acceptera inbjudan.
  3. När du ansluter till den andra datorn måste du eventuellt ange ett lösenord. Kontakta den person som skickade inbjudan om du inte känner till lösenordet.

Om avsändaren är ansluten till Internet och tillåter att anslutningen upprättas kommer ni kunna chatta online och du kommer att kunna se precis det din kompis ser på sin dator. Den person du hjälper kan aldrig se eller kontrollera din dator.

Anmärkningar

  • Både du och den person du hjälper måste vara anslutna till Internet hela den tid som ni använder Fjärrhjälp.
  • Om ni finns på ett lokalt nätverk kanske nätverkets brandvägg förhindrar att ni använder Fjärrhjälp. I sådant fall bör ni kontakta nätverksadministratören och be om ytterligare information.
  • Om du vill läsa mer om Fjärrhjälp kan du klicka på Start i Windows XP, klicka på Hjälp och support och sedan skriva Fjärrhjälp i fältet Sök.
