远程协助

通过远程协助提供帮助

有时，帮助别人解决问题的最佳办法是演示解决方式。远程协助便于您连接到朋友的电脑(操作系统都必须是 Microsoft Windows XP)，一起解决问题。

以下信息适用于收到远程协助邀请的用户。

连接后，您能看到朋友的计算机屏幕并能联机进行实时聊天，谈论两人都能看见的东西。通过您的朋友的许可，您甚至可以用您的鼠标和键盘，跟您的朋友一起在计算机上进行操作。请在邀请中列出的过期时间之前响应。

注意只接受您知道并信任的人发出的邀请。电子邮件可以含有病毒或其他有害附件。

   要接受这个请求，
 
  1. 打开电子邮件带的文件。请确定其类型是 MsRCincident。如果得到打开或保存的提示，请将其打开。
  2. 要接受邀请，请单击“是”。
  3. 得到连接到发送人的计算机上时，如果需要，请键入密码。如果不知道密码，请跟发送人联系。

如果发送人处于联机状态并接受您的连接，您将能够联机交谈并查看发送人的计算机桌面。您帮助的人无法在任何时候看到或控制您的计算机。

注释

  • 使用远程协助时，您和您正在帮助的人都必须连接到 Internet。
  • 如果在局域网上，防火墙可能会使您无法使用远程协助。在这种情况下，请跟网络管理员联系。
  • 要进一步了解远程协助，请单击 Windows XP 中的「开始」，单击“帮助和支持”，然后在搜索框中键入“远程协助”。
