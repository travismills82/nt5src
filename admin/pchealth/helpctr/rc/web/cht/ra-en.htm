遠端協助

使用遠端協助來提供幫助

有時候幫助別人解決問題的最好方法就是示範解決問題的方法。遠端協助可以方便使用者連線到朋友的電腦 (兩人都必須執行 Microsoft Windows XP) 向朋友逐步示範解決方法。

以下資訊適用於已經收到遠端協助邀請的使用者。

連上線後，您將可以檢視您的朋友的電腦螢幕，於線上即時交談你們所看到的問題。在取得您朋友的許可之後，您甚至可以使用您的滑鼠和鍵盤，和您的朋友一起解決他或她的電腦問題。請在邀請期限過期之前回應。

注意 請只接受您所認識和信任的人所發出的邀請。電子郵件訊息可能包含病毒或其他有害的附件。

   要接受這項要求的話
 
  1. 開啟附在電子郵件訊息中的檔案。請先確定它的類型是 MsRCincident。如果收到開啟或儲存的提示，請務必開啟它。
  2. 按「是」接受邀請。
  3. 收到連線到寄件者電腦的提示時，如果必要，請輸入密碼。如果您不知道密碼，請向寄件者詢問。

如果寄件者在線上並接受您的連線，您將可以與他/她在線上即時聊天並檢視他/她的電腦桌面。接受您幫助的使用者無論何時都無法看到或控制您的電腦。

注意事項

  • 使用遠端協助時，您和接受您幫助的人，都必須連線到網際網路。
  • 如果您是在區域網路上，防火牆可能會防止您使用遠端協助。如果是這樣的話，請與您的網路系統管理員連絡，取得詳細資訊。
  • 如果您想瞭解關於遠端協助的詳細資訊，請按 Windows XP 的「開始」，然後按「說明及支援」，在搜尋方塊中鍵入「遠端協助」。
