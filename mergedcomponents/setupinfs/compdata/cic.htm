The Handwriter version 2.0 software by CIC is not compatible with this version of Microsoft Windows.

The tablet will no longer function, and the following error message will appear each time the system is started:

CIC Could Not Set Priority

This error does not prevent Windows from being installed, and you can uninstall the CIC Handwriter software after Windows Setup is complete.

For more information about this device, or to obtain a version of the software that runs under this version of Windows, visit the manufacturer's Web site. Web addresses can change, so you might be unable to connect to this Web site.

For a list of devices supported by this version of Windows, see the list of compatible hardware on the Microsoft Web site.
