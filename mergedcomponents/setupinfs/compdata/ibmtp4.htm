The currently installed driver for TrackPoint is not compatible with this version of Microsoft Windows and will be disabled during the upgrade. You should uninstall this device for a more complete solution.

To uninstall this device

  1. Click Start, point to Settings, and then click Control Panel.
  2. Double-click Add/Remove Programs.
  3. On the Install/Uninstall tab, click IBM TrackPoint Support.
  4. Click Add/Remove, and then click OK.
  5. Restart the computer, and then restart Windows Setup.

For more information about this driver, visit the manufacturer's Web site. Web addresses can change, so you might be unable to connect to this Web site.

For a list of devices supported by this version of Windows, see the list of compatible hardware on the Microsoft Web site.
