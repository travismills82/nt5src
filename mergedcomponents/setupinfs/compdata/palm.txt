HotSync Manager, a component that synchronizes 3Com Palm computing devices with your computer, allows Palm devices to synchronize with a network. After upgrading to to this version of Microsoft Windows, network synchronization will fail because the device will have problems locating its primary computer.

To solve this problem, synchronize the device through the serial port before attempting a network synchronization.

For more information, visit the manufacturer's Web site. 

For a list of devices supported by this version of Windows, see the Microsoft Windows Compatibility List at http://go.microsoft.com/fwlink/?LinkId=9890.
