The PROBEPRO software by Fujitsu is not compatible with this version of Microsoft Windows.  Make sure that the software is uninstalled before Windows upgrade.

1. Click Start, point to Programs, point to PROBEPRO V1.0LXX, and then click Uninstall PROBEPRO.
2. Select 'Yes' for the PROBEPRO removal prompt.
3. After uninstallation is completed, restart your computer, and then continue with Windows Setup.

For more information about this software, visit the manufacturer's Web site.
