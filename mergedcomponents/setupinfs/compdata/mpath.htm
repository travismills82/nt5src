The software for Siemens MultiPath is not compatible with this version of Microsoft Windows and will be disabled during the upgrade. Drivers for this version of Windows do not support multiple controllers for a single hard disk.

You must uninstall this software and reconfigure your system before upgrading to this version of Windows.

  1. Click Start, point to Settings, and then click Control Panel.
  2. Double-click Add/Remove Programs, and then uninstall the MultiPath software.
  3. When you are prompted, restart the system.
  4. Reconfigure your system with a single controller attached to each hard disk storage unit.
  5. Restart the Windows installation.

For more information about this software, visit the manufacturer's Web site.

For a list of software supported by this version of Windows, see the list of compatible software on the Microsoft Web site.
