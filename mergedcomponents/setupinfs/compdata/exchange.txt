The software for Microsoft Exchange Server 5.5 Standard Edition and Microsoft Exchange Server 5.5 Enterprise Edition is not compatible with this version of Microsoft Windows. You must install Exchange Server 5.5 Server Pack 3 or later in order for these programs to run under this version of Windows.

For more information about this software and to download Exchange Server 5.5 Service Pack 3 or later, visit the manufacturer's web site at http://www.microsoft.com/exchange.

For a list of software supported by this version of Windows, see the Microsoft Windows Compatibility List at http://go.microsoft.com/fwlink/?LinkId=9946.
