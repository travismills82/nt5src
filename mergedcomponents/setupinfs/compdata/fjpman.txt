"Fujitsu Power MANagement for Windows" is not compatible with this version of Microsoft Windows and will be disabled during the upgrade process.

Make sure "Power MANagement for Windows" for Windows NT 4.0 is uninstalled before Windows upgrade.

To uninstall this software:

1. Select "Power MANagement for Windows" uninstall program.
2. After restarting your system, you can run Windows Setup to install Windows on your system.
