Setup detected that the Cluster service is already installed on this server. This version of Windows requires that the Cluster Service Account be granted the "Act as part of the operating system" user right. Setup determined that the Cluster Service Account does not have that right. You must grant the "Act as part of the operating system" right to the account, then restart Setup to continue upgrading this node in the cluster.

For security reasons, you must grant this user right to the specific account that is used by the Cluster Server service. You cannot correct this problem by granting the user right to a security group of which the service account is a member. Typically, you must grant this user right as a local user right; it cannot be a domain-level user right. However, if your node is a domain controller, you can use the domain level user right.

For more information on setting user rights, refer to "To edit a security setting" in the Windows 2000 help documentation, or on Windows NT 4.0, open User Manager for Domains, click the Help menu in User Manager, and refer to "Managing the User Rights Policy."
