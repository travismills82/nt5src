Microsoft Windows Setup has detected an Iomega Zip drive connected to the parallel port on your computer. Some installed tools for the Iomega Zip drive will not work properly after upgrading to this version of Windows if your Iomega Zip drive is connected to the parallel port on your computer.

Although this version of Windows does contain embedded support for the Iomega Zip drive, the drivers for this device will be disabled during the upgrade. It is recommended that you uninstall the Iomega Zip drive tools before continuing.
		 
For more information about this driver, visit the manufacturer's Web site.

For a list of devices supported by this version of Windows, see the Microsoft Windows Compatibility List at http://go.microsoft.com/fwlink/?LinkId=9890.
