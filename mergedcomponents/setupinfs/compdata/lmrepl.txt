The software for Directory Replicator Service (LMRepl) is not compatible with this version of Microsoft Windows and will be disabled during the upgrade. Directory Replicator Service is replaced by the File Replication System in this version of Windows.

For more information, see the "Determining Domain Upgrade Strategies" chapter in the Deployment information for this version of Windows, which is part of the Resource Kit for this version of Windows and available on the CD-ROM.

For a list of software supported by this version of Windows, see the Microsoft Windows Compatibility List at http://go.microsoft.com/fwlink/?LinkId=9946.
