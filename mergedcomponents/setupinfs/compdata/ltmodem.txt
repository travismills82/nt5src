The currently installed driver for the Lucent Technologies Modem is not compatible with this version of Microsoft Windows and will be disabled during the upgrade. If your modem is supported by this version of Windows, a new driver will be installed during Windows Setup. However, it is recommended that you uninstall your modem driver and software before continuing the upgrade. 

For more information about this driver, visit the manufacturer's Web site.
