A conflict occurs during an upgrade to this version of Microsoft Windows when Equis MetaStock 6.5 is installed on Microsoft Windows NT 4.0 (with Service Pack 3 or below) and a password is created for the Downloader for Windows program. You should delete the Downloader password before continuing with the upgrade.

To delete the Downloader for Windows password

  1. Open the Downloader for Windows program.
  2. Under Tool, click Vendor Options.
  3. Delete the user name and password, and then click OK.

You can enter a new password after the upgrade is complete.

For more information about this software, visit the manufacturer's Web site.

For a list of software supported by this version of Windows, see the list of compatible software on the Microsoft Web site.
