The currently installed drivers for the Creative Video Blaster WebCam Go and Go Plus cameras are not compatible with the Microsoft Windows upgrade process. The camera MUST BE UNPLUGGED during the upgrade.

For more information about these drivers, visit the manufacturer's Web site.

For a list of devices supported by this version of Windows, see the Microsoft Windows Compatibility List at http://go.microsoft.com/fwlink/?LinkId=9890.
