WARNING!

Read before continuing with this upgrade!

Windows upgrade process has detected that this is a system which requires a custom HAL. Customers must have the custom HAL diskette from their OEM to support this system in order to proceed with this upgrade.

If the custom HAL is not available, please cancel the upgrade until it is available.

If the custom HAL is available on floppy, then please continue. After the copying of the installation files, the system will restart. After the system restart, Windows Setup will also start again. At this point a text message will appear near the bottom of the screen: "Press F6 if you need to install a third-party SCSI or RAID driver". You MUST press F5 and follow the directions that will appear on the screen. (You can also press the F6 key if necessary for third-party SCSI or RAID driver).

For more information, contact your OEM representative.
