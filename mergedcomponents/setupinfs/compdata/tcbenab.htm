The enabler for Toshiba Cardbus LANCard (LANCB00A) on your system is not compatible with this version of Microsoft Windows, and most of the Cardbus feature will be provided by Windows itself. Therefore this software must be uninstalled before the upgrade can continue.

To uninstall this software

  1. Click Start, and then click Run.
  2. In Run, type regedt32, and then click OK.
  3. Select HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services.
  4. Delete CBTSB Key.
  5. After restarting your system and logging back on, run Windows Setup to install Windows on your system.

For more information about the extension of power management on this version of Windows, visit the Toshiba Web site.
