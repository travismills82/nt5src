Your existing fax server application will not work after upgrading to a Microsoft Windows Server 2003 operating system. Uninstall this application using Add/Remove Programs in Control Panel, and then restart the upgrade.

During the upgrade, Windows Server 2003 Fax will be installed automatically.  Existing fax settings are saved and are available after the upgrade. 

For a list of software supported by this version of Windows, see the Microsoft Windows Compatibility List at http://go.microsoft.com/fwlink/?LinkId=9946.
