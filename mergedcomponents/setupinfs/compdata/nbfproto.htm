The currently installed driver for the NETBEUI Transport Protocol is not compatible with this version of Microsoft Windows and will be uninstalled during the upgrade. This protocol is removed from this new version of Windows.

For more information about this driver, visit the manufacturer's Web site. Web addresses can change, so you might be unable to connect to this Web site.

For a list of protocols supported by this version of Windows, see the list of compatible hardware on the Microsoft Web site.
