During setup the drivers currently installed for Yamaha AC-XG NT4 Audio Drivers will be replaced with an updated version that is compatible with this version of Microsoft Windows. The previous version will be disabled during the upgrade. It is recommended that you uninstall this software from Add/Remove Programs in Control Panel before continuing with the upgrade.

For more information about this software, visit the manufacturer's Web site.

For a list of devices supported by this version of Windows, see the list of compatible hardware on the Microsoft Web site.
