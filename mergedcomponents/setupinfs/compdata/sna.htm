The software for Microsoft SNA Server is not compatible with this version of Microsoft Windows. You must install SNA Server 4.0 Service Pack 3 for this program to run under this version of Windows.

For more information about this software, and to download SNA Server 4.0 Service Pack 3, visit the Microsoft SNA Server page at http://www.microsoft.com/sna. Web addresses can change, so you might be unable to connect to the Web site mentioned here.

For a list of software supported by this version of Windows, see the list of compatible software on the Microsoft Web site.
