Access to Internet Information Services (IIS) Manager is restricted to Localhost

As a security precaution, Setup restricts IIS Manager to allow access only from the server itself (Localhost, IP address 127.0.0.1).

Please try the following:

To enable remote use of IIS Manager to administer this server, go to the server and use IIS Manager to update the IP address restrictions for the Web site IISADMIN.

Important   IP address restrictions can only be modified on the server if it is running IIS on the Windows 2000 Server or Windows 2000 Advanced Server platform. IP address restrictions cannot be modified by using IIS on the Windows 2000 Professional platform.

In IIS Manager:

  1. Expand the Web site you are connecting to.
  2. Right-click the IISADMIN virtual directory, and click Properties.
  3. Click the Directory Security tab.
  4. Edit the IP address and domain name restrictions.
