Select Computer Overview

Use Select Computer to select the computer that hosts the component you want to connect to.

    To Do this
    Manage a UDDI Services component on the computer on which this console is running Click Local computer (the one this console is running on).
    Manage a UDDI Services component on a remote computer Click Another computer, and then type the name or IP address of the remote computer hosting the UDDI Services component.
