Use this search page to locate services with bindings to instances of tModels that are categorized with the following categorization:

Categorization Scheme: uddi-org:types
tModelKey: C1ACF26D-9672-4404-9D70-39B756E62AB4
Key Name: Specification for a web service described in WSDL
Key Value: wsdlSpec

This categorization marks a Web service as having an exposed interface and associated WSDL description allowing you to quickly import its supported functions and attributes into your application or Web service. For more information about this categorization, and other UDDI Services information, see Additional Resources on the Web.

Use any one of the search options available to locate Web services matching your search criteria.

  1. Search by Service name

    Allows you to search this UDDI Services site for Web services by the name of the service. Type all or part of the name of the service you are searching for, and click Search. Use % as a wildcard character.

  2. Search by Provider name

    Allows you to search this UDDI Services site for Web services by the name of their provider. Type all or part of the name of the provider you want to search by, and click Search. Use % as a wildcard character.

  3. Select a categorization scheme

    Allows you to search this UDDI Services site for Web services and providers by a selected categorization. Select the categorization scheme and categorization you want to search by. The categorizations available are those categorizations that have been marked as browseable in this installation of UDDI Services. Contact your UDDI Services coordinator or administrator if no categorizations are available.

    Once you have selected a categorization, click Search to search for providers or services that have been categorized by your selected categorization and the WSDL interface categorization described earlier.
