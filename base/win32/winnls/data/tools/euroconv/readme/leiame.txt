Ferramenta de conversão para o Euro
------------------------------------

Descrição:
   A 'Ferramenta de conversão para o Euro' é uma ferramenta simples
   que permite a migração de unidades monetárias para o Euro nos
   países membros da zona Euro. Esta ferramenta funciona com o
   Windows 95, Windows 98, Windows NT4, Windows 2000 e Windows XP. 

   Esta ferramenta de conversão funciona para aqueles utilizadores
   que tenham a sua configuração regional ('Padrões e formatos' no
   Windows XP) definida para um país membro da zona Euro. Esta
   ferramenta, por predefinição, não irá mudar as definições de 
   unidade monetária para os utilizadores com definições regionais 
   fora da zona Euro.  
   
   Para o Windows 95, Windows 98 e Windows NT4, necessita de executar
   um patch anterior para concluir a migração. Este patch incluí 
   actualizações a tipos de letras, teclados e páginas de código. 
   Este patch está disponível no website da Microsoft em
   http://www.microsoft.com/Windows/Euro.asp.

   As alterações, por predefinição, são aplicadas a todos os utilizadores.
   Contudo, pode aplicar um parâmetro que permite fazer a alteração apenas 
   para o utilizador actual (ver mais abaixo o parâmetro).

Utilização:
   O utilitário euroconv.exe aceita os seguintes parâmetros na 
   linha de comandos:

   [nada]
   Exemplo: euroconv.exe
   Função:  Muda a unidade monetária para o Euro para
            todos os utilizadores.

   /?
   Exemplo: euroconv.exe /?
   Função:  Mostra a utilização.

   /[h/H]
   Exemplo: euroconv.exe /h
   Função:  Mostra a utilização.

   /[c|C]
   Exemplo: euroconv.exe /c
   Função:  Muda a unidade monetária para o Euro apenas para
            o utilizador actual.

   /[s|S]
   Exemplo: euroconv.exe /s
   Função:  Muda a unidade monetária para o Euro para todos
            os utilizadores de forma silenciosa (sem caixas
            de diálogo).

   /[s|S] /[c|C]
   Exemplo: euroconv.exe /c /s
   Função:  Muda a unidade monetária para o Euro  para
            o utilizador actual de forma silenciosa (sem caixas
            de diálogo).

   /[a|A]:w1, "x1",”y1”,”z1”;x2,”y2”,”z2”;…;wn, "xn",”yn”,”zn”
   Exemplo: euroconv.exe /a:0x00000807,”,”,”2”,”.”
   Função:  Muda a unidade monetária para o Euro para todos os 
            utilizadores que cumpram as condições e adiciona uma definição
            regional se aplicável. Ou seja, a informação de unidade 
            monetária do utilizador também será modificada se a definição
            regional do utilizador for igual a wi. Se a definição regional
            de utilizador (LCID) for igual a wi, o separador decimal
            monetário será modificado para xi, o número de dígitos
            decimais para os formatos de unidade monetária será 
            modificado para yi e o separador monetário dos milhares
            será modificado para zi. (Nota: xi, yi e zi podem todos 
            estar vazios).

            Esta opção pode ser utilizada para adicionar uma definição
            regional à lista de processamento. Esta opção também pode
            ser utilizada para substituir informação para uma determinada
            definição regional.

            (Veja a parte final deste 'Leia-me' para um exemplo de 
            como isto funciona.)

   /[c|C] /[a|A]:w1,"x1",”y1”,”z1”;w2,"x2",”y2”,”z2”;…;wn,"xn",”yn”,”zn”
   Exemplo: euroconv.exe /c /a:0x00000807,”,”,”2”,”.”
   Função:  Muda a unidade monetária para o Euro apenas para o utilizador actual
   e adiciona uma excepção de definição regional se aplicável (foi adicionado
   parâmetro "c").

   /[s|S] /[c|C] /[a|A]:w1,"x1",”y1”,”z1”;w2,"x2",”y2”,”z2”;…;wn,"xn",”yn”,”zn”
   Exemplo: euroconv.exe /s /c /a:0x00000807,”,”,”2”,”.”
   Função:  Muda a unidade monetária para o Euro  para o utilizador actual de 
   forma silenciosa (sem caixas de diálogo) e adiciona uma excepção de definição 
   regional se aplicável (foi adicionado parâmetro "s"). 

Notas:

   - "wi" representa a forma hexadecimal do ID de definição regional (LCID). 
   Para mais informações acerca de LCIDs, consulte o website Microsoft MSDN em 
   http://www.msdn.microsoft.com

   - "xi" representa o separador decimal monetário, que pode ser uma 
   cadeia vazia. Existe um limite de três (3) caracteres. A cadeia pode
   estar vazia. Se a cadeia estiver vazia o separador decimal não
   será actualizado.

   - "yi" representa o número de dígitos decimais para os formatos 
   monetários e está limitado a um máximo de dois (2) caracteres. 
   A cadeia pode estar vazia. Se a cadeia estiver vazia o número 
   de dígitos não será actualizado.

   - "zi" representa o separador de milhares monetário, e está limitado a 
   um máximo de três (3) caracteres. Se a cadeia estiver vazia o separador 
   de milhares não será actualizado.

Exemplo:

euroconv /a:0x00000809,”,”,”2”,”.”
    em que:
      /a significa adicionar uma excepção à lista.

          wi: Inglês (Reino Unido) (0x00000809)
          xi: vírgula (“,”)
          yi: dois (“2”)
          zi: ponto (“.”)
