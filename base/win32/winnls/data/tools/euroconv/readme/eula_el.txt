ΕΝΗΜΕΡΩΜΕΝΗ ΕΚΔΟΣΗ ΤΩΝ MICROSOFT WINDOWS

ΕΑΝ ΕΓΚΑΤΑΣΤΗΣΕΤΕ, ΑΝΤΙΓΡΑΨΕΤΕ Ή ΧΡΗΣΙΜΟΠΟΙΗΣΕΤΕ ΜΕ ΟΠΟΙΟΝΔΗΠΟΤΕ ΑΛΛΟ ΤΡΟΠΟ ΤΑ ΣΥΣΤΑΤΙΚΑ ΜΕΡΗ ΛΕΙΤΟΥΡΓΙΚΟΥ ΣΥΣΤΗΜΑΤΟΣ, ΣΥΜΦΩΝΕΙΤΕ ΜΕ ΤΟΥΣ ΟΡΟΥΣ ΚΑΙ ΤΙΣ ΠΡΟΫΠΟΘΕΣΕΙΣ ΤΗΣ ΙΣΧΥΟΥΣΑΣ ΑΔΕΙΑΣ ΤΟΥ ΠΡΟΪΟΝΤΟΣ ΛΕΙΤΟΥΡΓΙΚΟΥ ΣΥΣΤΗΜΑΤΟΣ ΚΑΙ ΑΥΤΗΣ ΤΗΣ ΣΥΜΠΛΗΡΩΜΑΤΙΚΗΣ ΑΔΕΙΑΣ. ΕΑΝ ΔΕΝ ΣΥΜΦΩΝΕΙΤΕ ΜΕ ΑΥΤΟΥΣ ΤΟΥΣ ΟΡΟΥΣ ΚΑΙ ΤΙΣ ΠΡΟΫΠΟΘΕΣΕΙΣ, ΜΗΝ ΕΓΚΑΤΑΣΤΗΣΕΤΕ, ΑΝΤΙΓΡΑΨΕΤΕ Ή ΧΡΗΣΙΜΟΠΟΙΗΣΕΤΕ ΤΑ ΣΥΣΤΑΤΙΚΑ ΜΕΡΗ ΛΕΙΤΟΥΡΓΙΚΟΥ ΣΥΣΤΗΜΑΤΟΣ.

ΣΗΜΕΙΩΣΗ:  ΕΑΝ ΔΕΝ ΕΧΕΤΕ ΑΝΤΙΓΡΑΦΟ ΜΕ ΕΓΚΥΡΗ ΑΔΕΙΑ ΧΡΗΣΗΣ ΟΠΟΙΑΣΔΗΠΟΤΕ ΕΚΔΟΣΗΣ ΤΟΥ ΛΕΙΤΟΥΡΓΙΚΟΥ ΣΥΣΤΗΜΑΤΟΣ MICROSOFT WINDOWS 95, WINDOWS 98, WINDOWS NT 4.0, WINDOWS 2000 Ή ΟΠΟΙΟΔΗΠΟΤΕ ΛΕΙΤΟΥΡΓΙΚΟ ΣΥΣΤΗΜΑ ΤΗΣ MICROSOFT ΤΟ ΟΠΟΙΟ ΑΠΟΤΕΛΕΙ ΣΥΝΕΧΕΙΑ ΟΠΟΙΟΥΔΗΠΟΤΕ ΑΠΟ ΑΥΤΑ ΤΑ ΛΕΙΤΟΥΡΓΙΚΑ ΣΥΣΤΗΜΑΤΑ (καθένα από τα οποία αποτελεί "Προϊόν λειτουργικού συστήματος"), ΔΕΝ ΣΑΣ ΠΑΡΕΧΕΤΑΙ ΤΟ ΔΙΚΑΙΩΜΑ ΝΑ ΕΓΚΑΤΑΣΤΗΣΕΤΕ, ΝΑ ΑΝΤΙΓΡΑΨΕΤΕ Ή ΝΑ ΧΡΗΣΙΜΟΠΟΙΗΣΕΤΕ ΜΕ ΟΠΟΙΟΝΔΗΠΟΤΕ ΑΛΛΟΝ ΤΡΟΠΟ ΤΑ ΣΥΣΤΑΤΙΚΑ ΜΕΡΗ ΛΕΙΤΟΥΡΓΙΚΟΥ ΣΥΣΤΗΜΑΤΟΣ ΚΑΙ Η ΠΑΡΟΥΣΑ ΣΥΜΠΛΗΡΩΜΑΤΙΚΗ ΑΔΕΙΑ ΔΕΝ ΣΑΣ ΠΑΡΕΧΕΙ ΚΑΝΕΝΑ ΔΙΚΑΙΩΜΑ.

Οι όροι που αναφέρονται με κεφαλαία σε αυτήν τη Συμπληρωματική Άδεια και δεν ορίζονται διαφορετικά στην παρούσα θα έχουν την έννοια που τους έχει αποδοθεί στην ισχύουσα Άδεια του Προϊόντος λειτουργικού συστήματος.

Γενικά. Τα Συστατικά Μέρη Λειτουργικού Συστήματος σας παρέχονται από τη Microsoft για την ενημέρωση, τη συμπλήρωση ή την αντικατάσταση της υφιστάμενης λειτουργικότητας του ισχύοντος Προϊόντος λειτουργικού συστήματος. Η Microsoft σας παραχωρεί την άδεια χρήσης των Συστατικών Μερών Λειτουργικού Συστήματος σύμφωνα με τους όρους και τις προϋποθέσεις της Άδειας του ισχύοντος Προϊόντος λειτουργικού συστήματος (η οποία ενσωματώνεται στην παρούσα εξ αναφοράς, εκτός από τα όσα ορίζονται στη συνέχεια), τους όρους και τις προϋποθέσεις που ορίζονται στην παρούσα Συμπληρωματική Άδεια, και τους όρους και τις προϋποθέσεις οποιασδήποτε πρόσθετης άδειας χρήσης τελικού χρήστη ενδέχεται να συνοδεύει τα μεμονωμένα Συστατικά Μέρη Λειτουργικού Συστήματος (καθεμία από τις οποίες αποτελεί "Μεμονωμένη Άδεια"), υπό τον όρο ότι συμμορφώνεστε με όλους τους εν λόγω όρους και προϋποθέσεις.  Στην έκταση που υπάρχει σύγκρουση μεταξύ οποιωνδήποτε όρων και προϋποθέσεων που ισχύουν για τα Συστατικά Μέρη Λειτουργικού Συστήματος, θα υπερισχύει η ακόλουθη ιεραρχία:  1) οι όροι και προϋποθέσεις της Μεμονωμένης Άδειας, 2) οι όροι και προϋποθέσεις αυτής της Συμπληρωματικής Άδειας, και 3) οι όροι και προϋποθέσεις της ισχύουσας Άδειας του Προϊόντος λειτουργικού συστήματος.

Πρόσθετα δικαιώματα και περιορισμοί.  

*	Επιτρέπεται να εγκαταστήσετε και να χρησιμοποιήσετε ένα μόνο αντίγραφο των Συστατικών Μερών Λειτουργικού Συστήματος. Εάν έχετε πολλά αντίγραφα του ισχύοντος Προϊόντος λειτουργικού συστήματος με έγκυρη άδεια χρήσης, θα πρέπει να συνδεθείτε ξανά σε αυτήν την τοποθεσία και να κάνετε λήψη ενός πρόσθετου αντιγράφου των Συστατικών Μερών Λειτουργικού Συστήματος για κάθε ηλεκτρονικό υπολογιστή στον οποίο εκτελείται το ισχύον Προϊόν λειτουργικού συστήματος.

*	Επιτρέπεται επίσης να αναπαραγάγετε ένα πρόσθετο αντίγραφο των Συστατικών Μερών Λειτουργικού Συστήματος αποκλειστικά για σκοπούς αρχειοθέτησης ή επανεγκατάστασης των Συστατικών Μερών Λειτουργικού Συστήματος στον ίδιο ηλεκτρονικό υπολογιστή στον οποίο έχουν ήδη εγκατασταθεί τα Συστατικά Μέρη Λειτουργικού Συστήματος.  Η Microsoft διατηρεί κάθε δικαίωμα κυριότητας και νομής και κάθε συμφέρον που αφορά τα Συστατικά Μέρη Λειτουργικού Συστήματος. Η Microsoft επιφυλάσσεται όλων των δικαιωμάτων τα οποία δεν παραχωρούνται ρητά.

*	Τα Συστατικά Μέρη Λειτουργικού Συστήματος ενδέχεται να περιλαμβάνουν τεχνολογία που επιτρέπει την κοινή χρήση εφαρμογών ανάμεσα σε δύο ή περισσότερους ηλεκτρονικούς υπολογιστές, ακόμα και στην περίπτωση που μια εφαρμογή έχει εγκατασταθεί σε έναν μόνο ηλεκτρονικό υπολογιστή.  Επιτρέπεται να χρησιμοποιήσετε αυτήν την τεχνολογία με όλα τα προϊόντα εφαρμογών της Microsoft για τη σύγκληση διασκέψεων ανάμεσα σε πολλούς χρήστες.  Για εφαρμογές άλλων κατασκευαστών εκτός της Microsoft, θα πρέπει να συμβουλευτείτε την άδεια χρήσης που συνοδεύει κάθε προϊόν ή να επικοινωνήσετε με τον χορηγό της άδειας για να διαπιστώσετε εάν επιτρέπει την κοινή χρήση εφαρμογών.
