Herramienta de conversión para el euro
--------------------------------------

Descripción:
   La Herramienta de conversión para el euro es una utilidad simple que
   permite la migración de la unidad monetaria al euro para los estados
   miembros de la zona del euro. Esta herramienta funciona con
   Windows 95, Windows 98, Windows NT4, Windows 2000 y Windows XP. 

   Esta herramienta de conversión funciona para aquellos usuarios que 
   tengan la configuración regional ("estándares y formatos" en
   Windows XP) establecida para un estado de la zona del euro. Esta 
   herramienta no cambiará de forma predeterminada la configuración de
   moneda de los usuarios con una configuración regional que no
   corresponda a los estados de la zona del euro.

   En Windows 95, Windows 98 y Windows NT4 debe ejecutar primero una
   revisión anterior para poder completar la migración. Esta revisión
   incluye la actualización de fuentes, teclados y páginas de código y
   está disponible en el sitio Web de Microsoft
   (http://www.microsoft.com/Windows/Euro.asp).

   De forma predeterminada se aplicarán los cambios a todos los usuarios.
   Sin embargo, puede agregar un modificador que le permitirá realizar el
   cambio sólo para el usuario actual (ver más abajo).

Uso:
   La utilidad euroconv.exe acepta los parámetros siguientes en la línea
   de comandos:

   [ninguno]
   Ejemplo: euroconv.exe
   Función: cambia la configuración de moneda de todos los usuarios al
            euro.

   /?
   Ejemplo: euroconv.exe /?
   Función: muestra información de uso.

   /[h/H]
   Ejemplo: euroconv.exe /h
   Función: muestra información de uso.

   /[c|C]
   Ejemplo: euroconv.exe /c
   Función: cambia la configuración de moneda del usuario actual al
            euro.

   /[s|S]
   Ejemplo: euroconv.exe /s
   Función: cambia la configuración de moneda de todos los usuarios al euro,
            de forma silenciosa (sin cuadros de diálogo).

   /[s|S] /[c|C]
   Ejemplo: euroconv.exe /c /s
   Función: cambia la configuración de moneda del usuario actual al euro, de
            forma silenciosa (sin cuadros de diálogo).

   /[a|A]:w1, "x1","y1","z1" [;wn, "xn","yn","zn"]
   Ejemplo: euroconv.exe /a:0x00000807,",","2","."
   Función: cambia la configuración de moneda de todos los usuarios
            correspondientes al euro y agrega una configuración regional
            de excepción si corresponde. Es decir, también se cambiará la
            configuración de moneda al euro para los usuarios cuya 
            configuración regional sea igual a wi. Si la configuración
            regional del usuario (LCID) coincide con wi, el separador
            decimal monetario será cambiado a xi, el número de dígitos
            decimales para los formatos monetarios será cambiado a yi y el
            separador de miles monetario será cambiado a zi. (Note que
            xi, yi y zi pueden dejarse en blanco.)

            Esta opción se puede utilizar para agregar una configuración
            regional a la lista de procesamiento. También puede usarse para
            sustituir información de una configuración regional definida.

            (Al final del archivo Léame hay un ejemplo de esta
            funcionalidad).

   /[c|C] /[a|A]:w1, "x1","y1","z1" [;wn, "xn","yn","zn"]
   Ejemplo: euroconv.exe /c /a:0x00000807,",","2","."
   Función: cambia la configuración de moneda al euro sólo para este
            usuario y agrega una configuración regional de excepción
            si corresponde (se ha agregado el parámetro "c").

   /[s|S] /[c|C] /[a|A]:w1, "x1","y1","z1" [;wn, "xn","yn","zn"]
   Ejemplo: euroconv.exe /s /c /a:0x00000807,",","2","."
   Función: cambia la configuración de moneda del usuario actual al euro
            de forma silenciosa (sin cuadros de diálogo) y agrega una
            configuración regional de excepción si corresponde (se ha
            agregado el parámetro "s").

Notas:

   - "wi" representa la forma hexadecimal del Id. de configuración regional
  (LCID) Para obtener más información acerca del los valores LCID visite el
  sitio Web de Microsoft MSDN (http://www.msdn.microsoft.com).

   - "xi" representa el separador decimal monetario; puede dejarse en
   blanco. Está limitado a un máximo de tres (3) caracteres. Si la cadena
   se deja en blanco, el separador decimal no se actualizará.

   - "yi" representa el número de dígitos decimales para los formatos
   monetarios y está limitado a un máximo de dos (2) caracteres. Si la
   cadena se deja en blanco, no se actualizará el número de dígitos.

   - "zi" representa el separador de miles monetario y está limitado a un
   máximo de tres (3) caracteres. Si la cadena se deja en blanco, no se 
   actualizará el separador de miles.

Ejemplo:

euroconv /a:00000809,",","2","."
    donde:
      /a agrega una excepción a la lista.

          wi: Inglés (Gran Bretaña) (0x00000809)
          xi: coma (",")
          yi: dos ("2")
          zi: punto (".")

Nota: se ha utilizado Gran Bretaña como ejemplo porque aún no es
      parte de la zona del euro pero es posible que lo sea en el futuro.
