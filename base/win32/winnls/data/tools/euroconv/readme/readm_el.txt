Εργαλείο προσαρμογής ρυθμίσεων ευρώ
-----------------------------------

Περιγραφή:
   Το εργαλείο προσαρμογής ρυθμίσεων ευρώ είναι ένα απλό εργαλείο που 
   επιτρέπει την προσαρμογή των νομισματικών ρυθμίσεων στο ευρώ, για τις
   χώρες που ανήκουν στην Ευρωζώνη. Αυτό το εργαλείο εφαρμόζεται στα
   Windows 95, Windows 98, Windows NT4, Windows 2000 και Windows XP. 

   Το εργαλείο αυτό προκαλεί την προσαρμογή της ρύθμισης νομίσματος για
   τους χρήστες εκείνους που οι ρυθμίσεις τοποθεσίας τους ("Πρότυπα και 
   μορφές" στα Windows XP) είναι ορισμένες για μια τοποθεσία εντός της
   Ευρωζώνης.  Από προεπιλογή, αυτό το εργαλείο δεν θα αλλάξει τις 
   νομισματικές ρυθμίσεις για ένα χρήστη εκτός της Ευρωζώνης.
   
   Για τα Windows 95, Windows 98 και Windows NT4, θα πρέπει να εκτελέσετε ένα
   προηγούμενο πρόγραμμα ενημέρωσης κώδικα για να ολοκληρώσετε την μετάβαση.
   Αυτό το πρόγραμμα ενημέρωσης περιλαμβάνει ενημερώσεις γραμματοσειρών, 
   πληκτρολογίων και κωδικοσελίδων. Το πρόγραμμα ενημέρωσης κώδικα είναι 
   διαθέσιμο στην τοποθεσία web της Microsoft 
   (http://www.microsoft.com/Windows/Euro.asp).

   Από προεπιλογή, οι αλλαγές εφαρμόζονται σε όλους τους χρήστες. Ωστόσο, 
   μπορείτε να προσθέσετε ένα διακόπτη που σας επιτρέπει να κάνετε την
   αλλαγή μόνο για τον τρέχοντα χρήστη (δείτε παρακάτω για τη σχετική
   παράμετρο).

Χρήση:
   Το euroconv.exe δέχεται τις ακόλουθες παραμέτρους στη 
   γραμμή εντολών:
   
   [καμία]
   Πρότυπο   : euroconv.exe
   Λειτουργία: Αλλαγή του νομίσματος σε ευρώ για όλους τους χρήστες.

   /?
   Πρότυπο   : euroconv.exe /?
   Λειτουργία: Εμφάνιση χρήσης.

   /[h/H]
   Πρότυπο   : euroconv.exe /h
   Λειτουργία: Εμφάνιση χρήσης.

   /[c|C]
   Πρότυπο   : euroconv.exe /c
   Λειτουργία: Αλλαγή του νομίσματος σε ευρώ για τον τρέχοντα χρήστη μόνο.

   /[s|S]
   Πρότυπο   : euroconv.exe /s
   Λειτουργία: Αλλαγή του νομίσματος σε ευρώ για όλους τους χρήστες σιωπηλά
               (χωρίς παράθυρο διαλόγου).

   /[s|S] /[c|C]
   Πρότυπο   : euroconv.exe /c /s
   Λειτουργία: Αλλαγή του νομίσματος σε ευρώ για τον τρέχοντα χρήστη σιωπηλά
               (χωρίς παράθυρο διαλόγου).

   /[a|A]:w1, "x1",y1,z1;x2,y2,z2;;wn, "xn",yn,zn
   Πρότυπο   : euroconv.exe /a:0x00000807,,,2,.
   Λειτουργία: Αλλαγή του νομίσματος σε ευρώ για όλους τους χρήστες και πρόσθεση 
               μιας τοποθεσίας εξαίρεσης εάν υπάρχει διαθέσιμη.  Με άλλα λόγια,
               η νομισματική πληροφορία επίσης θα αλλάξει σε ευρώ εάν η τοποθεσία
               χρήστη ισούται με wi. Εάν η τοποθεσία χρήστη (LCID) συμφωνεί με το 
               wi, ο νομισματικός διαχωριστής δεκαδικού θα αλλάξει επίσης σε
               xi, ο αριθμός των δεκαδικών ψηφίων για τις νομισματικές μορφές της
               τοποθεσίας θα αλλάξει σε yi, και ο νομισματικός διαχωριστής χιλιάδων
               θα αλλάξει σε zi.  (Σημειώστε ότι τα xi, yi και zi μπορούν όλα να 
               είναι κενά.)
 
               Αυτή η επιλογή μπορεί να χρησιμοποιηθεί για να προστεθεί μια 
               τοποθεσία στη λίστα επεξεργασίας. Επίσης μπορεί να χρησιμοποιηθεί 
               για να γίνει υπέρβαση ρυθμίσεων για μια συγκεκριμένη τοποθεσία.

               (Δείτε στο τέλος του αρχείου readme για ένα παράδειγμα του τρόπου με
               τον οποίο αυτό δουλεύει.)

   /[c|C] /[a|A]:w1,"x1",y1,z1;w2,"x2",y2,z2;;wn,"xn",yn,zn
   Πρότυπο   : euroconv.exe /c /a:0x00000807,,,2,.
   Λειτουργία: Αλλαγή του νομίσματος σε ευρώ για τον τρέχοντα χρήστη μόνο και 
               πρόσθεση μιας τοποθεσίας εξαίρεσης εάν υπάρχει διαθέσιμη (η παράμετρος 
               "c" έχει προστεθεί).

   /[s|S] /[c|C] /[a|A]:w1,"x1",y1,z1;w2,"x2",y2,z2;;wn,"xn",yn,zn
   Πρότυπο   : euroconv.exe /s /c /a:0x00000807,,,2,.
   Λειτουργία: Αλλαγή του νομίσματος σε ευρώ για τον τρέχοντα χρήστη σιωπηλά (χωρίς 
               παράθυρο διαλόγου) και πρόσθεση μιας τοποθεσίας εξαίρεσης εάν υπάρχει 
               διαθέσιμη (η παράμετρος "s" έχει προστεθεί). 
Σημειώσεις:

   - Το "wi" αντιπροσωπεύει την δεκαεξαδική μορφή του αναγνωριστικού τοποθεσίας(LCID). 
   Για περισσότερες πληροφορίες σχετικά με τα LCID, δείτε στην τοποθεσία web του 
   Microsoft MSDN (http://www.msdn.microsoft.com).

   - Το "xi" αντιπροσωπεύει τον νομισματικό διαχωριστή δεκαδικού, ο οποίος μπορεί να 
   είναι κενή συμβολοσειρά. Αυτός περιορίζεται σε ένα μέγιστο τριών (3) χαρακτήρων. 
   Η συμβολοσειρά μπορεί να είναι κενή. Εάν είναι κενή, ο διαχωριστής δεκαδικού δεν 
   ενημερώνεται.

   - Το "yi" αντιπροσωπεύει τον αριθμό δεκαδικών ψηφίων για τις νομισματικές μορφές 
   και είναι περιορισμένος σε ένα μέγιστο δύο (2) χαρακτήρων. Η συμβολοσειρά μπορεί 
   να είναι κενή. Εάν είναι κενή, ο αριθμός των δεκαδικών ψηφίων δεν ενημερώνεται.

   - Το "zi" αντιπροσωπεύει τον νομισματικό διαχωριστή χιλιάδων και είναι περιορισμένος 
   σε ένα μέγιστο τριών (3) χαρακτήρων. Εάν η συμβολοσειρά είναι κενή, ο νομισματικός 
   διαχωριστής χιλιάδων δεν ενημερώνεται.
   
Παράδειγμα:

euroconv /a:00000409,,,2,.
    όπου:
      /a σημαίνει πρόσθεση εξαίρεσης στη λίστα.

          wi: Αγγλικά - Ηνωμένων Πολιτειών (0x00000409)
          xi: κόμμα (,)
          yi: δύο (2)
          zi: τελεία (.)
