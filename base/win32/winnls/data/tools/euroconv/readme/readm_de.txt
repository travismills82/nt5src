Euro-Konvertierungsprogramm
---------------------------

Beschreibung:
   Das Euro-Konvertierungsprogramm ist ein leicht zu bedienendes Hilfsprogramm,
   mit dem Währungen von Ländern der Europäischen Union in Euro konvertiert
   werden können. Dieses Programm ist mit Windows 95, Windows 98, Windows NT4, 
   Windows 2000 und Windows XP kompatibel. 

   Dieses Euro-Konvertierungsprogramm kann auf Benutzerkonten, die Benutzer-
   gebietsschemen ("Standards und Formate" auf Windows XP) der Europäischen
   Union verwenden, angewendet werden. Die Währungseinstellungen von Benutzern,
   die standardmäßig kein Benutzergebietsschema der Europäischen Union
   verwenden, werden nicht geändert.
   
   Auf Windows 95, Windows 98 und Windows NT4 muss zuerst ein vorheriger Patch
   ausgeführt werden, damit die Migration abgeschlossen werden kann. Der Patch
   enthält Updates von Schriftarten, Tastaturen und Codepages. Diesen Patch
   finden Sie auf der Microsoft-Website unter:
   http://www.microsoft.com/Windows/Euro.asp

   Änderungen werden auf alle Benutzer standardmäßig angewendet. Sie können
   jedoch einen Parameter hinzufügen, so dass die Änderung nur für den aktuellen
   Benutzer übernommen wird. (Diesen Parameter finden Sie weiter unten.)

Syntax:
   Folgende Parameter können für euroconv.exe auf Befehlszeilenebene verwendet
   werden:

   [keiner]
   Prototyp: euroconv.exe
   Funktion: Übernimmt die Euro-Währung für alle Benutzer.

   /?
   Prototyp: euroconv.exe /?
   Funktion: Zeigt die Hilfe an.

   /[h/H]
   Prototyp: euroconv.exe /h
   Funktion: Zeigt die Hilfe an.

   /[c|C]
   Prototyp: euroconv.exe /c
   Funktion: Übernimmt die Euro-Währung nur für den aktuellen Benutzer.

   /[s|S]
   Prototyp: euroconv.exe /s
   Funktion: Übernimmt die Euro-Währung für alle Benutzer im stillen
             Modus (kein Dialogfeld wird angezeigt).

   /[s|S] /[c|C]
   Prototyp: euroconv.exe /c /s
   Funktion: Übernimmt die Euro-Währung für den aktuellen Benutzer im
             stillen Modus (kein Dialogfeld wird angezeigt).

   /[a|A]:w1, "x1",”y1”,”z1”;x2,”y2”,”z2”;…;wn, "xn",”yn”,”zn”
   Prototyp: euroconv.exe /a:0x00000807,”,”,”2”,”.”
   Funktion: Übernimmt die Euro-Währung für alle entsprechenden Benutzer,
             und fügt, falls zutreffend, ein Ausnahmegebietsschema hinzu.
             D. h. die Währungsinformationen eines Benutzers werden in Euro
             geändert, falls das Gebietsschema des Benutzers mit wi
             übereinstimmt. Falls das Benutzergebietsschema (LCID) mit wi
             übereinstimmt, wird das Geldbetrag-Dezimaltrennzeichen für xi,
             die Anzahl der Dezimalziffern für die Geldbetragsformate des
             Gebietsschema für yi und das Geldbetrag-Tausendertrennzeichen
             für zi geändert. (Hinweis: xi, yi und zi können leer sein.)

             Diese Option kann zum Hinzufügen eines Gebietsschemas zur
             Migrationsliste verwendet werden. Diese Option kann auch
             verwendet werden, um Informationen für ein definiertes
             Gebietsschema außer Kraft zu setzen.

             (Beispiele hierzu finden Sie am Ende dieser Readme-Datei.)

   /[c|C] /[a|A]:w1,"x1",”y1”,”z1”;w2,"x2",”y2”,”z2”;…;wn,"xn",”yn”,”zn”
   Prototyp: euroconv.exe /c /a:0x00000807,”,”,”2”,”.”
   Funktion: Übernimmt die Euro-Währung nur für den aktuellen Benutzer,
             und fügt eine Ausnahme, falls zutreffend, hinzu (Parameter "c"
             wurde hinzugefügt).

   /[s|S] /[c|C] /[a|A]:w1,"x1",”y1”,”z1”;w2,"x2",”y2”,”z2”;…;wn,"xn",”yn”,”zn”
   Prototyp: euroconv.exe /s /c /a:0x00000807,”,”,”2”,”.”
   Funktion: Übernimmt die Euro-Währung für den aktuellen Benutzer im stillen
             Modus (kein Dialogfeld wird angezeigt) und fügt, falls zutreffend,
             ein Ausnahmegebietsschema hinzu (Parameter "s" wurde hinzugefügt). 

Hinweise:

   - "wi" stellt das Hexadezimalformat der Gebietsschemakennung (LCID) dar.
   Weitere Informationen über LCIDs erhalten Sie auf der Microsoft MSDN-Website
   unter http://www.msdn.microsoft.com.

   - "xi" stellt das Geldbetrag-Dezimaltrennzeichen dar, das aus einer leeren
   Zeichenfolge bestehen kann. Diese Zeichenfolge darf höchstens aus drei (3)
   Zeichen bestehen. Die Zeichenfolge kann leer sein. Das Dezimaltrennzeichen
   wird nicht aktualisiert, wenn die Zeichenfolge leer ist.

   - "yi" stellt die Anzahl der Dezimalziffern für die Geldbetragformate dar
   und darf höchsten aus zwei (2) Zeichen bestehen. Die Zeichenfolge kann leer
   sein. Die Zifferanzahl wird nicht aktualisiert, wenn die Zeichenfolge leer
   ist.

   - "zi" stellt das Geldbetrag-Tausendertrennzeichen dar und darf höchstens
   aus drei (3) Zeichen bestehen. Das Tausendertrennzeichen wird nicht
   aktualisiert, wenn die Zeichenfolge leer ist.

Beispiel:

euroconv /a:00000409,”,”,”2”,”.”
    Beschreibung:
      /a Fügt der Liste eine Ausnhame hinzu.

          wi: Englisch – US (0x00000409)
          xi: Komma (“,”)
          yi: zwei (“2”)
          zi: Punkt (“.”)
