Outil de configuration de l'euro par défaut
-------------------------------------------

Description :
   L'Outil de configuration de l'euro par défaut est un utilitaire qui
   permet la migration du symbole monétaire vers l'euro pour les pays
   membres de la zone euro. Cet outil fonctionne sous Windows 95,
   Windows 98, Windows NT4, Windows 2000 et Windows XP. 

   Cet outil de conversion ne fonctionne que pour les utilisateurs dont
   le paramètre régional utilisateur ("Standards et formats" sur
   Windows XP) appartient à la zone euro. Cet outil ne modifie
   pas les paramètres monétaires des utilisateurs dont le paramètre
   régional utilisateur n'appartient pas à la zone euro.

   Pour Windows 95, Windows 98 et Windows NT4, vous devez avoir exécuté
   un autre correctif logiciel Microsoft avant de commencer la migration.
   Ce correctif met à jour les polices, les claviers et les pages de codes.
   Le correctif est disponible sur le site Web Microsoft à l'adresse :
        http://www.microsoft.com/france/euro/etapes/windows.asp

   Les modifications sont appliquées par défaut à tous les utilisateurs.
   Cependant, un commutateur vous permet de n'appliquer la modification
   qu'à l'utilisateur en cours (voir ci-dessous pour la syntaxe).

Utilisation :
   L'Outil de configuration de l'euro par défaut accepte les paramètres
   suivants à la ligne de commande :

   [aucun]
   Exemple  : euroconv.exe
   Fonction : Définit l'euro pour tous les utilisateurs.

   /?
   Exemple  : euroconv.exe /?
   Fonction : Affiche l'aide sur la syntaxe.

   /[h/H]
   Exemple  : euroconv.exe /h
   Fonction : Affiche l'aide sur la syntaxe.

   /[c|C]
   Exemple  : euroconv.exe /c
   Fonction : Définit l'euro pour l'utilisateur en cours uniquement.

   /[s|S]
   Exemple  : euroconv.exe /s
   Fonction : Définit l'euro pour tous les utilisateurs sans interaction
              utilisateur.

   /[s|S] /[c|C]
   Exemple  : euroconv.exe /c /s
   Fonction : Définit l'euro pour l'utilisateur en cours sans interaction
              utilisateur.

   /[a|A]:wi, "xi","yi","zi" [;wn, "xn","yn","zn"]
   Exemple  : euroconv.exe /a:0x00000807,",","2","."
   Fonction : Définit l'euro pour tous les utilisateurs qui remplissent les
              conditions, et ajoute un paramètre régional exceptionnel si
              applicable. En d'autres termes, le symbole monétaire d'un
              utilisateur sera défini à l'euro si le paramètre régional
              utilisateur est égal à wi. Si le paramètre régional
              utilisateur (LCID) correspond à wi, le symbole décimal de la
              monnaie (séparateur décimal) sera remplacé par xi, le nombre
              de décimales de la monnaie sera remplacé par yi et le symbole
              de groupement de chiffres (séparateur de milliers) sera
              remplacé par zi. (Notez que xi, yi et zi peuvent être vides.)

              Cette option peut être utilisée pour ajouter un paramètre
              régional à la liste à traiter. Cette option permet également
              de remplacer les informations concernant un paramètre
              régional spécifique.

              (Consultez l'exemple au bas de ce document.)

   /[c|C] /[a|A]:wi, "xi","yi","zi" [;wn, "xn","yn","zn"]
   Exemple  : euroconv.exe /c /a:0x00000807,",","2","."
   Fonction : Définit l'euro pour l'utilisateur en cours et ajoute un
              paramètre régional exceptionnel si applicable.
              (Le paramètre "c" a été ajouté.)

   /[s|S] /[c|C] /[a|A]:wi, "xi","yi","zi" [;wn, "xn","yn","zn"]
   Exemple  : euroconv.exe /s /c /a:0x00000807,",","2","."
   Fonction : Définit l'euro pour l'utilisateur en cours sans interaction
              utilisateur et ajoute un paramètre régional exceptionnel si
              applicable. (Le paramètre "s" a été ajouté.) 

Remarques :

   - "wi" représente la forme hexadécimale de l'identificateur de paramètre
     régional (LCID). Pour obtenir plus d'informations sur ces identificateurs,
     consultez le site Web Microsoft MSDN (http://www.microsoft.com/france/msdn).

   - "xi" représente le symbole décimal de la monnaie (séparateur décimal),
     qui peut être une chaîne vide. Ce symbole est une chaîne composée de
     trois (3) caractères au maximum. Si la chaîne est vide, le symbole décimal
     de la monnaie n'est pas mis à jour.

   - "yi" représente le nombre de décimales du format monétaire. Ce nombre est
     une chaîne, qui peut être vide, composée de deux (2) caractères au maximum.
     Si la chaîne est vide, le nombre de décimales n'est pas mis à jour.

   - "zi" représente le symbole de groupement de chiffres (séparateur de
      milliers), qui peut être une chaîne vide. Ce symbole est une chaîne
      composée de trois (3) caractères au maximum. Si la chaîne est vide, le
      symbole de groupement de chiffres n'est pas mis à jour.

Exemple :

euroconv /a:00000809,",","2","."
    où :
      /a signifie l'ajout d'exceptions à la liste.

          wi: anglais - Grande-Bretagne (0x00000809)
          xi: virgule (",")
          yi: deux ("2")
          zi: point (".")
