Webdav Redirector

The Webdav Redirector component enables Windows to file I/O against file servers using the Webdav protocol. The Webdav protocol is used by many Web servers such as the Microsoft Internet Information Server (IIS) or the Apache WebServer, and by Internet service provider (ISP) services such as the Microsoft Network Communities section.

Component Configuration

There are no configuration requirements for this component.
