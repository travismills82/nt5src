SMB Server

The SMB Server component is the server side of the Microsoft File and Print Services solution. It allows Windows to provide file and print services to other machines through the Server Message Block (SMB) protocol.

Component Configuration

There are no configuration requirements for this component.

For a list of registry parameters that you can use to modify this component, see the Windows Resource Kit.

For More Information

For more information on this component, see the Windows Resource Kit and this Microsoft Web site.
