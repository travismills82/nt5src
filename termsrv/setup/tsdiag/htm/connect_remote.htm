Can't connect to remote computer

This test checks a terminal server or other remote computer running Windows Whistler Professional or Server for common connection problems. To successfully run this test, you must be logged in on an account that has administrative priveleges on the computer where you are running the test.

To run diagnostics on the remote computer related to this problem
  1. In Servername, type the DNS name or IP address of the server you want to test.
  2. Click Diagnose.
