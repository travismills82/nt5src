Mise à jour du logiciel

Utilisez cette page pour installer les mises à jour logicielles sur votre système serveur.

Pour mettre à jour le logiciel de votre système serveur

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Mise à jour du logiciel.
  3. Conformez-vous aux instructions de l'Assistant Mise à jour du logiciel.
