Configuration DNS

Le DNS (système des noms de domaine) est un service statique de noms hiérarchisés pour les hôtes TCP/IP. L'administrateur réseau configure le DNS avec la liste des noms d'hôtes et leurs adresses IP. Cela permet aux utilisateurs du réseau d'effectuer des requêtes DNS en spécifiant les systèmes distants par leur nom d'hôte plutôt que par leur adresse IP.

Remarque   Cette page de propriétés a pour but de vous permettre d'entrer des adresses de serveurs DNS externes. Le système serveur ne contient pas de serveur DNS.

Une station de travail configurée pour utiliser la résolution de noms DNS, par exemple, peut utiliser la commande ping hôte_distant plutôt que ping 1.2.3.4 si le mappage du système nommé hôte_distant est contenu dans la base de données DNS. Ne confondez pas les domaines DNS avec les domaines Windows.

Dans le modèle client-serveur DNS, le serveur contenant des informations sur une partie de la base de données DNS, celle qui met les noms d'ordinateur à la disposition des clients, effectue ses requêtes de résolution de noms sur Internet.

Pour configurer le système serveur afin qu'il obtienne automatiquement les informations du serveur DNS à partir d'un serveur DHCP

  1. Dans la barre de navigation principale, choisissez Réseau.
  2. Choisissez Interfaces.
  3. Sélectionnez la connexion réseau à modifier.
  4. Dans la liste Tâches, choisissez DNS.
  5. Cliquez sur le bouton Obtenir la configuration à partir du serveur DHCP.
  6. Cliquez sur OK.

Pour configurer manuellement les serveurs DNS à utiliser avec le système serveur

  1. Dans la page Réseau, choisissez Interfaces.
  2. Sélectionnez la connexion réseau à modifier.
  3. Dans la liste Tâches, choisissez DNS.
  4. Cliquez sur le bouton Configurer manuellement.
  5. Entrez l'adresse IP appropriée dans la zone placée à côté du bouton Ajouter, puis cliquez sur Ajouter.
  6. Pour ajouter un autre serveur DNS, reprenez l'étape 5.
  7. Lorsque vous avez fini d'ajouter des serveurs DNS, cliquez sur OK.

    Remarque   Si vous avez choisi d'obtenir l'adresse IP à partir du serveur DHCP et si vous configurez le DNS manuellement, le système acceptera les entrées manuelles, et les propriétés enregistrées dans le système serveur seront automatiquement définies à Configurer manuellement. Cependant, la colonne Configuration actuelle du Sélecteur d'objet/de tâche de la page Interfaces affiche quand même le DHCP comme source de l'adresse IP. Vous pouvez retourner dans la page de propriétés Paramètres DNS pour confirmer que la configuration manuelle a été enregistrée.
