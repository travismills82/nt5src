Configuration des propriétés de partage Web (HTTP)

Utilisez cette page pour spécifier quels clients HTTP ont une autorisation d'accès à chaque partage. L'accès peut être autorisé ou interdit en fonction du nom d'hôte du client. L'accès peut aussi être autorisé ou interdit en fonction de groupes de clients, lorsqu'un groupe de clients contient un ou plusieurs noms d'hôtes clients.

Pour accorder aux clients une autorisation sur un partage Web

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Dans la page Partages, choisissez Partages.
  3. Sélectionnez le partage pour lequel vous souhaitez créer un accès client HTTP puis choisissez Propriétés.
  4. Sous l'onglet Général, activez la case à cocher Web (HTTP).
  5. Sélectionnez l'onglet Partage HTTP, puis
    • activez la case à cocher Lecture pour autoriser l'accès en lecture
Ou
  • activez la case à cocher Écriture pour autoriser l'accès en écriture.
  1. Cliquez sur OK.
