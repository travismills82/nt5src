Activation de l'enregistrement FTP

Vous pouvez enregistrer les connexions entrantes FTP dans le fichier journal FTP en activant Enregistrement FTP. Par défaut, les fichiers journaux FTP sont stockés dans %windir%\system32\logfiles\msftpsvc1.

Les administrateurs ont accès à ces fichiers depuis leur station de travail, soit en accédant à un partage administrateur—par exemple, \\nom_système_serveur\admin$\winnt\system32—soit en créant un nouveau partage pour ce dossier.

Pour activer l'enregistrement FTP

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Protocoles de partage.
  3. Sélectionnez Protocole FTP, puis Propriétés.
  4. Sélectionnez l'onglet Enregistrement.
  5. Activez la case à cocher Activer la journalisation, puis cliquez sur OK.
