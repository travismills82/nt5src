Configuration des mappages de groupe et d'utilisateur

Pour mapper des utilisateurs et des groupes NFS

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Protocoles de partage.
  3. Sélectionnez Protocole NFS, puis Propriétés.
  4. Choisissez Mappages de groupe et d'utilisateur.

Pour configurer l'utilisation d'un serveur NIS

  1. Dans l'onglet Général, cliquez sur le bouton Utiliser le serveur NIS.
  2. Dans la zone Domaine NIS, entrez le nom du domaine dans lequel sont obtenues les informations d'utilisateur et de groupe UNIX.
  3. Facultatif : dans la zone Serveur NIS (facultatif), entrez le nom du serveur à mapper.
  4. Pour spécifier le délai qu'attendra le système serveur avant de rafraîchir les informations d'utilisateur et de groupe, entrez une durée dans les zones Heures et Minutes.
  5. Cliquez sur OK.

Pour configurer l'utilisation de fichiers de mots de passe et de groupes

  1. Cliquez sur le bouton Utiliser les fichiers de mots de passe et de groupes.
  2. Dans la zone Fichier de mots de passe, entrez le nom du fichier de mots de passe à utiliser.
Voici un fichier au format mot de passe d'un système UNIX contenant les comptes d'utilisateurs UNIX susceptibles d'être mappés.
  1. Dans la zone Fichier du groupe, tapez le nom du fichier de groupes que vous voulez utiliser.
  2. Cliquez sur OK.
