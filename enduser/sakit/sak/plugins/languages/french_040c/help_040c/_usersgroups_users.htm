Utilisateurs

Dans cette page vous pouvez créer, modifier et supprimer des groupes et des utilisateurs locaux sur le système serveur. Vous pouvez aussi changer les membres de chaque groupe. Si le système serveur est membre d'un domaine, vous n'aurez pas à créer des utilisateurs sur le système serveur lui-même. La raison d'être principale de cette page est la création d'un ou plusieurs membres de domaine dans le groupe local.

Vous voulez peut-être également utiliser les comptes d'utilisateurs et de groupes pour contrôler l'accès aux ressources du système serveur ; ou bien, utiliser les outils de gestion de domaine pour gérer les utilisateurs de domaine et les groupes de domaine.
