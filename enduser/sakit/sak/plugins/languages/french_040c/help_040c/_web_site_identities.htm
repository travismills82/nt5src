Modification des identités d'un site

Dans la page Identités du site vous pouvez assigner l'adresse IP et le numéro de port TCP de votre système serveur, et aussi ajouter ou modifier un nom d'en-tête d'hôte au site Web que vous êtes en train de créer ou de modifier.

L'Adresse IP est définie à Non assignées par défaut. Les adresses IP de toutes les cartes d'interface réseau installées sur le système serveur apparaissent dans la liste.

Le port TCP est fixé à 80 par défaut. Ce numéro de port par défaut constitue la norme dans la profession, et il est sous-entendu lorsque vous entrez une adresse Web dans un navigateur, par exemple www.tonsite.com. Si vous changez ce numéro de port, pour 8055 par exemple, lorsque les utilisateurs entreront l'adresse Web du site ils devront également entrer deux points puis le numéro de port, par exemple www.tonsite.com:8055.

Pour modifier les paramètres dans Identités du site

  1. Dans la page de navigation principale, choisissez Sites.
  2. Sélectionnez le site Web à modifier.
  3. Dans la liste Tâches, choisissez Modifier.
  4. Sélectionnez l'onglet Identités du site.
  5. Dans la liste Adresse IP, sélectionnez l'adresse IP à assigner au site Web.
  6. Dans la zone Port TCP, entrez le numéro du port voulu.
  7. Dans la zone En-têtes des hôtes, entrez le nom d'en-tête de l'hôte.
  8. Choisissez OK ou sélectionnez un onglet pour effectuer d'autres modifications sur les propriétés du site Web.
