Suffixes DNS

Les suffixes DNS ont deux raisons d'être principales :

  • Ajoutés à la fin du nom d'hôte du système serveur, les suffixes DNS constituent le nom complet du système serveur.
  • Les suffixes DNS sont utilisés pour résoudre les adresses IP. Si votre système serveur est un membre du domaine Windows NT 4, d'un annuaire Active Directory Microsoft ou d'un groupe de travail, le suffixe DNS est fonction de l'environnement du domaine.

Un suffixe DNS peut compter jusqu'à 155 caractères, y compris les traits d'union et les points.
