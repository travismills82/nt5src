Affichage des messages FTP

Pour afficher les messages de bienvenue, de fin et du nombre maximal de connexions

  1. Dans la barre des tâches principale, choisissez Serveur Web.
  2. Choisissez Messages FTP.
  3. Entrez vos messages de bienvenue, de fin et du nombre maximal de connexions dans les zones appropriées.
    • Message de bienvenue—Les utilisateurs verront ce message en se connectant au système serveur via FTP.
    • Message de fin—Ce message s'affichera lorsque les utilisateurs se déconnecteront du système serveur via FTP.
    • Message du nombre maximal de connexions—Ce message informe un utilisateur que le nombre maximal de connexions qu'accepte le système serveur est atteint.
  4. Cliquez sur OK.
