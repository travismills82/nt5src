Configuration des propriétés de protocole

Utilisez la page de propriétés du protocole désigné pour configurer les protocoles réseau voulus.

Pour configurer les propriétés d'un protocole réseau

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Protocoles de partage.
  3. Sélectionnez le protocole que vous souhaitez configurer.
  4. Dans la liste Tâches, choisissez Propriétés.
