Affichage des détails des entrées du journal

Vous pouvez afficher les détails des entrées de journaux d'applications, système ou de sécurité. Dans cette page vous pouvez consulter la date, l'heure, la source, l'identifiant d'un événement, sa description et les données de fichiers journaux spécifiques.

Pour afficher les détails d'un fichier journal

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Journaux.
  3. Choisissez le type de fichier que vous souhaitez afficher.
  4. Sélectionnez l'entrée de journal que vous souhaitez afficher.
  5. Cliquez sur Détails.
  6. Utilisez Précédent et Suivant pour faire défiler les fichiers journaux.
  7. Choisissez Précédent pour retourner au sélecteur d'objet/de tâche pour le type de journal spécifique que vous avez choisi.
