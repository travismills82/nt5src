Présentation des protocoles de partage

Les protocoles de partage permettent aux ordinateurs clients d'accéder aux fichiers de votre système serveur.

Votre système serveur peut accepter certains ou tous les types de protocoles de partage qui suivent :

  • Windows (CIFS)

    Le protocole CIFS (Common Internet File System) est utilisé par des clients équipés d'un système d'exploitation Windows.

  • UNIX (NFS)

    Le protocole Network File System est utilisé par des clients équipés de UNIX.

  • FTP

    Le protocole File Transfer Protocol est un autre moyen d'accéder à un partage de fichiers depuis n'importe quel système d'exploitation

  • Web (HTTP)

    Le protocole Hypertext Transfer Protocol permet d'accéder à un partage de fichiers à partir de navigateurs Web.

  • Netware

    Ce protocole permet d'accéder à un partage de fichiers à partir de clients équipés de NetWare.

  • AppleTalk

    Protocole d'accès à un partage de fichiers depuis des clients Macintosh.
