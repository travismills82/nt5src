Configuring UPS

On the UPS Configuration  page, you can configure the uninterruptible power supply service on your server appliance.

To configure the UPS service

  1. On the primary navigation bar, choose Maintenance.
  2. Choose UPS.
  3. Select the manufacturer of your UPS device.
  4. Select the model of your UPS device.
  5. Select the serial port to which the UPS is connected.
  6. Select the Enable the UPS service on this appliance check box.
  7. Choose OK.
