Modèle de navigation de l'interface Web

Dans le haut de chaque page de l'interface Web se trouvent une zone d'état ainsi que des barres de navigation principale et secondaire. Le corps de chaque page de l'interface Web constitue la zone du contenu.

Voici la description de ces sections :

Zone d'état

Dans la bande supérieure de la fenêtre, qui constitue la zone d'état, s'affichent, de gauche à droite :

  • le logo du constructeur du système serveur Windows Powered (facultatif)
  • le nom d'hôte du système serveur et son état
  • le logo Microsoft Windows Powered

Il existe quatre affichages possibles pour État :

  • Normal (texte vert)
  • Information (texte gris)
  • Avertissement (texte jaune)
  • Critique (texte rouge)

Vous pouvez cliquer sur État : <type d'état> pour obtenir des informations détaillées sur l'état du système serveur.

Barres de navigation

Immédiatement au-dessous de la zone d'état se trouvent les barres de navigation. L'illustration ci-dessous présente les barres de navigation principale (bleue) et secondaire (blanche) d'un système serveur Web standard.

La barre de navigation principale donne la liste des tâches disponibles sur le système serveur, classées par type. La barre de navigation secondaire donne la liste des tâches secondaires disponibles pour la tâche principale sélectionnée. La barre de navigation secondaire est dynamique, et les types de tâches disponibles changent en fonction du groupe de tâches sélectionné.

Les info-bulles des éléments des barres de navigation fournissent des informations complémentaires. Placez le curseur de la souris sur chaque élément pour afficher son info-bulle.

Zone de contenu

Les informations de la page s'affichent dans la zone de contenu, située sous les barres de menus. Cette section de l'interface Web décrit les activités de gestion que vous pouvez exécuter dans cette page. Ce texte fournit parfois également des instructions sur les outils disponibles pour l'exécution des tâches sélectionnées.

De nombreuses pages de tâches comportent un sélecteur d'objet/de tâche. Un sélecteur d'objet/de tâche est simplement une table donnant la liste des objets que vous pouvez gérer ou configurer, leur description et les tâches que vous pouvez exécuter. La colonne de gauche du sélecteur d'objet/de tâche contient un bouton radio sur lequel vous cliquez pour sélectionner un objet donné. Lorsqu'il est possible de sélectionner plusieurs objets dans le sélecteur d'objet/de tâche, la colonne de gauche contient une case à cocher au lieu d'un bouton radio. La colonne de droite donne la liste des tâches que vous pouvez exécuter.

Pour naviguer dans l'interface Web

  1. Dans la barre de navigation principale, cliquez sur le type général de tâche que vous souhaitez exécuter.
  2. Dans la barre de navigation secondaire ou dans la liste des tâches figurant dans la zone de contenu de la page, cliquez sur le type spécifique de tâche que vous souhaitez exécuter.
  3. Dans la zone de contenu de la page suivante :
    1. S'il existe un sélecteur d'objet/de page, sélectionnez l'objet que vous voulez gérer ou configurer en cliquant sur le bouton radio placé à gauche de son nom. Sélectionnez ensuite la tâche à exécuter dans la liste Tâches située à droite. Selon les pages, il se peut que le sélecteur d'objet/de tâche propose une fonction de recherche, ainsi que la possibilité de consulter l'ensemble des informations contenues dans la table.
    2. En l'absence du sélecteur d'objet/de page, entrez les données dans les champs indiqués pour accomplir la tâche choisie.
  4. Lorsque vous avez terminé chaque tâche, cliquez sur OK pour confirmer vos modifications ou sur Annuler pour rétablir les anciens paramètres.
Une fois le changement effectué ou annulé, la page précédente s'affichera.

Si vous êtes dans une page de propriétés et si vous sélectionnez un autre élément dans la barre de navigation, une fenêtre s'affiche avec le message Cliquez sur OK pour ignorer les modifications. Ceci vous permet soit de valider soit d'annuler les modifications avant de passer à la page sélectionnée suivante.
