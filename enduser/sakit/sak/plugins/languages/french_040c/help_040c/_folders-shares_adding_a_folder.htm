Ajout d'un dossier

Vous pouvez créer dans le système serveur autant de dossiers que vous le souhaitez.

Pour créer un nouveau dossier

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Dossiers.
  3. Sélectionnez le volume sur lequel vous souhaitez travailler, puis cliquez sur Ouvrir.
  4. Ouvrez le document dans lequel vous souhaitez ajouter un sous-dossier.
  5. Choisissez Nouvelle dans la listeTâches.
  6. Dans l'onglet Général, entrez le nom du nouveau dossier dans la zone Nom.
  7. Définissez les attributs d'accès en sélectionnant la case à cocher appropriée.
  8. Facultatif : Si vous souhaitez compresser le contenu du nouveau dossier pour gagner de la place, sélectionnez l'onglet Compresser.
  9. Cliquez sur OK.

    Le sélecteur d'objet/de tâche contient maintenant le dossier que vous avez ajouté. Si votre nouveau dossier n'apparaît pas immédiatement dans le tableau, faites défiler la liste pour le trouver.
