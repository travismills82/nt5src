Définition d'une langue

Si d'autres langues sont installées sur votre système serveur, vous pouvez changer la langue qu'il utilise dans l'interface Web, l'interface locale et les fichiers d'Aide.

Pour changer la langue qu'utilise votre système serveur

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Langue.
  3. Dans la page Définir la langue, sélectionnez la langue voulue.
  4. Cliquez sur OK.
