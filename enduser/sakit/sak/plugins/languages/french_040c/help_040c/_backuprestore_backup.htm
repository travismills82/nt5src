Sauvegarde et restauration du système d'exploitation

Dans cette page, vous pouvez choisir de sauvegarder ou de restaurer le système d'exploitation (OS) du système serveur.

Pour sauvegarder ou restaurer le système d'exploitation du système serveur

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Sauvegarder.
  3. Connectez-vous au client avancé des services Terminal Server. L'application de sauvegarde démarre automatiquement.
  4. Lorsque vous avez terminé, fermez l'application en cliquant sur le X dans l'angle supérieur droit. Ceci vous déconnecte du client avancé des services Terminal Server.

    Remarque   La fermeture de la session peut demander quelques instants lorsque vous quittez l'application.

Client des services Terminal Server

Le client des services Terminal Server est l'outil qui permet de sauvegarder et de restaurer le système d'exploitation du système serveur. Il n'accepte que deux connexions simultanées. En outre, si vous naviguez vers une autre page lorsqu'une session est ouverte, le client sera déconnecté, mais la session sera préservée. Ceci peut empêcher d'autres utilisateurs d'accéder à une session de services Terminal Server.
