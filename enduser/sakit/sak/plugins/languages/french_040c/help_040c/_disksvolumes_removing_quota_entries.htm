Suppression d'entrées de quota

Pour supprimer une entrée de quota

  1. Dans la barre de navigation principale, choisissez Disques.
  2. Choisissez Quota de disque.
  3. Sélectionnez le volume que vous souhaitez gérer.
  4. Dans la liste Tâches, choisissez Entrées de quota.
  5. Dans la page Entrées de quota, sélectionnez le nom d'ouverture de session pour lequel vous souhaitez supprimer l'entrée de quota.
  6. Dans la liste Tâches, choisissez Supprimer.
  7. Cliquez sur OK.
