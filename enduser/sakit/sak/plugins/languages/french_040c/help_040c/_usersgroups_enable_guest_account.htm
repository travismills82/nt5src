Activation du compte invité

Par défaut, le compte invité est désactivé. Pour les groupes de travail qui ont des ordinateurs clients Windows 95 et Windows 98, l'activation du compte invité est le moyen le plus rapide de leur donner accès aux ressources d'un système serveur. Cependant, si vous activez le compte invité, tout utilisateur connecté au réseau aura accès aux ressources du système serveur. Une autre solution consiste à créer un compte utilisateur pour chaque utilisateur sur le réseau.

Pour activer le compte utilisateur

  1. Dans la barre de navigation principale, choisissez Utilisateurs.
  2. Choisissez Utilisateurs locaux.
  3. Dans la colonne Nom, sélectionnez Invité.
  4. Dans la liste Tâches, choisissez Propriétés.
  5. Sous l'onglet Général, désactivez la case à cocher Désactiver ce compte utilisateur.
  6. Cliquez sur OK.
