Modification des groupes de clients NFS

Pour ajouter des membres à un groupe de clients NFS

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Protocoles de partage.
  3. Sélectionnez Protocole NFS, puis Propriétés.
  4. Choisissez Groupes de clients.
  5. Sélectionnez le groupe que vous souhaitez modifier.
  6. Dans la liste Tâches, choisissez Modifier.
  7. Dans la page Modifier le groupe de clients NFS, entrez l'adresse IP ou le nom de l'ordinateur du membre à ajouter au groupe.
  8. Choisissez Ajouter.
  9. Cliquez sur OK.

Pour supprimer des membres d'un groupe de clients NFS

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Protocoles de partage.
  3. Sélectionnez Protocole NFS, puis Propriétés.
  4. Choisissez Groupes de clients.
  5. Sélectionnez le groupe que vous souhaitez modifier.
  6. Dans la liste Tâches, choisissez Modifier.
  7. Dans la page Modifier le groupe de clients NFS, sélectionnez l'adresse IP ou le nom de l'ordinateur du membre à supprimer du groupe.
  8. Choisissez Supprimer.
  9. Cliquez sur OK.
