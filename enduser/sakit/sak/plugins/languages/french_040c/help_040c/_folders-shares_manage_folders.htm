Gestion des dossiers

La page Volumes vous permet d'ouvrir ou de partager, un certain nombre de volumes réseau. Elle affiche un sélecteur d'objet/de page doté des colonnes suivantes :

  • Nom du volume—Affiche la liste de tous les volumes par leur nom. Pour créer, ouvrir, supprimer ou configurer les propriétés d'un volume donné, sélectionnez la case à cocher en regard du nom du volume concerné.
  • Taille totale—Affiche la taille totale du volume.
  • Espace libre—Affiche la quantité d'espace disponible dans le volume.
  • Type de partage—Indique le type de partage affecté au dossier :

    W = Partage Windows (CIFS)

    U = Partage UNIX (NFS)

    F = Partage FTP

    H = Partage HTTP

    A = Partage AppleTalk

    N = Partage NetWare

Le sélecteur d'objet/de tâche affiche les cent premiers dossiers trouvés. Pour parcourir la liste des dossiers à l'aide du sélecteur d'objet/de tâche, vous pouvez effectuer une recherche sur les champs disponibles dans la liste Rechercher, puis entrer les critères de recherche dans la zone située à gauche du bouton Aller à, ou bien faire défiler la liste. En outre, s'il existe plus de cent dossiers, vous pouvez les afficher par groupes de 100 à l'aide des boutons Page précédente et Page suivante placés à droite du bouton Aller à.

Pour partager les dossiers

  1. Dans la page Partages, choisissez Dossiers.
  2. Sélectionnez le ou les volume(s) dont vous voulez partager les dossiers, puis choisissez Partage dans la liste Tâches.
  3. Entrez les informations que vous demandent les invites. Pour plus d'informations sur la façon de remplir la page de propriétés d'une méthode de partage de fichiers donnée, consultez le fichier d'aide spécifique à ce protocole de partage.

Pour gérer les dossiers

  1. Dans la page Partages, choisissez Dossiers.
  2. Sélectionnez le ou les volume(s) dont vous voulez consulter ou gérer les dossiers ou les partages, et choisissez Ouvrir dans la liste Tâches.

    La page Dossiers vous permet de créer, d'ouvrir, de supprimer ou de configurer un certain nombre de dossiers réseau. Elle affiche un sélecteur d'objet/de page doté des colonnes suivantes :

    Nom du dossier—Donne la liste de chaque dossier par son nom.

    Date de modification—Indique la date de dernière modification du dossier.

    Attributs—Affiche un ou plusieurs des attributs de dossier qui suivent :

    R = Lecture seule

    A = Prêt pour l'archivage

    H = Caché

    C = Compressé

    S = Dossier système

    Type de partage—Indique le type de partage affecté au dossier.

  3. Sélectionnez un dossier, puis choisissez la tâche à exécuter dans la liste Tâches.
  4. Pour vous rendre dans les répertoires secondaires d'un répertoire racine, sélectionnez ce répertoire racine puis choisissez Ouvrir dans la liste Tâches.
  5. Pour remonter d'un répertoire secondaire à son parent, choisissez Précédent dans la liste Tâches. Si les dossiers racines sont déjà affichés dans le sélecteur d'objet/de tâche, il n'y a pas de dossier parent à choisir.
