Protocoles de partage

La page Protocoles de partage vous permet d'activer, désactiver, arrêter ou configurer les protocoles réseau applicables. La page Protocoles de partage affiche le sélecteur d'objet/de tâche avec les colonnes suivantes :

  • Nom—Affiche chaque protocole par son nom. Pour activer, désactiver ou modifier les propriétés d'un protocole donné, cliquez sur le bouton en regard du protocole concerné.
  • État—Indique que le protocole est En cours d'exécution, Arrêté ou En pause.
  • Type de démarrage—Indique si le protocole doit démarrer automatiquement lorsque le système serveur démarre, ou bien s'il doit être lancé manuellement ou être désactivé.
  • Description—Affiche une brève description du protocole.

    La liste Tâches est située à côté du sélecteur d'objet/de tâche. Dans la colonne Nom du sélecteur d'objet/de tâche, sélectionnez un protocole. Pour exécuter une tâche, choisissez cette tâche dans la liste Tâches.
