Protocole FTP

FTP, protocole de transfert de fichiers, est intégré dans le modèle de sécurité Windows. Les utilisateurs qui se connectent avec FTP sont authentifiés d'après les comptes utilisateurs enregistrés dans le système serveur Windows Powered. L'accès leur est accordé en fonction de ces profils utilisateurs. Sachez cependant que le serveur FTP a pour caractéristique de pouvoir envoyer les mots de passe des utilisateurs sur le réseau sans cryptage. Par conséquent, un utilisateur bénéficiant d'un accès physique pourrait voir ces mots de passe durant le processus de validation FTP.

FTP reconnaît toutes les commandes de clients FTP Microsoft Windows ; lorsqu'un système serveur exécute FTP, d'autres ordinateurs dotés de l'utilitaire FTP peuvent se connecter au serveur et transférer des fichiers. Cependant, des versions non Microsoft de clients FTP peuvent contenir des commandes non reconnues par le protocole serveur FTP.
