Réseau

Dans la page Réseau, vous pouvez choisir les propriétés de système serveur relatives au réseau que vous souhaitez configurer :

  • Identification—Définissez le nom et le domaine d'appartenance du système serveur.
  • Paramètres globaux—Configurez les paramètres réseau applicables à toutes les cartes réseau du système serveur.
  • Interfaces—Configurez les propriétés de chaque carte réseau du système serveur.
  • Administrateur—Changez le mot de passe du compte utilisateur qui vous donne accès au site Web d'administration du système serveur.
  • Site Web d'administration—Spécifiez la ou les adresses IP et le port à utiliser pour accéder au site Web d'administration.
  • Telnet—Configurez Telnet pour administrer le système serveur.
