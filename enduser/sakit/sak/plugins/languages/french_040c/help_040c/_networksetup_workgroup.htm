Groupe de travail

Un groupe de travail est un regroupement simple d'ordinateurs, conçu uniquement pour aider les utilisateurs à trouver des imprimantes et des dossiers partagés à l'intérieur de ce groupe. Votre système serveur peut faire partie d'un groupe de travail, d'un environnement d'annuaire Active Directory ou d'un domaine Windows NT 4. Dans Microsoft Windows 2000, les groupes de travail n'offrent pas les comptes utilisateurs centralisés ni l'authentification qu'offrent les domaines.

Un nom de groupe de travail ne peut reprendre le nom de l'ordinateur. Un nom de groupe de travail peut être composé de 15 caractères au maximum et contenir tout caractère, à l'exception des suivants : ; : " < > * + = \ | ? , .

Pour définir ou modifier l'appartenance du système serveur à un groupe de travail

  1. Dans la barre de navigation principale, choisissez Réseau.
  2. Choisissez Identification.
  3. Cliquez sur le bouton Groupe de travail et entrez le nom du groupe de travail auquel vous joindre.
  4. Si le système serveur appartenait à un domaine avant de rejoindre le groupe de travail, le système serveur sera exclu du domaine, et le compte de l'ordinateur sera désactivé.
  5. Cliquez sur OK.
  6. Vous allez être invité à redémarrer le système serveur.

    Choisissez si vous voulez le redémarrer.

    • Si vous cliquez sur OK, le système serveur redémarre, et une page s'affiche, signalant ce redémarrage. Lorsqu'il est de nouveau en ligne, la page par défaut de l'interface Web s'affiche, et vos modifications ont été prises en compte.
    • Si vous cliquez sur Annuler, les modifications à l'identité du système serveur ne sont pas prises en compte tant que vous n'avez pas redémarré.
