Gestion des journaux de partages Web (HTTP)

Le journal de partages Web (HTTP) contient les événements enregistrés par le serveur Web concernant l'accès aux partages HTTP.

Pour gérer les journaux de partages Web (HTTP)

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Journaux.
  3. Choisissez Journal des partages Web (HTTP).
