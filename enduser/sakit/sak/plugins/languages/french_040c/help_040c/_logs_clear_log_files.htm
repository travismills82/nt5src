Suppression des fichiers journaux

Dans cette page, vous pouvez supprimer des fichiers journaux spécifiques. Lorsque vous supprimez des journaux d'applications, système, NFS ou de sécurité, tout le contenu est supprimé, mais pour les journaux FTP, d'administration Web ou de partages Web (HTTP), vous pouvez sélectionner les fichiers spécifiques à supprimer.

Pour supprimer des journaux d'applications, système, NFS ou de sécurité

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Journaux.
  3. Sélectionnez le type des fichiers à supprimer.
  4. Sélectionnez Supprimer le journal.
  5. Cliquez sur OK.

Pour supprimer des journaux FTP, d'administration Web ou de partages Web (HTTP)

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Journaux.
  3. Sélectionnez le type des fichiers à supprimer
  4. Sélectionnez un ou plusieurs fichiers journaux à supprimer.
  5. Choisissez Supprimer le journal.
  6. Cliquez sur OK.
