Partage d'un dossier

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Dossiers.
  3. Sélectionnez le volume sur lequel vous souhaitez travailler, puis cliquez sur Ouvrir.
  4. Rendez-vous dans le répertoire contenant le dossier à partager.
  5. Sélectionnez le dossier à partager.
  6. Dans la liste Tâches, choisissez Partager.
    • Si ce dossier n'a pas encore été partagé, la page Nouveau partage s'affiche.
    • Si ce dossier a déjà été partagé, la page Propriétés du partage s'affiche.
    • Si ce dossier a été partagé sous plusieurs noms, sélectionnez la tâche Gérer le partage.
