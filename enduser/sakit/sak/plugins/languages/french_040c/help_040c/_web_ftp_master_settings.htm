Paramètres principaux FTP

Pour activer ou désactiver l'accès anonyme à votre système serveur depuis des clients FTP, utilisez la case à cocher Autoriser les connexions FTP anonymes.

La zone Répertoire racine FTP affiche le répertoire racine FTP auquel vous accédez lorsque vous vous connectez avec une connexion FTP anonyme.

La zone Style de listage de répertoire définit le format dans lequel les répertoires sont listés lorsqu'un client FTP envoie une opération LIST.

  • Par défaut, le style de listage MS-DOS affiche l'année dans un format à deux chiffres.
  • Le style de listage UNIX affiche l'année dans un format à quatre chiffres lorsque la date du fichier est différente de l'année du serveur FTP. Si l'année du fichier et celle du serveur sont identiques, aucune date d'année n'est retournée.

Nombre maximal de connexions représente le nombre maximal de connexions FTP autorisées simultanément sur le système serveur.

Délai de connexion (secondes) est le nombre de secondes sans activité avant que le serveur ne se déconnecte automatiquement.
