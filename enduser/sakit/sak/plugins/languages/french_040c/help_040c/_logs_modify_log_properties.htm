Modification des propriétés du journal

Vous pouvez modifier les propriétés de journaux d'applications, système ou de sécurité. Dans cette page vous pouvez spécifier la taille maximale du journal et déterminer comment le système gérera les entrées de journal si la capacité maximale du système serveur est atteinte.

Pour modifier les propriétés d'un fichier journal

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Journaux.
  3. Choisissez le journal Application, Système ou Sécurité que vous souhaitez configurer.
  4. Sélectionnez le journal que vous souhaitez configurer.
  5. Choisissez Propriétés.
  6. Dans la zone Taille maximale du journal, entrez la taille maximale du journal, en kilo-octets.
  7. Cliquez sur le bouton correspondant le mieux à la façon dont vous souhaitez traiter les entrées de journal si la taille maximale est atteinte.
