Désactivation des connexions FTP anonymes

Pour désactiver les connexions FTP anonymes

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Protocoles de partage.
  3. Sélectionnez Protocole FTP, puis Propriétés.
  4. Sélectionnez l'onglet Connexion anonyme.
  5. Désactivez la case à cocher Activer les connexions anonymes, puis cliquez sur OK.
