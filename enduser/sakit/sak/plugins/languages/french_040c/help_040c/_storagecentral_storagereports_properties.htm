Report Properties

Use the report properties page to change report filter selections, format selections, and set report output location.    

Filter tab

By name  Files may be filtered by name using standard file selection masks. You can enter multiple file selection masks by separating them with semicolons. For example *.exe;*.dll will select all the .exe and .dll files.

By screening group  Files may be filtered by screening groups. Screening groups are lists of file selection masks that can be inclusive or exclusive. For example, one predefined screening group is called Graphics Files.

Format tab

  • HTML  The file is outputted to an HTML format, which supports Microsoft(r) Internet Explorer 5 and Netscape 6.
  • Text  The file is outputted as a .txt file.

Output tab

Save the report to the following directory  You can choose the directory in which to save the report. This can be any valid Windows path name. If left blank, the default directory \<Installation Path>\Reports is used. 

Mail the report to the following e-mail address   You can type a valid Internet e-mail address using the format user@organization.domain to send the report as an e-mail attachment. If left blank, no e-mail is sent.
