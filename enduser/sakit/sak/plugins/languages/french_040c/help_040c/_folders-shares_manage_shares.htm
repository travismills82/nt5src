Gestion des partages

La page de gestion Partages vous permet de créer, d'ouvrir, de supprimer ou de configurer toutes sortes de partages de fichiers. La page Partages affiche un sélecteur d'objet/de tâche doté des colonnes suivantes :

  • Nom du partage—Énumère chaque dossier partagé par son nom. Pour créer, ouvrir, supprimer ou configurer les propriétés d'un partage donné, activez la case à cocher en regard du nom du partage concerné.
  • Chemin du partage—Affiche le chemin d'accès au partage.
  • Type—Indique l'un des types de partage suivants :

    W = Windows (CIFS)

    U = UNIX (NFS)

    F = FTP

    H = HTTP

  • Commentaire—Affiche une brève description du partage, si vous en avez entré une.

Utilisez le sélecteur d'objet/de tâche pour sélectionner un partage, puis choisissez dans la liste Tâches la tâche à exécuter.
