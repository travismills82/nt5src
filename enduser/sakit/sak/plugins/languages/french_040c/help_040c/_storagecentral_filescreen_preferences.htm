File Screening Preferences

Use the Preferences page to change the following file screening options in the audit database:

  • Database Location The location of the audit database.
  • Record Screening Activity in audit database Record screened files in the audit database.  If this option is not selected, all recording activity is halted for the appliance.
  • Record user name in audit database Records the user identity in the audit database.  The default is not to record the user.
