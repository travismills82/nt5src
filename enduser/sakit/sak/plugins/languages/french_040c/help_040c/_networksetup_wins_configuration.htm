Configuration WINS

Cette page de propriétés vous permet d'entrer les adresses de serveurs WINS externes. Le système serveur ne contient pas de serveur WINS. Pour les besoins de cette explication, le système serveur est un client WINS.

Les ordinateurs clients supportant WINS peuvent être configurés pour utiliser directement un serveur WINS. La plupart des ordinateurs clients WINS ont en général plusieurs noms NetBIOS (network basic input/output system) qu'ils doivent enregistrer sur le réseau. Ces noms permettent de publier différents types de services réseau, tels que le service Messagerie ou Station de travail, que chaque ordinateur peut utiliser de différentes manières pour communiquer avec d'autres ordinateurs du réseau.

Les clients supportant WINS communiquent avec le serveur WINS pour vous permettre d'exécuter les tâches suivantes :

  • inscrire les noms des clients dans la base de données WINS ;
  • renouveler les noms des clients avec la base de données WINS ;
  • libérer les noms des clients de la base de données WINS ;
  • résoudre les noms en obtenant de la base de données WINS des mappages pour les noms d'utilisateur, les noms NetBIOS, les noms DNS et les adresses IP.

Les clients non activés pour WINS peuvent utiliser des serveurs proxy WINS pour participer à ces processus de façon limitée. Si vous utilisez un serveur DHCP pour allouer des adresses IP de serveurs WINS, il n'est pas nécessaire d'ajouter des adresses de serveurs WINS.

N'oubliez pas que l'interface Web ne vous permet de manipuler que deux adresses WINS, et même dans ce cas, seulement si vous assignez de façon statique l'adresse IP de la carte réseau. Si vous disposez de DHCP, vous pouvez supprimer une ou deux adresses existantes et ajouter d'autres adresses, mais vous ne pouvez pas supprimer tous les serveurs WINS d'une carte réseau activée pour DHCP. Si vous supprimez deux adresses WINS et que vous n'en ajoutez pas au moins une, DHCP assignera automatiquement des adresses WINS.

Les clients WINS tentent d'inscrire leurs noms sur un serveur WINS lorsqu'ils démarrent ou rejoignent le réseau. Les clients WINS interrogent ensuite le serveur WINS afin de résoudre les noms distants, le cas échéant.

Pour modifier les paramètres WINS du système serveur

  1. Dans la barre de navigation principale, choisissez Réseau.
  2. Choisissez Interfaces.
  3. Sélectionnez la connexion réseau à modifier.
  4. Dans la liste Tâches, choisissez WINS.
  5. Dans la liste Serveurs WINS, sélectionnez l'adresse IP du serveur WINS à supprimer, puis cliquez sur Supprimer.
  6. Dans la zone Adresse du serveur WINS, entrez l'adresse IP du serveur WINS, puis cliquez sur Ajouter.
  7. Répétez les étapes 4 et 5 pour chaque adresse IP de serveur WINS à ajouter.
  8. Cliquez sur OK.
