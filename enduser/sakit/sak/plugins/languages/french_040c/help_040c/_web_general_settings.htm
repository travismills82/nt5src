Rubrique précédente

Configuration des propriétés générales d'identité du site

  1. Dans la zone Identificateur de site, entrez le nom unique du site Web.

    Un identificateur de site Web doit être unique et peut comporter jusqu'à douze caractères majuscules ou minuscules à l'exception des caractères suivants : " / \ [ ] : ; | = , + * ? < >. Un identificateur de site Web ne peut pas être entièrement composé de points (.) ou d'espaces.

  2. Votre système serveur remplira automatiquement la zone Répertoire avec un chemin de répertoire par défaut. Vous pouvez modifier ce chemin.
  3. Sélectionnez le type de compte utilisateur que vous souhaitez pour accéder à l'administration du site Web. Vous pouvez créer un compte d'utilisateur local ou de domaine, ou désigner un compte utilisateur existant.

    Remarque   Vous ne pouvez créer un compte de domaine que si le système serveur appartient à ce domaine.

  4. En option : Si vous choisissez d'assigner des privilèges administrateur à un compte utilisateur existant, entrez le nom de l'utilisateur dans la zone Administrateur du site.

    Remarque   Si le système serveur appartient à un domaine, et si vous choisissez d'assigner des privilèges administrateur à un compte utilisateur existant, vous devez entrer le nom de domaine et le nom d'utilisateur sous le format <nom_domaine>\<nom_utilisateur>.

  5. En option : Si vous choisissez de créer un compte utilisateur local ou de domaine, entrez puis confirmez le mot de passe adminstrateur dans les zones Mot de passe administrateur et Confirmer le mot de passe. Le mot de passe doit respecter les éventuelles règles de formation de mot de passe en vigueur pour le domaine auquel appartient le système serveur.
