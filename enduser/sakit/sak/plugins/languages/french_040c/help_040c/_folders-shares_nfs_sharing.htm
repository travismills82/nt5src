Configuration des propriétés de partage UNIX (NFS)

Utilisez cette page pour spécifier quels clients NFS ont une autorisation d'accès à chaque partage. L'accès peut être autorisé ou interdit en fonction du nom d'hôte du client. L'accès peut aussi être autorisé ou interdit en fonction de groupes de clients, lorsqu'un groupe de clients contient un ou plusieurs noms d'hôtes clients.

Pour ajouter un nouveau client ou groupe de clients NFS dans un partage

  1. Créez un nouveau groupe de clients comme expliqué dans Ajout de groupes de clients NFS.
  2. Exécutez la procédure qui suit pour ajouter un client ou un groupe de clients existants.

Pour ajouter un client ou un groupe de clients existants

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Dans la page Partages, choisissez Partages.
  3. Sélectionnez le partage pour lequel vous souhaitez ajouter un client ou un groupe de clients NFS.
  4. Dans la liste Tâches, choisissez Propriétés.
  5. Sélectionnez l'onglet Général.
  6. Activez la case à cocher UNIX (NFS).
  7. Cliquez sur l'onglet Partage NFS.
  8. Sélectionnez l'ordinateur ou le groupe voulu dans la liste de gauche, ou entrez le nom d'un ordinateur client NFS ou son adresse IP dans la zone de droite, puis cliquez sur Ajouter.
  9. Sélectionnez dans la liste Type d'accès le degré de contrôle que le client spécifié peut exercer sur les fichiers du partage.
  10. Cliquez sur OK.

Pour supprimer un client NFS

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Dans la page Partages, choisissez Partages.
  3. Sélectionnez le partage pour lequel vous souhaitez supprimer un client ou un groupe de clients NFS.
  4. Dans la liste Tâches, choisissez Propriétés.
  5. Sélectionnez l'onglet Général.
  6. Activez la case à cocher UNIX (NFS).
  7. Sélectionnez l'onglet Partage NFS.
  8. Sélectionnez dans la liste du milieu de page le système serveur ou le groupe de clients voulus, puis cliquez sur Supprimer.
  9. Cliquez sur OK.
