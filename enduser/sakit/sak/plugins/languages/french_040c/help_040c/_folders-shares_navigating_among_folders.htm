Navigation dans les dossiers

Utilisez le sélecteur d'objet/de tâche pour naviguer dans les dossiers. Pour chaque dossier possédant des sous-dossiers, il existe une tâche Ouvrir dans la liste Tâches. Pour chaque dossier possédant un dossier parent, il existe une tâche Précédent dans la liste Tâches.

Pour naviguer dans les dossiers

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Dossiers.
  3. Sélectionnez le volume sur lequel vous souhaitez travailler, puis cliquez sur Ouvrir.
  4. Sélectionnez le dossier de votre choix.
  5. Dans la liste Tâches, choisissez Ouvrir.
  6. Sélectionnez le dossier que vous souhaitez explorer, puis choisissez Ouvrir dans la liste Tâches.

    OU

    • Dans la liste Tâches, choisissez Précédent pour retourner dans le répertoire racine du volume.
