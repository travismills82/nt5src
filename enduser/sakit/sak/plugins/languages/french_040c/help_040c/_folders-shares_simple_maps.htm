Activation de mappages simples

S'ils sont activés, les mappages simples créent automatiquement des mappages entre utilisateurs UNIX et Microsoft Windows qui partagent le même nom d'utilisateur. Dans un mappage simple d'utilisateur, les utilisateurs d'un domaine Windows sont mappés implicitement un à un, en fonction de leur nom d'utilisateur. Lorsque le domaine Windows ainsi que les fichiers de groupes et le mot de passe UNIX ou le domaine NIS sont identifiés, la fonction mappages simples mappe les utilisateurs qui ont le même nom dans les domaines Windows et UNIX ou NIS. En l'absence de correspondance pour un nom d'utilisateur entre les deux domaines, cet utilisateur n'est pas mappé..

Remarque   Pour accéder à cette page, vous devez au préalable entrer un nom de serveur NIS valide dans l'onglet Général.

Pour activer les mappages simples

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Protocoles de partage.
  3. Sélectionnez Protocole NFS, puis Propriétés.
  4. Choisissez Mappages de groupe et d'utilisateur.
  5. Sélectionnez l'onglet Mappages simples.
  6. Activez la case à cocher Activer les mappages simples.
  7. Dans la liste Domaine Windows, sélectionnez le nom du système serveur, ou le domaine auquel il appartient.

    Si vous sélectionnez le nom du système serveur, les utilisateurs et les groupes locaux seront mappés.

  8. Cliquez sur OK.
