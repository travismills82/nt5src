Utilisation de l'aide

Vous pouvez rechercher des informations dans l'Aide par l'un des moyens suivants :

En parcourant les rubriques par catégorie

  • Pour parcourir les rubriques, cliquez sur le titre d'une rubrique dans la liste Sommaire.

Pour appeler l'aide contextuelle

  • À partir de la page pour laquelle vous avez besoin d'aide, cliquez sur l'icône ? à l'extrême droite de la barre de navigation principale.

    Remarque   Vous pouvez revenir en arrière dans l'Aide en cliquant sur le lien Rubrique précédente dans l'angle supérieur droit de chaque page.

Outre l'aide en ligne de votre système serveur spécifique, vous avez également accès à l'aide de Microsoft Windows 2000 avec Terminal Services qui se trouve dans la page Maintenance.

Pour ouvrir l'aide de Windows 2000

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Terminal Services.
  3. Connectez-vous avec votre nom d'utilisateur et votre mot de passe.
  4. Cliquez sur le bouton Démarrer, puis sur Aide.
