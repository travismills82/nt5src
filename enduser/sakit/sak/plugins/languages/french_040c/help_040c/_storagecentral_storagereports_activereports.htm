Using Active HTML Reports

Active HTML Reports provide the capability to search summary information for details on action that may be taken against the files.

Use one of the following methods to search for detailed information:

  1. Open the desired summary graph item.
  2. Open the report item in the report grid. 

To take action against the files

  1. Right click the file to view the Action menu.
  2. Choose one of the following actions from the Action menu:
    • Open Opens the file with its associated application.
    • Edit Opens the file in Notepad.
    • Explore Opens Windows Explorer at the folder which contains the file.
    • Properties Displays the file properties.
    • Delete Deletes the file.
    • Copy To Copies the file to a specified location.
    • Move To Moves the file to a specified location.
    • Find Finds a string in the report.
    • Find Next Repeats the last find.
    • Run Executes an application.
    • Select All Selects all elements in the table.
    • Clear All Clears the current selection.
