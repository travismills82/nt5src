Configuration des paramètres d'un journal FTP

Cette page sert à la configuration des paramètres d'un journal FTP.

Pour configurer les paramètres d'un journal FTP

  1. Dans la barre des tâches principale, choisissez Serveur Web.
  2. Choisissez Paramètres du journal FTP.
  3. Activez la case à cocher Activer la journalisation
  4. Sélectionnez le format de fichier journal approprié dans la zone Format du journal actif.
  5. Cliquez sur le bouton approprié pour régler le délai entre la génération de chaque nouveau journal.

    Si par exemple vous cliquez sur le bouton Tous les jours , un nouveau journal commencera toutes les 24 heures.

  6. Activez la case à cocher Utiliser l'heure locale du serveur pour la désignation et la substitution de fichier.

    Le système serveur adoptera son heure locale pour nommer le fichier et substituer le journal. Sinon, le serveur adoptera l'heure GMT.

    Remarque   Si vous cliquez sur le bouton Lorsque la taille du fichier atteint, la case à cocher Utiliser l'heure locale du serveur pour la désignation et la substitution de fichier se désactive.

  7. Assurez-vous que la zone Répertoire du fichier journal contient le chemin de répertoire correct de vos fichiers journaux.
  8. Cliquez sur OK.
