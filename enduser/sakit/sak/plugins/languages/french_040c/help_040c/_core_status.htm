Accès à la page d'état

Dans la page d'État, vous pouvez consulter les alertes d'état, les ressources système et les informations sur le système. Le volet supérieur gauche affiche les alertes système et le volet supérieur droit, les informations d'état relatives aux ressources système. Si vous sélectionnez une alerte ou une ressource système, des informations détaillées concernant votre sélection s'affichent dans le volet inférieur de votre écran.

Pour accéder à la page État

  • Cliquez sur État : <type d'état> dans la zone État dans le haut de la page.

    —OU—

  • Dans la barre de navigation principale, choisissez État.

Pour plus d'informations sur les alertes affichées dans la page État, consultez Alertes d'état.
