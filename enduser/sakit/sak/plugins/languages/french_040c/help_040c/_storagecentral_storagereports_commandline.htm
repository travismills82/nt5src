Storage Reports Command Line

Use the command line to execute reports from an MS-DOS window.  The following table shows the options the command line supports.

Command switch Description
/R Name of report to execute
/O Object about which the report is run
/M E-mail addresses separated by semicolon where the report is e-mailed
/H or /? Help

In the example, >StorageReports /r "Best Practice Reports..." /o "D:\", the Best Practice report is executed on the D: partition.
