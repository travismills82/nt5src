Administrateur

Le système serveur est configuré avec un jeu de comptes par défaut. Seul le compte administrateur possède des privilèges d'administration.

Remarque   Si un administrateur ajoute un compte de domaine dans le groupe administrateurs local, l'utilisateur du domaine peut accéder au système serveur et l'administrer. Cependant, l'administrateur ne peut pas utiliser la page Modifier le mot de passe administrateur pour changer le mot de passe de son compte de domaine. Cette page ne permet de changer que le mot de passe du compte administrateur local.

Lorsque vous modifiez le mot de passe administrateur, vous ne recevez pas de page de confirmation explicite. Cependant, la modification s'est bien déroulée si aucun message d'erreur ne s'affiche après soumission de ce changement.

Remarque   Vous ne pouvez pas changer le mot de passe administrateur si vous êtes connecté en tant qu'utilisateur de domaine, puisqu'il n'entre pas dans les compétences de l'interface Web du système serveur d'effectuer des changements dans les comptes utilisateurs de domaine, qui sont stockés dans le contrôleur de domaine, et non dans le système serveur.

Dans ce contexte, administrateur désigne le compte utilisateur qui est membre du groupe local administrateurs et qui est utilisé par un utilisateur courant pour établir une connexion. Ce terme ne se réfère pas au compte administrateur par défaut, appelé administrateur.

Si vous recevez le message d'erreur Impossible de changer le mot de passe de ce compte de domaine ou Impossible de changer le nom de compte de ce compte de domaine lorsque vous essayez de modifier le mot de passe administrateur ou son nom de compte, c'est parce que vous êtes connecté en tant qu'utilisateur de domaine. Pour pouvoir changer le mot de passe administrateur, vous devez être connecté comme administrateur du système serveur.

Pour modifier le nom de compte administrateur pour le système serveur

  1. Connectez-vous au système serveur en tant qu'Administrateur.
  2. Dans la barre de navigation principale, choisissez Réseau.
  3. Choisissez Administrateur.
  4. Entrez le mot de passe administrateur actuel dans la zone Mot de passe actuel.
  5. Entrez le nouveau mot de passe administrateur dans la zone Nouveau mot de passe.

    Remarque   Le nouveau mot de passe administrateur doit respecter les éventuelles règles de formation de mot de passe en vigueur pour le domaine auquel appartient le système serveur.

  6. Entrez de nouveau le mot de passe administrateur dans la zone Confirmer le nouveau mot de passe.
  7. Cliquez sur OK.

Pour modifier le nom de compte administrateur pour le système serveur

  1. Connectez-vous au système serveur en tant qu'Administrateur.
  2. Choisissez Réseau.
  3. Choisissez Administrateur.
  4. Entrez le nouveau nom de compte administrateur dans la zone Nom d'utilisateur.
  5. Cliquez sur OK.
