Monitoring File Screening Activity

View the activity monitor for real-time file screening activity on the server appliance. All screening records are displayed and you can sort them by any column.

To monitor screening activity

  1. On the primary navigation bar, choose Disks.
  2. Choose File Screening.    
  3. In the Tasks list, choose  Activity.

The monitor table contains the following information for each screened object:

  • Status The status of the screened object, either active or passive.
  • Time The time the file screening began.
  • Server The name of the server appliance on which the file screen occurred.
  • User The user who performed the operation.
  • File Name The name of the file that was screened.
  • Folder Location The folder where the screened file resides.
  • Policy Name The policy that caused the screening action.
