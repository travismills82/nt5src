Protocole Web (HTTP)

Le protocole HTTP (hypertext transfer protocol) est un protocole de communications conçu pour transférer des documents à liens hypertexte entre ordinateurs sur le Web. HTTP définit les actions que doivent exécuter les navigateurs et les serveurs Web en réponse à diverses commandes.

Serveur World Wide Web

Les commandes utilisées par le Web sont définies dans HTTP.

Pour spécifier l'emplacement d'une ressource, HTTP utilise des URL (Uniform Resource Locators). Les URL sont régies par une convention de dénomination qui identifie de façon unique l'emplacement d'un ordinateur, d'un répertoire ou d'un fichier sur Internet. L'URL spécifie également le protocole Internet—FTP ou HTTP, par exemple—nécessaire pour rechercher la ressource. Si vous connaissez l'URL d'une ressource, vous pouvez indiquer cette URL, ou insérer un lien depuis un document que vous souhaitez mettre à la disposition des usagers du Web.

Le protocol HTTP accepte la connexion anonyme, ainsi que l'authentification de base et l'authentification Windows.

Pour configurer les propriétés de partage Web (HTTP)

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Protocoles de partage.
  3. Sélectionnez Protocole HTTP, puis Propriétés.
  4. Choisissez d'autoriser toute adresse IP pour l'accès aux partages de données sur le système serveur, ou sélectionnez une adresse IP unique dans la liste.
  5. Entrez le numéro de port qui peut être utilisé pour accéder aux partages de données sur le système serveur. Le port par défaut est 80.
