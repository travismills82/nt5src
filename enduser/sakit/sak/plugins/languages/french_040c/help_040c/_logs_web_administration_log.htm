Gestion des journaux d'administration Web

Le journal d'administration Web contient des événements enregistrés par le serveur Web relatifs à l'accès au site d'administration Web.

Pour gérer les journaux d'administration Web

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Journaux.
  3. Choisissez Journal d'administration Web.
