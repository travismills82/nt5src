En-têtes des hôtes

Les noms d'en-têtes des hôtes sont des adresses de sites Web qui permettent à une seule adresse IP de prendre en charge plusieurs sites Web. Par exemple, un serveur d'entreprise pourrait héberger des sites Web pour divers départements internes, tels que le marketing, la comptabilité et les ressources humaines. Pour distinguer ces sites Web les uns des autres, puisqu'ils ont la même adresse IP, ils se voient assigner des noms d'en-têtes d'hôte uniques.

Le tableau qui suit montre comment utiliser des en-têtes d'hôtes uniques avec une seule adresse IP afin de distinguer les différents sites Web.

Département Adresse IP Exemple d'en-tête d'hôte
Site principal de l'entreprise 172.28.144.10 www.CompanyServer.com
Marketing 172.28.144.10 www.Marketing.CompanyServer.com
Comptabilité 172.28.144.10 www.Comptabilité.CompanyServer.com
Ressources humaines 172.28.144.10 www.RessourcesHumaines.CompanyServer.com
