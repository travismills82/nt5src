Paramètres IPX

Le protocole IPX (Internetwork Packet Exchange) est le protocole NetWare d'origine largement utilisé sur les premiers réseaux Novell.

Pour être accessible à partir des clients NetWare, votre système serveur doit leur fournir une adresse IPX.

Pour configurer l'adresse IPX

  1. Dans la page de navigation principale, choisissez Réseau.
  2. Choisissez Paramètres globaux.
  3. Cliquez sur l'onglet Paramètres IPX.
  4. Entrez une valeur dans la zone Numéro de réseau interne, ou laissez la valeur par défaut de ce paramètre à 00000000.
  5. Cliquez sur OK.
