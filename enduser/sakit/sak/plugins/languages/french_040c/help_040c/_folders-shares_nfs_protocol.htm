Protocole NFS

Vous pouvez utiliser l'option Protocole NFS pour configurer le système serveur afin qu'il se comporte en serveur NFS. Le Protocole NFS permet aux utilisateurs de partager leurs fichiers dans un environnement hétérogène d'ordinateurs, de systèmes d'exploitation et de réseaux.

Vous pouvez utiliser le Protocole NFS pour gérer des Groupes de clients NFS, le verrou NFS et les mappages de groupe et d'utilisateur NFS. Néanmoins, les Partages NFS se créent depuis la section Partages de l'interface Web.

Pour configurer le protocole NFS

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Protocoles de partage.
  3. Sélectionnez Protocole NFS, puis Propriétés.
  4. Sélectionnez la tâche de configuration NFS voulue.
