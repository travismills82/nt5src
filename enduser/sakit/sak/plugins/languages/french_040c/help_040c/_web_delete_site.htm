Suppression d'un site Web

Vous pouvez supprimer n'importe quel site Web que vous avez créé sur le système serveur.

Pour supprimer un site Web

  1. Dans la page de navigation principale, choisissez Sites.
  2. Sélectionnez le site Web à supprimer.
  3. Dans la liste Tâches, choisissez Supprimer.
  4. Vérifiez si le site indiqué est bien celui que vous souhaitez supprimer.
  5. Cliquez sur OK.
