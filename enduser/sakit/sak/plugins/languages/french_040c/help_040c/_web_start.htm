Démarrage d'un site Web

Le sélecteur d'objet/de tâche de la page Configuration du site Web indique l'état de chaque site Web résidant sur votre système serveur. Si nécessaire, vous pouvez démarrer tout site Web suspendu ou arrêté ; cependant, si le port d'un site arrêté est en cours d'utilisation par un autre site, le site Web arrêté ne peut pas redémarrer.

Pour démarrer un site Web suspendu ou arrêté

  1. Dans la page de navigation principale, choisissez Sites.
  2. Sélectionnez l'identificateur du site Web que vous souhaitez démarrer.
  3. Dans la liste Tâches, choisissez Démarrer.
