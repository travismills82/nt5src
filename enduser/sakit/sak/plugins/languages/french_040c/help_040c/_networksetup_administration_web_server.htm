Site d'administration Web

Cette fonctionnalité vous permet de changer la ou les adresses IP et le port utilisés pour accéder au site d'administration Web sur le système serveur.

L'adresse IP par défaut à laquelle répond (ou qu'écoute) le système serveur est en général modifiée lorsque le système serveur n'est géré que sur un sous-réseau déterminé ou un réseau de gestion séparé.

Les ports d'écoute par défaut pour les accès tant cryptés que non cryptés peuvent être modifiés selon les besoins pour travailler avec des configurations et des logiciels réseau existants, par exemple, lorsqu'aucun trafic n'est autorisé au-delà d'un numéro de port donné.

Pour changer les propriétés du site d'administration Web

  1. Dans la barre de navigation principale, choisissez Réseau.
  2. Choisissez Site Web d'administration.
  3. Dans la page Propriétés du site d'administration :
    1. Spécifiez s'il faut utiliser Toutes les adresses IP ou Uniquement cette adresse IP.
    2. Si vous choisissez Uniquement cette adresse IP, sélectionnez dans la liste l'adresse IP voulue.
    3. Si vous changez le port pour les accès non cryptés, entrez le nouveau numéro de port dans la zone Port à accès non crypté.
    4. Si vous changez le port pour les accès cryptés, entrez le nouveau numéro de port dans la zone Port à accès crypté (SSL).
  4. Cliquez sur OK.
