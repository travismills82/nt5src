Ajout d'entrées de quota

Pour ajouter une nouvelle entrée de quota

  1. Dans la barre de navigation principale, choisissez Disques.
  2. Choisissez Quota de disque.
  3. Sélectionnez le volume que vous souhaitez gérer.
  4. Dans la liste Tâches, choisissez Entrées de quota.
  5. Dans la liste Tâches, choisissez Nouvelle.
  6. Sélectionnez dans la liste un utilisateur local, ou tapez le nom d'un compte de domaine dans la zone de texte, en respectant le format <nom de domaine\nom d'utilisateur>.

Pour autoriser une utilisation illimitée du disque

  • Sélectionnez le bouton Ne pas limiter l'espace disque.

Pour limiter l'espace disque

  1. Sélectionnez l'option Limiter l'espace disque à.
  2. Dans le champ, entrez une valeur numérique pour spécifier la quantité d'espace disque à allouer à un utilisateur ou un groupe donné. Utilisez la liste pour préciser kilo-octets (Ko), méga-octets (Mo), giga-octets (Go), téraoctets (To), peta-octets (Po) ou exa-octets (Eo).
  3. Entrez à quel volume de remplissage de l'espace disque devra se déclencher un message avertissant l'utilisateur ou le membre du groupe qu'il atteint bientôt la limite de capacité autorisée. Utilisez la liste pour préciser Ko, Mo, Go, To, Po ou Eo.
  4. Cliquez sur OK.
