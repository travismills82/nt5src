Configuration de l'adresse IP

Chaque ordinateur du réseau doit posséder une adresse unique afin d'envoyer et de recevoir des données. Vous pouvez utiliser la page Configuration de l'adresse IP pour demander que votre système serveur reçoive automatiquement la configuration de l'adresse IP depuis le serveur DHCP (Dynamic Host Configuration Protocol). Ou bien, vous pouvez configurer la ou les adresses IP manuellement.

Vous pouvez aussi utiliser la page Configuration de l'adresse IP pour spécifier une ou plusieurs adresses de passerelle. Il s'agit de l'adresse d'un routeur IP local situé sur le même réseau que le système serveur et qui est utilisé pour transférer du trafic vers des destinations situées hors du réseau local. La valeur de chaque champ doit être un nombre compris entre 0 et 255.

Remarque   Le changement de l'adresse IP peut amener le client à perdre sa connexion avec le système serveur. Pour se reconnecter, l'utilisateur doit soit utiliser la nouvelle adresse IP, soit attendre que le serveur DNS soit mis à jour.

Pour définir ou modifier les paramètres IP dans l'onglet Général

  1. Dans la barre de navigation principale, choisissez Réseau.
  2. Choisissez Interfaces.
  3. Sélectionnez la connexion réseau à modifier.
  4. Dans la liste Tâches, choisissez IP.
  5. Sélectionnez l'onglet Général.
  6. Choisissez d'obtenir la configuration automatiquement par le serveur DHCP, ou de configurer de façon statique la ou les adresses IP
  7. Pour obtenir la configuration par le serveur DHCP, cliquez sur OK.
  8. Pour utiliser les paramètres IP statiques, entrez les adresses IP, le masque de sous-réseau et la passerelle par défaut dans les zones appropriées.

Pour définir ou modifier les paramètres IP dans l'onglet Avancé

  1. Dans la barre de navigation principale, choisissez Réseau.
  2. Choisissez Interfaces.
  3. Sélectionnez la connexion réseau à modifier.
  4. Dans la liste Tâches, choisissez IP.
  5. Sélectionnez l'onglet Avancé.
  6. Dans la zone Adresse IP située sur la droite, entrez l'adresse IP, puis cliquez sur Ajouter.
  7. Si vous avez une connexion sur un réseau local, entrez les informations de masque appropriées dans la zone Masque de sous-réseau.

    Un masque de sous-réseau est un nombre codé sur 32 bits constitué de quatre nombres compris entre 0 et 255, séparés par des points. En général, par défaut ces nombres utilisent les valeurs 0 ou 255, par exemple 255.255.255.0. Cependant on peut trouver d'autres valeurs numériques, indiquant que le sous-réseau est configuré pour un seul réseau TCP/IP. Ce nombre de valeur différente de 0 ou 255 est combiné avec le numéro d'adresse IP pourt identifier le réseau sur lequel réside votre ordinateur.

  8. Si nécessaire, mettez à jour la Métrique de connexion IP.

    La métrique indique le coût de l'utilisation des itinéraires associés à cette connexion ; sa valeur s'affiche dans la colonne Métrique de ces itinéraires dans la table de routage IP. S'il existe plusieurs itinéraires vers une destination, c'est l'itinéraire ayant la métrique la plus basse qui est utilisé. La valeur par défaut est 1.

  9. Répétez les étapes 1 à 3 pour toutes les autres adresses IP à ajouter.

Pour définir ou modifier les paramètres d'adresse de passerelle

  1. Dans les zones Passerelle et Métrique, tapez l'adresse IP de la passerelle par défaut et de la métrique, puis cliquez sur Ajouter.
  2. Répétez l'étape 1 pour chaque passerelle par défaut à ajouter.
  3. Lorsque vous avez fini de modifier les configurations dans cette page, cliquez sur OK.
