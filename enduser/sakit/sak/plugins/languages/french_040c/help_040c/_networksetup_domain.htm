Domaine

Votre système serveur peut se situer dans un groupe de travail, dans un environnement Active Directory ou dans un domaine Windows NT 4. Dans les environnements Microsoft Windows NT 4 et Microsoft Active Directory, un domaine est un ensemble d'ordinateurs définis par l'administrateur d'un réseau, qui se partagent une base de données d'annuaire commune.

Utilisés pour les informations d'utilisateurs et de groupes Windows, les domaines Windows ont un nom unique et donnent accès aux comptes utilisateurs et groupes centralisés gérés par l'administrateur du domaine. Chaque domaine utilise ses propres stratégies de sécurité et ses propres relations de sécurité avec d'autres domaines et représente une limite de sécurité unique pour un réseau d'ordinateurs Windows. Active Directory se compose d'un ou plusieurs domaines, qui peuvent s'étendre sur plusieurs emplacements physiques.

Pour DNS, un domaine est une arborescence ou une sous-arborescence à l'intérieur de l'espace de noms DNS. Bien que les noms des domaines DNS correspondent souvent aux domaines Active Directory, les domaines DNS ne doivent pas être confondus avec les domaines de mise en réseau Windows et Active Directory.

Par défaut, l'utilisateur d'un domaine doit entrer domaine\nom d'utilisateur lorsqu'il se connecte à un système serveur Web ou à un serveur d'accès réseau (NAS) doté d'un navigateur Netscape.

Pour définir le domaine par défaut utilisé pour la connexion

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Terminal Services.
  3. Connectez-vous à Terminal Services.
  4. Cliquez avec le bouton droit sur Poste de travail, puis cliquez sur Gérer.
  5. Ouvrez Services Internet (IIS).
  6. Cliquez avec le bouton droit sur Site Web d'administration et choisissez Propriétés.
  7. Sélectionnez Sécurité du répertoire.
  8. Dans la zone Connexions anonymes et contrôle d'authentification, choisissez Modifier.
  9. Dans la zone Authentification de base, choisissez Modifier.
  10. Entrez le nom de domaine par défaut à utiliser pour l'authentification.
