Configuration des propriétés de partage NetWare

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Dans la page Partages, choisissez Partages.
  3. Sélectionnez le partage pour lequel vous souhaitez créer un accès client NetWare puis choisissez Propriétés.
  4. Sous l'onglet Général, activez la case à cocher NetWare.
  5. Sélectionnez l'onglet Partage Netware, puis définissez le nombre maximal d'utilisateurs et les autorisations, et ajoutez ou supprimez des utilisateurs et des groupes.
  6. Cliquez sur OK.
