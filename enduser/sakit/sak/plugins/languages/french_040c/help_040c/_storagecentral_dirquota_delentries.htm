Deleting a Directory Quota

Deleting a file typically does not delete the file; instead, it renames it to the Recycle Bin folder. If the user’s quota is on his own directory, it will not apply to the user's directory quota. If, however, a user quota is on the root directory of the partition, the used disk space may actually increase after the file is deleted. This is because the Recycle Bin must store additional information to allow the file to be restored. 

To delete a directory quota

  1. On the primary navigation bar, choose Disks.
  2. Choose Directory Quota.
  3. Select the directory quota you want to delete.
  4. In the Tasks list, choose Delete.
  5. Choose OK.
