Suppression d'un partage

Vous pouvez totalement supprimer des partages, ou simplement désactiver un protocole donné. Dans ce cas, l'accès au partage est supprimé, mais les fichiers eux-mêmes demeurent sur le système serveur.

Pour supprimer un partage et tous ses protocoles

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Dans la page Partages, choisissez Partages.
  3. Sélectionnez le partage à supprimer.
  4. Dans la liste Tâches, choisissez Supprimer.
  5. Cliquez sur OK.

Pour supprimer des protocoles spécifiques

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Dans la page Partages, choisissez Partages.
  3. Sélectionnez le partage dont vous voulez modifier les propriétés.
  4. Dans la liste Tâches, choisissez Propriétés.
  5. Désactivez la ou les cases à cocher du ou des protocoles que vous souhaitez supprimer du partage.
  6. Cliquez sur OK.
