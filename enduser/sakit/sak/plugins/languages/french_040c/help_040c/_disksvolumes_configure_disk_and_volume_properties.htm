Configuration des propriétés des disques et volumes

Dans cette page, vous pouvez configurer les disques et les volumes du système serveur.

Pour gérer les disques et les volumes du système serveur

  1. Dans la barre de navigation principale, choisissez Disques.
  2. Choisissez Disques et volumes.
  3. Connectez-vous au client avancé des services Terminal Server.
  4. Lorsque vous avez terminé, fermez l'application.

    Remarque   La fermeture de la session peut demander quelques instants lorsque vous quittez l'application.

Client avancé des services Terminal Server

Le client avancé des services Terminal Server est un outil généraliste qui vous donne un accès illimité à votre système serveur. Si vous accédez au client avancé des services Terminal Server depuis la page Disques et volumes, il adopte un mode dédié et ne peut être utilisé que pour gérer les disques et les volumes de votre système serveur.

Le client avancé des services Terminal Server n'accepte que deux connexions simultanées. En outre, si vous naviguez vers une autre page lorsqu'une session est ouverte, le client sera déconnecté, mais la session sera préservée.
