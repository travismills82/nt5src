Suppression d'un dossier

Vous pouvez supprimer n'importe quel dossier que vous avez créé sur le système serveur.

Pour supprimer des dossiers

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Dossiers.
  3. À l'aide du sélecteur d'objet/de tâche, rendez-vous dans le dossier dont vous voulez supprimer un sous-dossier.
  4. Sélectionnez le ou les dossiers à supprimer.
  5. Dans la liste Tâches, choisissez Supprimer.
  6. Vérifiez si le dossier indiqué est bien celui que vous souhaitez supprimer.
  7. Cliquez sur OK.
