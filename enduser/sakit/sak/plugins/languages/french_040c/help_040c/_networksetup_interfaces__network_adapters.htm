Interfaces : Cartes réseau

Une carte réseau fournit l'interface physique (connecteur) et le matériel qui permettent à un ordinateur d'accéder à un réseau. Une carte réseau est également appelée adaptateur, NIC (Network interface card) ou simplement carte.

Dans la page Interfaces de l'interface Web du système serveur, vous pouvez exécuter l'une des tâches suivantes :

  • Changer le nom de la connexion.
  • Définir ou changer les adresses IP, les adresses de passerelle, les masques de sous-réseau et la métrique.
  • Définir ou changer le mode de résolution de noms DNS adopté par le système serveur.
  • Définir ou changer la configuration des clients WINS (Windows Internet Naming Service).
  • Configurer AppleTalk, s'il a été activé par le constructeur de votre système serveur.

Pour configurer une carte réseau

  1. Sélectionnez la carte voulue dans la colonne Description.
  2. Sélectionnez la tâche voulue dans la liste Tâches.
  3. Dans la page Tâche, utilisez les onglets du volet de gauche pour afficher la configuration de la carte réseau choisie.
