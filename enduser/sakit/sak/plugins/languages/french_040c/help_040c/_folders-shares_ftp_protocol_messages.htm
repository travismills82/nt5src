Ajout de messages FTP personnalisés

Vous pouvez créer des messages personnalisés de bienvenue et de fin adressés aux utilisateurs lorsqu'ils se connectent et se déconnectent sur votre système serveur.

Pour créer des messages personnalisés

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Protocoles de partage.
  3. Sélectionnez Protocole FTP, puis Propriétés.
  4. Sélectionnez l'onglet Messages.
  5. Dans la zone Message de bienvenue, tapez le message qui accueillera les utilisateurs qui se connecteront au système serveur.
  6. Dans la zone Message de fin, tapez le message qui s'affichera lorsque les utilisateurs se déconnecteront du système serveur.
  7. Cliquez sur OK.
