Modification des propriétés utilisateur

Les propriétés de l'utilisateur sont constituées du nom d'utilisateur, de son nom complet et de sa description. Dans la page Propriétés de l'utilisateur, vous pouvez aussi activer ou désactiver un compte utilisateur. Vous pouvez également modifier en une fois les propriétés de plusieurs comptes utilisateurs, en sélectionnant tous les comptes à modifier avant d'ouvrir la page Propriétés.

Pour accéder aux propriétés d'utilisateur

  1. Dans la barre de navigation principale, choisissez Utilisateurs.
  2. Choisissez Utilisateurs locaux.
  3. Sélectionnez le ou les comptes utilisateurs à modifier.
  4. Dans la liste Tâches, choisissez Propriétés.
  5. Modifiez les propriétés utilisateurs que vous voulez.

    Le champ Répertoire de base spécifie un nouveau répertoire qui sera créé, et auquel l'utilisateur aura une autorisation d'accès exclusif. Le nom de ce répertoire est identique au nom d'utilisateur défini précédemment, et sera situé dans le chemin spécifié.

  6. Cliquez sur OK.
