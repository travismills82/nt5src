Ajout d'un compte de groupe

Pour ajouter un compte de groupe

  1. Dans la barre de navigation principale, choisissez Utilisateurs.
  2. Choisissez Groupes locaux.
  3. Dans la liste Tâches, choisissez Nouvelle.
  4. Dans l'onglet Général, entrez le nom et la description du groupe à ajouter.
  5. Dans l'onglet Membres :
    • Pour ajouter des membres au groupe, sélectionnez dans la zone Ajouter un utilisateur ou un groupe un utilisateur ou un groupe à ajouter, puis cliquez sur Ajouter.

      Remarque   Seuls les utilisateurs locaux sont affichés dans cette liste. Pour entrer un compte d'utilisateur de domaine, tapez le domaine et le nom d'utilisateur (<domaine\nom_utilisateur>)

    • Pour supprimer des membres du groupe, sélectionnez dans la zone Membres un membre ou un groupe, puis cliquez sur Supprimer.
  6. Cliquez sur OK.
