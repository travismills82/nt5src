Modification des paramètres du site Web

Une fois votre site Web créé, vous pouvez modifier les paramètres généraux, d'identité du site et des applications selon vos besoins.

Pour modifier les paramètres d'un site Web existant

  1. Dans la page de navigation principale, choisissez Sites.
  2. Sélectionnez le site Web à modifier.
  3. Dans la liste Tâches, choisissez Modifier.

    La page Modifier un site Web s'ouvre et l'onglet Général est actif. Dans cet onglet vous pouvez modifier les paramètres, ou dans les onglets Identités des sites ou Paramètres de l'application, vous pouvez modifier les paramètres qui s'y trouvent.
