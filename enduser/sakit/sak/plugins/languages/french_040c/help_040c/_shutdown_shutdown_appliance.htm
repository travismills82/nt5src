Arrêt du système serveur

Utilisez cette page pour arrêter, redémarrer ou planifier l'arrêt ou le redémarrage du système serveur.

Pour arrêter ou redémarrer le système serveur

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Arrêter.
  3. Choisissez la tâche à exécuter.
  4. Cliquez sur OK pour confirmer votre décision.

    Si vous avez choisi de redémarrer le système serveur, la page Redémarrage en cours s'affiche.

Remarque   La page Redémarrage en cours contrôle régulièrement si le système serveur est de nouveau en ligne. Quand la page Redémarrage en cours détecte que le système est en ligne, elle revient automatiquement à la page par défaut.

Pour planifier un arrêt ou un redémarrage du système serveur

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Arrêter.
  3. Choisissez Arrêt planifié.
  4. Choisissez les paramètres d'arrêt planifié que vous souhaitez.
  5. Cliquez sur OK.

Remarque   Pour annuler un arrêt planifié, cliquez sur le bouton Aucun redémarrage ou arrêt planifié.
