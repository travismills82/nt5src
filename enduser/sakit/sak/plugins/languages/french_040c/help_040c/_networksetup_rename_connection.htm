Changement de nom de la connexion

Pour renommer une connexion d'interface

  1. Dans la page de navigation principale, choisissez Réseau.
  2. Choisissez Interfaces.
  3. Sélectionnez la connexion d'interface à modifier.
  4. Dans la liste Tâches, choisissez Renommer.
  5. Dans la zone Nouveau nom de connexion, entrez le nouveau nom de la connexion d'interface.
  6. Cliquez sur OK.
