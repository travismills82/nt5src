Configuration des propriétés du partage FTP

Utilisez cette page pour spécifier quels clients FTP ont une autorisation d'accès à chaque partage. L'accès peut être autorisé ou interdit en fonction du nom d'hôte du client.

Pour accorder aux clients une permission sur un partage FTP

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Dans la page Partages, choisissez Partages.
  3. Sélectionnez le partage pour lequel vous souhaitez créer un accès client FTP puis choisissez Propriétés.
  4. Sélectionnez l'onglet FTP, puis :
    • Activez la case à cocher Lecture pour autoriser l'accès en lecture.
Ou
  • Activez la case à cocher Écriture pour autoriser l'accès en écriture.
Vous pouvez accorder des autorisations en lecture seule, en écriture seule ou en lecture/écriture.
  1. Cliquez sur OK.

Pour enregistrer les visites des clients dans un partage FTP

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Dans la page Partages, choisissez Partages.
  3. Sélectionnez le partage pour lequel vous souhaitez créer un accès client FTP puis choisissez Propriétés.
  4. Sélectionnez l'onglet Partage FTP.
  5. Activez la case à cocher Accès au journal
  6. Cliquez sur OK.
