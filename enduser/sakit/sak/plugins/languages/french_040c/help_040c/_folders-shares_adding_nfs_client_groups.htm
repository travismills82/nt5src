Ajout de groupes de clients NFS

Pour ajouter un groupe de clients NFS

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Protocoles de partage.
  3. Sélectionnez Protocole NFS, puis Propriétés.
  4. Choisissez Groupes de clients.
  5. Dans la liste Tâches, choisissez Nouvelle.
  6. Dans la page Nouveau groupe de clients NFS, entrez le nom du groupe à ajouter dans la zone Nom du groupe.
  7. Dans la zone Nom du client ou adresse IP, entrez le nom ou l'adresse IP de l'ordinateur à ajouter dans le groupe.
  8. Choisissez Ajouter.
  9. Cliquez sur OK.
