Configuration des propriétés du partage Windows (CIFS)

Les ordinateurs clients sous Windows ont recours au protocole CIFS (Common Internet File System) pour partager les fichiers. Utilisez cette page pour modifier le nombre d'utilisateurs ayant accès à un partage, pour changer les options de mise en cache relatives à ce partage et pour définir ou modifier les autorisations des utilisateurs.

Paramètres du système serveur

Dans la section Nombre maximal d'utilisateurs, vous pouvez choisir d'autoriser le nombre maximal d'utilisateurs, ou spécifier le nombre de connexions autorisées à un moment donné.

Pour définir le nombre maximal d'utilisateurs

  • Cliquez sur le bouton Maximum autorisé pour autoriser la connexion sur le système serveur d'autant de personnes qu'il peut en recevoir.

    OU

  • Cliquez sur le bouton Autoriser les utilisateurs, puis spécifiez le nombre d'utilisateurs à autoriser.
  • Si vous autorisez la mise en cache des fichiers du dossier partagé, spécifiez dans la liste Configuration l'option de mise en cache à utiliser. Les options de mise en cache sont décrites dans la rubrique Windows (CIFS).

Autorisations

Vous pouvez aussi définir des autorisations pour des utilisateurs ou des groupes qui se voient autoriser ou interdire l'accès au système serveur.

Pour définir des autorisations utilisateurs

  1. Dans la zone Ajouter un utilisateur ou un groupe, entrez le nom d'un utilisateur ou d'un groupe à ajouter à la liste des autorisations, ou sélectionnez un utilisateur dans la liste placée au-dessous.
Vous pouvez sélectionner dans cette liste des utilisateurs ou des groupes locaux. Pour ajouter des utilisateurs de domaine ou des groupes de domaine, vous devez entrer le compte sous la forme <nom de domaine\nom d'utilisateur> ou <nom de domaine\nom de groupe>.
  1. Choisissez Ajouter.
  2. Dans la liste Autoriser, définissez le degré de contrôle dont disposeront les utilisateurs sélectionnés dans la liste Autorisations sur les fichiers du système serveur.
Ces utilisateurs peuvent n'avoir aucun contrôle, ou un accès en lecture seule, un accès en modification, un accès en lecture et modification ou un contrôle total.
  1. Utilisez la liste Refuser pour refuser un degré de contrôle aux utilisateurs et aux groupes sélectionnés dans la liste Autorisations.
  2. Pour supprimer un utilisateur ou un groupe dans la liste Autorisations, sélectionnez-le dans la liste puis cliquez sur Supprimer.
  3. Cliquez sur OK.
