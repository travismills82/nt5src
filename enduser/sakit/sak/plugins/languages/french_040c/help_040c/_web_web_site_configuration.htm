Configuration du site Web

Dans la page Configuration du site Web, vous pouvez exécuter l'une des tâches suivantes :

  • Création d'un site Web
  • Modification des paramètres du site Web
  • Suppression d'un site Web
  • Suspension d'un site Web
  • Arrêt d'un site Web
  • Démarrage d'un site Web

Cette page vous permet également de rechercher des sites commençant par une valeur déterminée. Vous pouvez par exemple rechercher tous les en-têtes d'hôtes commençant par « www », ou tous les sites qui ont pour État Arrêté.

Pour afficher tous les sites qui sont en état Arrêté

  1. Sélectionnez la valeur État dans la liste déroulante située près de l'invite Rechercher.
  2. Entrez Arrêté comme critère de recherche.
  3. Cliquez sur OK.
