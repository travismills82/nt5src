Modification des propriétés de groupe

La page Propriétés du groupe affiche deux onglets : Général et Membres. Utilisez l'onglet Général pour définir ou pour modifier le nom du groupe et sa description. Utilisez l'onglet Membres pour ajouter ou supprimer des utilisateurs et des groupes.

Pour définir ou modifier le nom d'un groupe ou sa description

  1. Dans la barre de navigation principale, choisissez Utilisateurs.
  2. Choisissez Groupes locaux.
  3. Sélectionnez le compte de groupe que vous souhaitez modifier.
  4. Dans la liste Tâches, choisissez Propriétés.
  5. Dans l'onglet Général, entrez un nom et une description pour le groupe voulu.
  6. Cliquez sur OK.

Pour définir ou modifier l'adhésion à un groupe

  1. Dans la barre de navigation principale, choisissez Utilisateurs.
  2. Choisissez Groupes locaux.
  3. Sélectionnez le compte de groupe que vous souhaitez modifier.
  4. Dans la liste Tâche, choisissez Propriétés, puis sélectionnez l'onglet Membres.
  5. Pour ajouter un nouveau membre :
    • Dans la zone Ajouter un utilisateur ou groupe, sélectionnez un groupe ou un utilisateur local dans la liste, puis cliquez sur le bouton Ajouter.

    OU

    • Entrez le domaine et le nom du groupe (<domaine\nom_groupe>) d'un groupe de domaine, ou un compte d'utilisateur de domaine (<domaine\nom_utilisateur>) puis cliquez sur Ajouter. Cependant, si vous ajoutez Domaine\Groupe, vous devez également entrer les autorisations qui permettront l'ajout depuis ce domaine.
  6. Pour supprimer un membre, sélectionnez un nom d'utilisateur dans la liste Membres, puis choisissez Supprimer.
  7. Cliquez sur OK.
