Modification des propriétés du dossier

Dans la page Propriétés du dossier, vous pouvez définir ou modifier le nom du dossier, obtenir des précisions sur le type du dossier, sa taille, son emplacement et compresser les données de ce dossier.

Pour modifier le nom d'un dossier

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Dans la page Partages, choisissez Dossiers.
  3. À l'aide du sélecteur d'objet/de tâche, rendez-vous dans le répertoire contenant le dossier à modifier.
  4. Activez la case à cocher correspondant au dossier souhaité.
  5. Dans la liste Tâches, choisissez Propriétés.
  6. Dans l'onglet Général, entrez le nouveau nom du dossier dans la zone Nom.
  7. Cliquez sur OK.

Pour compresser un dossier

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Dans la page Partages, choisissez Dossiers.
  3. À l'aide du sélecteur d'objet/de tâche, rendez-vous dans le répertoire dans lequel vous voulez ajouter le nouveau dossier.
  4. Dans la liste Tâches, choisissez Propriétés.
  5. Sélectionnez l'onglet Compresser, puis activez la case à cocher Compresser le contenu pour minimiser l'espace disque nécessaire.
  6. Choisissez si vous voulez N'appliquer les modifications qu'à ce dossier, ou Appliquer les modifications à ce dossier et à tous les sous-dossiers et fichiers.
  7. Cliquez sur OK.
