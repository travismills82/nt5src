Suppression d'un compte de groupe

Vous pouvez supprimer tout compte de groupe que vous avez créé. Cependant, un compte de groupe supprimé ne peut être recréé. Sachez que vous pouvez supprimer plusieurs comptes de groupe en une fois, en sélectionnant tous ces comptes de groupe avant de lancer la tâche Supprimer.

Pour supprimer un compte de groupe

  1. Dans la page de navigation principale, choisissez Utilisateurs.
  2. Choisissez Groupes locaux.
  3. Sélectionnez les comptes de groupe à supprimer.
  4. Dans la liste Tâches, choisissez Supprimer.
  5. Assurez-vous que le groupe identifié est bien celui à supprimer, puis cliquez sur OK.
