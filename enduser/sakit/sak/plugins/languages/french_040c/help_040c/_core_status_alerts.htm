Alertes d'état

Lorsque vous choisissez État : <type d'état> dans la zone d'état, une page Alerte s'ouvre.

La liste qui suit indique le type de contenu d'alerte que produit chaque type d'état :

  • État Normal—La page Alerte indique qu'il n'y a pas de messages.
  • État Information—La page Alerte donne la liste des points traités sous forme de liens hypertexte, avec une icône contenant la lettre i devant chaque point que le système a rencontré.
  • État Avertissement—La page Alerte donne la liste des erreurs sous forme de liens hypertexte, avec un triangle jaune entourant un point d'exclamation (!) devant chaque erreur rencontrée par le système.
  • État Critique—La page Alerte donne la liste des erreurs sous forme de liens hypertexte, avec un cercle rouge entourant un x blanc devant chaque erreur rencontrée par le système.

Pour annuler une alerte

  1. Dans la barre de navigation principale, choisissez État.
  2. Choisissez l'alerte à annuler.
  3. Choisissez l'action à entreprendre pour corriger les conditions d'alerte, ou choisissez Supprimer le message pour supprimer le message d'alerte sans modifier la configuration de votre système serveur.
