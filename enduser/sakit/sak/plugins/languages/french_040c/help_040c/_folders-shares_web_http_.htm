Web (HTTP)

HTTP est le protocole Internet utilisé par les navigateurs et les serveurs du World Wide Web pour échanger des informations. Ce protocole définit quelles actions les serveurs et les navigateurs Web doivent exécuter en réponse à diverses commandes, permettant ainsi à l'utilisateur de recourir à un programme client pour entrer une URL, choisir un lien hypertexte, rechercher du texte, des graphiques, des sons et d'autres informations numériques dans un serveur Web. Toutes les URL des fichiers sur serveurs Web commencent par http://.
