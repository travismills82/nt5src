Paramètres principaux du Web

Cette page vous permet de configurer les paramètres génériques de serveur Web qui s'appliqueront à tous les sites Web de votre système serveur.

Le chemin du répertoire par défaut est affiché dans la zone Répertoire racine du site Web. Vous pouvez modifier ce chemin de répertoire. Dans ce cas, vous modifiez le répertoire par défaut des sites Web suivants. Les sites Web créés avant cette modification demeurent dans leur répertoire d'origine.

Pour changer de répertoire racine par défaut, entrez le nouveau chemin dans la zone Répertoire racine du site Web.

Remarque   Les changements apportés au répertoire racine du site Web n'affectent pas le répertoire FTP.

Les options suivantes sont disponibles :

  • Délai d'exécution du script ASP (secondes)—Spécifie combien de temps ASP alloue à un script pour s'exécuter. Si l'exécution du script n'est pas terminée à l'expiration du délai, ASP arrête le script et enregistre un événement dans le journal d'événements Windows. La valeur du délai est comprise entre 1 et 2147483647. Vous pouvez ignorer cette option dans une page ASP à l'aide de la méthode Server.ScriptTimeout.
  • Indexer les sites Web—Permet d'interroger les fichiers en texte complet à l'aide du Service d'indexation Microsoft.

Remarque   Dans tous les cas, les cases à cocher doivent être activées pour activer la fonctionnalité.

Enfin, vous pouvez définir le nombre maximal de connexions à des sites Web hébergés dans votre système serveur.
