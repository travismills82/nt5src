Ajout d'un partage

Pour créer un partage, vous devez fournir un nom de partage qui soit unique et un chemin de partage. Certains protocoles acceptent également l'insertion d'un commentaire ou d'une brève description du partage. En outre, vous devez activer un au moins des protocoles disponibles.

Bien qu'il n'existe qu'une seule interface utilisateur pour créer un partage pour tous les protocoles, en réalité un partage différent est créé pour chaque protocole. Vous pouvez supprimer d'un partage des protocoles de partage donnés sans supprimer le partage lui-même. Néanmoins, si vous supprimez d'un partage tous les protocoles de partage, vous supprimez toutes les versions de ce partage.

Pour ajouter un partage

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Dans la page Partages, choisissez Partages.
  3. Dans la liste Tâches, choisissez Nouvelle.
  4. Dans l'onglet Général, entrez le nom du partage et le chemin d'accès au partage.
  5. Activez la ou les cases appropriées pour spécifier quels types de protocoles vous souhaitez activer.
  6. Utilisez les onglets des protocoles pour configurer les propriétés spécifiques de chaque type de partage.
