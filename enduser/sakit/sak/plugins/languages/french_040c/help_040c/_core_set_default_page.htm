Définition de la page par défaut

Vous pouvez définir la page par défaut de votre système serveur. Ce sera la page qui s'affichera chaque fois que vous accéderez au système serveur. Vous pouvez décider que la page par défaut soit la page de Bienvenue ou la page d'État.

Pour définir la page par défaut du système serveur

  1. Dans la page de navigation principale, choisissez Bienvenue.
  2. Dans la page Bienvenue, choisissez Définir la page par défaut.
  3. Dans Définir la page par défaut, choisissez quelle page sera la page par défaut.
  4. Cliquez sur OK.
