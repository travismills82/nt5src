Suppression d'un compte utilisateur

À l'exception du dernier compte non supprimé et de votre compte personnel, vous pouvez supprimer tous les comptes utilisateurs que vous avez créés sur le système serveur Microsoft Windows Powered. Vous pouvez même supprimer plusieurs comptes utilisateurs en une fois. Cependant, si vous supprimez le seul compte utilisateur du système serveur, la sécurité est désactivée.

En supprimant des comptes utilisateurs, gardez présentes à l'esprit les considérations suivantes :

  • Les utilisateurs prédéfinis ne peuvent pas être supprimés.
  • Il est impossible de récupérer un utilisateur supprimé.

    Si vous supprimez un compte utilisateur, puis que vous en créez un autre portant le même nom, vous devez définir de nouvelles autorisations pour le nouvel utilisateur, car il n'hérite pas de celles qui étaient attribuées à l'ancien.

Pour supprimer des comptes utilisateurs

  1. Dans la barre de navigation principale, choisissez Utilisateurs.
  2. Choisissez Utilisateurs locaux.
  3. Sélectionnez le ou les comptes utilisateurs à supprimer.
  4. Dans la liste Tâches, choisissez Supprimer.
  5. Assurez-vous que les comptes utilisateurs indiqués sont bien ceux que vous souhaitez supprimer, puis cliquez sur OK.
