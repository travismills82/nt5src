Suspension d'un site Web

Le sélecteur d'objet/de tâche de la page Configuration du site Web indique l'état de chaque site Web résidant sur votre système serveur. Si nécessaire, vous pouvez suspendre tout site Web en cours d'exécution.

En suspendant un site Web, vous empêchez les services Internet d'accepter de nouvelles connexions, mais vous n'affectez pas les requêtes en cours de traitement. En revanche, en arrêtant un site Web, vous empêchez les services Internet d'accepter de nouvelles connexions, et vous arrêtez toutes les requêtes en cours de traitement.

Pour suspendre un site Web en activité

  1. Dans la page de navigation principale, choisissez Sites.
  2. Sélectionnez l'identificateur du site Web que vous souhaitez suspendre.
  3. Dans la liste Tâches, choisissez Suspendre.
