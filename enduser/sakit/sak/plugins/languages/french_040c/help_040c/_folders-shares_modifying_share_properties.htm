Modification des propriétés du partage

Utilisez la page Partages pour afficher et modifier les propriétés d'un partage.

Pour modifier les propriétés d'un partage

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Dans la page Partages, choisissez Partages.
  3. Sélectionnez le partage dont vous voulez modifier les propriétés.
  4. Dans la liste Tâches, choisissez Propriétés.

    La page de propriétés Général s'affiche. Utilisez cette page pour changer le type de client dans lequel ce partage est accessible.

  5. Cliquez sur OK.
