Gestion des journaux NFS

Le journal NFS contient les événements enregistrés par le serveur NFS.

Pour gérer les journaux NFS

  1. Dans la barre de navigation principale, choisissez Maintenance.
  2. Choisissez Journaux.
  3. Choisissez Journaux NFS.
