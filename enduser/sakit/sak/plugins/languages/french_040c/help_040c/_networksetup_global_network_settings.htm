Paramètres globaux

Dans cette page, vous pouvez changer les paramètres globaux de réseau pour votre système serveur, en configurant les paramètres IPX et en spécifiant les suffixes DNS et le fichier LMHOSTS et HOSTS à utiliser. Ces fichiers permettent de résoudre les noms de tout ordinateur ou périphérique. Sachez que le suffixe DNS utilisé ici s'applique lorsque le système serveur essaie de résoudre un nom de domaine ou d'hôte.

Pour définir ou modifier automatiquement des suffixes DNS

  1. Dans la barre de navigation principale, choisissez Réseau.
  2. Choisissez Paramètres globaux.
  3. Cliquez sur l'onglet Résolution DNS.
  4. Cliquez sur le bouton Ajouter le suffixe DNS principal.
  5. Facultatif : vous pouvez décider d'Ajouter des suffixes parents du suffixe DNS principal en activant cette case à cocher.
  6. Cliquez sur OK.

Pour ajouter manuellement des suffixes DNS spécifiques

  1. Dans la page Réseau, choisissez Paramètres globaux.
  2. Cliquez sur l'onglet Résolution DNS.
  3. Cliquez sur le bouton Ajouter les suffixes DNS suivants, dans l'ordre d'utilisation.
  4. Dans la zone Suffixe du domaine, entrez le suffixe DNS que vous souhaitez ajouter, puis cliquez sur Ajouter.
  5. Cliquez sur OK.

Pour supprimer manuellement des suffixes DNS spécifiques

  1. Dans la page Réseau, choisissez Paramètres globaux.
  2. Cliquez sur l'onglet Résolution DNS.
  3. Cliquez sur le bouton Ajouter les suffixes DNS suivants, dans l'ordre d'utilisation.
  4. Utilisez les boutons Précédent et Suivant pour faire défiler la liste des suffixes de domaines. Cliquez sur le suffixe que vous souhaitez supprimer, puis sur Supprimer.
  5. Cliquez sur OK.
