Définition d'un mot de passe utilisateur

Pour définir le mot de passe d'un utilisateur

  1. Dans la page de navigation principale, choisissez Utilisateurs.
  2. Choisissez Utilisateurs locaux.
  3. Sélectionnez le compte utilisateur dont vous voulez modifier le mot de passe.
  4. Dans la liste Tâches, choisissez Définir un mot de passe.
  5. Entrez le nouveau mot de passe, puis confirmez-le dans les zones prévues.

    Le nouveau mot de passe doit respecter les éventuelles règles de formation de mot de passe en vigueur pour le domaine auquel appartient le système serveur.

  6. Cliquez sur OK.
