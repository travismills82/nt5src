Définition des autorisations d'exécution Web

Cette page vous permet de définir les autorisations par défaut pour les sites du système serveur, soit à scripts et exécutables, soit à scripts seulement soit à aucune.

Pour définir les autorisations d'exécution Web

  1. Dans la barre des tâches principale, choisissez Serveur Web.
  2. Choisissez Autorisations d'exécution Web.
  3. Sélectionnez l'autorisation d'exécution voulue dans la liste Autorisations d'exécution par défaut.
  4. Cliquez sur ce bouton pour appliquer ce paramètre à tous les sites Web qui utilisent la valeur par défaut.
  5. Cliquez sur OK.
