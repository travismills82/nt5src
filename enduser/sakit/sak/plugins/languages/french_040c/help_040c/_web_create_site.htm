Création d'un site Web

Dans la page Créer un site Web, vous pouvez créer un nouveau site Web et configurer ses propriétés.

Pour créer un site Web

  1. Dans la page de navigation principale, choisissez Sites.
  2. Dans la liste Tâches, choisissez Créer.
  3. Dans cette page, fournissez les informations nécessaires pour créer un site Web.
  4. Cliquez sur OK.
