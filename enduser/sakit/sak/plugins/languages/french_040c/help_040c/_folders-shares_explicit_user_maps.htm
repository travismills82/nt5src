Configuration de mappages d'utilisateur explicites

Le Mappage d'utilisateurs et de groupes vous permet de créer des mappages interplates-formes et multiplates-formes entre comptes d'utilisateurs et de groupes Microsoft Windows et UNIX, même si les noms d'utilisateurs et de groupes ne sont pas identiques dans les deux environnements.

Le Mappage d'utilisateurs et de groupes vous permet aussi de créer des mappages univoques interplates-formes et multiplates-formes entre utilisateurs et groupes Windows et UNIX. Par exemple, un nom d'utilisateur Windows pourrait être mappé sur un nom d'utilisateur UNIX, ou un groupe UNIX pourrait être mappé sur un ou plusieurs comptes utilisateurs Windows. Les mappages utilisateur explicites peuvent aussi s'utiliser lorsque la même personne a un nom d'utilisateur différent dans des comptes Windows et UNIX. En utilisant l'option Mappages d'utilisateur explicites, vous pouvez entretenir une seule base de données de mappage pour l'ensemble de l'entreprise.

Pour créer des mappages utilisateur explicites

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Protocoles de partage.
  3. Sélectionnez Protocole NFS, puis Propriétés.
  4. Choisissez Mappages de groupe et d'utilisateur.
  5. Complétez l'onglet Général, en veillant bien à indiquer soit un domaine NIS soit un mot de passe PCNFS et les fichiers de groupe. Pour plus d'informations sur la façon de remplir l'onglet Général, consultez Configuration des mappages de groupe et d'utilisateur.
  6. Sélectionnez l'onglet Mappages utilisateur explicites.
  7. Facultatif : entrez dans le champ Domaine NIS le nom du serveur NIS sur lequel effectuer le mappage.
  8. Choisissez le bouton Liste d'utilisateurs UNIX pour remplir la zone Utilisateurs UNIX.
  9. Dans chaque groupe, sélectionnez un utilisateur, puis cliquez sur Ajouter.

    Les utilisateurs mappés s'affichent dans la zone Utilisateurs mappés explicitement. Si vous souhaitez mapper un utilisateur DOMAINE, vous devez entrer le nom DOMAINE\utilisateur, sélectionner un utilisateur UNIX puis cliquer sur le bouton Ajouter à côté de la zone de saisie DOMAINE\utilisateur.

    Remarque   Vous pouvez mapper les utilisateurs depuis un domaine Windows vers plusieurs domaines UNIX, mais vous ne pouvez pas mapper un utilisateur UNIX vers plusieurs utilisateurs Windows. 

Pour définir l'un des mappages comme mappage principal pour un utilisateur donné

  1. Sélectionnez le mappage dans la liste Utilisateurs mappés explicitement.
  2. Choisissez Définir le groupe principal.
  3. Cliquez sur OK.

Pour supprimer des mappages utilisateur explicites

  1. Exécutez les étapes 1-4 de la procédure Pour créer des mappages utilisateur explicites afin de vous rendre sur la page Mappages utilisateur explicites.
  2. Dans la liste Utilisateurs mappés explicitement, sélectionnez le mappage utilisateur à supprimer.
  3. Cliquez sur Supprimer.
  4. Cliquez sur OK.
