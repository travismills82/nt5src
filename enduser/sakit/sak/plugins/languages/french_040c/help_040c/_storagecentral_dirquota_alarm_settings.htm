Directory Quota Alarm Settings

There are two types of alarms, severe and warning.  Each alarm contains the attributes which are differentiated by threshold values.  The threshold value equals the percentage of the space limit used.  

An alarm is only activated once until the used space drops below the threshold. For example, if the severe alarm threshold value is 90% and warning alarm threshold value is 70% and the space used is 71%, the warning alarm is activated.  The warning alarm will not be be activated again, unless the value first drops below 70%.

If the current space limit exceeds any of the thresholds, the corresponding quota actions are executed. Alarms that activate numerous times within a few seconds are discarded.
