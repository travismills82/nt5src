Ajout d'un compte d'utilisateur

Lorsque vous créez un compte d'utilisateur, vous devez fournir un nom d'utilisateur, le nom complet de l'utilisateur, un bref descriptif de ce compte et un mot de passe de compte.

N'oubliez pas que les noms d'utilisateurs doivent être uniques, et ne doivent pas reprendre le nom de quelque groupe existant que ce soit.

Un nom d'utilisateur doit être différent de celui de tout autre utilisateur ou groupe sur l'ordinateur en cours de gestion. Il peut contenir jusqu'à 20 caractères quelconques, en majuscules ou en minuscules, sauf : " / \ [ ] : ; | = , + * ? < >. De plus, un nom d'utilisateur ne peut pas être entièrement composé de points (.) ou d'espaces.

Dans les zones Mot de passe et Confirmer le mot de passe, vous pouvez taper un mot de passe (127 caractères au maximum). Cependant, si vous utilisez Windows 2000 sur un réseau qui comprend également des ordinateurs exécutant Windows 95 ou Windows 98, utilisez de préférence des mots de passe d'une longueur inférieure à 14 caractères. Windows 95 et Windows 98 acceptent les mots de passe allant jusqu'à 14 caractères. Si votre mot de passe est plus long, vous ne pourrez peut-être pas ouvrir de session sur le réseau à partir de ces ordinateurs.

Les seuls nouveaux utilisateurs que vous devriez ajouter au groupe administrateurs sont ceux qui n'exécuteront que des tâches administratives.

Pour ajouter un compte d'utilisateur

  1. Dans la page de navigation principale, choisissez Utilisateurs.
  2. Dans la page Utilisateurs, choisissez Utilisateurs locaux.
  3. Dans la liste Tâches, choisissez Nouvelle.
  4. Entrez les informations du nouveau compte d'utilisateur.

    Le champ Répertoire de base spécifie un nouveau répertoire qui sera créé, et auquel l'utilisateur aura une autorisation d'accès exclusif. Le nom de ce répertoire est identique au nom d'utilisateur défini précédemment, et sera situé dans le chemin spécifié.

  5. Cliquez sur OK.
