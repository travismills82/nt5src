Modification des propriétés de quota

Pour modifier les propriétés d'une entrée de quota

  1. Dans la barre de navigation principale, choisissez Disques.
  2. Choisissez Quota de disque.
  3. Sélectionnez le volume que vous souhaitez gérer.
  4. Dans la liste Tâches, choisissez Entrées de quota.
  5. Dans la page Entrées de quota du volume concerné, sélectionnez un compte d'utilisateur.
  6. Dans la liste Tâches, choisissez Propriétés.
  7. Dans la page Entrée de quota pour l'utilisateur, exécutez l'une de ces tâches :

    Pour autoriser une utilisation illimitée du disque

    • Sélectionnez le bouton Ne pas limiter l'espace disque.

    —OU—

    Pour limiter l'espace disque

    1. Sélectionnez le bouton Limiter l'espace disque à.
    2. Dans le champ, entrez une valeur numérique pour spécifier la quantité d'espace disque à allouer à un utilisateur ou un groupe donné. Utilisez la liste pour préciser Ko, Mo, Go, To, Po ou Eo.
    3. Entrez à quel volume de remplissage de l'espace disque devra se déclencher un message avertissant l'utilisateur ou le membre du groupe qu'il atteint bientôt la limite de capacité autorisée. Utilisez la liste pour préciser Ko, Mo, Go, To, Po ou Eo.
    4. Cliquez sur OK.

  5. Dans la page Entrées de quota du volume concerné, sélectionnez plusieurs comptes d'utilisateurs.
  6. Dans la liste Tâches, choisissez Propriétés.
  7. Dans la page Entrée de quota pour l'utilisateur, exécutez l'une de ces tâches :
