Configuration de mappages de groupe explicites

Le Mappage d'utilisateurs et de groupes vous permet de créer des mappages interplates-formes et multiplates-formes entre comptes d'utilisateurs et de groupes Microsoft Windows et UNIX, même si les noms d'utilisateurs et de groupes ne sont pas identiques dans les deux environnements.

Le Mappage d'utilisateurs et de groupes vous permet aussi de définir des mappages univoques entre utilisateurs Windows et utilisateurs et groupes UNIX. Par exemple, un nom d'utilisateur Windows pourrait être mappé sur un nom d'utilisateur UNIX, ou un groupe UNIX pourrait être mappé sur un ou plusieurs comptes utilisateurs Windows. Les mappages explicites peuvent aussi s'utiliser lorsque la même personne a un nom d'utilisateur différent dans des comptes Windows et UNIX. En utilisant l'option Mappages de groupe explicites, vous pouvez entretenir une seule base de données de mappage pour l'ensemble de l'entreprise.

Pour créer des mappages de groupe explicites

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Protocoles de partage.
  3. Sélectionnez Protocole NFS, puis Propriétés.
  4. Choisissez Mappages de groupe et d'utilisateur.
  5. Complétez l'onglet Général, en veillant bien à indiquer soit un domaine NIS soit un mot de passe PCNFS et les fichiers de groupe. Pour plus d'informations sur la façon de remplir l'onglet Général, consultez Configuration des mappages de groupe et d'utilisateur.
  6. Sélectionnez l'onglet Mappages de groupe explicites.
  7. Facultatif : entrez dans le champ Domaine NIS le nom du serveur NIS sur lequel effectuer le mappage.
  8. Pour alimenter la liste Groupes UNIX, cliquez sur le bouton Liste des groupes UNIX.
  9. Dans chaque groupe, sélectionnez un groupe, puis cliquez sur Ajouter.

    Les groupes mappés s'affichent dans la zone Groupes mappés explicitement. Si vous souhaitez mapper un groupe DOMAINE, vous devez entrer le nom DOMAINE\groupe, sélectionner un groupe UNIX, puis cliquer sur le bouton Ajouter à côté de la zone de saisie DOMAINE\groupe.

    Remarque   Vous pouvez mapper des groupes depuis un domaine Windows vers plusieurs domaines UNIX, mais pas l'inverse.

Pour définir l'un des mappages comme mappage principal pour un groupe donné

  1. Sélectionnez le mappage dans la liste Groupes mappés explicitement.
  2. Choisissez Définir le groupe principale.
  3. Cliquez sur OK.

Pour supprimer des mappages de groupe explicites

  1. Exécutez les étapes 1-4 de la procédure Pour créer des mappages de groupe explicites afin de vous rendre sur la page Mappages de groupe explicites.
  2. Dans la liste Groupes mappés explicitement, sélectionnez le mappage de groupe à supprimer.
  3. Cliquez sur Supprimer.
  4. Cliquez sur OK.
