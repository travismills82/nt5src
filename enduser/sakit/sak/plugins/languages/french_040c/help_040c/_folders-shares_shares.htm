Partages

Un dossier de votre système serveur peut être partagé avec d'autres ordinateurs du réseau, qu'ils soient équipés d'un système d'exploitation Microsoft Windows ou UNIX.

Si les protocoles spécifiques de partage de fichiers à votre disposition peuvent différer selon le constructeur de votre système serveur, un système serveur peut en principe prendre en charge les protocoles suivants :

  • Windows (CIFS)—Le protocole CIFS (Common Internet File System) est utilisé par des clients équipés d'un système d'exploitation Windows.
  • NFS—Le protocole Network File System est utilisé par des clients équipés de UNIX.
  • FTP—Le protocole File Transfer Protocol est un autre moyen d'accéder à un partage de fichiers depuis n'importe quel système d'exploitation.
  • HTTP—Le protocole Hypertext Transfer Protocol permet d'accéder à un partage de fichiers à partir de navigateurs Web.
  • NetWare—Ce protocole permet d'accéder à un partage de fichiers à partir de clients équipés de NetWare.
  • AppleTalk—Protocole d'accès à un partage de fichiers depuis des clients Macintosh.

Lorsque vous créez un partage sur le serveur système, vous pouvez activer n'importe lequel ou tous les protocoles disponibles ou installés sur votre système serveur.
