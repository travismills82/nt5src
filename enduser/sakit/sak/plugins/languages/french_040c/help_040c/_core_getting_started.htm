Mise en route

Votre système serveur exécuté sous Microsoft® Windows® est un système serveur directement associé au réseau informatique. Un système serveur se gère et se pilote dans une interface utilisateur Web et peut être géré à distance à partir d'un ordinateur client attaché au réseau. C'est un matériel potentiellement autonome sans moniteur ni périphériques d'entrée (clavier ou souris par exemple) qui lui soient propres. Parce que les systèmes serveurs sont basés sur le même code que Microsoft® Windows® 2000, de nombreuses méthodes de gestion à distance existant sur la plate-forme Windows 2000 peuvent également être utilisées pour gérer les systèmes serveurs sous Windows.
