Modifying a File Screening Object

 To modify file screening properties

  1. On the primary navigation bar, choose Disks.
  2. Choose  File Screening.
  3. Select the directory you want to modify.
  4. In the Tasks list, choose Properties.
  5. To add a policy, select it from the Available policies list, and then choose the button with arrows pointing to the left.
  6. To remove a policy select it from the Selected policies list, and then choose the button with arrows pointing to the right.
  7. Choose OK.
