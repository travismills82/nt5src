Désactivation de protocoles

Pour désactiver des protocoles de partage

  1. Dans la barre de navigation principale, choisissez Partages.
  2. Choisissez Protocoles de partage.
  3. Sélectionnez le protocole que vous souhaitez désactiver.
  4. Dans la liste Tâches, choisissez Désactiver.
