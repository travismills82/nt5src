Connexion FTP anonyme

Les utilisateurs dotés d'une connexion anonyme au serveur FTP peuvent se connecter avec le nom d'utilisateur anonyme—ou ftp, qui est synonyme d'anonyme. Aucun mot de passe n'est nécessaire, mais l'utilisateur est invité à donner une adresse électronique comme mot de passe. Par défaut, les connexions anonymes sont autorisées.

Remarque   Vous ne pouvez accéder au serveur FTP à partir d'un compte utilisateur Microsoft Windows Powered nommé anonyme. Le nom d'utilisateur anonyme est réservé, dans le serveur FTP, à la fonction d'ouverture de session anonyme. Les utilisateurs qui ouvrent une session sur le serveur avec le nom d'utilisateur anonyme reçoivent des autorisations selon la configuration du serveur FTP pour les ouvertures de session anonymes.

Une fois le logiciel de protocole FTP installé sur votre ordinateur, vous devez configurer ce logiciel pour pouvoir fonctionner. Le paramétrage du protocole de serveur FTP produira l'une des configurations suivantes :

  • Aucune connexion anonyme FTP autorisée sur le serveur. Dans ce cas, chaque utilisateur doit fournir un nom d'utilisateur et un mot de passe Windows valides. Pour configurer ainsi le protocole de serveur FTP, désactivez la case à cocher Activer les connexions anonymes dans la page Propriétés du protocole FTP.
  • Connexion d'utilisateurs anonymes et d'utilisateurs Windows autorisées sur le serveur FTP. Cette configuration permet d'utiliser soit une connexion anonyme soit un nom d'utilisateur et un mot de passe Windows. Pour configurer ainsi le protocole de serveur FTP, n'activez que la case à cocher Activer les connexions anonymes dans la page Propriétés du protocole FTP.
  • Connexions FTP anonymes seules autorisées sur le serveur. Dans ce cas, un utilisateur ne peut pas se connecter sur le serveur FTP avec un nom d'utilisateur et un mot de passe Windows. Pour configurer ainsi le protocole de serveur FTP, activez à la fois la case à cocher Activer les connexions anonymes et la case à cocher Autoriser uniquement les connexions anonymes dans la page Propriétés du protocole FTP.

Si vous autorisez les connexions anonymes, vous devez indiquer le nom d'utilisateur et le mot de passe Windows qui fournissent l'accès anonyme au serveur FTP. Lors d'un transfert FTP anonyme, Windows vérifie le nom d'utilisateur indiqué dans cette boîte de dialogue afin de déterminer si l'accès aux fichiers est autorisé.
