Vorheriges Thema

Öffnen eines Ordners

So öffnen Sie einen Ordner

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Ordner aus.
  3. Wählen Sie das Volume aus, mit dem Sie arbeiten möchten, und wählen Sie dann Öffnen aus.
  4. Wählen Sie den zu öffnenden Ordner aus.
  5. Wählen Sie in der Liste Tasks den Eintrag Öffnen aus.
