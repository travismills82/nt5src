Vorheriges Thema

Administrator

Die Serverappliance umfasst eine Reihe von Standardkonten. Ausschließlich dem Administratorkonto sind Administratorrechte zugewiesen.

Anmerkung   Fügt ein Administrator der lokalen Administratorengruppe ein Domänenkonto hinzu, kann der Domänenbenutzer auf die Serverappliance zugreifen und sie verwalten. Der Administrator ist jedoch nicht in der Lage, sein Kennwort für das Domänenkonto unter Verwendung der Seite Administratorkennwort ändern zu ändern. Diese Seite dient lediglich zum Ändern des Kontokennworts des lokalen Administrators.

Wenn Sie das Administratorkennwort ändern, wird keine explizite Bestätigungsseite angezeigt. Das Kennwort wurde jedoch erfolgreich geändert, wenn nach dem Übergeben der Änderung keine Fehlermeldung angezeigt wird.

Anmerkung   Sie können das Administratorkennwort nicht ändern, wenn Sie als Domänenbenutzer angemeldet sind. Funktionen zum Durchführen von Änderungen an Domänenbenutzerkonten, die statt auf der Serverappliance auf dem Domänencontroller gespeichert sind, sind in der Webbenutzeroberfläche der Serverappliance nicht enthalten.

In diesem Zusammenhang bezieht sich Administrator auf das Benutzerkonto, das ein Mitglied der lokalen Administratorengruppe ist und von einem aktuellen Benutzer zur Anmeldung verwendet wird, und nicht auf das Standardadministratorkonto Administrator.

Wenn beim Ändern des Administratorkennworts oder des Kontonamens die Fehlermeldung Das Kennwort kann für das Domänenkonto nicht geändert werden oder Der Kontoname kann für dieses Domänenkonto nicht geändert werden angezeigt wird, sind Sie als Domänenbenutzer angemeldet. Sie müssen als Administrator der Serverappliance angemeldet sein, um das Administratorkennwort ändern zu können.

So ändern Sie das Kennwort des Administratorkontos für die Serverappliance

  1. Melden Sie sich bei der Serverappliance als Administrator an.
  2. Wählen Sie auf der primären Navigationsleiste Netzwerk aus.
  3. Wählen Sie Administrator aus.
  4. Geben Sie das aktuelle Administratorkennwort im Feld Aktuelles Kennwort ein.
  5. Geben Sie das neue Administratorkennwort im Feld Neues Kennwort ein.

    Anmerkung   Das neue Administratorkennwort muss mit den geltenden Richtlinien für die Komplexität von Kennwörtern der Domäne übereinstimmen, zu der die Serverappliance gehört.

  6. Geben Sie das neue Administratorkennwort im Feld Neues Kennwort bestätigen ein.
  7. Klicken Sie auf OK.

So ändern Sie den Namen des Administratorkontos für die Serverappliance

  1. Melden Sie sich bei der Serverappliance als Administrator an.
  2. Klicken Sie auf Netzwerk.
  3. Wählen Sie Administrator aus.
  4. Geben Sie den neuen Namen für das Administratorkonto im Feld Benutzername ein.
  5. Klicken Sie auf OK.
