Aktivieren von Protokollen

Gemäß den Empfehlungen von Microsoft sollten lediglich die notwendigen Netzwerkprotokolle aktiviert werden. Durch Eingrenzen der Anzahl aktivierter Netzwerkprotokolle wird die Leistungsfähigkeit anderer Netzwerkprotokolle verbessert. Wenn ferner ein Problem mit einer Netzwerk- oder DFÜ-Verbindung auftritt, wird über jedes installierte und aktivierte Netzwerkprotokoll versucht, die Verbindung wiederherzustellen. Indem Sie lediglich die Protokolle aktivieren, die das System verwenden kann, wird eine bessere Leistung der Serverappliance sowie das Einsparen von Ressourcen ermöglicht.

So aktivieren Sie eine Protokollfreigabe

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Protokolle freigeben aus.
  3. Markieren Sie das zu aktivierende Protokoll.
  4. Wählen Sie in der Liste Tasks den Eintrag Aktivieren aus.
