Entfernen eines Gruppenkontos

Sie können jedes von Ihnen erstellte Gruppenkonto entfernen. Ein Gruppenkonto, das entfernt wurde, kann jedoch nicht wiederhergestellt werden. Beachten Sie, dass Sie mehrere Gruppenkonten gleichzeitig entfernen können, indem Sie alle betreffenden Gruppenkonten vor dem Starten des Tasks Löschen auswählen.

So entfernen Sie ein Gruppenkonto

  1. Wählen Sie auf der primären Navigationsleiste Benutzer aus.
  2. Wählen Sie dann Lokale Gruppen aus.
  3. Wählen Sie die zu entfernenden Gruppenkonten aus.
  4. Wählen Sie in der Liste Tasks den Eintrag Löschen aus.
  5. Stellen Sie sicher, dass es sich bei der angegebenen Gruppe um das zu löschende Gruppenkonto handelt, und klicken Sie dann auf OK.
