Webmastereinstellungen

Auf dieser Seite können Sie allgemeine Webservereinstellungen konfigurieren, die für alle Websites auf der Serverappliance gelten.

Der Standardverzeichnispfad wird im Feld Website-Stammverzeichnis angezeigt. Sie können diesen Verzeichnispfad ändern. Dadurch wird das Standardverzeichnis für nachfolgende Websites geändert. Websites, die vor der Änderung erstellt wurden, bleiben in ihren ursprünglichen Verzeichnissen.

Um das Standardstammverzeichnis zu ändern, geben Sie den neuen Pfad im Website-Stammverzeichnis ein.

Anmerkung   Änderungen am Website-Stammverzeichnis wirken sich nicht auf das FTP-Verzeichnis aus.

Die folgenden Optionen stehen ebenfalls zur Verfügung:

  • ASP-Skriptzeitlimit (in Sekunden) Gibt die von ASP gewährte Zeit für die Ausführung eines Skripts an. Wenn das Zeitlimit bei der Skriptausführung nicht eingehalten wird, stoppt ASP das Skript und schreibt ein Ereignis in das Windows-Ereignisprotokoll. Für das Zeitlimit kann ein Wert zwischen 1 und 2147483647 festgelegt werden. Sie können diese Option auf einer ASP-Seite durch Anwenden der Methode Server.ScriptTimeout außer Kraft setzen.
  • Indexwebsites Ermöglicht dem Indexdienst von Microsoft eine Volltextsuche innerhalb der Dateien.

Anmerkung   In jedem Fall müssen die Kontrollkästchen aktiviert sein, um die jeweilige Funktionalität zu aktivieren.

Abschließend können Sie noch die maximale Anzahl von Verbindungen mit den Websites festlegen, die auf der Serverappliance gehostet werden.
