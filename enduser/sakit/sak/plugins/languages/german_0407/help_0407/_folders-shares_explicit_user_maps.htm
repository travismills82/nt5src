Vorheriges Thema

Konfigurieren expliziter Benutzerzuordnungen

Mit Benutzer- und Gruppenzuordnungen können Sie plattforminterne und -übergreifende Zuordnungen zwischen Benutzer- und Gruppenkonten unter Microsoft Windows und UNIX erstellen, auch wenn die Benutzer- und Gruppennamen in beiden Umgebungen nicht übereinstimmen.

Benutzer- und Gruppenzuordnungen ermöglicht ebenfalls das Einrichten plattforminterner und -übergreifender Zuordnungen im Verhältnis 1:1 zwischen Benutzern und Gruppen aus Windows und UNIX. Ein Windows-Benutzername könnte beispielsweise einem UNIX-Benutzernamen bzw. eine UNIX-Gruppe könnte mindestens einem Windows-Benutzerkonto zugeordnet werden. Die Verwendung expliziter Zuordnungen ist ebenfalls möglich, wenn ein Windows- und ein UNIX-Konto derselben Person verschiedene Benutzernamen aufweisen. Mit Hilfe der Option Explizite Benutzerzuordnungen können Sie eine einzelne Zuordnungsdatenbank für das gesamte Unternehmen verwalten.

So erstellen Sie explizite Benutzerzuordnungen

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Protokolle freigeben aus.
  3. Wählen Sie NFS-Protokoll und dann Eigenschaften aus.
  4. Wählen Sie Benutzer- und Gruppenzuordnungen aus.
  5. Nehmen Sie auf der Registerkarte Allgemein die erforderlichen Angaben vor. Stellen Sie sicher, dass Sie entweder eine NIS-Domäne oder eine PCNFS-passwd und Gruppendatei angeben. Weitere Informationen zu den Angaben auf dieser Registerkarte erhalten Sie unter Konfigurieren von Benutzer- und Gruppenzuordnungen.
  6. Wählen Sie die Registerkarte Explizite Benutzerzuordnungen aus.
  7. Optional: Geben Sie den Namen des zuzuordnenden NIS-Servers im Feld NIS-Domäne ein.
  8. Wählen Sie UNIX-Benutzer auflisten aus, um Daten in das Feld UNIX-Benutzer einzugeben.
  9. Wählen Sie aus jeder Gruppe einen Benutzer aus, und klicken Sie dann auf Hinzufügen.

    Die zugeordneten Benutzer werden im Feld Explizite zugeordnete Benutzer angezeigt. Zum Zuordnen eines Domänenbenutzers müssen Sie DOMÄNE\Benutzername eingeben, einen UNIX-Benutzer auswählen und auf die Schaltfläche Hinzufügen neben dem Eingabefeld für DOMÄNE\Benutzer klicken.

    Anmerkung   Sie können Benutzer aus einer Windows-Domäne zu mehreren UNIX-Domänen zuordnen, die Zuordnung eines UNIX-Benutzers zu mehreren Windows-Benutzern ist jedoch nicht möglich. 

So legen Sie eine Zuordnung als primäre Zuordnung für einen bestimmten Benutzer fest

  1. Wählen Sie die Zuordnung in der Liste Explizite zugeordnete Benutzer aus.
  2. Wählen Sie Primär festlegen aus.
  3. Klicken Sie auf OK.

So löschen Sie explizite Benutzerzuordnungen

  1. Führen Sie Schritte 1 bis 4 der Vorgehensweise So erstellen Sie explizite Benutzerzuordnungen aus, um zur Seite Explizite Benutzerzuordnungen zu navigieren.
  2. Wählen Sie in der Liste Explizite zugeordnete Benutzer die zu löschende Benutzerzuordnung aus.
  3. Klicken Sie auf Entfernen.
  4. Klicken Sie auf OK.
