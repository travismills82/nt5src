Vorheriges Thema

Konfigurieren von Datenträger- und Volumeeigenschaften

Über diese Seite können Sie die Datenträger oder Volumes auf der Serverappliance konfigurieren.

So konfigurieren Sie Datenträger und Volumes auf der Serverappliance

  1. Wählen Sie auf der primären Navigationsleiste Datenträger aus.
  2. Wählen Sie Datenträger und Volumes aus.
  3. Melden Sie sich bei Terminaldienste Advanced Client an.
  4. Sobald Sie fertig sind, schließen Sie die Anwendung.

    Anmerkung   Die Abmeldung der Terminaldienste Advanced Client-Sitzung beim Schließen der Anwendung nimmt einige Zeit in Anspruch.

Terminaldienste Advanced Client

Bei Terminaldienste Advanced Client handelt es sich um ein Tool für allgemeine Tasks, das Ihnen den vollständigen Zugriff auf die Serverappliance gewährt. Beim Zugriff über die Seite Datenträger und Volumes wird Terminaldienste Advanced Client im dedizierten Modus ausgeführt und kann lediglich zur Verwaltung der Datenträger und Volumes der Serverappliance verwendet werden.

Terminaldienste Advanced Client unterstützt lediglich zwei gleichzeitige Verbindungen. Wenn Sie während einer bestehenden Sitzung zu einer anderen Seite navigieren, wird die Verbindung zum Client getrennt, die Sitzung jedoch beibehalten.
