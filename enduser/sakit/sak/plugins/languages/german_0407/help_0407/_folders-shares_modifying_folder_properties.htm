Ändern von Ordnereigenschaften

Auf der Seite Ordnereigenschaften können Sie den Ordnernamen angeben oder ändern, Informationen zu Typ, Größe und Speicherort des Ordners anzeigen sowie die Daten eines Ordners komprimieren.

So ändern Sie den Namen eines Ordners

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Ordner auf der Seite Freigaben aus.
  3. Navigieren Sie mit der Objekt-/Taskauswahl zu dem Verzeichnis, in dem sich der zu bearbeitende Ordner befindet.
  4. Aktivieren Sie das Kontrollkästchen für den gewünschten Ordner.
  5. Wählen Sie in der Liste Tasks den Eintrag Eigenschaften aus.
  6. Wählen Sie die Registerkarte Allgemein aus, und geben Sie im Feld Name den Namen des neuen Ordners ein.
  7. Klicken Sie auf OK.

So komprimieren Sie einen Ordner

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Ordner auf der Seite Freigaben aus.
  3. Navigieren Sie mit der Objekt-/Taskauswahl zu dem Verzeichnis, dem Sie den neuen Ordner hinzufügen möchten.
  4. Wählen Sie in der Liste Tasks den Eintrag Eigenschaften aus.
  5. Wählen Sie die Registerkarte Komprimieren aus, und aktivieren Sie das Kontrollkästchen Inhalte dieses Ordners komprimieren, um Speicherplatz zu sparen.
  6. Wählen Sie entweder Änderungen nur für diesen Ordner übernehmen oder Änderungen für diesen Ordner, die untergeordneten Ordner und Dateien übernehmen aus.
  7. Klicken Sie auf OK.
