Telnet

Das Telnet-Feature zur Verwaltungskonfiguration ermöglicht die Verwaltung der Windows® Powered-Server Appliance über ein Remotesystem mit Hilfe des Telnet-Protokolls. Sie können sich von einem Telnet-Remoteclient aus am System anmelden und zeichenorientierte Anwendungen auf der Serverappliance ausführen. Der in der Serverappliance enthaltene Telnet-Server unterstützt maximal zwei Telnet-Clients gleichzeitig, wenn vom Hersteller der Hardware für die Serverappliance nichts anderes festgelegt wurde.

So konfigurieren Sie die Serverappliance für die Verwaltung von Telnet

  1. Wählen Sie auf der primären Navigationsleiste Netzwerk aus.
  2. Wählen Sie Telnet aus.
  3. Aktivieren Sie das Kontrollkästchen Aktivieren Sie den Zugriff über Telnet zu dieser Appliance.
  4. Klicken Sie auf OK.
