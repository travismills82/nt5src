Vorheriges Thema

Aktivieren der FTP-Protokollierung

Sie können eingehende FTP-Verbindungen durch Aktivieren der FTP-Protokollierung im FTP-Protokoll erfassen. Standardmäßig werden FTP-Protokolle im Pfad %Windir%\System32\Logfiles\Msftpsvc1 gespeichert.

Dem Administrator ist der Zugriff auf diese Dateien über seine Arbeitsstation auf zwei Arten möglich: Er greift auf eine administrative Freigabe, wie z. B. \\Appliancename\Admin$\Winnt\System32, zu, oder er erstellt eine neue Freigabe für den Ordner.

So aktivieren Sie die FTP-Protokollierung

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Protokolle freigeben aus.
  3. Wählen Sie FTP-Protokoll und dann Eigenschaften aus.
  4. Wählen Sie die Registerkarte Protokollierung aus.
  5. Aktivieren Sie das Kontrollkästchen Protokollierung aktivieren, und klicken Sie auf OK.
