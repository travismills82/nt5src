Verwenden der Hilfe

Im Folgenden werden einige Vorgehensweisen zum Suchen nach Informationen in der Hilfe aufgeführt:

So durchsuchen Sie die Themen nach Kategorie

  • Um die Themen zu durchsuchen, klicken Sie in der Liste Inhaltsverzeichnis auf den Thementitel.

So rufen Sie die kontextbezogene Hilfe auf

  • Klicken Sie auf der Seite, für die Sie Hilfe benötigen, auf das Symbol ? ganz rechts auf der primären Navigationsleiste.

    Anmerkung   Sie können zum vorherigen Hilfethema wechseln, indem Sie auf den Link Vorheriges Thema in der oberen rechten Ecke der einzelnen Hilfeseiten klicken.

Ferner kann die spezifische Onlinehilfe der Serverappliance unter Verwendung von Terminaldienste auf der Seite Wartung ebenfalls auf die Hilfe von Microsoft Windows 2000 zugreifen.

So öffnen Sie die Windows 2000-Hilfe

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie Terminaldienste aus.
  3. Melden Sie sich mit Ihrem Benutzernamen und Kennwort an.
  4. Klicken Sie auf die Schaltfläche Start, und wählen Sie Hilfe aus.
