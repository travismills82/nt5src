Vorheriges Thema

Verwalten von Sicherheitsprotokollen

Das Sicherheitsprotokoll enthält Sicherheitsereignisse, wie gültige und ungültige Anmeldeversuche, sowie Ereignisse zur Verwendung von Ressourcen, beispielsweise das Erstellen, Öffnen oder Löschen von Dateien. Die im Sicherheitsprotokoll aufzuzeichnenden Ereignisse werden von einem Administrator festgelegt.

So verwalten Sie Sicherheitsprotokolle

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie Protokolle aus.
  3. Wählen Sie Sicherheitsprotokoll aus.
