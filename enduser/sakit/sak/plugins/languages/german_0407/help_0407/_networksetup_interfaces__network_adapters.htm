Schnittstellen: Netzwerkadapter

Ein Netzwerkadapter stellt die physische Schnittstelle bzw. Verbindung und die Hardware zur Verfügung, um einem Computer den Zugriff auf ein Netzwerk zu ermöglichen. Ein Netzwerkadapter wird auch als Adapterkarte, Netzwerkkarte (NIC) oder einfach als Karte bezeichnet.

Auf der Seite Schnittstellen in der Webbenutzeroberfläche der Serverappliance können Sie folgende Tasks durchführen:

  • Den Verbindungsnamen ändern.
  • Die IP-Adressen (Internet Protocol), Gateway-Adressen, Subnetzmasken sowie die Metrik festlegen oder ändern.
  • Die Auflösung von DNS-Namen durch die Serverappliance einrichten oder ändern.
  • Die Konfiguration von WINS-Clients (Windows Internet Naming Service) festlegen oder ändern.
  • AppleTalk konfigurieren, wenn AppleTalk vom Hersteller der Serverappliance aktiviert wurde.

So konfigurieren Sie einen Netzwerkadapter

  1. Wählen Sie den gewünschten Adapter in der Spalte Beschreibung aus.
  2. Wählen Sie den gewünschten Task in der Liste Tasks aus.
  3. Verwenden Sie die Registerkarten im linken Ausschnitt der Seite Tasks, um die Konfiguration des ausgewählten Netzwerkadapters anzuzeigen.
