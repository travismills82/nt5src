Globale Einstellungen

Auf dieser Seite können Sie die gesamten Netzwerkeinstellungen für die Serverappliance ändern, indem Sie die IPX-Einstellungen konfigurieren sowie die DNS-Suffixe, die LMHOSTS- und die HOSTS-Datei angeben, die zu verwenden sind. Diese Dateien ermöglichen die Auflösung eines beliebigen Computer- oder Gerätenamens. Beachten Sie, dass das hier verwendete DNS-Suffix bei Auflösung eines Host- oder Domänennamens durch die Serverappliance verwendet wird.

So richten Sie DNS-Suffixe automatisch ein oder ändern sie

  1. Wählen Sie auf der primären Navigationsleiste Netzwerk aus.
  2. Wählen Sie Globale Einstellungen aus.
  3. Wählen Sie die Registerkarte DNS-Auflösung aus.
  4. Wählen Sie Übergeordnetes DNS-Suffix anhängen aus.
  5. Optional: Aktivieren Sie das Kontrollkästchen Übergeordnete Suffixe der primären DNS-Suffixe anhängen.
  6. Klicken Sie auf OK.

So fügen Sie bestimmte DNS-Suffixe manuell hinzu

  1. Wählen Sie Globale Einstellungen auf der Seite Netzwerk aus.
  2. Wählen Sie die Registerkarte DNS-Auflösung aus.
  3. Wählen Sie Folgende DNS-Suffixe in Verwendungsreihenfolge anhängen aus.
  4. Geben Sie im Feld Domänensuffix das hinzuzufügende DNS-Suffix ein, und klicken Sie dann auf Hinzufügen.
  5. Klicken Sie auf OK.

So entfernen Sie bestimmte DNS-Suffixe manuell

  1. Wählen Sie Globale Einstellungen auf der Seite Netzwerk aus.
  2. Wählen Sie die Registerkarte DNS-Auflösung aus.
  3. Wählen Sie Folgende DNS-Suffixe in Verwendungsreihenfolge anhängen aus.
  4. Führen Sie unter Verwendung der Schaltflächen Nach unten und Nach oben einen Bildlauf in der Liste der Domänensuffixe durch. Wählen Sie das zu löschende Suffix aus, und klicken Sie dann auf Entfernen.
  5. Klicken Sie auf OK.
