Vorheriges Thema

Anonymer FTP-Zugriff

Der anonyme Zugriff auf den FTP-Server ermöglicht Benutzern die Anmeldung mit dem Benutzernamen anonymous oder dem Synonym ftp. Ein Kennwort ist nicht notwendig, der Benutzer wird jedoch aufgefordert, eine E-Mail-Adresse als Kennwort anzugeben. Standardmäßig sind anonyme Verbindungen zulässig.

Anmerkung   Der Zugriff auf den FTP-Server über ein Benutzerkonto Powered by Microsoft Windows unter dem Namen anonymous ist nicht möglich. Dieser Benutzername ist auf dem FTP-Server für die Funktion zur anonymen Anmeldung reserviert. Benutzer, die sich mit dem Benutzernamen anonymous beim Server anmelden, erhalten Berechtigungen auf Grundlage der FTP-Serverkonfiguration für anonyme Anmeldungen.

Nach Installation der Software für das FTP-Protokolls auf Ihrem Computer muss diese ordnungsgemäß konfiguriert werden. Aus den Einstellungen des FTP-Serverprotokolls ergibt sich eine der folgenden Konfigurationen:

  • Der Server lässt keine anonyme FTP-Verbindung zu. In diesem Fall muss jeder Benutzer einen gültigen Windows-Benutzernamen und ein entsprechendes Kennwort eingeben. Um diese Einstellung für das FTP-Serverprotokoll festzulegen, deaktivieren Sie das Kontrollkästchen Anonymen Zugriff freigeben auf der Seite FTP-Protokolleigenschaften.
  • Sowohl anonyme Benutzer als auch Windows-Benutzer können Verbindungen zum FTP-Server herstellen. Bei dieser Konfiguration kann ein Benutzer entweder eine anonyme Verbindung oder einen Windows-Benutzernamen und ein entsprechendes Kennwort verwenden. Um diese Einstellung für das FTP-Serverprotokoll festzulegen, aktivieren Sie nur das Kontrollkästchen Anonymen Zugriff freigeben auf der Seite FTP-Protokolleigenschaften.
  • Der Server lässt nur anonyme FTP-Verbindungen zu. In diesem Fall kann ein Benutzer nicht unter Verwendung eines Windows-Benutzernamens und eines entsprechenden Kennworts auf den FTP-Server zugreifen. Um diese Einstellung für das FTP-Serverprotokoll festzulegen, aktivieren Sie sowohl das Kontrollkästchen Anonymen Zugriff freigeben als auch das Kontrollkästchen Nur anonymen Zugriff zulassen auf der Seite FTP-Protokolleigenschaften.

Wenn anonyme Verbindungen zulässig sind, müssen Sie den Windows-Benutzernamen und das entsprechende Kennwort für den anonymen Zugriff auf den FTP-Server bereitstellen. Erfolgt eine anonyme FTP-Übertragung, überprüft Windows den in diesem Dialogfeld zugewiesenen Benutzernamen, um zu ermitteln, ob der Zugriff auf die Dateien zulässig ist.
