Zugreifen auf die Statusseite

Auf der Statusseite können Sie Statusbenachrichtigungen, Systemressourcen und -informationen anzeigen. Der obere linke Bereich enthält Systembenachrichtungen und der obere rechte Teil Statusinformationen zu Systemressourcen. Nach Auswahl einer Benachrichtigung oder Systemressource werden in der unteren Bildschirmhälfte detaillierte Informationen zur jeweiligen Auswahl eingeblendet.

So greifen Sie auf die Statusseite zu

  • Klicken Sie im Bereich Status oben auf der Seite auf Status: <Statustyp>.

    - ODER -

  • Wählen Sie auf der primären Navigationsleiste Status aus.

Weitere Informationen zu den auf der Statusseite angezeigten Benachrichtigungen erhalten Sie unter Statusbenachrichtungen.
