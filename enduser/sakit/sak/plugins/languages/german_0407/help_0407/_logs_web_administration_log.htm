Vorheriges Thema

Verwalten von Webverwaltungsprotokollen

Das Webverwaltungsprotokoll enthält Ereignisse, die im Zusammenhang mit Zugriffen auf die Verwaltungswebsite stehen und vom Webserver erfasst wurden.

So verwalten Sie Webverwaltungsprotokolle

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie Protokolle aus.
  3. Wählen Sie Webverwaltungsprotokoll aus.
