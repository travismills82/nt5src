NFS-Protokoll

Mit Hilfe der Option NFS-Protokoll können Sie die Serverappliance für die Verwendung als NFS-Server konfigurieren. Das NFS-Protokoll ermöglicht Benutzern die gemeinsame Nutzung von Dateien in einer gemischten Computer-, Betriebssystem- und Netzwerkumgebung.

Mithilfe des NFS-Protokolls können Sie NFS-Clientgruppen, NFS-Sperren sowie NFS-Benutzer und -Gruppenzuordnungen verwalten. NFS-Freigaben werden jedoch im Abschnitt Freigaben der Webbenutzeroberfläche erstellt.

So konfigurieren Sie das NFS-Protokoll

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Protokolle freigeben aus.
  3. Wählen Sie NFS-Protokoll und dann Eigenschaften aus.
  4. Wählen Sie den gewünschten Task der NFS-Konfiguration aus.
