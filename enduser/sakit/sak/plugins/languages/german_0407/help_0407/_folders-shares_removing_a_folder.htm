Vorheriges Thema

Entfernen eines Ordners

Sie können jeden auf der Serverappliance erstellten Ordner entfernen.

So löschen Sie Ordner

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Ordner aus.
  3. Navigieren Sie mit der Objekt-/Taskauswahl zu dem Ordner, in dem sich der zu entfernende Unterordner befindet.
  4. Wählen Sie den zu löschenden Ordner aus.
  5. Wählen Sie in der Liste Tasks den Eintrag Löschen aus.
  6. Stellen Sie sicher, dass Sie den angezeigten Ordner entfernen möchten.
  7. Klicken Sie auf OK.
