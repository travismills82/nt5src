Vorheriges Thema

Verwalten von Ordnern

Auf der Seite Ordner können Sie eine Reihe von Netzwerkvolumes öffnen oder freigeben. Die Seite enthält eine Objekt-/Taskauswahl mit den folgenden Spalten:

  • Volumename - Zeigt jedes Volume mit Namen an. Zum Erstellen, Öffnen, Löschen oder Konfigurieren der Eigenschaften eines bestimmten Volumes aktivieren Sie das Kontrollkästchen neben dem zu bearbeitenden Volume.
  • Gesamter Speicherplatz - Zeigt den gesamten Speicherplatz auf dem Volume an.
  • Freier Speicherplatz - Gibt den verfügbaren Speicherplatz auf dem Volume an.
  • Freigabetyp - Gibt den Freigabetyp an, der auf den Ordner angewendet wird:

    W = Windows-Freigabe (CIFS)

    U = UNIX-Freigabe (NFS)

    F = FTP-Freigabe

    H = HTTP-Freigabe

    A = AppleTalk-Freigabe

    N = NetWare-Freigabe

In der Objekt-/Taskauswahl werden die ersten 100 ermittelten Ordner aufgelistet. Zum Navigieren mit der Objekt-/Taskauswahl in der Ordnerliste können Sie die Felder der Liste Suchen verwenden und die Suchkriterien im Feld links neben der Schaltfläche Gehe zu eingeben oder einen Bildlauf in der Liste durchführen. Wenn mehr als 100 Ordner vorhanden sind, besteht zusätzlich die Möglichkeit, die Ordner mit Hilfe der Schaltflächen zum Durchführen eines Bildlaufs nach oben oder unten links neben der Schaltfläche Gehe zu in Hundertergruppen anzuzeigen.

So geben Sie Ordner frei

  1. Wählen Sie Ordner auf der Seite Freigaben aus.
  2. Wählen Sie die Volumes aus, für die Ordner freizugeben sind, und wählen Sie dann in der Liste Tasks den Eintrag Freigabe aus.
  3. Geben Sie die erforderlichen Informationen ein. Weitere Informationen zur Dateneingabe auf der Eigenschaftenseite für ein bestimmtes Dateifreigabeverfahren erhalten Sie in der Hilfedatei des jeweiligen Freigabeprotokolls.

So verwalten Sie Ordner

  1. Wählen Sie Ordner auf der Seite Freigaben aus.
  2. Wählen Sie die Volumes aus, für die Ordner oder Freigaben anzuzeigen oder zu verwalten sind, und wählen Sie in der Liste Tasks den Eintrag Öffnen aus.

    Auf der Seite Ordner können Sie zahlreiche Netzwerkordner erstellen, öffnen, löschen oder konfigurieren. Die Seite enthält eine Objekt-/Aufgabenauswahl mit den folgenden Spalten:

    Ordnername - Zeigt jeden Ordner mit Namen an.

    Geändert am - Zeigt das Datum der letzten Änderung am Ordner an.

    Attribute - Zeigt eines der folgenden Ordnerattribute an:

    R = Schreibgeschützte Dateien

    A = Zu archivierende Dateien

    H = Versteckte Dateien

    C = Komprimierte Dateien

    S = Systemordner

    Freigabetyp - Gibt den Freigabetyp an, der auf den Ordner angewendet wird.

  3. Markieren Sie einen Ordner, und wählen Sie dann in der Liste Tasks den durchzuführenden Task aus.
  4. Um zu den Unterverzeichnissen eines Stammordners zu wechseln, markieren Sie den Stammordner, und wählen Sie dann in der Liste Tasks den Eintrag Öffnen aus.
  5. Um von einem Unterverzeichnis zum übergeordneten Element zu wechseln, wählen Sie in der Liste Tasks den Eintrag Nach unten aus.
