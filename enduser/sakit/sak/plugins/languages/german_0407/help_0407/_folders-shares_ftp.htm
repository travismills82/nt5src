Vorheriges Thema

FTP

Das FTP-Protokoll (File Transfer Protocol) wird zum Kopieren von Dateien zwischen zwei Computern im Internet verwendet. Beide Computer müssen ihre jeweilige FTP-Funktion unterstützen: Bei dem einen Computer muss es sich um einen FTP-Client, bei dem anderen um einen FTP-Server handeln.

Sie können die Serverappliance als FTP-Server konfigurieren. Clientcomputer können dann mit Windows® Explorer oder einem FTP-Befehlszeilenprogramm wie Ftp.exe auf die Serverappliance zugreifen; dazu muss der URL in der Adressleiste im Format ftp://Standortname/ eingegeben werden.
