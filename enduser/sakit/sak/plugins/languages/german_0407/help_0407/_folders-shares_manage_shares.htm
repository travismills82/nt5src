Vorheriges Thema

Verwalten von Freigaben

Auf der Seite Freigaben können Sie verschiedene Dateifreigaben erstellen, öffnen, löschen oder konfigurieren. Die Seite enthält eine Objekt-/Taskauswahl mit den folgenden Spalten:

  • Freigabename - Zeigt jeden Ordner mit Namen an. Zum Erstellen, Öffnen, Löschen oder Konfigurieren der Eigenschaften bestimmter Freigaben aktivieren Sie das Kontrollkästchen neben den zu bearbeitenden Freigaben.
  • Freigabepfad - Zeigt den Pfad der Freigabe an.
  • Typ - Zeigt einen der folgenden Freigabetypen an:

    W = Windows (CIFS)

    U = UNIX (NFS)

    F = FTP

    H = HTTP

  • Kommentar - Zeigt eine kurze Beschreibung der Freigabe an, sofern eine Beschreibung angegeben wurde.

Wählen Sie mit der Objekt-/Taskauswahl eine Freigabe aus, und wählen Sie dann in der Liste Tasks den durchzuführenden Task aus.
