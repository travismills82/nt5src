Vorheriges Thema

Hinzufügen einer Freigabe

Zur Erstellung einer Freigabe müssen Sie einen Freigabenamen angeben, der unter sämtlichen Freigaben sowie innerhalb des Freigabepfads eindeutig ist. Einige Protokolle unterstüzten außerdem einen Kommentar zur Freigabe oder eine kurze Beschreibung. Ferner muss mindestens ein verfügbares Protokoll aktiviert werden.

Während die Erstellung einer Freigabe für alle Protokolle über eine einzelne Benutzeroberfläche erfolgt, wird tatsächlich jedoch für jedes Protokoll eine gesonderte Freigabe angelegt. Sie können einzelne Protokolle aus einer Freigabe entfernen, ohne die Freigabe selbst zu entfernen. Beim Entfernen aller Protokolle aus einer Freigabe werden jedoch sämtliche Versionen der Freigabe gelöscht.

So fügen Sie eine Freigabe hinzu

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Freigaben auf der Seite Freigaben aus.
  3. Wählen Sie in der Liste Tasks den Eintrag Neu aus.
  4. Geben Sie auf der Registerkarte Allgemein den Namen und den Pfad der Freigabe ein.
  5. Aktivieren Sie die entsprechenden Kontrollkästchen, um die zu aktivierenden Protokolltypen anzugeben.
  6. Konfigurieren Sie die spezifischen Eigenschaften der einzelnen Freigabetypen anhand der Protokollregisterkarten.
