Konfigurieren der Webprotokolleinstellungen

Auf dieser Seite werden die Web- oder IIS-Protokolleinstellungen konfiguriert.

So konfigurieren Sie die gewünschten Einstellungen

  1. Wählen Sie auf der primären Navigationsleiste Webserver aus.
  2. Wählen Sie Webprotokolleinstellungen aus.
  3. Aktivieren Sie das Kontrollkästchen Protokollierung aktivieren.
  4. Wählen Sie im Feld Aktives Protokollformat das geeignete Protokolldateiformat aus.
  5. Wählen Sie die entsprechende Option aus, um das Zeitintervall festzulegen, nach dessen Ablauf ein neues Protokoll generiert wird.

    Wenn Sie beispielsweise Täglich auswählen, wird alle 24 Stunden mit der Erstellung eines neuen Protokolls begonnen.

  6. Aktivieren Sie das Kontrollkästchen Verwenden Sie für das Benennen von Dateien und das Rollover die Ortszeit des Servers.

    Die Serverappliance verwendet die lokale Zeit für Dateibenennung und Protokollrollover. Andernfalls verwendet der Server Greenwich Mean Time (GMT).

    Anmerkung   Wenn Sie Wenn die Dateigröße folgenden Wert in MB erreicht auswählen, wird das Kontrollkästchen Verwenden Sie für das Benennen von Dateien und das Rollover die Ortszeit deaktiviert.

  7. Überprüfen Sie, ob das Feld Protokolldateiverzeichnis den richtigen Verzeichnispfad für die Protokolldateien enthält.
    8. Wählen Sie die entsprechende Option aus, um die Einstellungen auf alle Websites anzuwenden, die den Standardwert verwenden.
  8. Klicken Sie auf OK.
