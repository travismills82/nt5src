Netzwerk

Auf der Seite Netzwerk können Sie die folgenden netzwerkbezogenen Eigenschaften der Serverappliance konfigurieren:

  • Identifikation - Den Namen und die Domänenmitgliedschaft der Serverappliance festlegen.
  • Globale Einstellungen - Die Netzwerkeinstellungen konfigurieren, die auf sämtliche Netzwerkadapter auf der Serverappliance angewendet werden.
  • Schnittstellen - Die Eigenschaften der einzelnen Netzwerkadapter auf der Serverappliance konfigurieren.
  • Administrator - Das Kennwort des Benutzerkontos ändern, mit dem Sie auf die Verwaltungswebsite der Serverappliance zugreifen.
  • Verwaltungswebsite - Die IP-Adressen und den Port angeben, die für den Zugriff auf die Verwaltungswebsite verwendet werden.
  • Telnet - Telnet für die Verwaltung der Serverappliance konfigurieren.
