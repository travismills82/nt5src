Vorheriges Thema

Bearbeiten von NFS-Clientgruppen

So fügen Sie Mitglieder zu einer NFS-Clientgruppe hinzu

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Protokolle freigeben aus.
  3. Wählen Sie NFS-Protokoll und dann Eigenschaften aus.
  4. Wählen Sie Clientgruppen aus.
  5. Wählen Sie die Gruppe aus, die Sie bearbeiten möchten.
  6. Wählen Sie in der Liste Tasks den Eintrag Bearbeiten aus.
  7. Geben Sie auf der Seite NFS-Clientgruppen bearbeiten die IP-Adresse oder den Computernamen des Mitglieds ein, das der Gruppe hinzuzufügen ist.
  8. Klicken Sie auf Hinzufügen.
  9. Klicken Sie auf OK.

So entfernen Sie Mitglieder aus einer NFS-Clientgruppe

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Protokolle freigeben aus.
  3. Wählen Sie NFS-Protokoll und dann Eigenschaften aus.
  4. Wählen Sie Clientgruppen aus.
  5. Wählen Sie die Gruppe aus, die Sie bearbeiten möchten.
  6. Wählen Sie in der Liste Tasks den Eintrag Bearbeiten aus.
  7. Geben Sie auf der Seite NFS-Clientgruppen bearbeiten die IP-Adresse oder den Computernamen des Mitglieds ein, das aus der Gruppe zu entfernen ist.
  8. Klicken Sie auf Entfernen.
  9. Klicken Sie auf OK.
