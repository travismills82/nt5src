Vorheriges Thema

Verwalten von Anwendungsprotokollen

Das Anwendungsprotokoll enthält Ereignisse, die von Anwendungen oder Programmen aufgezeichnet wurden. Ein Textverarbeitungsprogramm könnte beispielsweise einen Dateifehler im Anwendungsprotokoll erfassen. Die erfassten Ereignisse richten sich nach der jeweiligen Anwendung.

So verwalten Sie Anwendungsprotokolle

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie Protokolle aus.
  3. Wählen Sie Anwendungsprotokoll aus.
