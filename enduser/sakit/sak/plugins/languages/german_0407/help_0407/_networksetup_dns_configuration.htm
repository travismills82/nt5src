DNS-Konfiguration

Domain Name System (DNS) stellt einen statischen, hierarchischen Namensdienst für TCP/IP-Hosts dar. Der Netzwerkadministrator konfiguriert DNS mit einer Liste von Hostnamen und IP-Adressen. Auf diese Weise wird es Netzwerkbenutzern ermöglicht, Abfragen an DNS zu senden, in denen die Angabe von Remotesystemen anhand von Hostnamen statt IP-Adressen angefordert wird.

Anmerkung   Der Zweck dieser Eigenschaftenseite besteht darin, dem Benutzer die Eingabe von Adressen externer DNS-Server zu ermöglichen. Die Serverappliance enthält keinen DNS-Server.

So könnte beispielsweise eine Arbeitsstation, die für die Verwendung der DNS-Namensauflösung konfiguriert wurde, statt des Befehls ping 1.2.3.4 den Befehl ping remotehost verwenden, sofern eine entsprechende Zuordnung für das System namens remotehost in der DNS-Datenbank enthalten ist. DNS-Domänen dürfen nicht mit Microsoft Windows-Domänen verwechselt werden.

Im dem nach Client/Server-Prinzip gestalteten DNS-Modell sendet der Server, auf dem Informationen zu dem Teil der DNS-Datenbank gespeichert sind, der Computernamen an Clients ausgibt, Anforderungen zur Namensauflösung über das Internet.

So richten Sie die Serverappliance zum automatischen Abrufen der DNS-Serverinformationen von einem DHCP-Server ein

  1. Wählen Sie auf der primären Navigationsleiste Netzwerk aus.
  2. Wählen Sie Schnittstellen aus.
  3. Wählen Sie die zu ändernde Netzwerkverbindung aus.
  4. Wählen Sie in der Liste Tasks den Eintrag DNS aus.
  5. Wählen Sie Konfiguration des DHCP-Servers erhalten aus.
  6. Klicken Sie auf OK.

So legen Sie die von der Serverappliance verwendeten DNS-Server manuell fest

  1. Wählen Sie Schnittstellen auf der Seite Netzwerk aus.
  2. Wählen Sie die zu ändernde Netzwerkverbindung aus.
  3. Wählen Sie in der Liste Tasks den Eintrag DNS aus.
  4. Wählen Sie Manuell konfigurieren aus.
  5. Geben Sie die entsprechende IP-Adresse in dem Feld neben der Schaltfläche Hinzufügen ein, und klicken Sie auf Hinzufügen.
  6. Um einen weiteren DNS-Server hinzuzufügen, wiederholen Sie Schritt 5.
  7. Wenn sämtliche DNS-Server hinzugefügt wurden, klicken Sie auf OK.

    Anmerkung   Wenn Sie festlegen, dass die IP-Adresse von DHCP übernommen und DNS manuell eingerichtet werden soll, akzeptiert das System die manuelle Eingabe, und die Eigenschaften auf der Serverappliance werden automatisch auf Manuell konfigurieren gesetzt. In der Spalte Aktuelle Konfiguration der Objekt-/Taskauswahl auf der Seite Schnittstellen wird DHCP jedoch weiterhin als Quelle der IP-Adresse angezeigt. Sie können zur Eigenschaftenseite mit den DNS-Einstellungen zurückkehren, um das Speichern der manuellen Konfiguration zu bestätigen.
