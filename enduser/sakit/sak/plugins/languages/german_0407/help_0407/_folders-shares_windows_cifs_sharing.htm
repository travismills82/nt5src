Festlegen der Eigenschaften von Windows-Freigaben (CIFS)

Windows-Clientcomputer verwenden das CIFS-Protokoll (Common Internet File System). Auf dieser Seite können Sie die Anzahl der Benutzer, die über Zugriff auf eine Freigabe verfügen, und die Zwischenspeicherungsoptionen bezüglich der Freigabe ändern sowie Benutzerberechtigungen festlegen oder bearbeiten.

Applianceeinstellungen

Im Abschnitt Benutzerlimit können Sie die maximale Anzahl an Benutzern zulassen oder die Anzahl an Verbindungen angeben, die zu einem bestimmten Zeitpunkt hergestellt werden können.

So richten Sie das Benutzerlimit ein

  • Wählen Sie Zulässiges Maximum aus, um der maximal zulässigen Anzahl an Benutzern die Anmeldung an der Serverappliance zu ermöglichen.

    - ODER -

  • Wählen Sie Maximum aus, und geben Sie dann die zulässige Anzahl an Benutzern an.
  • Wenn Sie die Zwischenspeicherung von Dateien in dem freigegebenen Ordner ermöglichen, geben Sie anhand der Liste Einstellungen die zu verwendende Zwischenspeicherungsoption an. Die Zwischenspeicherungsoptionen werden unter Windows (CIFS) beschrieben.

Berechtigungen

Sie können auch Berechtigungen für Benutzer oder Gruppen festlegen, denen der Zugriff auf die Serverappliance gewährt oder verweigert wird.

So legen Sie Benutzerberechtigungen fest

  1. Geben Sie im Feld Benutzer oder Gruppe hinzufügen den Benutzer- oder Gruppennamen ein, der in der Liste von Berechtigungen hinzuzufügen ist, oder wählen Sie in der Liste unter dem Feld einen Benutzer aus.
Sie können in der Liste lokale Benutzer oder lokale Gruppen auswählen. Zum Hinzufügen von Domänenbenutzern oder -gruppen müssen Sie das jeweilige Konto im Format <Domänenname\Benutzername> oder <Domänenname\Gruppenname> eingeben.
  1. Klicken Sie auf Hinzufügen.
  2. Legen Sie anhand der Liste Maximum die Art des Zugriffs fest, der den in der Liste Berechtigungen ausgewählten Benutzern für Dateien auf der Serverappliance gewährt wird.
Benutzern kann der Zugriff verweigert oder der Lesezugriff, Schreibzugriff, Schreib- und Lesezugriff oder Vollzugriff zugewiesen werden.
  1. Verwenden Sie die Liste Verweigern, um den in der Liste Berechtigungen ausgewählten Benutzern und Gruppen eine Zugriffsart zu verweigern.
  2. Um einen Benutzer oder eine Gruppe aus der Liste Berechtigungen zu entfernen, markieren Sie den Benutzer oder die Gruppe in der Liste, und wählen Sie dann Entfernen aus.
  3. Klicken Sie auf OK.
