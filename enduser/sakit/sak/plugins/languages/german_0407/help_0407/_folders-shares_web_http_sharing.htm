Vorheriges Thema

Festlegen der Eigenschaften von Webfreigaben (HTTP)

Auf dieser Seite können Sie den Zugriff von HTTP-Clients auf die einzelnen Freigaben festlegen. Der Zugriff kann basierend auf dem Clienthostnamen erteilt oder verweigert werden. Das Gewähren oder Verweigern des Zugriffs auf Grundlage von Clientgruppen ist ebenfalls möglich, wenn eine Clientgruppe mindestens einen Clienthostnamen enthält.

So weisen Sie Clients Berechtigungen für eine Webfreigabe zu

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Freigaben auf der Seite Freigaben aus.
  3. Wählen Sie die Freigabe aus, für die Sie den HTTP-Clientzugriff hinzufügen möchten, und wählen Sie dann Eigenschaften aus.
  4. Aktivieren Sie auf der Registerkarte Allgemein das Kontrollkästchen Web (HTTP).
  5. Wählen Sie die Registerkarte Webfreigabe aus.
    • Aktivieren Sie das Kontrollkästchen Lesen, um Lesezugriff zu gewähren.
- ODER -
  • Aktivieren Sie das Kontrollkästchen Schreiben, um Schreibzugriff zu gewähren.
  1. Klicken Sie auf OK.
