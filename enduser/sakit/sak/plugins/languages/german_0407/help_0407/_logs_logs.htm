Protokolle

In einer Protokolldatei werden Meldungen bzw. Ereignisprotokolle gespeichert, die von einer Anwendung, einem Dienst oder Microsoft® Windows® generiert wurden. Mit Hilfe dieser Meldungen können Sie die von der Serverappliance durchgeführten Vorgänge nachverfolgen.

Die Protokollfunktion ermöglicht das Anzeigen, Löschen, Herunterladen und Konfigurieren der folgenden Ereignisprotokolltypen, die vom System bereitgestellten werden:

  • Verwalten von Anwendungsprotokollen
  • Verwalten von FTP-Protokollen
  • Verwalten von NFS-Protokollen
  • Verwalten von Sicherheitsprotokollen
  • Verwalten von Systemprotokollen
  • Verwalten von Webfreigabenprotokollen (HTTP)
  • Verwalten von Webverwaltungsprotokollen
  • Löschen von Protokolldateien
  • Downloaden von Protokolldateien
  • Ändern von Protokolleigenschaften
  • Anzeigen von Protokolldetails
