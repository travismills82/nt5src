Vorheriges Thema

Ändern von Protokolleigenschaften

Es besteht die Möglichkeit, die Eigenschaften von Anwendungs-, System- und Sicherheitsprotokollen zu konfigurieren. Auf dieser Seite können Sie die maximale Protokollgröße angeben und festlegen, wie Protokolleinträge bei Erreichen der Höchstkapazität der Serverappliance verarbeitet werden.

So ändern Sie die Eigenschaften einer Protokolldatei

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie Protokolle aus.
  3. Wählen Sie das zu konfigurierende Anwendungs-, System- oder Sicherungsprotokoll aus.
  4. Wählen Sie das zu konfigurierende Protokoll aus.
  5. Wählen Sie Protokolleigenschaften aus.
  6. Geben Sie im Feld Maximale Protokollgröße die maximale Protokollgröße in KB an.
  7. Wählen Sie die Schaltfläche aus, die am besten darstellt, wie Protokolleinträge behandelt werden sollen, wenn die maximale Protokollgröße erreicht wird.
