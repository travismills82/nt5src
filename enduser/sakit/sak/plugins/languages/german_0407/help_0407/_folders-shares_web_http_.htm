Vorheriges Thema

Web (HTTP)

Bei HTTP handelt es sich um das Internetprotokoll, das von WWW-Browsern und -Servern zum Datenaustausch verwendet wird. Das Protokoll legt die Aktionen fest, die Webserver und -browser als Reaktion auf verschiedene Befehle durchführen müssen; so wird einem Benutzer die Verwendung eines Clientprogramms zum Eingeben eines URLs oder Auswählen eines Hyperlinks und das Abrufen von Text, Grafiken, Sound und anderen digitalen Informationen von einem Webserver ermöglicht. Sämtliche URLs von Dateien auf Webservern beginnen mit http://.
