Vorheriges Thema

Ändern von Benutzereigenschaften

Zu den Benutzereigenschaften gehören der Benutzername, der vollständige Name sowie die Beschreibung. Auf der Eigenschaftenseite eines Benutzers können Sie ferner das Benutzerkonto aktivieren oder deaktivieren. Darüber hinaus besteht ebenfalls die Möglichkeit, die Eigenschaften mehrerer Benutzerkonten zu ändern, indem Sie sämtliche zu bearbeitende Konten vor dem Öffnen der Eigenschaftenseite auswählen.

So rufen Sie die Benutzereigenschaften auf

  1. Wählen Sie auf der primären Navigationsleiste Benutzer aus.
  2. Wählen Sie Lokale Benutzer aus.
  3. Wählen Sie die zu ändernden Benutzerkonten aus.
  4. Wählen Sie in der Liste Tasks den Eintrag Eigenschaften aus.
  5. Nehmen Sie die gewünschten Änderungen an den Benutzereigenschaften vor.

    Im Feld Stammverzeichnis wird ein neues Verzeichnis angegeben, das erstellt wird und auf das der Benutzer exklusiv zugreifen kann. Der Verzeichnisname stimmt mit dem oben definierten Benutzernamen überein und wird im angegebenen Pfad angelegt.

  6. Klicken Sie auf OK.
