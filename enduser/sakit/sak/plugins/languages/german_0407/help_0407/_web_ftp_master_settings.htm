FTP-Haupteinstellungen

Mithilfe von Anonyme FTP-Verbindungen zulassen können Sie den anonymen Zugriff auf die Serverappliance über FTP-Clients zulassen oder verweigern.

Im Feld FTP-Stammverzeichnis wird das FTP-Stammverzeichnis angezeigt, auf das bei der Verwendung einer anonymen FTP-Verbindung zugegriffen wird.

Das Feld Verzeichnisauflistungsstil definiert das Format, in dem Verzeichnisse beim Versenden von LIST-Vorgängen durch FTP-Clients aufgelistet werden.

  • Bei Verwendung der MS-DOS-Auflistung wird die Jahreszahl standardmäßig im zweistelligen Format angezeigt.
  • Bei Verwendung der UNIX-Auflistung wird die Jahreszahl im vierstelligen Format angezeigt, wenn die Jahreszahl aus dem Dateidatum nicht mit der Jahreszahl auf dem FTP-Server übereinstimmt. Wenn die Jahreszahlen aus dem Dateidatum und auf dem Server übereinstimmen, wird keine Jahreszahl ausgegeben.

Maximale Anzahl von Verbindungen gibt die maximale Anzahl der zulässigen FTP-Verbindungen an, die gleichzeitig zur Serverappliance hergestellt werden können.

Verbindungszeitlimit in Sekunden gibt die Leerlaufzeit in Sekunden an, nach deren Ablauf die Verbindung zur Serverappliance automatisch getrennt wird.
