Vorheriges Thema

Freigeben eines Ordners

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Ordner aus.
  3. Wählen Sie das Volume aus, mit dem Sie arbeiten möchten, und wählen Sie dann Öffnen aus.
  4. Navigieren Sie zu dem Verzeichnis, das den freizugebenden Ordner enthält.
  5. Wählen Sie den freizugebenden Ordner aus.
  6. Wählen Sie in der Liste Tasks den Eintrag Freigaben aus.
    • Wenn der Ordner nicht bereits freigegeben wurde, wird die Seite Neue Freigabe angezeigt.
    • Wenn der Ordner freigegeben wurde, wird die Seite Freigabeeigenschaften angezeigt.
    • Wenn der Ordner unter mehreren Namen freigegeben wurde, wählen Sie den Task Freigabe verwalten aus.
