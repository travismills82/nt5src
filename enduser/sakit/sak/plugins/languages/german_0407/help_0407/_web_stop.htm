Vorheriges Thema

Beenden einer Website

Die Objekt-/Taskauswahl auf der Seite Websitekonfiguration gibt den Status jeder Website auf der Serverappliance an. Gegebenenfalls können Sie jede Website beenden, die angehalten wurde oder ausgeführt wird.

Das Beenden einer Website verhindert die Annahme neuer Verbindungen durch Internetdienste und stoppt alle Anforderungen, die derzeit verarbeitet werden. Das Anhalten einer Website verhindert die Annahme neuer Verbindungen durch Internetdienste, wirkt sich jedoch nicht auf die Anforderungen aus, die aktuell verarbeitet werden.

So beenden Sie eine Website

  1. Wählen Sie auf der primären Navigationsleiste Sites aus.
  2. Wählen Sie die Kennung der zu beendenden Website aus.
  3. Wählen Sie in der Liste Tasks den Eintrag Beenden aus.
