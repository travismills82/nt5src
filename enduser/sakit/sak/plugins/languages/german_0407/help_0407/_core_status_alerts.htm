Statusbenachrichtigungen

Wenn Sie im Statusbereich Status: <Statustyp> auswählen, wird eine Benachrichtigungsseite angezeigt.

Im Folgenden werden die durch die einzelnen Statustypen erzeugten Benachrichtigungsinhalte aufgeführt:

  • Status Normal - Auf der Benachrichtigungsseite wird angegeben, dass keine Benachrichtigungen vorliegen.
  • Status Information - Die vorliegenden Probleme werden auf der Benachrichtigungsseite als Hyperlinks aufgeführt, wobei neben jedem im System aufgetretenen Problem der Buchstabe i angezeigt wird.
  • Status Warnung - Die Fehler werden auf der Benachrichtigungsseite als Hyperlinks aufgeführt, wobei neben jedem im System aufgetretenen Fehler ein gelbes Dreieck mit einem Ausrufungszeichen (!) angezeigt wird.
  • Status Kritisch - Die Fehler werden auf der Benachrichtigungsseite als Hyperlinks aufgeführt, wobei neben jedem im System aufgetretenen Fehler ein roter Kreis mit einem weißen x angezeigt wird.

So löschen Sie eine Benachrichtigung

  1. Wählen Sie auf der primären Navigationsleiste Status aus.
  2. Wählen Sie die zu löschende Benachrichtigung aus.
  3. Wählen Sie die Aktion aus, welche die Warnsituationen ansprechen soll, oder wählen Sie Meldung löschen aus, um die Benachrichtigung ohne Änderung der Serverappliancekonfiguration zu löschen.
