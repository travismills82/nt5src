Einstellen der Sprache

Wenn auf der Serverappliance andere Sprachen installiert sind, können Sie die Sprache ändern, die in der Webbenutzeroberfläche, der lokalen Benutzeroberfläche und den Hilfedateien verwendet wird.

So ändern Sie die von der Serverappliance verwendete Sprache

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie dann Sprache aus.
  3. Wählen Sie auf der Seite Sprache festlegen die Sprache für die Serverappliance aus.
  4. Klicken Sie auf OK.
