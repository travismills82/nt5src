Verwaltungswebsite

Mit dieser Funktion können Sie die IP-Adressen und den Port für den Zugriff auf die Verwaltungswebsite auf der Serverappliance ändern.

Die standardmäßige IP-Adresse für die Kommunikation mit der Serverappliance wird in der Regel geändert, wenn die Serverappliance nur für ein gesondertes Verwaltungsnetzwerk oder ein bestimmtes Subnetz verwaltet wird.

Die standardmäßigen Abhörports für den verschlüsselten und nicht verschlüsselten Zugriff können je nach vorhandener Netzwerksoftware und bestehenden Konfigurationen geändert werden, beispielsweise wenn für eine bestimmte Portnummer kein Datenverkehr zulässig ist.

So ändern Sie die Eigenschaften der Verwaltungswebsite

  1. Wählen Sie auf der primären Navigationsleiste Netzwerk aus.
  2. Wählen Sie Verwaltungswebsite aus.
  3. Gehen Sie auf der Seite Verwaltungs-Websiteeigenschaften wie folgt vor:
    1. Wählen Sie Alle IP-Adressen oder Nur diese IP-Adresse aus.
    2. Wenn Sie Nur diese IP-Adresse verwenden, wählen Sie in der Liste die gewünschte IP-Adresse aus.
    3. Wenn Sie den Port für den nicht verschlüsselten Zugriff ändern, geben Sie die neue Portnummer im Feld Port für nicht verschlüsselten Zugriff ein.
    4. Wenn Sie den Port für den verschlüsselten Zugriff ändern, geben Sie die neue Portnummer im Feld Port für verschlüsselten (SSL) Zugriff ein.
  4. Klicken Sie auf OK.
