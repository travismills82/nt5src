Vorheriges Thema

Konfigurieren von Protokolleigenschaften

Verwenden Sie die Eigenschaftenseite des angegebenen Protokolls zur Konfiguration der gewünschten Netzwerkprotokolle.

So konfigurieren Sie Eigenschaften von Netzwerkprotokollen

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Protokolle freigeben aus.
  3. Markieren Sie das zu konfigurierende Protokoll.
  4. Wählen Sie in der Liste Tasks den Eintrag Eigenschaften aus.
