Erste Schritte

Die Server Appliance Powered by Microsoft® Windows® ist unmittelbar mit dem Computernetzwerk verbunden. Eine Serverappliance wird unter Verwendung einer Webbenutzeroberfläche verwaltet; die Verwaltung kann über einen an das Netzwerk angeschlossenen Remoteclientcomputer erfolgen. Es handelt sich in der Regel um ein nicht vom Benutzer gesteuertes Gerät ohne eigenen Bildschirm oder zugehörige Eingabegeräte wie Tastatur oder Maus. Da Serverappliances auf demselben Code wie Microsoft® Windows® 2000 basieren, können zahlreiche Remoteverwaltungsmethoden, die auf der Windows 2000-Plattform verfügbar sind, ebenfalls zur Verwaltung von Serverappliances unter Windows eingesetzt werden.

Verwandte Themen

Navigationsmodell der Webbenutzeroberfläche

Konfigurieren von netzwerkverbundenem Speicher

Konfigurieren einer Web Server Appliance
