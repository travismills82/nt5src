Vorheriges Thema

Erstellen einer Website

Auf der Seite Website erstellen können Sie eine neue Website erstellen und die zugehörigen Eigenschaften konfigurieren.

So erstellen Sie eine Website

  1. Wählen Sie auf der primären Navigationsleiste Sites aus.
  2. Wählen Sie in der Liste Tasks den Eintrag Erstellen aus.
  3. Vervollständigen Sie die Informationen auf der Seite, um eine neue Website zu erstellen.
  4. Klicken Sie auf OK.
