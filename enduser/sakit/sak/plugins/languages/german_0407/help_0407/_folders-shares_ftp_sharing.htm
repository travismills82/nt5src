Vorheriges Thema

Festlegen der Eigenschaften von FTP-Freigaben

Über diese Seite können Sie festlegen, welche FTP-Clients zum Zugriff auf die einzelnen Freigaben berechtigt sind. Der Zugriff kann basierend auf dem Clienthostnamen erteilt oder verweigert werden.

So weisen Sie Clients Berechtigungen für eine FTP-Freigabe zu

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Freigaben auf der Seite Freigaben aus.
  3. Wählen Sie die Freigabe aus, für die Sie den FTP-Clientzugriff hinzufügen möchten, und wählen Sie dann Eigenschaften aus.
  4. Wählen Sie die Registerkarte FTP-Freigabe aus.
    • Aktivieren Sie das Kontrollkästchen Lesen, um Lesezugriff zu gewähren.
- ODER -
  • Aktivieren Sie das Kontrollkästchen Schreiben, um Schreibzugriff zu gewähren.
Es besteht die Möglichkeit, Berechtigungen zum Lesen oder Schreiben bzw. Berechtigungen zum Lesen und Schreiben zuzuweisen.
  1. Klicken Sie auf OK.

So protokollieren Sie Clientzugriffe auf eine FTP-Freigabe

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Freigaben auf der Seite Freigaben aus.
  3. Wählen Sie die Freigabe aus, für die Sie den FTP-Clientzugriff hinzufügen möchten, und wählen Sie dann Eigenschaften aus.
  4. Wählen Sie die Registerkarte FTP-Freigabe aus.
  5. Aktivieren Sie das Kontrollkästchen Besuche protokollieren.
  6. Klicken Sie auf OK.
