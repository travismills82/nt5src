Herunterfahren der Serverappliance

Auf dieser Seite können Sie die Serverappliance herunterfahren oder neu starten bzw. das Herunterfahren oder Neustarten planen.

So fahren Sie die Serverappliance herunter oder starten sie neu

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie dann Herunterfahren aus.
  3. Wählen Sie den durchzuführenden Task aus.
  4. Klicken Sie zum Bestätigen auf OK.

    Wenn die Serverappliance neu gestartet werden soll, wird die Seite Neustarten angezeigt.

Anmerkung   Die Seite Neustarten überprüft in regelmäßigen Abständen, ob die Serverappliance wieder online ist. Ist dies der Fall, wird automatisch die Standardseite angezeigt.

So planen Sie das Herunterfahren oder Neustarten der Serverappliance

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie dann Herunterfahren aus.
  3. Wählen Sie Geplantes Herunterfahren aus.
  4. Legen Sie die gewünschten Einstellungen fest.
  5. Klicken Sie auf OK.

Anmerkung   Um ein aktuell geplantes Ereignis abzubrechen, wählen Sie Kein Herunterfahren oder Neustarten geplant aus.
