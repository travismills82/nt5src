Vorheriges Thema

Verwalten von Systemprotokollen

Das Systemprotokoll enthält Ereignisse, die von den Microsoft® Windows® 2000-Systemkomponenten aufgezeichnet wurden. So wird beispielsweise das Fehlschlagen des Ladens eines Gerätetreibers oder einer anderen Systemkomponente während des Startens im Systemprotokoll aufgezeichnet.

So verwalten Sie Systemprotokolle

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie Protokolle aus.
  3. Wählen Sie Systemprotokoll aus.
