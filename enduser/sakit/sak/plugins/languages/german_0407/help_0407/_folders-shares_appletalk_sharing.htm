Vorheriges Thema

Festlegen der Eigenschaften von AppleTalk-Freigaben

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Freigaben auf der Seite Freigaben aus.
  3. Wählen Sie die Freigabe aus, für die Sie den AppleTalk-Clientzugriff hinzufügen möchten, und wählen Sie dann Eigenschaften aus.
  4. Aktivieren Sie auf der Registerkarte Allgemein das Kontrollkästchen AppleTalk-Freigabe.
  5. Wählen Sie die Registerkarte AppleTalk-Freigabe aus, und legen Sie die Benutzerlimits sowie das Kennwort für den Freigabezugriff fest.
  6. Legen Sie die Zugriffsberechtigungen Schreibgeschützt und Gast fest.
  7. Klicken Sie auf OK.
