Vorheriges Thema

Starten einer Website

Die Objekt-/Taskauswahl auf der Seite Websitekonfiguration gibt den Status jeder Website auf der Serverappliance an. Sie können jede beendete oder angehaltene Website starten. Wenn der Port einer beendeten Website jedoch von einer anderen Site verwendet wird, kann die beendete Website nicht gestartet werden.

So starten Sie eine beendete oder angehaltenen Website

  1. Wählen Sie auf der primären Navigationsleiste Sites aus.
  2. Wählen Sie die Kennung der zu startenden Site aus.
  3. Wählen Sie in der Liste Tasks den Eintrag Starten aus.
