Windows (CIFS)

Freigeben von Dateien mit Microsoft Windows

Wenn Sie einen Ordner freigeben, können Sie Berechtigungen auswählen, die anderen Netzwerkbenutzern Zugriff auf die Dateien in dem jeweiligen Ordner ermöglichen oder verweigern. Für Clientcomputer unter Microsoft Windows besteht ebenfalls die Möglichkeit anzugeben, ob andere Windows-Benutzer den freigegebenen Ordner offline verfügbar machen können.

Um eine freigegebene Netzwerkdatei offline verfügbar zu machen, wird eine Version der Datei in einem zu diesem Zweck reservierten Teil der Festplatte des Clientcomputers, dem Cache, gespeichert. Der Computer kann auch dann auf den Cache zugreifen, wenn er nicht mit dem Netzwerk verbunden ist. Beim Freigeben von Dateien stehen folgende Zwischenspeicherungsoptionen zur Verfügung:

Manuelle Zwischenspeicherung von Dokumenten

Mit dieser Option wird der Offlinezugriff lediglich für die Dateien gewährt, die der Benutzer des freigegebenen Ordners der Serverappliance explizit (manuell) angibt. Diese Zwischenspeicherungsoption eignet sich insbesondere für einen freigegebenen Serverapplianceordner mit Dateien, die von mehreren Benutzern verwendet und bearbeitet werden sollen. Beim Einrichten eines freigegebenen Ordners für die Offlineverwendung ist dies die Standardoption.

Automatische Zwischenspeicherung von Dokumenten

Diese Option macht sämtliche Dateien des freigegebenen Serverapplianceordners für alle Benutzer offline verfügbar, die diese Dateien öffnen.

Durch das automatische Zwischenspeichern ist der Inhalt des Ordners offline verfügbar, und zwar unabhängig davon, ob ein Benutzer, der den freigegebenen Serverapplianceordner verwendet, diesen ausdrücklich als verfügbar angibt oder nicht. Dies umfasst auch Dokumente, Grafiken, Programmdateien und andere Dateien.

Nur Dateien, die von einem Benutzer im freigegebenen Serverapplianceordner geöffnet werden, sind für diesen Benutzer weiterhin verfügbar, wenn er offline arbeitet.

Automatische Zwischenspeicherung von Programmen

Diese Option stellt schreibgeschützten Offlinezugriff für die Dateien des freigegebenen Ordners bereit. Sie eignet sich insbesondere, um Dateien offline zur Verfügung zu stellen, die nur gelesen, referenziert oder ausgeführt, aber nicht geändert werden dürfen. Mit dieser Option verringern Sie die Netzwerkauslastung, weil die Offlinedateien direkt ohne Zugriff auf die Dateiversionen im Netzwerk geöffnet werden und in der Regel schneller gestartet und ausgeführt werden.

Bei Verwendung der automatischen Zwischenspeicherung von Programmen sollten Sie die Berechtigungen für die Dateien des freigegebenen Ordners auf schreibgeschützten Zugriff beschränken.
