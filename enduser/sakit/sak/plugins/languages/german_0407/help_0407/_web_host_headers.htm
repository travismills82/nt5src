Hostheader

Hostheadernamen sind Adressen von Websites, die die Unterstützung von mehreren Websites durch eine IP-Adresse ermöglichen. Ein Firmenserver kann beispielsweise als Host für Websites verschiedener interner Abteilungen, wie Marketingabteilung, Buchhaltung und Personalabteilung, eingesetzt werden. Um diese Websites, die über die gleiche IP-Adresse verfügen, voneinander zu unterscheiden, werden ihnen eindeutige Hostheadernamen zugewiesen.

Die folgende Tabelle zeigt, wie Sie eindeutige Hostheader mit einer IP-Adresse zur Unterscheidung verschiedener Websites verwenden können.

Abteilung IP-Adresse Beispiel für Hostheader
Firmenhauptsite 172.28.144.10 www.Firmenserver.de
Marketingabteilung 172.28.144.10 www.Marketing.Firmenserver.de
Buchhaltung 172.28.144.10 www.Buchhaltung.Firmenserver.de
Personalabteilung 172.28.144.10 www.Personal.Firmenserver.de
