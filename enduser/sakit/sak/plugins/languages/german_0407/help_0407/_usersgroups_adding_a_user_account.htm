Vorheriges Thema

Hinzufügen eines Benutzerkontos

Beim Hinzufügen eines Benutzerkontos müssen Sie einen Benutzernamen, den vollständigen Namen des Benutzers, eine kurze Beschreibung des Kontos und ein Kontokennwort angeben.

Beachten Sie, dass Benutzernamen eindeutig sein müssen und nicht mit den Namen vorhandener Gruppen übereinstimmen dürfen.

Ein Benutzername darf mit keinem anderen Benutzer- oder Gruppennamen auf dem verwalteten Computer identisch sein. Er kann bis zu 20 Zeichen in Groß- und Kleinschreibung enthalten. Folgende Zeichen sind ausgenommen: " / \ [ ] : ; | = , + * ? < >. Darüber hinaus darf ein Benutzername nicht ausschließlich aus Punkten (.) oder Leerzeichen bestehen.

Geben Sie in den Feldern Kennwort und Kennwort bestätigen ein bis zu 127 Zeichen langes Kennwort ein. Wenn Sie jedoch Microsoft® Windows® 2000 in einem Netzwerk verwenden, in dem auch Microsoft Windows 95- oder Microsoft Windows 98-Computer eingesetzt werden, sollten Sie Kennwörter mit weniger als 14 Zeichen verwenden. Windows 95 und Windows 98 unterstützen Kennwörter nur bis zu einer Länge von 14 Zeichen. Mit einem längeren Kennwort kann die Netzwerkanmeldung von diesen Computern aus fehlschlagen.

Der Administratorgruppe sollten nur neue Benutzer hinzugefügt werden, die ausschließlich Verwaltungstasks durchführen.

So fügen Sie ein Benutzerkonto hinzu

  1. Wählen Sie auf der primären Navigationsleiste Benutzer aus.
  2. Wählen Sie Lokale Benutzer auf der Seite Benutzer aus.
  3. Wählen Sie in der Liste Tasks den Eintrag Neu aus.
  4. Geben Sie die Informationen für das neue Benutzerkonto ein.

    Im Feld Stammverzeichnis wird ein neues Verzeichnis angegeben, das erstellt wird und auf das der Benutzer exklusiv zugreifen kann. Der Verzeichnisname stimmt mit dem oben definierten Benutzernamen überein und wird im angegebenen Pfad angelegt.

  5. Klicken Sie auf OK.
