Vorheriges Thema

Entfernen von NFS-Clientgruppen

So entfernen Sie eine NFS-Clientgruppe

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Protokolle freigeben aus.
  3. Wählen Sie NFS-Protokoll und dann Eigenschaften aus.
  4. Wählen Sie Clientgruppen aus.
  5. Wählen Sie die zu entfernenden NFS-Clientgruppen aus.
  6. Wählen Sie in der Liste Tasks den Eintrag Löschen aus.
  7. Klicken Sie auf der Seite NFS-Clientgruppen löschen auf OK.
