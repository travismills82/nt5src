Vorheriges Thema

Festlegen der Eigenschaften von UNIX-Freigaben (NFS)

Auf dieser Seite können Sie den Zugriff von NFS-Clients auf die einzelnen Freigaben festlegen. Der Zugriff kann basierend auf dem Clienthostnamen erteilt oder verweigert werden. Das Gewähren oder Verweigern des Zugriffs auf Grundlage von Clientgruppen ist ebenfalls möglich, wenn eine Clientgruppe mindestens einen Clienthostnamen enthält.

So fügen Sie einen neuen NFS-Client oder eine neue NFS-Clientgruppe zu einer Freigabe hinzu

  1. Erstellen Sie eine neue Clientgruppe gemäß den Anweisungen unter Hinzufügen von NFS-Clientgruppen.
  2. Führen Sie die im Folgenden beschriebenen Schritte aus, um einen vorhandenen Client bzw. eine bestehende Clientgruppe hinzuzufügen.

So fügen Sie einen vorhandenen Client oder eine bestehende Clientgruppe hinzu

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Freigaben auf der Seite Freigaben aus.
  3. Wählen Sie die Freigabe aus, der ein NFS-Client oder eine NFS-Clientgruppe hinzuzufügen ist.
  4. Wählen Sie in der Liste Tasks den Eintrag Eigenschaften aus.
  5. Wählen Sie die Registerkarte Allgemein aus.
  6. Aktivieren Sie das Kontrollkästchen UNIX (NFS).
  7. Wählen Sie die Registerkarte NFS-Freigabe aus.
  8. Wählen Sie den gewünschten Computer oder die betreffende Gruppe in der links angezeigten Liste aus, oder geben Sie den Namen eines NFS-Clientcomputers bzw. eine IP-Adresse im rechts angezeigten Feld ein, und klicken Sie dann auf Hinzufügen.
  9. Wählen Sie in der Liste Zugriffsart die Art des Zugriffs aus, der dem angegebenen Client für Dateien gewährt wird.
  10. Klicken Sie auf OK.

So entfernen Sie einen NFS-Client

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Freigaben auf der Seite Freigaben aus.
  3. Wählen Sie die Freigabe aus, für die ein NFS-Client oder eine NFS-Clientgruppe zu entfernen ist.
  4. Wählen Sie in der Liste Tasks den Eintrag Eigenschaften aus.
  5. Wählen Sie die Registerkarte Allgemein aus.
  6. Aktivieren Sie das Kontrollkästchen UNIX (NFS).
  7. Wählen Sie die Registerkarte NFS-Freigabe aus.
  8. Wählen Sie die gewünschte Client Appliance oder Clientgruppe in der Liste in der Seitenmitte aus, und klicken Sie dann auf Entfernen.
  9. Klicken Sie auf OK.
