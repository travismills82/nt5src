Vorheriges Thema

Konfigurieren expliziter Gruppenzuordnungen

Mit Benutzer- und Gruppenzuordnungen können Sie plattforminterne und -übergreifende Zuordnungen zwischen Benutzer- und Gruppenkonten aus Microsoft Windows und UNIX erstellen, auch wenn die Benutzer- und Gruppennamen in beiden Umgebungen nicht übereinstimmen.

Benutzer- und Gruppenzuordnungen ermöglicht ebenfalls das Einrichten von Zuordnungen im Verhältnis 1:1 zwischen Windows-Benutzern und UNIX-Benutzern und -Gruppen. Ein Windows-Benutzername könnte beispielsweise einem UNIX-Benutzernamen bzw. eine UNIX-Gruppe könnte mindestens einem Windows-Benutzerkonto zugeordnet werden. Die Verwendung expliziter Zuordnungen ist ebenfalls möglich, wenn ein Windows- und ein UNIX-Konto derselben Person verschiedene Benutzernamen aufweisen. Mit Hilfe der Option Explizite Gruppenzuordnungen können Sie eine einzelne Zuordnungsdatenbank für das gesamte Unternehmen verwalten.

So erstellen Sie explizite Gruppenzuordnungen

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Protokolle freigeben aus.
  3. Wählen Sie NFS-Protokoll und dann Eigenschaften aus.
  4. Wählen Sie Benutzer- und Gruppenzuordnungen aus.
  5. Nehmen Sie auf der Registerkarte Allgemein die erforderlichen Angaben vor. Stellen Sie sicher, dass Sie entweder eine NIS-Domäne oder eine PCNFS-passwd und Gruppendatei angeben. Weitere Informationen zu den Angaben auf dieser Registerkarte erhalten Sie unter Konfigurieren von Benutzer- und Gruppenzuordnungen.
  6. Wählen Sie die Registerkarte Explizite Gruppenzuordnungen aus.
  7. Optional: Geben Sie den Namen des zuzuordnenden NIS-Servers im Feld NIS-Domäne ein.
  8. Um die Liste UNIX-Gruppen mit Daten zu füllen, wählen Sie UNIX-Gruppen auflisten aus.
  9. Wählen Sie aus jeder Gruppe eine Gruppe aus, und klicken Sie dann auf Hinzufügen.

    Die zugeordneten Gruppen werden im Feld Explizit zugeordnete Gruppen angezeigt. Zum Zuordnen einer Domänengruppe müssen Sie DOMÄNE\Gruppenname eingeben, eine UNIX-Gruppe auswählen und auf die Schaltfläche Hinzufügen neben dem Eingabefeld für die DOMÄNE\Gruppe klicken.

    Anmerkung   Gruppen aus einer Windows-Domäne können mehreren UNIX-Domänen zugeordnet werden; die umgekehrte Vorgehensweise ist jedoch nicht möglich.

So legen Sie eine Zuordnung als primäre Zuordnung für eine bestimmte Gruppe fest

  1. Wählen Sie die Zuordnung in der Liste Explizit zugeordnete Gruppen aus.
  2. Wählen Sie Primär festlegen aus.
  3. Klicken Sie auf OK.

So löschen Sie explizite Gruppenzuordnungen

  1. Führen Sie Schritte 1 bis 4 der Vorgehensweise So erstellen Sie explizite Gruppenzuordnungen aus, um zur Seite Explizite Gruppenzuordnungen zu navigieren.
  2. Wählen Sie in der Liste Explizit zugeordnete Gruppen die zu löschende Gruppenzuordnung aus.
  3. Klicken Sie auf Entfernen.
  4. Klicken Sie auf OK.
