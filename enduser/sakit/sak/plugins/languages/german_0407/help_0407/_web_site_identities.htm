Ändern von Siteidentitäten

Auf der Seite Siteidentitäten können Sie die IP-Adresse und die TCP-Portummer der Serverappliance zuweisen. Außerdem können Sie auf der Website, die Sie erstellen oder ändern, einen Hostheadernamen hinzufügen oder ändern.

Die IP-Adresse wird standardmäßig auf Alle nicht zugeordnet gesetzt. Die Liste enthält die IP-Adressen aller Netzwerkkarten (NIC), die auf der Serverappliance installiert sind.

Der TCP-Port wird standardmäßig auf 80 gesetzt. Diese Standardportnummer ist der Industriestandard und wird verwendet, wenn Sie eine Webadresse in einen Browser eingeben, z. B. www.ihresite.de. Wenn Sie für die Portnummer einen anderen Wert als 80 verwenden, beispielsweise. 8055, müssen die Benutzer zur Eingabe der Websiteadresse einen Doppelpunkt und die Portnummer eingeben, z. B. www.ihresite.de:8055.

So ändern Sie die Parameter für Siteidentitäten

  1. Wählen Sie auf der primären Navigationsleiste Sites aus.
  2. Wählen Sie die Website aus, die Sie ändern möchten.
  3. Wählen Sie in der Liste Tasks den Eintrag Bearbeiten aus.
  4. Wählen Sie die Registerkarte Siteidentitäten aus.
  5. Wählen Sie die der Website zuzuweisende IP-Adresse in der Liste IP-Adresse aus.
  6. Geben Sie die gewünschte TCP-Portnummer in das Feld TCP-Port ein.
  7. Geben Sie den Hostheadernamen in das Feld Hostheader ein.
  8. Klicken Sie auf OK, oder wählen Sie eine Registerkarte aus, um weitere Änderungen an den Websiteeigenschaften vorzunehmen.
