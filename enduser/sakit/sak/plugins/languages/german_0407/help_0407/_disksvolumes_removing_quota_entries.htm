Vorheriges Thema

Entfernen von Kontingenteinträgen

So entfernen Sie einen Kontingenteintrag

  1. Wählen Sie auf der primären Navigationsleiste Datenträger aus.
  2. Wählen Sie Datenträgerkontingent aus.
  3. Wählen Sie das zu verwaltende Volume aus.
  4. Wählen Sie in der Liste Tasks den Eintrag Kontingenteinträge aus.
  5. Wählen Sie auf der Seite Kontingenteinträge den Anmeldenamen aus, für den der Kontingenteintrag gelöscht werden soll.
  6. Wählen Sie in der Liste Tasks den Eintrag Löschen aus.
  7. Klicken Sie auf OK.
