Hinzufügen eines Ordners

Sie können beliebig viele Ordner auf der Serverappliance erstellen.

So erstellen Sie einen neuen Ordner

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Ordner aus.
  3. Wählen Sie das Volume aus, mit dem Sie arbeiten möchten, und wählen Sie dann Öffnen aus.
  4. Wählen Sie den Ordner aus, dem ein Unterordner hinzugefügt werden soll.
  5. Wählen Sie in der Liste Tasks den Eintrag Neu aus.
  6. Geben Sie auf der Registerkarte Allgemein im Feld Name den Namen des neuen Ordners ein.
  7. Legen Sie die Zugriffsattribute durch Auswahl des entsprechenden Kontrollkästchens fest.
  8. Optional: Wenn Sie den Inhalt des neuen Ordners komprimieren möchten, um Speicherplatz zu sparen, wählen Sie die Registerkarte Komprimieren aus.
  9. Klicken Sie auf OK.

    Die Objekt-/Taskauswahl enthält jetzt den hinzugefügten Ordner. Wenn Sie den neuen Ordner in der Tabelle nicht unmittelbar sehen, führen Sie einen Bildlauf in der Liste durch.
