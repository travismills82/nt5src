Vorheriges Thema

Datenträgerkontingente

Mithilfe der Datenträgerkontingente wird die Speichernutzung auf Volumes verfolgt und gesteuert. Sie können die Volumes für die Serverappliance zur Durchführung folgender Tasks konfigurieren:

  • Unterbinden einer weiteren Speicherbelegung und Protokollieren eines Ereignisses, wenn ein Benutzer das festgelegte Kontingentlimit überschreitet.
  • Protokollieren eines Ereignisses, wenn ein Benutzer die Warnstufe für den belegten Speicherplatz überschreitet.

Wenn Sie Datenträgerkontingente aktivieren, können Sie sowohl das Limit als auch die Warnstufe eines Datenträgerkontingents festlegen. Mit dem Datenträgerkontingentlimit beschränken Sie den Speicherplatz, den ein Benutzer maximal belegen darf. Die Warnstufe bezeichnet den Punkt, an dem ein Benutzer sein Datenträgerkontingent nahezu vollständig genutzt hat. Setzen Sie beispielsweise das Limit für das Datenträgerkontingent auf 50 Megabytes (MB) und die Warnstufe auf 45 MB. In diesem Fall kann der Benutzer nicht mehr als 50 MB Speicherplatz auf dem Volume belegen. Sobald der Benutzer mehr als 45 MB auf dem Volume speichert, wird ein Systemereignis durch das Datenträgerkontingentsystem protokolliert.

Ferner können Sie ebenfalls angeben, dass Benutzern die Überschreitung des Kontingentlimits ermöglicht wird. Aktivieren Sie Datenträgerkontingente, ohne eine Beschränkung für den Speicherplatz anzugeben, wenn Sie den Benutzern nicht den Zugriff auf den betreffenden Datenträger verweigern und dennoch die Speichernutzung für die einzelnen Benutzer verfolgen möchten. Darüber hinaus können Sie festlegen, ob ein Ereignis protokolliert werden soll, wenn ein Benutzer die Warnstufe oder das Limit für das Datenträgerkontingent überschreitet.

Wenn Sie Datenträgerkontingente für ein Volume aktivieren, wird die Volumenutzung von diesem Zeitpunkt an automatisch verfolgt, auf vorhandene Benutzer des Volumes werden jedoch keine Datenträgerkontingente angewendet. Um Datenträgerkontingente für vorhandene Benutzer des Volumes festzulegen, fügen Sie neue Einträge auf der Seite Kontingenteinträge hinzu.
