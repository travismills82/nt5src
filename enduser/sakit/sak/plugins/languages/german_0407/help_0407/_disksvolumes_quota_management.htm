Aktivieren der Kontingentverwaltung

Wenn Sie Datenträgerkontingente auf einem Volume aktivieren, wird die Datenträgernutzung der einzelnen Benutzer überwacht und in Abhängigkeit von den Kontingentverwaltungseinstellungen des jeweiligen Benutzers unterschiedlich gehandhabt. Beispielsweise können Benutzer, die über Schreibzugriff auf das Volume verfügen und ihr Kontingentlimit nicht überschritten haben, Daten auf dem Volume speichern. Beim ersten Speichern von Daten auf einem Volume mit aktivierten Kontingenten werden einem Benutzer automatisch die Standardwerte für die Kontingentsgrenze und die Warnstufe zugewiesen.

So aktivieren oder deaktivieren Sie die Kontingentverwaltung für ein Volume

  1. Wählen Sie auf der primären Navigationsleiste Datenträger aus.
  2. Wählen Sie Datenträgerkontingent aus.
  3. Wählen Sie das zu verwaltende Volume aus.
  4. Wählen Sie in der Liste Tasks den Eintrag Kontingente aus.
  5. Aktivieren oder deaktivieren Sie auf der Seite Standardkontingent für (Volume) das Kontrollkästchen Kontingentverwaltung aktivieren.
  6. Wählen Sie das standardmäßige Kontingentlimit für neue Benutzer auf diesem Volume aus.
  7. Geben Sie die zu protokollierenden Kontingentereignisse an.
  8. Klicken Sie auf OK.
