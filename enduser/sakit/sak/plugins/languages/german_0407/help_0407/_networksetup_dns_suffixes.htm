DNS-Suffixe

Im Folgenden werden die Hauptbedeutungen von DNS-Suffixen (Domain Name System) aufgeführt:

  • An den Hostnamen der Serverappliance angehängt stellen DNS-Suffixe den vollqualifizierten Serverappliancenamen dar.
  • DNS-Suffixe werden zur Auflösung von IP-Adressen verwendet. Wenn die Serverappliance Mitglied einer Windows NT 4-Domäne, einer Arbeitsgruppe oder von Microsoft Active Directory ist, richtet sich das DNS-Suffix nach der Domänenumgebung.

Ein DNS-Suffix kann bis zu 155 Zeichen umfassen (einschließlich Bindestrichen und Punkten).
