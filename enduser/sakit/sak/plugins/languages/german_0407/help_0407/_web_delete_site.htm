Vorheriges Thema

Löschen einer Website

Sie können jede auf der Serverappliance erstellte Website entfernen.

So löschen Sie eine Website

  1. Wählen Sie auf der primären Navigationsleiste Sites aus.
  2. Wählen Sie die Website aus, die Sie entfernen möchten.
  3. Wählen Sie in der Liste Tasks den Eintrag Löschen aus.
  4. Überprüfen Sie, ob Sie die angezeigte Website entfernen möchten.
  5. Klicken Sie auf OK.
