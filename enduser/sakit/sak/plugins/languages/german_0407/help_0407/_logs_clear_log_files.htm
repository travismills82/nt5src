Vorheriges Thema

Löschen von Protokolldateien

Auf dieser Seite können Sie spezifische Protokolldateien löschen. Beim Löschen von NFS-Protokollen, Anwendungs-, Sicherheits- oder Systemprotokollen wird der gesamte Protokollinhalt entfernt. Wenn Sie hingegen FTP-Protokolle, Webverwaltungs- oder Webfreigabenprotokolle (HTTP) löschen, können Sie spezifische Protokolldateien für den Löschvorgang auswählen.

So löschen Sie NFS-Protokolle, Anwendungs-, Sicherheits- oder Systemprotokolle

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie Protokolle aus.
  3. Wählen Sie den Typ des zu löschenden Protokolls aus.
  4. Wählen Sie Protokoll löschen aus.
  5. Klicken Sie auf OK.

So löschen Sie FTP-Protokolle, Webverwaltungs- oder Webfreigabenprotokolle (HTTP)

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie Protokolle aus.
  3. Wählen Sie den Typ des zu löschenden Protokolls aus.
  4. Wählen Sie mindestens eine zu löschende Protokolldatei aus.
  5. Wählen Sie Löschen aus.
  6. Klicken Sie auf OK.
