Vorheriges Thema

Anhalten einer Website

Die Objekt-/Taskauswahl auf der Seite Websitekonfiguration gibt den Status aller Websites an, die auf der Serverappliance verfügbar sind. Falls erforderlich, können Sie jede Website anhalten, die ausgeführt wird.

Sobald Sie eine Website anhalten, akzeptieren Internetdienste keine neuen Verbindungen mehr; Anfragen, die sich bereits in der Verarbeitung befinden, werden jeoch nicht beeinflusst. Wenn Sie eine Website beenden, akzeptieren Internetdienste keine neuen Verbindungen mehr, und die Verarbeitung aktiver Anfragen wird ebenfalls beendet.

So halten Sie eine ausgeführte Website an

  1. Wählen Sie auf der primären Navigationsleiste Sites aus.
  2. Wählen Sie die Websitekennung der Site aus, die Sie anhalten möchten.
  3. Wählen Sie in der Liste Tasks den Eintrag Anhalten aus.
