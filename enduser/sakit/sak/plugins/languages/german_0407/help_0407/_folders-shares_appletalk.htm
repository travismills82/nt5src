Vorheriges Thema

AppleTalk

Die AppleTalk-Netzwerkintegration von Microsoft Windows 2000 Server ermöglicht die Freigabe von Dateien und Drucker zwischen der Serverappliance und sämtlichen mit dem Netzwerk verbundenen Apple Macintosh-Clients.

Wird die AppleTalk-Netzwerkintegration verwendet, muss auf Macintosh-Computern nur das Macintosh-Betriebssystem installiert sein, damit sie als Clients arbeiten können. Zusätzliche Software ist nicht erforderlich.

Die AppleTalk-Netzwerkintegration erleichtert die Verwaltung, da anstelle einzelner Benutzerkonten nur eine Gruppe von Benutzerkonten verwaltet werden muss, beispielsweise eine Gruppe auf dem Macintosh-Server und eine weitere auf dem Windows 2000 Server-Computer.
