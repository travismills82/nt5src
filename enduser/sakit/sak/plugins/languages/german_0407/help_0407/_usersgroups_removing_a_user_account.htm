Vorheriges Thema

Entfernen eines Benutzerkontos

Sie können bis auf das letzte verbleibende Konto und Ihr eigenes Konto alle Benutzerkonten entfernen, die auf der Serverappliance unter Microsoft Windows von Ihnen erstellt wurden. Ferner besteht die Möglichkeit, mehrere Benutzerkonten gleichzeitig zu entfernen. Wenn Sie jedoch das einzige Benutzerkonto auf der Serverappliance entfernen, wird die Sicherheit deaktiviert.

Beim Entfernen von Benutzerkonten sollten Sie folgende Richtlinien berücksichtigen:

  • Integrierte Benutzer können nicht gelöscht werden.
  • Ein gelöschter Benutzer kann nicht wiederhergestellt werden.

    Wenn Sie ein Benutzerkonto löschen und anschließend ein anderes Benutzerkonto mit demselben Benutzernamen erstellen, müssen dem neuen Benutzer neue Berechtigungen zugewiesen werden. Der neue Benutzer übernimmt die Berechtigungen des alten Benutzers nicht.

So entfernen Sie Benutzerkonten

  1. Wählen Sie auf der primären Navigationsleiste Benutzer aus.
  2. Wählen Sie Lokale Benutzer aus.
  3. Wählen Sie die zu entfernenden Benutzerkonten aus.
  4. Wählen Sie in der Liste Tasks den Eintrag Löschen aus.
  5. Stellen Sie sicher, dass Sie die angegebenen Benutzerkonten löschen möchten, und klicken Sie dann auf OK.
