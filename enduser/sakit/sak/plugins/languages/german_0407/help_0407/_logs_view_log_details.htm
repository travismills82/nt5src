Vorheriges Thema

Anzeigen von Protokolldetails

Es besteht die Möglichkeit, die Details der Protokolleinträge aus Anwendungs-, System- und Sicherheitsprotokollen anzuzeigen. Auf dieser Seite können Sie das Datum, die Uhrzeit, die Quelle, die Ereignis-ID, die Beschreibung und die Daten bestimmter Protokolldateien anzeigen.

So zeigen Sie die Details einer Protokolldatei an

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie Protokolle aus.
  3. Wählen Sie den Typ des anzuzeigenden Protokolls aus.
  4. Wählen Sie den anzuzeigenden Protokolleintrag aus.
  5. Wählen Sie Details aus.
  6. Führen Sie mit Hilfe von Nach unten und Nach oben in den Protokolldateien einen Bildlauf durch.
  7. Wählen Sie Zurück aus, um zur Objekt-/Taskauswahl für den angegebenen Protokolltyp zurückzukehren.
