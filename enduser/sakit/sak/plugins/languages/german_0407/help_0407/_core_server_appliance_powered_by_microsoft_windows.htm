Server Appliance Powered by Microsoft Windows

Willkommen in der Onlinehilfe zur Server Appliance Powered by Microsoft® Windows®. Diese Onlinehilfe enthält Informationen zur Konfiguration der Serverappliance vor der erstmaligen Verwendung, zur Navigation innerhalb der zugehörigen webbasierten Benutzeroberfläche sowie zum Einsatz der Serverappliance in Ihrem Unternehmen. Darüber hinaus kann auf jedem Bildschirm der Webbenutzeroberfläche die kontextabhängige Hilfe aufgerufen werden. Klicken Sie dazu auf das Fragezeichen (?) ganz rechts auf der primären Navigationsleiste.

Verwandte Themen

  • Erste Schritte
  • Navigationsmodell der Webbenutzeroberfläche
  • Verwenden der Hilfe
  • Einstellen der Standardseite
