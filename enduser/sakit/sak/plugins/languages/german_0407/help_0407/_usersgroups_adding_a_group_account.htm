Vorheriges Thema

Hinzufügen eines Gruppenkontos

So fügen Sie ein Gruppenkonto hinzu

  1. Wählen Sie auf der primären Navigationsleiste Benutzer aus.
  2. Wählen Sie dann Lokale Gruppen aus.
  3. Wählen Sie in der Liste Tasks den Eintrag Neu aus.
  4. Geben Sie auf der Registerkarte Allgemein den Namen und die Beschreibung der hinzuzufügenden Gruppe ein.
  5. Wählen Sie die Registerkarte Mitglieder aus.
    • Um der Gruppe Mitglieder hinzuzufügen, wählen Sie einen Benutzer oder eine Gruppe im Feld Benutzer oder Gruppe hinzufügen aus, und klicken Sie auf Hinzufügen.

      Anmerkung   Die Liste enthält nur lokale Benutzer. Zur Angabe eines Domänenbenutzerkontos geben Sie die Domäne und den Benutzernamen ein (<Domäne\Benutzername>).

    • Um Mitglieder aus der Gruppe zu entfernen, wählen Sie im Feld Mitglieder ein Mitglied oder eine Gruppe aus, und klicken Sie dann auf Löschen.
  6. Klicken Sie auf OK.
