Vorheriges Thema

Protokolle freigeben

Auf der Seite Protokolle freigeben können Sie relevante Netzwerkprotokolle aktivieren, deaktivieren, beenden oder konfigurieren. Die Seite enthält die Objekt-/Taskauswahl mit den folgenden Spalten:

  • Name - Zeigt jedes Protokoll mit Namen an. Zum Aktivieren, Deaktivieren oder Ändern der Eigenschaften eines bestimmten Protokolls klicken Sie auf die Schaltfläche neben dem zu ändernden Protokoll.
  • Status - Zeigt an, dass das Protokoll ausgeführt, beendet oder angehalten wird.
  • Starttyp - Gibt an, ob das Protokoll beim Start der Serverappliance automatisch aufgerufen, manuell gestartet oder deaktiviert wird.
  • Beschreibung - Zeigt eine kurze Beschreibung des Protokolls an.

    Die Liste Tasks befindet sich neben der Objekt-/Taskauswahl. Verwenden Sie die Spalte Name der Objekt-/Taskauswahl, um ein Protokoll auszuwählen. um einen Task durchzuführen, wählen Sie den entsprechenden Task in der Liste Tasks aus.
