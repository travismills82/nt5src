Festelegen der Standardseite

Sie können die Standardseite der Serverappliance festlegen. Diese Seite wird bei jedem Zugriff auf die Serverappliance angezeigt. Als Standardseite der Serverappliance kann die Willkommen- oder Statusseite festgelegt werden.

So legen Sie die Standardseite der Serverappliance fest

  1. Wählen Sie auf der primären Navigationsleiste Willkommen aus.
  2. Klicken Sie auf der Willkommenseite auf Standardseite festlegen.
  3. Legen Sie unter Standardseite festlegen die Standardseite fest.
  4. Klicken Sie auf OK.
