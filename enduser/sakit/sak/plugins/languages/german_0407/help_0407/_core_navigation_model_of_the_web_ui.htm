Vorheriges Thema

Navigationsmodell der Webbenutzeroberfläche

Oben auf den Seiten der Webbenutzeroberfläche befinden sich ein Statusbereich sowie die primäre und die sekundäre Navigationsleiste. Der Hauptteil jeder Seite besteht aus dem Inhaltsbereich.

Im Folgenden erhalten Sie eine Beschreibung der einzelnen Bereiche:

Statusbereich

Als oberstes Band des Fensters enthält der Statusbereich von links nach rechts gesehen folgende Elemente:

  • Logo des Herstellers der Serverappliance unter Windows (optional)
  • Name und Status des Serverappliancehosts
  • Logo für Powered by Microsoft Windows

Folgende Status können angezeigt werden:

  • Normal (grüner Text)
  • Information (grauer Text)
  • Warnung (gelber Text)
  • Kritisch (roter Text)

Sie können auf Status: <Statustyp> klicken, um detaillierte Informationen zum Status der Serverappliance anzuzeigen.

Navigationsleisten

Unmittelbar unter dem Statusbereich befinden sich die Navigationsleisten. In der folgenden Abbildung sind die primäre (blau) und sekundäre (weiß) Navigationsleiste einer standardmäßigen Web Server Appliance dargestellt.

In der primären Navigationsleiste werden die verfügbaren Serverappliancetasks nach Typ angezeigt. Die sekundäre Navigationsleiste listet die untergeordneten Tasks auf, die für den ausgewählten primären Task zur Verfügung stehen. Diese Navigationsleiste wird dynamisch aufgebaut, und die verfügbaren Tasktypen richten sich nach der ausgewählten Taskgruppe.

Der Rollovertext für Elemente in den Navigationsleisten enthält weitere Informationen. Bewegen Sie den Cursor über die einzelnen Elemente, um den Rollovertext anzuzeigen.

Inhaltsbereich

Die Informationen der einzelnen Seiten werden im Inhaltsbereich unter den Menüleisten angezeigt. In diesem Abschnitt der Webbenutzeroberfläche werden die Verwaltungsaktivitäten beschrieben, die auf der jeweiligen Seite durchgeführt werden können. Dieser Text enthält unter Umständen ebenfalls Anweisungen zu Tools, die für die Ausführung der ausgewählten Tasks verfügbar sind.

Ein Großteil der Taskseiten enthält eine Objekt-/Taskauswahl. Dabei handelt es sich um eine Tabelle mit den zu verwaltenden oder zu konfigurierenden Objekten, deren Beschreibungen sowie den durchführbaren Tasks. Die erste Spalte der Objekt-/Taskauswahl enthält ein Optionsfeld, auf das Sie zur Auswahl eines bestimmten Objekts klicken. Falls mehrere Objekte in der Objekt-/Taskauswahl ausgewählt werden können, enthält die erste Spalte anstelle des Optionsfeldes ein Kontrollkästchen. In der ganz rechten Spalte werden die durchführbaren Tasks aufgeführt.

So navigieren Sie durch die Webbenutzeroberfläche

  1. Wählen Sie auf der primären Navigationsleiste den allgemeinen Typ des durchzuführenden Tasks aus.
  2. Wählen Sie auf der sekundären Navigationsleiste oder in der Taskliste im Inhaltsbereich der Seite den spezifischen Typ des durchzuführenden Tasks aus.
  3. Gehen Sie im Inhaltsbereich der nachfolgenden Seite folgendermaßen vor:
    1. Wenn eine Objekt-/Taskauswahl zur Verfügung steht, wählen Sie das zu verwaltende oder zu konfigurierende Objekt durch Klicken auf das Optionsfeld links neben dem Objektnamen aus. Wählen Sie dann den durchzuführenden Task rechts in der Liste Tasks aus. In Abhängigkeit von der angezeigten Seite stellt die Objekt-/Taskauswahl Suchfunktionen sowie eine Funktion zum Anzeigen der Tabelleninformationen bereit.
    2. Steht keine Objekt-/Taskauswahl zur Verfügung, geben Sie die zur Ausführung des ausgewählten Tasks erforderlichen Daten in den Feldern ein.
  4. Klicken Sie nach Beendigung der einzelnen Tasks auf OK, um die Änderungen zu bestätigen, oder auf Abbrechen, um die vorherigen Einstellungen beizubehalten.
Nach Verarbeitung der Änderungen oder des Abbruchs wird die vorherige Seite angezeigt.

Wenn Sie sich auf einer Eigenschaftenseite befinden und ein anderes Navigationsleistenelement auswählen, wird ein Popupfenster mit der Meldung Klicken Sie auf "OK", um Änderungen zu verwerfen angezeigt. Auf diese Weise erhalten Sie die Möglichkeit, die Änderungen zu übergeben oder abzulehnen, bevor Sie zur nächsten ausgewählten Seite wechseln.
