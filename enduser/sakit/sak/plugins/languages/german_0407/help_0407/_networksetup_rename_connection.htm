Vorheriges Thema

Umbenennen einer Verbindung

So benennen Sie eine Schnittstellenverbindung um

  1. Wählen Sie auf der primären Navigationsleiste Netzwerk aus.
  2. Wählen Sie Schnittstellen aus.
  3. Wählen Sie die Schnittstellenverbindung aus, die umzubenennen ist.
  4. Wählen Sie in der Liste Tasks den Eintrag Umbenennen aus.
  5. Geben Sie im Feld Verbindungsname den neuen Namen der Schnittstellenverbindung ein.
  6. Klicken Sie auf OK.
