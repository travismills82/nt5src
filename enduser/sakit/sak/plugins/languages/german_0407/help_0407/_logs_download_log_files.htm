Vorheriges Thema

Downloaden von Protokolldateien

Auf dieser Seite können Sie bestimmte Protokolldateien auf die Serverappliance downloaden.

So downloaden Sie Anwendungs-, Sicherheits- oder Systemprotokolle

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie Protokolle aus.
  3. Wählen Sie den Typ des Protokolls aus, das Sie downloaden möchten.
  4. Wählen Sie Protokoll herunterladen aus.
  5. Wählen Sie Datei auf Datenträger speichern aus.
  6. Klicken Sie auf OK, um die Datei zu downloaden.

So downloaden Sie NFS-Protokolle

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie Protokolle aus.
  3. Wählen Sie NFS-Protokolle aus.
  4. Wählen Sie Protokoll herunterladen aus.
  5. Wählen Sie Datei auf Datenträger speichern aus.
  6. Klicken Sie auf OK.

So downloaden Sie FTP-Protokolle, Webverwaltungs- oder Webfreigabenprotokolle (HTTP)

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie Protokolle aus.
  3. Wählen Sie das Protokoll aus, das Sie downloaden möchten.
  4. Wählen Sie die Protokolldatei aus, die Sie downloaden möchten.
  5. Wählen Sie Protokoll herunterladen aus.
  6. Wählen Sie Datei auf Datenträger speichern aus.
  7. Klicken Sie auf OK.
