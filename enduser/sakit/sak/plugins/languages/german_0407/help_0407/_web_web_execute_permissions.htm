Festlegen von Webausführungsberechtigungen

Auf dieser Seite können Sie die Standardberechtigungen für die Sites auf der Serverappliance festlegen. Dabei kann angegeben werden, ob Skripts und ausführbare Dateien, nur Skripts oder weder Skripts noch ausführbare Dateien ausgeführt werden dürfen.

So legen Sie die Webausführungsberechtigungen fest

  1. Wählen Sie auf der primären Navigationsleiste Webserver aus.
  2. Wählen Sie Webausführungsberechtigungen aus.
  3. Wählen Sie die gewünschte Standardberechtigung in der Liste Standardausführungsberechtigungen aus.
  4. Wählen Sie die Option aus, um diese Einstellung auf alle Websites anzuwenden, die den Standardwert verwenden.
  5. Klicken Sie auf OK.
