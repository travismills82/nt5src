Aktivieren des anonymen FTP-Zugriffs

So aktivieren Sie den anonymen FTP-Zugriff

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Protokolle freigeben aus.
  3. Wählen Sie FTP-Protokoll und dann Eigenschaften aus.
  4. Wählen Sie die Registerkarte Anonymer Zugriff aus.
  5. Aktivieren Sie das Kontrollkästchen Anonymen Zugriff freigeben, und klicken Sie auf OK.
