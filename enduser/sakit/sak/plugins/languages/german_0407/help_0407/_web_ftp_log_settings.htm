Konfigurieren der FTP-Protokolleinstellungen

Über diese Seite können Sie die FTP-Protokolleinstellungen konfigurieren.

So konfigurieren Sie die FTP-Protokolleinstellungen

  1. Klicken Sie in der primären Navigationsleiste auf Webserver.
  2. Wählen Sie FTP-Protokolleinstellungen aus.
  3. Aktivieren Sie das Kontrollkästchen Protokollierung aktivieren.
  4. Wählen Sie das entsprechende Protokolldateiformat in dem Feld Aktives Protokollformat aus.
  5. Wählen Sie die entsprechende Option aus, um das Zeitintervall festzulegen, nach dessen Ablauf ein neues Protokoll generiert wird.

    Wenn Sie beispielsweise Täglich auswählen, wird alle 24 Stunden mit der Erstellung eines neuen Protokolls begonnen.

  6. Aktivieren Sie das Kontrollkästchen Verwenden Sie für das Benennen von Dateien und das Rollover die Ortszeit des Servers .

    Die Serverappliance verwendet die lokale Zeit für Dateibenennung und Protokollrollover. Andernfalls verwendet der Server Greenwich Mean Time (GMT).

    Anmerkung   Wenn Sie Wenn die Dateigröße folgenden Wert in MB erreicht auswählen, wird das Kontrollkästchen Verwenden Sie die Ortszeit für das Benennen von Dateien und das Rollover deaktiviert.

  7. Überprüfen Sie, ob das Feld Protokolldateiverzeichnis den richtigen Verzeichnispfad für Ihre Protokolldateien enthält.
  8. Klicken Sie auf OK.
