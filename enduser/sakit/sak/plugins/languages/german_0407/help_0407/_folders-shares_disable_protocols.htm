Deaktivieren von Protokollen

So deaktivieren Sie Protokollfreigaben

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Protokolle freigeben aus.
  3. Markieren Sie das zu deaktivierende Protokoll.
  4. Wählen Sie in der Liste Tasks den Eintrag Deaktivieren aus.
