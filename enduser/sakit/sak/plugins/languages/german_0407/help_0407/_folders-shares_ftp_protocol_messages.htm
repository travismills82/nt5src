Vorheriges Thema

Hinzufügen benutzerdefinierter FTP-Nachrichten

Sie können benutzerdefinierte Willkommens- und Beendigungsnachrichten erstellen, die beim Herstellen oder Trennen von Verbindungen zur Serverappliance an Benutzer ausgegeben werden.

So fügen Sie benutzerdefinierte Nachrichten hinzu

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Protokolle freigeben aus.
  3. Wählen Sie FTP-Protokoll und dann Eigenschaften aus.
  4. Wählen Sie die Registerkarte Meldungen aus.
  5. Geben Sie im Feld Willkommensnachricht den Text der Nachricht ein, mit der die Benutzer beim Erstellen der Serverappliance-Verbindung begrüßt werden sollen.
  6. Geben Sie im Feld Beendigungsnachricht den Text der Nachricht ein, die beim Trennen der Serverappliance-Verbindung anzuzeigen ist.
  7. Klicken Sie auf OK.
