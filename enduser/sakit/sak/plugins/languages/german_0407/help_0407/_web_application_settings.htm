Vorheriges Thema

Festlegen von allgemeinen Anwendungseinstellungen

Auf der Seite Anwendungseinstellung können Sie folgende Tasks durchführen:

  • Festlegen von Berechtigungen
  • Angeben der Standardseite
  • Aktivieren und Deaktivieren der Verwendung von Erweiterungen für Microsoft® FrontPage® (standardmäßig deaktiviert)
  • Erstellen einer FTP-Website für die von der Serverappliance verwalteten Websites

So legen Sie Anwendungseinstellungen fest oder ändern sie

  1. Wählen Sie auf der primären Navigationsleiste Sites aus.
  2. Wählen Sie die Website aus, die Sie ändern möchten.
  3. Wählen Sie in der Liste Tasks den Eintrag Ändern aus.
  4. Wählen Sie die Registerkarte Anwendungseinstellungen aus.
  5. Verwenden Sie die Liste Standardausführungsberechtigungen, um eine der folgenden Skriptausführungsebenen auszuwählen:
    • Keine - Keine Skripts, z. B. ASP-Anwendungen (Active Server Pages) oder ausführbare Dateien, auf dem Server ausführen.
    • Nur Skripts - Nur Skripts, z. B. ASP-Anwendungen, auf dem Server ausführen.
    • Skripts und ausführbare Dateien - Sowohl Skripts, z. B. ASP-Anwendungen, und ausführbare Dateien auf dem Server ausführen.
  6. Verwenden Sie die Liste Standardseite zur Auswahl der entsprechenden Standardwebseite.
    • Um eine Seite zu der Liste der Standardseiten hinzuzufügen, geben Sie den Dateinamen der hinzuzufügenden Seite im Feld Dateiname ein, und wählen Sie dann Hinzufügen aus.
    • Um eine Seite aus der Liste der Standardseiten zu entfernen, wählen Sie den Dateinamen der zu entfernenden Seite aus, und klicken Sie dann auf Enfernen.
  7. Erweiterungen für Microsoft FrontPage ermöglichen Administratoren das Anzeigen und Verwalten von Websites über eine grafische Benutzeroberfläche. Wenn Sie Websites auf der Serverappliance über FrontPage anzeigen und verwalten möchten, aktivieren Sie das Kontrollkästchen Microsoft FrontPage-Erweiterungen zulassen.
  8. Aktivieren Sie das Kontrollkästchen FTP-Website zum Bearbeiten des Inhalts erstellen, wenn Sie Änderungen von Dateien auf einer Website durch Benutzer zulassen wollen, die über das Administratorkonto eine FTP-Verbindung zur Site herstellen.
  9. Aktivieren Sie das Kontrollkästchen Anonymen Websitezugriff zulassen, um Remotebenutzern den Zugriff auf die Serverappliance ohne Angabe eines Benutzernamen oder Kennworts zu ermöglichen. Benutzer, die sich anonym anmelden, verfügen nur über die dem Gastkonto zugewiesenen Berechtigungen. Der anonyme Zugriff wird in der Regel für Websites im Internet verwendet.
