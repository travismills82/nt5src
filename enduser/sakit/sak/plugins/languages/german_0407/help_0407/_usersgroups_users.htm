Vorheriges Thema

Benutzer

Auf dieser Seite können Sie lokale Benutzer und Gruppen der Serverappliance erstellen, bearbeiten und löschen. Sie können auch die Mitglieder jeder Gruppe ändern. Wenn sich die Serverappliance in einer Domäne befindet, sollten Sie auf der Serverappliance selbst keine Benutzer erstellen. Der Hauptzweck dieser Seite liegt darin, der lokalen Gruppe ein oder mehrere Domänenmitglieder hinzuzufügen.

Sie sollten außerdem Domänenbenutzer- und -gruppenkonten verwenden, um den Zugriff auf die Ressourcen der Serverappliance zu überwachen. Sie sollten des weiteren Domänenverwaltungstools verwenden, um Domänenbenutzer und -gruppen zu verwalten.

Verwandte Themen

  • Informationen zum Verwalten von lokalen Benutzern erhalten Sie unter Lokale Benutzer verwalten. Weitere Informationen zum Verwalten von lokalen Gruppen erhalten Sie unter Lokale Gruppen verwalten.
