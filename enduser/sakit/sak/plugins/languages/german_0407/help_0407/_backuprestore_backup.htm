Vorheriges Thema

Sichern und Wiederherstellen des Betriebssystems

Diese Seite enthält Anweisungen zum Sichern oder Wiederherstellen des Betriebssystems der Serverappliance.

So sichern Sie das Betriebssystem der Serverappliance oder stellen es wieder her

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie dann Sicherung aus.
  3. Melden Sie sich bei Terminaldienste Advanced Client an. Die Sicherungsanwendung wird automatisch gestartet.
  4. Sobald die Sicherung abgeschlossen ist, schließen Sie die Anwendung, indem Sie auf das X in der oberen rechten Ecke klicken. Sie werden dann bei Terminaldienste Advanced Client abgemeldet.

    Anmerkung   Die Abmeldung der Sitzung beim Schließen der Anwendung nimmt einige Zeit in Anspruch.

Terminaldiensteclient

Bei dem Terminaldiensteclient handelt es sich um das Tool, das zum Sichern und Wiederherstellen des Betriebssystems der Serverappliance verwendet wird. Lediglich zwei gleichzeitige Verbindungen werden unterstützt. Wenn Sie während einer bestehenden Sitzung zu einer anderen Seite navigieren, wird die Verbindung zum Client getrennt, die Sitzung jedoch beibehalten. So wird der Zugriff auf eine Terminaldienstesitzung durch andere Benutzer vermieden.
