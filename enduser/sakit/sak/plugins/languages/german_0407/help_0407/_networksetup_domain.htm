Vorheriges Thema

Domäne

Die Serverappliance kann zu einer Arbeitsgruppe, einer Active Directory-Umgebung oder einer Windows NT 4-Domäne gehören. In Microsoft Windows NT 4- und Microsoft Active Directory-Umgebungen stellt eine Domäne eine vom Netzwerkadministrator definierte Ansammlung von Computern dar, die eine gemeinsame Verzeichnisdatenbank nutzen.

Windows-Domänen werden für Informationen von Windows-Benutzern und -Gruppen verwendet, sie weisen daher einen eindeutigen Namen auf und stellen Zugriff auf zentralisierte Benutzer- und Gruppenkonten bereit, die vom Systemadministrator verwaltet werden. Jede Domäne verfügt über eigene Sicherheitsrichtlinien und -stellungen zu anderen Domänen, und jede Domäne stellt einen geschlossenen Sicherheitsbereich in einem Windows 2000-Computernetzwerk dar. Active Directory setzt sich aus mindestens einer Domäne zusammen, von denen sich jede über mehrere physische Standort erstrecken kann.

Im DNS stellt eine Domäne eine beliebige Struktur oder Teilstruktur innerhalb des DNS-Namespace dar. Obwohl die Namen von DNS-Domänen häufig mit denen von Active Directory-Domänen übereinstimmen, dürfen DNS-Domänen nicht mit den Netzwerkdomänen von Microsoft Windows und Active Directory verwechselt werden.

Standardmäßig muss ein Domänenbenutzer Domäne\Benutzername eingeben, wenn er sich an einer Web Serverappliance bzw. NAS mit Hilfe eines Netscape-Browsers anmeldet.

So richten Sie die zur Anmeldung verwendete Standarddomäne ein

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie Terminaldienste aus.
  3. Melden Sie sich an Terminaldienste an.
  4. Klicken Sie mit der rechten Maustaste auf Arbeitsplatz, und klicken Sie dann auf Verwalten.
  5. Öffnen Sie Internet-Informationsdienste.
  6. Klicken Sie mit der rechten Maustaste auf Administration, und klicken Sie dann auf Eigenschaften.
  7. Wählen Sie Verzeichnissicherheit aus.
  8. Wählen Sie Bearbeiten im Bereich Steuerung des anonymen Zugriffs und der Authentifizierung aus.
  9. Wählen Sie Bearbeiten im Bereich Standardauthentifizierung aus.
  10. Geben Sie den standardmäßigen Domänennamen ein, der für die Authentifizierung zu verwenden ist.
