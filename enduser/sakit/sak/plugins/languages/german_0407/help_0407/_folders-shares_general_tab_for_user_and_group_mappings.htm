Vorheriges Thema

Konfigurieren von Benutzer- und Gruppenzuordnungen

So ordnen Sie NFS-Benutzer und -Gruppen zu

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Protokolle freigeben aus.
  3. Wählen Sie NFS-Protokoll und dann Eigenschaften aus.
  4. Wählen Sie Benutzer- und Gruppenzuordnungen aus.

So erstellen Sie eine Konfiguration zur Verwendung eines NIS-Servers

  1. Wählen Sie NIS-Server verwenden auf der Registerkarte Allgemein aus.
  2. Geben Sie im Feld NIS-Domäne den Namen der Domäne ein, aus der die Daten der UNIX-Benutzer und -Gruppen stammen.
  3. Optional: Geben Sie im Feld NIS-Server (optional) den Namen des zuzuordnenden Servers ein.
  4. Geben Sie den Zeitraum an, nach dessen Ablauf die Serverappliance die Benutzer- und Gruppendaten aktualisiert. Verwenden Sie dazu die Felder Stunden und Minuten.
  5. Klicken Sie auf OK.

So erstellen Sie eine Konfiguration zur Verwendung von Kennwort- und Gruppendateien

  1. Wählen Sie Kennwörter und Gruppendateien verwenden aus.
  2. Geben Sie im Feld Kennwortdatei den Namen der zu verwendenden Kennwortdatei ein.
Hierbei handelt es sich um eine Datei im Kennwortformat aus einem UNIX-System, das sämtliche Benutzerkonten enthält, die zugeordnet werden können.
  1. Geben Sie im Feld Gruppendatei den Namen der zu verwendenden Gruppendatei ein.
  2. Klicken Sie auf OK.
