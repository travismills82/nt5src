Vorheriges Thema

Verstehen von Freigabeprotokollen

Freigabeprotokolle ermöglichen Clientcomputern den Zugriff auf Dateien der Serverappliance.

Ihre Serverappliance unterstützt mindestens einen der folgenden Freigabeprotokolltypen:

  • Windows (CIFS)

    Das CIFS-Protokoll (Common Internet File System) wird von Clients verwendet, auf denen ein Windows-Betriebssystem ausgeführt wird.

  • UNIX (NFS)

    Das NFS-Protokoll (Network File System) wird von UNIX-Clients verwendet.

  • FTP

    Das FTP-Protokoll (File Transfer Protocol) stellt eine alternative Möglichkeit für den Zugriff auf eine Dateifreigabe über ein beliebiges Betriebssystem dar.

  • Web (HTTP)

    Das HTTP-Protokoll (Hypertext Transfer Protocol) wird verwendet, um mit Webbrowsern auf eine Dateifreigabe zuzugreifen.

  • NetWare

    Hierbei handelt es sich um das Protokoll, mit dem NetWare-Clients auf eine Dateifreigabe zugreifen.

  • AppleTalk

    Hierbei handelt es sich um das Protokoll, mit dem Macintosh-Clients auf eine Dateifreigabe zugreifen.
