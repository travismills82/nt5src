Konfigurieren von Kontingenteinträgen

Auf der Seite Kontingenteinträge können Sie Datenträgerkontingente für einen beliebigen Benutzer der Serverappliance hinzufügen, löschen oder konfigurieren. Kontingente werden unter Verwendung der Objekt-/Taskauswahl verwaltet. Folgende Spalten stehen zur Verfügung:

  • Anmeldename - Diese Spalte enthält den Anmeldenamen jedes Benutzers, dem Zugriff auf die Serverappliance gewährt wurde.
  • Status - Diese Spalte gibt an, ob der Benutzer das zugewiesene Kontingentlimit überschritten hat.
  • Verwendeter Speicherplatz - Diese Spalte gibt den Speicherplatz an, der derzeit von einem bestimmten Benutzer verwendet wird.
  • Kontingentlimit - Diese Spalte enthält den maximalen Speicherplatz, den ein Benutzer auf einem Volume belegen kann.

Das Verhalten der Serverappliance bei Überschreitung des Kontingentslimits richtet sich nach den auf der Eigenschaftenseite Kontingente vorgenommenen Einstellungen; der Zugriff erfolgt über die Registerkarte Datenträgerkontingent. Ist das Kontrollkästchen Benutzern, die das Kontingentlimit überschreiten, Datenträgerspeicherplatz verweigern aktiviert, kann der Benutzer das Limit nicht überschreiten. Wenn das Kontrollkästchen Ereignis bei Kontingentüberschreitung protokollieren aktiviert ist, wird eine Ereignisprotokollmeldung erfasst. Sind beide Kontrollkästchen deaktiviert, wird keine Aktion durchgeführt.

  • Benachrichtigungslimit - Diese Spalte gibt den maximalen Speicherplatz an, den ein bestimmter Benutzer verwenden kann, bevor eine Warnung mit dem Hinweis angezeigt wird, dass das Limit nahezu erreicht ist.

    Anmerkung   Eine Warnung wird nur erzeugt, wenn der Benutzer das auf der Seite Kontingenteinträge angegebene Limit überschreitet und wenn die Ereignisprotokollierung auf der Seite Standardkontingent aktiviert ist. Ist das Kontrollkästchen deaktiviert, wird keine Warnung generiert, und diese Spalte enthält keinen Eintrag. In der Regel liegt der für Benachrichtigungslimit festgelegte Wert knapp unter dem Kontingentlimit.

So legen Sie die Kontingenteinträge auf der Serverappliance fest oder ändern sie

  1. Wählen Sie auf der primären Navigationsleiste Datenträger aus.
  2. Wählen Sie Datenträgerkontingent aus.
  3. Wählen Sie das zu verwaltende Volume aus.
  4. Wählen Sie in der Liste Tasks den Eintrag Kontingenteinträge aus.
