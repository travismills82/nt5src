Aktualisieren der Software

Auf dieser Seite können Sie Softwareupdates auf die Serverappliance anwenden.

So aktualisieren Sie die Software auf der Serverappliance

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie dann Softwareupdate aus.
  3. Folgen Sie den Anweisungen des Softwareupdate-Assistenten.
