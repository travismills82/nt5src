Aktivieren des Gastkontos

Standardmäßig ist das Gastkonto deaktiviert. In Arbeitsgruppen, die Windows 95- und Windows 98-Clientcomputern enthalten, wird durch Aktivieren des Gastkontos schnellstmöglich Zugriff auf Ressourcen auf einer Serverappliance bereitgestellt. Durch Aktivieren des Gastkontos erhält jedoch jeder mit dem Netzwerk verbundene Benutzer Zugriff auf die Ressourcen der Serverappliance. Eine Alternative bietet die Erstellung eines Benutzerkontos für jeden Benutzer im Netzwerk.

So aktivieren Sie das Gastkonto

  1. Wählen Sie auf der primären Navigationsleiste Benutzer aus.
  2. Wählen Sie Lokale Benutzer aus.
  3. Wählen Sie Gast in der Spalte Name aus.
  4. Wählen Sie in der Liste Tasks den Eintrag Eigenschaften aus.
  5. Deaktivieren Sie auf der Registerkarte Allgemein das Kontrollkästchen Dieses Benutzerkonto deaktivieren.
  6. Klicken Sie auf OK.
