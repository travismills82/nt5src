IPX-Einstellungen

IPX (Internet Packet Exchange) ist das systemeigene NetWare-Protokoll, das in zahlreichen früheren Novell-Netzwerken verwendet wird.

Damit NetWare-Clients auf die Serverappliance zugreifen können, müssen Sie eine IPX-Adresse bereitstellen.

So konfigurieren Sie die IPX-Adresse

  1. Wählen Sie auf der primären Navigationsleiste Netzwerk aus.
  2. Wählen Sie Globale Einstellungen aus.
  3. Wählen Sie die Registerkarte IPX-Einstellungen aus.
  4. Geben Sie im Feld Interne Netzwerknummer einen Wert ein, oder übernehmen Sie den standardmäßigen Parameterwert (00000000).
  5. Klicken Sie auf OK.
