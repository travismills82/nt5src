Anzeigen von FTP-Meldungen

So zeigen Sie Willkommens- und Beendennachricht sowie die Nachricht für die maximale Anzahl von Verbindungen an

  1. Wählen Sie auf der primären Navigationsleiste Webserver aus.
  2. Klicken Sie auf FTP-Meldungen.
  3. Geben Sie Ihre Willkommens- und Beendigungsnachricht sowie die Nachricht für die maximale Anzahl von Verbindungen in die entsprechenden Felder ein:
    • Willkommensnachricht -Benutzer sehen diese Nachricht, wenn sie sich über FTP an der Serverappliance anmelden.
    • Beendigungsnachricht - Diese Nachricht wird den Benutzern angezeigt, wenn sie sich über FTP von der Serverappliance abmelden.
    • Maximale Verbindungsanzahlnachricht - Mit dieser Nachricht wird der Benutzer darüber informiert, dass die maximale Anzahl an FTP-Verbindungen, die die Serverappliance verarbeiten kann, erreicht wurde.
  4. Klicken Sie auf OK.
