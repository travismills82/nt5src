Einstellen von Datum und Uhrzeit

Auf der Seite Datum/Uhrzeit können Sie das Datum, die Uhrzeit sowie die Zeitzone für die Serverappliance einstellen.

So stellen Sie das Datum, die Uhrzeit und die Zeitzone der Serverappliance ein

  1. Wählen Sie auf der primären Navigationsleiste Wartung aus.
  2. Wählen Sie Datum/Uhrzeit aus.
  3. Geben Sie das Datum und die Uhrzeit in dem Format ein, das für die im Browser ausgewählte Standardsprache definiert ist.
  4. Wählen Sie in der Liste die entsprechende Zeitzone aus.

    Sie können ebenfalls festlegen, dass die Serverappliance automatisch auf Sommer- und Winterzeit umgestellt wird. Diese Einstellung wird empfohlen.

  5. Klicken Sie auf OK.
