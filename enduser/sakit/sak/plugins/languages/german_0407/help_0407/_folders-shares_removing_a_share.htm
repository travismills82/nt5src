Entfernen einer Freigabe

Sie können Freigaben vollständig entfernen oder lediglich ein bestimmtes Protokoll deaktivieren. Als Konsequenz wird der Zugriff auf die Freigabe entfernt, die eigentlichen Dateien verbleiben jedoch auf der Serverappliance.

So entfernen Sie eine Freigabe und alle zugehörigen Protokolle

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Freigaben auf der Seite Freigaben aus.
  3. Wählen Sie das zu entfernende Verzeichnis aus.
  4. Wählen Sie in der Liste Tasks den Eintrag Löschen aus.
  5. Klicken Sie auf OK.

So entfernen Sie bestimmte Protokolle

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Freigaben auf der Seite Freigaben aus.
  3. Wählen Sie die Freigabe aus, deren Eigenschaften Sie ändern möchten.
  4. Wählen Sie in der Liste Tasks den Eintrag Eigenschaften aus.
  5. Deaktivieren Sie die Kontrollkästchen der Protokolle, die aus der Freigabe zu entfernen sind.
  6. Klicken Sie auf OK.
