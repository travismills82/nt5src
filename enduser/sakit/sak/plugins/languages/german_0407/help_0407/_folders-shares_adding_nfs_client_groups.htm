Vorheriges Thema

Hinzufügen von NFS-Clientgruppen

So fügen Sie eine NFS-Clientgruppe hinzu

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Protokolle freigeben aus.
  3. Wählen Sie NFS-Protokoll und dann Eigenschaften aus.
  4. Wählen Sie Clientgruppen aus.
  5. Wählen Sie in der Liste Tasks den Eintrag Neu aus.
  6. Geben Sie auf der Seite Neu NFS-Clientgruppe im Feld Gruppenname den gewünschten Gruppennamen ein.
  7. Geben Sie im Feld Clientnamen oder IP-Adressen den Namen oder die IP-Adresse des Computers ein, welcher der Gruppe hinzuzufügen ist.
  8. Klicken Sie auf Hinzufügen.
  9. Klicken Sie auf OK.
