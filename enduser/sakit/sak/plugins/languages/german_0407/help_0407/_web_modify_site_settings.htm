Ändern von Websiteeinstellungen

Nach der Erstellung Ihrer Website können Sie die allgemeine Siteidentität und die Anwendungseinstellungen nach Bedarf ändern.

So ändern Sie die Einstellungen einer bestehenden Website

  1. Wählen Sie auf der primären Navigationsleiste Sites aus.
  2. Wählen Sie die Website aus, die Sie ändern möchten.
  3. Wählen Sie in der Liste Tasks den Eintrag Bearbeiten aus.

    Die Seite Website ändern wird geöffnet, und die Registerkarte Allgemein ist aktiviert. Sie können die Einstellungen auf dieser Registerkarte ändern, oder Sie können Änderungen an allen Einstellungen auf den Registerkarten Siteidentitäten oder Anwendungseinstellungen vornehmen.
