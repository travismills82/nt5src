Vorheriges Thema

Hinzufügen von Kontingenteinträgen

So fügen Sie einen neuen Kontingenteintrag hinzu

  1. Wählen Sie auf der primären Navigationsleiste Datenträger aus.
  2. Wählen Sie Datenträgerkontingent aus.
  3. Wählen Sie das zu verwaltende Volume aus.
  4. Wählen Sie in der Liste Tasks den Eintrag Kontingenteinträge aus.
  5. Wählen Sie in der Liste Tasks den Eintrag Neu aus.
  6. Wählen Sie in der Liste einen lokalen Benutzer aus, oder geben Sie den Namen eines Domänenkontos im Format <Domänenname\Benutzername> in das Feld ein.

So ermöglichen Sie die unbegrenzte Datenträgernutzung

  • Wählen Sie Datenträgernutzung nicht begrenzen aus.

So begrenzen Sie die Datenträgernutzung

  1. Wählen Sie Datenträgernutzung begrenzen auf aus.
  2. Geben Sie in das Feld einen numerischen Wert ein, um den Speicherplatz anzugeben, der einem bestimmten Benutzer oder einer gewissen Gruppe zuzuweisen ist. Wählen Sie in der Liste die Maßeinheit Kilobytes (KB), Megabytes (MB), Gigabytes (GB), Terabytes (TB), Petabytes (PB) oder Exabytes (EB) aus.
  3. Geben Sie den Anteil des Speicherplatzes an, nach dessen Belegung an den Benutzer oder das Gruppenmitglied eine Warnung mit dem Hinweis ausgegeben wird, dass das Kapazitätslimit des Datenträgers nahezu erreicht ist. Wählen Sie in der Liste die entsprechende Maßeinheit aus (KB, MB, GB, TB, PB oder EB).
  4. Klicken Sie auf OK.
