Vorheriges Thema

Festlegen eines Benutzerkennworts

So legen Sie das Benutzerkennwort fest

  1. Wählen Sie auf der primären Navigationsleiste Benutzer aus.
  2. Wählen Sie Lokale Benutzer aus.
  3. Wählen Sie das Benutzerkonto aus, dessen Kennwort Sie ändern möchten.
  4. Wählen Sie in der Liste Tasks den Eintrag Kennwort festlegen aus.
  5. Geben Sie das neue Kennwort ein, und bestätigen Sie es anschließend in den dazu vorgesehenen Feldern.

    Das neue Kennwort muss mit den geltenden Kennwortrichtlinien für die Domäne übereinstimmen, zu der die Serverappliance gehört.

  6. Klicken Sie auf OK.
