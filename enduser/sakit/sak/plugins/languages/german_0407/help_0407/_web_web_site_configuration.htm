Vorheriges Thema

Websitekonfiguration

Auf der Seite Websitekonfiguration können Sie einen der folgenden Tasks durchführen:

  • Erstellen einer Website
  • Ändern von Websiteeinstellungen
  • Löschen einer Website
  • Anhalten einer Website
  • Beenden einer Website
  • Starten einer Website

Diese Seite ermöglicht Ihnen außerdem die Suche nach Sites, die mit einem bestimmten Wert beginnen. Sie können beispielsweise nach allen Hostheadern suchen, die mit "www" beginnen, oder nach allen Sites mit dem Status Beendet.

So zeigen Sie alle Sites mit dem Status "Beendet" an

  1. Wählen Sie den Statuswert in der Dropdownliste neben der Eingabeaufforderung Suchen aus.
  2. Geben Sie Beendet als Suchkriterium ein.
  3. Wählen Sie Gehe zu aus.
