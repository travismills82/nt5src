Ändern von Freigabeeigenschaften

Auf der Seite Freigaben können Sie Freigabeeigenschaften anzeigen und ändern.

So ändern Sie Freigabeeigenschaften

  1. Wählen Sie auf der primären Navigationsleiste Freigaben aus.
  2. Wählen Sie Freigaben auf der Seite Freigaben aus.
  3. Wählen Sie die Freigabe aus, deren Eigenschaften geändert werden sollen.
  4. Wählen Sie in der Liste Tasks den Eintrag Eigenschaften aus.

    Die Eigenschaftenseite Allgemein wird angezeigt. Verwenden Sie diese Seite, um den Clienttyp zu ändern, dem Zugriff auf die Freigabe erteilt wird.

  5. Klicken Sie auf OK.
