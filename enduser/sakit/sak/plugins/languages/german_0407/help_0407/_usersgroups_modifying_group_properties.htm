Vorheriges Thema

Ändern von Gruppeneigenschaften

Die Seite für die Gruppeneigenschaften zeigt die Registerkarten Allgemein und Mitglieder an. Auf der Registerkarte Allgemein können Sie den Gruppennamen sowie die zugehörige Beschreibung festlegen und ändern. Die Registerkarte Mitglieder dient zum Hinzufügen oder Entfernen von Benutzern und Gruppen.

So legen Sie Namen bzw. Beschreibungen für Gruppen fest oder ändern sie

  1. Wählen Sie auf der primären Navigationsleiste Benutzer aus.
  2. Wählen Sie dann Lokale Gruppen aus.
  3. Wählen Sie das Gruppenkonto aus, das Sie ändern möchten.
  4. Wählen Sie in der Liste Tasks den Eintrag Eigenschaften aus.
  5. Geben Sie auf der Registerkarte Allgemein einen Namen und eine Beschreibung für die gewünschte Gruppe ein.
  6. Klicken Sie auf OK.

So legen Sie die Gruppenmitgliedschaft fest oder ändern sie

  1. Wählen Sie auf der primären Navigationsleiste Benutzer aus.
  2. Wählen Sie dann Lokale Gruppen aus.
  3. Wählen Sie das Gruppenkonto aus, das Sie bearbeiten möchten.
  4. Wählen Sie in der Liste Tasks den Eintrag Eigenschaften aus, und wählen sie dann die Registerkarte Mitglieder aus.
  5. Gehen Sie folgendermaßen vor, um ein neues Mitglied hinzuzufügen:
    • Wählen Sie im Feld Benutzer oder Gruppe hinzufügen einen lokalen Benutzer oder eine lokale Gruppe aus, und klicken Sie dann auf Hinzufügen.

    - ODER -

    • Geben Sie den Domänen- und Gruppennamen (<Domäne\Gruppenname>) einer Domänengruppe oder ein Domänenbenutzerkonto (<Domäne\Benutzername>) ein, und klicken Sie dann auf Hinzufügen. Bei Angabe einer Domäne und eines Gruppennamens müssen Sie jedoch ebenfalls die Anmeldeinformationen eingeben, die das Hinzufügen einer Gruppe aus dieser Domäne ermöglichen. 
  6. Um ein Mitglied zu entfernen, wählen Sie in der Liste Mitglieder einen Benutzernamen aus und klicken dann auf Löschen.
  7. Klicken Sie auf OK.
