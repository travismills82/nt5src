Web (HTTP)

HTTP es el protocolo de Internet que utilizan los exploradores y los servidores de World Wide Web para intercambiar información. El protocolo define qué acciones deben realizar los servidores y los exploradores Web como respuesta a varios comandos, lo que permite que un usuario utilice un programa cliente para escribir una dirección URL, o elija un hipervínculo, y recupere texto, gráficos, sonido u otro tipo de información digital desde un servidor Web. Todas las direcciones URL de los archivos de los servidores Web comienzan con http://.
