Apagar el módulo servidor

Utilice esta página para apagar, reiniciar o programar un apagado o un reinicio del módulo servidor.

Para apagar o reiniciar el módulo servidor

  1. En la barra de exploración principal, elija Mantenimiento.
  2. Haga clic en Apagar.
  3. Elija la tarea que desea realizar.
  4. Haga clic en Aceptar para confirmar su decisión.

    Si ha elegido reiniciar el módulo servidor, aparecerá la página Reiniciando.

Nota   La página Reiniciando comprueba periódicamente para determinar si el módulo vuelve a estar en funcionamiento. Si la página Reiniciando detecta que el módulo está en funcionamiento, volverá automáticamente a la página predeterminada.

Para programar un apagado o un reinicio del módulo servidor

  1. En la barra de exploración principal, elija Mantenimiento.
  2. Haga clic en Apagar.
  3. Elija Apagado programado.
  4. Elija las opciones de apagado programado que desee.
  5. Haga clic en Aceptar.

Nota   Para cancelar un suceso programado actualmente, haga clic en el botón No hay ningún apagado ni reinicio programado
