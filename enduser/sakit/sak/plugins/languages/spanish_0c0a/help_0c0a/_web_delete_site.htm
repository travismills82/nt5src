Eliminar un sitio Web

Puede quitar cualquier sitio Web que haya creado en el módulo servidor.

Para eliminar un sitio Web

  1. En la barra de exploración principal, elija Sitios.
  2. Seleccione el sitio Web que desea quitar.
  3. En la lista Tareas, elija Eliminar.
  4. Compruebe que el sitio Web indicado es el que desea quitar.
  5. Haga clic en Aceptar.
