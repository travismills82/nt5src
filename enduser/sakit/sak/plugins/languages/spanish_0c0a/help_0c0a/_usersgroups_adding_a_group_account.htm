Agregar una cuenta de grupo

Para agregar una cuenta de grupo

  1. En la barra de exploración principal, elija Usuarios.
  2. Elija Grupos locales.
  3. En la lista Tareas, elija Nueva.
  4. En la ficha General, escriba el nombre y la descripción del grupo que desea agregar.
  5. En la ficha Miembros:
    • Para agregar miembros al grupo, seleccione un usuario o un grupo en el cuadro Agregar un usuario o un grupo y, a continuación, elija Agregar.

      Nota   En la lista sólo se muestran los usuarios locales. Para escribir una cuenta de usuario de dominio, escriba el nombre de dominio y de usuario (<dominio\nombreDeUsuario>)

    • Para quitar miembros del grupo, seleccione un miembro o un grupo en el cuadro Miembros y, a continuación, elija Quitar.
  6. Haga clic en Aceptar.
