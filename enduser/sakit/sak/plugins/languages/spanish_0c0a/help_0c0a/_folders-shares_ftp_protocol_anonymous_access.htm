Acceso anónimo a FTP

Permitir el acceso anónimo al servidor FTP hace que los usuarios puedan conectar con el nombre de usuario anonymous, o ftp, que es sinónimo de anonymous. La contraseña no es necesaria, pero se pide al usuario que proporcione su dirección de correo electrónico como contraseña. De forma predeterminada, las conexiones anónimas están permitidas.

Nota   No puede tener acceso al servidor FTP desde una cuenta de usuario de Microsoft Windows Powered con el nombre anonymous. El nombre de usuario anonymous está reservado en el servidor FTP para la función de inicio de sesión anónimo. Los usuarios que inician sesiones en el servidor con el nombre de usuario anonymous reciben permisos basados en la configuración del servidor FTP para los inicios de sesión anónimos.

Después de instalar el software del protocolo FTP en el equipo, debe configurar el software. Las opciones del protocolo del servidor FTP darán como resultado una de las configuraciones siguientes:

  • No se permiten la conexiones anónimas a FTP en el servidor. En este caso, todos los usuarios deben proporcionar un nombre de usuario y una contraseñas válidos de Windows. Para configurar esta opción en el protocolo del servidor FTP, desactive la casilla de verificación Habilitar conexión anónima en la página Propiedades del protocolo FTP.
  • Permitir que usuarios anónimos y usuarios de Windows conecten con el servidor FTP. Esta configuración permite que un usuario utilice una conexión anónima o un nombre de usuario y una contraseña de Windows. Para configurar esta opción en el protocolo del servidor FTP, active únicamente la casilla de verificación Habilitar conexión anónima en la página Propiedades del protocolo FTP.
  • Permitir sólo conexiones anónimas a FTP en el servidor. En este caso, un usuario no puede conectar con el servidor FTP mediante un nombre de usuario y una contraseña de Windows. Para configurar esta opción en el protocolo del servidor FTP, active las casillas de verificación Habilitar conexión anónima y Permitir sólo acceso anónimo en la página Propiedades del protocolo FTP.

Si las conexiones anónimas están permitidas, debe especificar el nombre de usuario y la contraseña de Windows que proporcionan acceso anónimo al servidor FTP. Cuando tiene lugar una transferencia FTP anónima, Windows comprueba el nombre de usuario asignado en este cuadro de diálogo para determinar si el acceso a los archivos está permitido.
