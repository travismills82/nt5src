AppleTalk

La integración de Microsoft Windows 2000 Server con redes AppleTalk le permite compartir archivos e impresoras entre el módulo servidor y los clientes Apple Macintosh conectados a la red.

Con la integración de red AppleTalk, los equipos Macintosh sólo necesitan el software del sistema operativo de Macintosh para funcionar como clientes; no se necesita ningún otro software adicional.

La integración de red AppleTalk facilita la administración ya que permite mantener un único conjunto de cuentas de usuario en lugar de tener dos grupos de cuentas de usuario diferentes; por ejemplo, uno en el servidor Macintosh y otro en el equipo que ejecuta Windows 2000 Server.
