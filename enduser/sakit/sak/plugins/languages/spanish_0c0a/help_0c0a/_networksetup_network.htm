Red

En la página Red puede elegir cuáles de las siguientes propiedades del módulo servidor relacionadas con la red va a configurar:

  • Identificación: establezca el nombre del módulo servidor y el dominio al que pertenece.
  • Configuración global: configure las opciones de la red que se aplican a todos los adaptadores de red del módulo servidor.
  • Interfaces: configure las propiedades de cada adaptador de red del módulo servidor.
  • Administrador: cambie la contraseña de la cuenta de usuario que utiliza para tener acceso al sitio Web de administración del módulo servidor.
  • Sitio Web de administración: especifique las direcciones IP y el puerto que se utilizan para tener acceso al sitio Web de administración.
  • Telnet: configure Telnet para administrar el módulo servidor.
