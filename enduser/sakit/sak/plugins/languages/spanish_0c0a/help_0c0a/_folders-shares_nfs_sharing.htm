Establecer las propiedades de uso compartido de UNIX (NFS)

Utilice esta página para especificar qué clientes NFS tienen acceso a cada recurso compartido. Se puede conceder o denegar acceso para cada nombre de host del cliente. El acceso también se puede conceder o denegar por grupos de clientes, donde cada grupo de clientes contiene uno o varios nombres de host cliente.

Para agregar un nuevo cliente o grupo de clientes NFS a un recurso compartido

  1. Cree un nuevo grupo de clientes como se describe en Agregar grupos de clientes NFS.
  2. Siga los pasos que se describen en el procedimiento siguiente para agregar un cliente o un grupo de clientes existente.

Para agregar un cliente o grupo de clientes existente

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. En la página Recursos compartidos, elija Recursos compartidos.
  3. Seleccione el recurso compartido al que desea agregar un cliente o grupo de clientes NFS.
  4. En la lista Tareas, elija Propiedades.
  5. Seleccione la ficha General.
  6. Active la casilla de verificación UNIX (NFS).
  7. Seleccione la ficha Uso compartido NFS.
  8. Seleccione la máquina o el grupo deseado en la lista de la izquierda, o escriba el nombre o la dirección IP del equipo cliente NFS en el cuadro de la derecha y, a continuación, haga clic en Agregar.
  9. En la lista Tipo de acceso, seleccione el grado de control que el cliente especificado puede ejercer sobre los archivos del recurso compartido.
  10. Haga clic en Aceptar.

Para quitar un cliente NFS

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. En la página Recursos compartidos, elija Recursos compartidos.
  3. Seleccione el recurso compartido del que desea quitar un cliente o grupo de clientes NFS.
  4. En la lista Tareas, elija Propiedades.
  5. Seleccione la ficha General.
  6. Active la casilla de verificación UNIX (NFS).
  7. Seleccione la ficha Uso compartido NFS.
  8. Seleccione el cliente o el grupo de clientes deseado en la lista del centro de la página y, a continuación, haga clic en Quitar.
  9. Haga clic en Aceptar.
