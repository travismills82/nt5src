Sufijos DNS

Los sufijos del sistema de nombres de dominio (DNS) tienen los dos propósitos principales siguientes:

  • Cuando se anexan al nombre de host del módulo servidor, los sufijos DNS constituyen el nombre completo del módulo servidor.
  • Los sufijos DNS se utilizan para resolver direcciones IP. Si el módulo servidor es miembro de un dominio de Windows NT 4, Active Directory de Microsoft o un grupo de trabajo, el sufijo DNS depende del entorno del dominio.

Un sufijo DNS puede contener hasta 155 caracteres, incluidos guiones y puntos.
