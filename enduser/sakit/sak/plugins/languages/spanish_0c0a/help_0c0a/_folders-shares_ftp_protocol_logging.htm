Habilitar el registro FTP

Puede registrar las conexiones FTP entrantes en el registro de FTP si habilita Registro FTP. De forma predeterminada, los registros de FTP se almacenan en %windir%\system32\logfiles\msftpsvc1.

Los administradores puede tener acceso a estos archivos desde su estación de trabajo si tienen acceso a un recurso compartido administrativo (por ejemplo, \\nombreMódulo\admin$\winnt\system32) o si crean un nuevo recurso compartido para dicha carpeta.

Para habilitar el registro FTP

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. Elija Protocolos para uso compartido.
  3. Seleccione Protocolo FTP y, a continuación, elija Propiedades.
  4. Seleccione la ficha Registro.
  5. Active la casilla de verificación Habilitar registro y, a continuación, haga clic en Aceptar.
