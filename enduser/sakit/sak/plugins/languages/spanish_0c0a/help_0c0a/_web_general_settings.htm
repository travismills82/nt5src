Establecer las propiedades de la identidad general del sitio

  1. En el cuadro Identificador del sitio, escriba el nombre exclusivo del sitio Web.

    El identificador de un sitio Web debe ser único y puede contener hasta 12 caracteres en mayúsculas o minúsculas, excepto los siguientes: " / \ [ ] : ; | = , + * ? < >. Además, un identificador de sitio Web no puede constar únicamente de puntos (.) o espacios en blanco.

  2. El módulo servidor llenará automáticamente el cuadro Directorio con la ruta del directorio predeterminado. Puede modificar dicha ruta.
  3. Seleccione el tipo de cuenta de usuario que desea para tener acceso a la administración del sitio Web. Puede elegir entre crear una cuenta de usuario local o del dominio, o puede designar una cuenta de usuario existente.

    Nota   Sólo podrá crear una cuenta de dominio si el módulo servidor forma parte de dicho dominio.

  4. Opcionalmente: Si elige asignar privilegios administrativos a una cuenta de usuario existente, escriba el nombre de usuario en el cuadro Administrador del sitio Web.

    Nota   Si el módulo servidor forma parte de un dominio y elige asignar privilegios administrativos a una cuenta de usuario existente, debe escribir el nombre del dominio y el nombre del usuario en el formato <nombre de dominio>\<nombre de usuario>.

  5. Opcionalmente: Si elige crear una cuenta de usuario local o del dominio, escriba y confirme la contraseña administrativa en los cuadros Contraseña del administrador y Confirmar contraseña. La contraseña debe cumplir las reglas de complejidad de contraseñas vigentes para el dominio al que pertenece el módulo servidor.
