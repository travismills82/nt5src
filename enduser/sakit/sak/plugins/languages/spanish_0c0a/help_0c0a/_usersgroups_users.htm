Usuarios

Desde esta página puede crear, modificar y eliminar usuarios y grupos locales del módulo servidor. También puede modificar los miembros de cada grupo. Si el módulo servidor es miembro de un dominio, no le interesará crear usuarios en el propio módulo servidor. El principal propósito de esta página es agregar uno o varios miembros del dominio al grupo local.

Puede que también desee utilizar cuentas de usuario y de grupo del dominio para controlar el acceso a los recursos del módulo servidor. También le puede interesar el uso de herramientas de administración de dominios para administrar usuarios y grupos del dominio.
