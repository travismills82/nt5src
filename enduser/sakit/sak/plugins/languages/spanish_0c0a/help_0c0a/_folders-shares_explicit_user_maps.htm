Configurar asignaciones explícitas de usuarios

Asignaciones de usuarios y grupos le permite crear asignaciones entre plataformas y a través de plataformas entre cuentas de usuario y de grupo de Microsoft Windows y UNIX, incluso cuando los nombres de usuario y de grupo no son idénticos en ambos entornos.

Asignaciones de usuarios y grupos también le permite establecer asignaciones unívocas entre plataformas y a través de plataformas entre usuarios y grupos de Windows y UNIX. Por ejemplo, un nombre de usuario de Windows podría asignarse a un nombre de usuario de UNIX, o un grupo de UNIX podría asignarse a una o más cuentas de usuario de Windows. Las asignaciones explícitas de usuarios también pueden utilizarse cuando la misma persona tiene nombres de usuario diferentes en sus cuentas de Windows y de UNIX. El uso de la opción Asignaciones explícitas de usuarios le permite mantener una única base de datos de asignaciones para toda la empresa.

Para crear asignaciones explícitas de usuarios

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. Elija Protocolos para uso compartido.
  3. Seleccione Protocolo NFS y, a continuación, elija Propiedades.
  4. Elija Asignaciones de usuarios y grupos.
  5. Complete la ficha General , asegurándose de que ha proporcionado una contraseña de dominio NIS o PCNFS y los archivos de grupo. Para obtener más información acerca de cómo completar la ficha General, consulte Configurar asignaciones de usuarios y grupos.
  6. Seleccione la ficha Asignación explícita de usuarios.
  7. Opcionalmente: Escriba el nombre del servidor NIS al que se va a asignar en el campo del dominio NIS.
  8. Haga clic en el botón Enumerar usuarios de UNIX para llenar el cuadro Usuarios de UNIX.
  9. Seleccione un usuario de cada grupo y, a continuación, haga clic en Agregar.

    Los usuarios asignados aparecerán en el cuadro Usuarios asignados de forma explícita. Si desea asignar un usuario de DOMINIO, debe escribir el nombre DOMINIO\usuario, seleccionar un usuario de UNIX y, a continuación, hacer clic en el botón Agregar que se encuentra junto al cuadro de entrada DOMINIO\usuario.

    Nota   Puede asignar usuarios de un dominio de Windows a varios dominios de UNIX, pero no puede asignar un usuario de UNIX a varios usuarios de Windows. 

Para establecer una de las asignaciones como principal para un usuario dado

  1. Seleccione la asignación en la lista Usuarios asignados de forma explícita.
  2. Elija Establecer principal.
  3. Haga clic en Aceptar.

Para eliminar asignaciones explícitas de usuarios

  1. Siga los pasos 1 a 4 del procedimiento Para crear asignaciones explícitas de usuarios para ir hasta la página Asignaciones explícitas de usuarios.
  2. En la lista Usuarios asignados de forma explícita, seleccione la asignación de usuario que desea eliminar.
  3. Elija Quitar.
  4. Haga clic en Aceptar.
