Establecer las propiedades generales de las aplicaciones

En la página Configuración de la aplicación puede

  • establecer permisos
  • especificar la página predeterminada
  • habilitar o deshabilitar el uso de las extensiones de Microsoft® FrontPage® (deshabilitadas de forma predeterminada)
  • crear un sitio Web FTP para los sitios Web administrados por el módulo servidor

Para establecer o modificar la configuración de las aplicaciones

  1. En la barra de exploración principal, elija Sitios.
  2. Seleccione el sitio Web que desea modificar.
  3. En la lista Tareas, elija Modificar.
  4. Seleccione la ficha Configuración de la aplicación.
  5. Utilice la lista Permisos de ejecución predeterminados para seleccionar uno de los siguientes niveles de ejecución de secuencias de comandos:
    • Ninguno: no ejecutar secuencias de comandos, incluyendo aplicaciones de páginas Active Server (ASP), o ejecutables del servidor.
    • Sólo secuencias de comandos: ejecutar únicamente secuencias de comandos, como aplicaciones ASP, en el servidor.
    • Secuencias de comandos y ejecutables: ejecutar secuencias de comandos, como aplicaciones ASP, y ejecutables del servidor.
  6. Utilice la lista Página predeterminada para seleccionar la página Web predeterminada apropiada.
    • Para agregar una página a la lista de páginas predeterminadas, escriba el nombre de archivo de la página que va a agregar en el cuadro Nombre de archivo y, a continuación, haga clic en Agregar.
    • Para quitar una página de la lista de páginas predeterminadas, seleccione el nombre de archivo de la página que va a quitar y, a continuación, haga clic en Quitar.
  7. Las extensiones de Microsoft FrontPage permiten que los administradores vean y administren sitios Web a través de la interfaz gráfica de usuario (GUI) de FrontPage. Si desea ver y administrar los sitios Web del módulo servidor con FrontPage, active la casilla de verificación Permitir las extensiones de Microsoft FrontPage.
  8. Active la casilla de verificación Crear un sitio Web FTP para modificar el contenido si desea crear un sitio Web al que la cuenta del administrador pueda conectarse mediante FTP para modificar archivos.
  9. Active la casilla de verificación Permitir acceso anónimo para permitir que los usuarios remotos se conecten al módulo servidor sin un nombre de usuario y una contraseña. Los usuarios que inician sesión de forma anónima sólo tienen los permisos asignados a la cuenta de invitado. El acceso anónimo suele utilizarse para los sitios Web de Internet.
