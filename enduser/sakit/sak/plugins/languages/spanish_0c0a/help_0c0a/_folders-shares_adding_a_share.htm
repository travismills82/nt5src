Agregar un recurso compartido

Para crear un recurso compartido, debe suministrar un nombre que sea único entre todos los recursos compartidos y la ruta de acceso a dicho recurso. Algunos protocolos aceptan también la inclusión de un comentario o una breve descripción del recurso compartido. Además, debe habilitar al menos uno de los protocolos disponibles.

Aunque sólo se proporciona una interfaz de usuario para crear un recurso compartido para todos los protocolos, en realidad se crea un recurso compartido distinto para cada protocolo. Puede quitar protocolos individuales para uso compartido de un recurso compartido sin quitar el propio recurso. Sin embargo, al quitar todos los protocolos para uso compartido de un recurso se eliminarán todas las versiones del recurso compartido.

Para agregar un recurso compartido

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. En la página Recursos compartidos, elija Recursos compartidos.
  3. En la lista Tareas, elija Nueva.
  4. En la ficha General, escriba el nombre y la ruta de acceso del recurso compartido.
  5. Active las casillas de verificación apropiadas para especificar los tipos de protocolos que desea habilitar.
  6. Utilice las fichas de los protocolos para configurar las propiedades específicas de cada tipo de recurso compartido.
