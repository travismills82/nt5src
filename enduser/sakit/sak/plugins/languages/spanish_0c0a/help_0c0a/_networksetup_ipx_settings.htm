Configuración IPX

Intercambio de paquetes entre redes (IPX) es el protocolo nativo de NetWare y se emplea en muchas redes Novell antiguas.

Para que sea accesible desde clientes NetWare, su módulo servidor debe proporcionar una dirección IPX.

Para configurar la dirección IPX

  1. En la barra de exploración principal, elija Red.
  2. Elija Configuración global.
  3. Seleccione la ficha Configuración IPX.
  4. Escriba un valor en el cuadro Número de red interna o deje el valor predeterminado del parámetro, 00000000.
  5. Haga clic en Aceptar.
