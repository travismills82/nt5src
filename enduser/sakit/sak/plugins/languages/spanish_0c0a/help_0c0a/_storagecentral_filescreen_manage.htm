Managing File Screening

To manage file screening

  1. On the primary navigation bar, choose Disks.
  2. Choose File Screening.    
  3. Select the task you want.

You can screen a partition to limit or monitor the file types being saved on the drive. The following list contains the table information for each screened directory:

  • Directory Name The name of the directory that contains the screening policy.
  • Screening Policy The screening policies applied to the directory.
