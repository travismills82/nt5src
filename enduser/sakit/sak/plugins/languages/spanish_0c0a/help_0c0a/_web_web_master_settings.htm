Configuración del maestro Web

Esta página le permite configurar opciones genéricas del servidor Web que se aplicarán a todos los sitios Web del módulo servidor.

La ruta del directorio predeterminado se muestra en el cuadro Directorio raíz del sitio Web. Puede modificar esta ruta de directorio. Si lo hace, cambiará el directorio predeterminado de los siguientes sitios Web. Los sitios Web creados antes de la modificación permanecerán en sus directorios originales.

Para cambiar el directorio raíz predeterminado, escriba la nueva ruta en el cuadro Directorio raíz del sitio Web.

Nota   Todas las modificaciones efectuadas en el directorio raíz del sitio Web no afectan al directorio de FTP.

También dispone de las siguientes opciones:

  • Tiempo de espera de secuencias de comandos ASP (en segundos): especifica el período de tiempo que ASP permite que se ejecute una secuencia de comandos. Si la secuencia de comandos no termina su ejecución antes del período de tiempo de espera, ASP detendrá la secuencia de comandos y escribirá un suceso en el registro de sucesos de Windows. Puede establecer el período del tiempo de espera como un valor entre 1 y 2147483647. Puede pasar por alto esta opción en una página ASP si utiliza el método Server.ScriptTimeout.
  • Indizar sitios Web: permite la búsqueda de texto en archivos mediante Servicios de Microsoft Index Server.

Nota  En todos los casos, las casillas de verificación deben estar activadas para que se habilite la funcionalidad.

Finalmente, puede establecer el número máximo de conexiones con los sitios Web alojados en el módulo servidor.
