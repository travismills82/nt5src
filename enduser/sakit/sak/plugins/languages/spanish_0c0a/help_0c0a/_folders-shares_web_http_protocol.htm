Protocolo Web (HTTP)

El Protocolo de transferencia de hipertexto (HTTP) es un protocolo de comunicaciones diseñado para transferir documentos de hipertexto entre equipos a través del Web. HTTP define qué acciones deben tomar los servidores y los exploradores Web como respuesta a varios comandos.

Servidor de World Wide Web

Los comandos que se utilizan en el Web están definidos en HTTP.

Para especificar la ubicación de un recurso, HTTP utiliza Localizadores de recursos uniforme (direcciones URL). Las direcciones URL siguen una convención de nomenclatura que identifica de forma exclusiva la ubicación de un equipo, directorio o archivo en Internet. La dirección URL también especifica el protocolo de Internet (por ejemplo, FTP, HTTP) necesario para recuperar el recurso. Si conoce la dirección URL de un recurso, puede proporcionarla, o bien puede vincularse a ella desde un documento que desea que esté disponible para los usuarios del Web.

El protocolo HTTP acepta el acceso anónimo, así como la autenticación básica y la de Windows.

Para configurar las propiedades de uso compartido Web (HTTP)

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. Elija Protocolos para uso compartido.
  3. Seleccione Protocolo HTTP y, a continuación, elija Propiedades.
  4. Elija permitir que todas las direcciones IP tengan acceso a los recursos compartidos de datos del módulo servidor o seleccione una única dirección IP de la lista.
  5. Escriba el número de puerto que puede utilizarse para tener acceso a los recursos compartidos de datos del módulo servidor. El número de puerto predeterminado es el 80.
