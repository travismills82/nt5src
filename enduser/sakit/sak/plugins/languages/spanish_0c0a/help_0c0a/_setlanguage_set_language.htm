Establecer el idioma

Si en el módulo servidor hay instalados otros idiomas, puede cambiar el idioma utilizado en la interfaz de usuario Web, la interfaz de usuario local y los archivos de Ayuda del módulo servidor.

Para cambiar el idioma utilizado en el módulo servidor

  1. En la barra de exploración principal, elija Mantenimiento.
  2. Haga clic en Idioma.
  3. En la página Establecer el idioma, seleccione el idioma que desea utilizar para el módulo servidor.
  4. Haga clic en Aceptar.
