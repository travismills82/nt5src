Introducción

El módulo servidor alimentado por Microsoft® Windows® es un módulo servidor que se conecta directamente a la red de equipos. Un módulo servidor se administra y supervisa mediante una UI tipo Web, y se puede administrar de manera remota desde un equipo cliente conectado a la red. Puede ser un dispositivo sin monitor ni dispositivos de entrada propios, como un teclado o un mouse (ratón). Como los módulos servidores están basados en el mismo código que Microsoft® Windows® 2000, muchos métodos de administración remota disponibles en la plataforma Windows 2000 pueden utilizarse también para administrar módulos servidores alimentados por Windows.
