Registros

Un archivo de registro es un archivo que almacena mensajes, o sucesos, generados por una aplicación, un servicio o Microsoft® Windows®. Estos mensajes se utilizan para hacer un seguimiento de las operaciones realizadas en el módulo servidor.

Puede utilizar la función Registros para ver, borrar, descargar y configurar los siguientes tipos de registros de sucesos que proporciona el sistema:

  • Administrar los registros de aplicación
  • Administrar los registros de FTP
  • Administrar los registros de NFS
  • Administrar los registros de seguridad
  • Administrar los registros de sistema
  • Administrar los registros de recursos compartidos Web (HTTP)
  • Administrar los registros de administración Web
  • Borrar archivos de registro
  • Descargar archivos de registro
  • Modificar las propiedades de registro
  • Ver los detalles del registro
