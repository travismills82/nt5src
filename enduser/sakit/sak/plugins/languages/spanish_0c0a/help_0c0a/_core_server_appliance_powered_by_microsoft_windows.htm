Server Appliance Powered de Microsoft Windows

Ésta es la Ayuda en pantalla del módulo servidor alimentado por Microsoft® Windows®. Puede utilizar la Ayuda en pantalla para aprender cómo configurar este módulo servidor antes de utilizarlo por vez primera, cómo desplazarse por la interfaz de usuario (UI) basada en Web y cómo utilizar el módulo servidor en su organización. Además, hay Ayuda contextual disponible desde cualquier pantalla de la UI Web. Para abrir la Ayuda contextual, haga clic en el signo de interrogación (?) situado en el extremo derecho de la barra de exploración principal.
