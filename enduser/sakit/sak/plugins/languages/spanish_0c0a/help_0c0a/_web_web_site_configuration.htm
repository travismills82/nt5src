Configuración del sitio Web

En la página Configuración del sitio Web puede completar una de las siguientes tareas:

  • Crear un sitio Web
  • Modificar la configuración del sitio Web
  • Eliminar un sitio Web
  • Pausar un sitio Web
  • Detener un sitio Web
  • Iniciar un sitio Web

Esta página también le permite buscar sitios que comiencen con un valor determinado. Por ejemplo, podría buscar todos los encabezados de host que comenzaran con "www" o todos los sitios que tengan el Estado Detenido.

Para mostrar todos los sitios que estén en estado Detenido

  1. Seleccione el valor Estado en la lista desplegable que se encuentra junto a Buscar.
  2. Escriba Detenido como criterio de búsqueda.
  3. Haga clic en Ir.
