Deleting a File Screening Group

 To delete a file screening group

  1. On the primary navigation bar, choose Disks.
  2. Choose File Screening. 
  3. In the Tasks list, choose Policies.
  4. In the Tasks list, choose Groups.
  5. Select the file screening group you want to delete.
  6. In the Tasks list, choose Delete.
  7. Choose OK.
