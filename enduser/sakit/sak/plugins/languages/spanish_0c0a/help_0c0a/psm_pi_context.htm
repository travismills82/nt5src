Persistent Image and Group Information

After a persistent image is created from the schedule you specify, it becomes a member of an image group. The Persistent Image and Group Information page displays the following information about the image group:

Image name and location on volume
Displays the name of the image and its path.
Persistent image group name
Displays the name assigned to this group.
Number of images in group
Displays the maximum number of images that can be included in the group.
Volumes included in this image
Displays each volume included in the image.
Image attributes
Displays the read-only or read/write attribute of the image.
Retention weight
Displays the relative retention weight of the image.
Most recent image in group
Displays the date and time of the image most recently added to the group.
Oldest image in group
Displays the chronologically oldest image in the group.
Next image in group to be deleted
Displays the date and time of the image that will be deleted next so the system can stay within the saved images limit.
