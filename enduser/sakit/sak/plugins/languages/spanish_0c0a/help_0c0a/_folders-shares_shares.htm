Recursos compartidos

Las carpetas del módulo servidor se pueden compartir con otros equipos de la red, tanto si los equipos ejecutan sistemas operativos Microsoft Windows como sistemas operativos UNIX.

Los protocolos específicos para uso compartido de archivos pueden variar de acuerdo con el fabricante del módulo servidor, por lo que un módulo servidor puede ser compatible con los protocolos siguientes:

  • Windows (CIFS): el protocolo Sistema de archivos común de Internet (CIFS) se utiliza en los clientes que ejecutan sistemas operativos Windows.
  • NFS: el protocolo Sistema de archivos de red se utiliza en clientes que ejecutan sistemas operativos UNIX.
  • FTP: el Protocolo de transferencia de archivos es una alternativa para tener acceso a recursos compartidos de archivos desde cualquier sistema operativo.
  • HTTP: el Protocolo de transferencia de hipertexto es el protocolo para tener acceso a recursos compartidos de archivos desde exploradores Web.
  • NetWare: el protocolo para tener acceso a recursos compartidos de archivos desde clientes NetWare.
  • AppleTalk: el protocolo para tener acceso a recursos compartidos de archivos desde clientes Macintosh.

Cuando crea un recurso compartido en el módulo servidor, puede habilitar alguno o todos los protocolos mencionados que estén habilitados o instalados en el módulo servidor.
