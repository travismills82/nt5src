Hacer copia de seguridad y restaurar el SO

En esta página puede elegir entre hacer una copia de seguridad o restaurar el sistema operativo (SO) del módulo servidor.

Para hacer una copia de seguridad o restaurar el SO del módulo servidor

  1. En la barra de exploración principal, elija Mantenimiento.
  2. Elija Copia de seguridad.
  3. Inicie sesión en el Cliente avanzado de Servicios de Terminal Server. La aplicación de copia de seguridad se iniciará automáticamente.
  4. Cuando haya terminado, haga clic en el cuadro de cierre (X) de la esquina superior derecha para cerrar la aplicación. Esto cerrará la sesión del Cliente avanzado de Servicios de Terminal Server.

    Nota   Puede que al cerrar la aplicación se tarde unos momentos en cerrar la sesión.

Cliente de Servicios de Terminal Server

Cliente de Servicios de Terminal Server es la herramienta que se utiliza para hacer copia de seguridad y restaurar el sistema operativo del módulo servidor. Sólo acepta dos conexiones simultáneas. Además, si se desplaza a otra página durante una sesión abierta, se desconectará el cliente pero se mantendrá la sesión. Esto puede impedir que otros usuarios tengan acceso a la sesión de Servicios de Terminal Server.
