Crear un sitio Web

En la página Crear sitio puede establecer un nuevo sitio Web y configurar sus propiedades.

Para crear un sitio Web

  1. En la barra de exploración principal, elija Sitios.
  2. En la lista Tareas, elija Crear.
  3. Complete la información de la página para crear un nuevo sitio Web.
  4. Haga clic en Aceptar.
