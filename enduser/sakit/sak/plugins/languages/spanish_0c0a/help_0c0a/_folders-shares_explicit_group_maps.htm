Configurar asignaciones explícitas de grupos

Asignaciones de usuarios y grupos le permite crear asignaciones entre plataformas y a través de plataformas entre cuentas de usuario y de grupo de Microsoft Windows y UNIX, incluso cuando los nombres de usuario y de grupo no son idénticos en ambos entornos.

Asignaciones de usuarios y grupos también le permite establecer asignaciones unívocas entre usuarios de Windows y usuarios y grupos de UNIX. Por ejemplo, un nombre de usuario de Windows podría asignarse a un nombre de usuario de UNIX, o un grupo de UNIX podría asignarse a una o más cuentas de usuario de Windows. Las asignaciones explícitas también pueden utilizarse cuando la misma persona tiene nombres de usuario diferentes en sus cuentas de Windows y de UNIX. El uso de la opción Asignaciones explícitas de grupos le permite mantener una única base de datos de asignaciones para toda la empresa.

Para crear asignaciones explícitas de grupos

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. Elija Protocolos para uso compartido.
  3. Seleccione Protocolo NFS y, a continuación, elija Propiedades.
  4. Elija Asignaciones de usuarios y grupos.
  5. Complete la ficha General , asegurándose de que ha proporcionado una contraseña de dominio NIS o PCNFS y los archivos de grupo. Para obtener más información acerca de cómo completar la ficha General, consulte Configurar asignaciones de usuarios y grupos.
  6. Seleccione la ficha Asignaciones explícitas de grupos.
  7. Opcionalmente: Escriba el nombre del servidor NIS al que se va a asignar en el campo del dominio NIS.
  8. Para llenar la lista Grupos de UNIX, haga clic en el botón Enumerar grupos de UNIX.
  9. Seleccione un grupo de cada grupo y, a continuación, haga clic en Agregar.

    Los grupos asignados aparecerán en el cuadro Grupos asignados de forma explícita. Si desea asignar un grupo de DOMINIO, debe escribir el nombre DOMINIO\grupo, seleccionar un grupo de UNIX y, a continuación, hacer clic en el botón Agregar que se encuentra junto al cuadro de entrada DOMINIO\grupo.

    Nota   Puede asignar grupos de un dominio de Windows a varios dominios de UNIX, pero no al contrario.

Para establecer una de las asignaciones como principal para un grupo dado

  1. Seleccione la asignación en la lista Grupos asignados de forma explícita.
  2. Elija Establecer principal.
  3. Haga clic en Aceptar.

Para eliminar asignaciones explícitas de grupos

  1. Siga los pasos 1 a 4 del procedimiento Para crear asignaciones explícitas de grupos para ir hasta la página Asignaciones explícitas de grupos.
  2. En la lista Grupos asignados de forma explícita, seleccione la asignación de grupo que desea eliminar.
  3. Elija Quitar.
  4. Haga clic en Aceptar.
