Modelo de desplazamiento de la interfaz de usuario Web

La parte superior de cada página de la interfaz de usuario (UI) Web consta de un área de estado, así como de barras de exploración principal y secundaria. El cuerpo de cada página de la UI Web está formado por el área de contenido.

A continuación se incluye una descripción de estas secciones:

Área de estado

La banda superior de la ventana, el área de estado, muestra lo siguiente de izquierda a derecha:

  • Logotipo del fabricante del módulo servidor basado en Windows (opcional)
  • Nombre de host y estado del módulo servidor
  • Logotipo de Microsoft Windows Powered

Hay cuatro valores posibles para Estado:

  • Normal (texto de color verde)
  • Información (texto de color gris)
  • Advertencia (texto de color amarillo)
  • Crítico (texto de color rojo)

Puede hacer clic en Estado: <tipoDeEstado> para obtener información detallada acerca del estado del módulo servidor.

Barras de exploración

Inmediatamente debajo del área de estado se encuentran las barras de exploración. La siguiente ilustración muestra las barras de exploración principal (azul) y secundaria (blanca) de un módulo servidor Web estándar.

La barra de exploración principal muestra las tareas disponibles del módulo servidor por tipo. La barra de exploración secundaria muestra las subtareas disponibles para la tarea principal seleccionada. La barra de exploración secundaria es dinámica y los tipos de tareas disponibles cambian dependiendo del grupo de tareas seleccionado.

El texto rotativo para los elementos de las barras de exploración proporciona más información. Mueva el cursor del mouse (ratón) sobre cada elemento para mostrar el texto rotativo.

Área de contenido

La información de la página se muestra en el área de contenido, que se encuentra bajo las barras de menús. En esta sección de la interfaz de usuario Web se describen las actividades de administración que puede realizar en dicha página. Este texto también puede proporcionar instrucciones acerca de las herramientas disponibles para realizar las tareas seleccionadas.

Muchas de las páginas de tareas incluyen un selector de objeto o de tarea. El selector de objeto o de tarea es una tabla en la que se enumeran los objetos que puede administrar o configurar, sus descripciones y las tareas que puede realizar. La columna izquierda del selector de objeto o de tarea contiene un botón de opción en el que debe hacer clic para seleccionar un objeto dado. En caso de que puedan seleccionarse varios objetos en el selector de objeto o de tarea, la columna izquierda contendrá una casilla de verificación en lugar de un botón de opción. La columna derecha muestra las tareas que puede realizar.

Para desplazarse por la interfaz de usuario Web

  1. En la barra de exploración principal, haga clic en el tipo de tarea general que desee realizar.
  2. En la barra de exploración secundaria o en la lista de tareas del área de contenido de la página, haga clic en el tipo de tarea específico que desee realizar.
  3. En el área de contenido de la página siguiente:
    1. Si hay un selector de objeto o de tarea disponible, haga clic en el botón de opción que está a la izquierda del nombre del objeto para seleccionar el objeto que desea administrar o configurar. A continuación, seleccione la tarea que desea realizar en la lista Tareas de la derecha. Dependiendo de la página, el selector de objeto y tarea puede proporcionar la posibilidad de realizar búsquedas, así como la posibilidad de recorrer la información contenida en la tabla.
    2. Si no hay disponible ningún selector de objeto y tarea, escriba los datos en los campos indicados para realizar la tarea elegida.
  4. Cuando haya terminado cada tarea, haga clic en Aceptar para confirmar las modificaciones o en Cancelar para conservar la configuración anterior.
Una vez procesada la modificación o la cancelación, se mostrará la página anterior.

Si se encuentra en una página de propiedades y selecciona otro elemento de la barra de exploración, una ventana emergente mostrará el mensaje Haga clic en Aceptar para descartar cualquier cambio realizado. Esto le ofrece la posibilidad de confirmar o rechazar las modificaciones antes de pasar a la siguiente página seleccionada.
