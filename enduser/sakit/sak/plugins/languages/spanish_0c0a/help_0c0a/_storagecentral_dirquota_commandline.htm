Directory Quota Command Line

Use the command line to manage directory quotas from Windows.  The following table shows the available command-line options.

Command switch Description
/show Shows existing quotas
/a Add
/d[y][a] Delete [y=Confirmation] [a=all]
/m Modify
/s Scan
/o Object name
/t Policy name, overrides /p,/e qualifiers
/p Space limit,  in bytes
/[-]e Active limit (default), -e Passive Limit


In the example >DirQuota /a /o c:\test /t "100 MB Limit", the command would add a directory quota to the c:\test directory and applies the 100 MB Limit policy to it.
