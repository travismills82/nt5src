FTP

El Protocolo de Transferencia de archivos (FTP) se utiliza para copiar archivos entres dos equipos en Internet. Ambos equipos deben admitir sus funciones FTP correspondientes: uno debe ser un cliente FTP y el otro un servidor FTP.

Puede configurar el módulo servidor para que actúe como servidor FTP. Los equipos cliente pueden tener acceso al módulo servidor con el Explorador de Windows® o con un programa FTP de línea de comandos, como ftp.exe, si escriben la dirección URL en la barra de direcciones con el formato ftp://nombreDeSitio/.
