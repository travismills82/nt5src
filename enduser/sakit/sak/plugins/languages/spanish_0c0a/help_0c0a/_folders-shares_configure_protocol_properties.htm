Configurar las propiedades del protocolo

Utilice la página de propiedades del protocolo designado para configurar los protocolos de red que desee.

Para configurar las propiedades del protocolo de red

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. Elija Protocolos para uso compartido.
  3. Seleccione el protocolo que desea configurar.
  4. En la lista Tareas, elija Propiedades.
