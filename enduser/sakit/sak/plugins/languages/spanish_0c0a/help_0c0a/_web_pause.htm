Pausar un sitio Web

El selector de objeto o de tarea de la página Configuración del sitio Web indica el estado de cada uno de los sitios Web que residen en el módulo servidor. Si es necesario, puede pausar cualquier sitio Web que esté en funcionamiento.

La pausa de un sitio Web impide que los servicios de Internet acepten conexiones nuevas pero no afecta a las solicitudes que ya se están procesando. La detención de un sitio Web impide que los servicios de Internet acepten conexiones nuevas y detendrá todas las solicitudes que ya se estén procesando.

Para pausar un sitio Web que está en funcionamiento

  1. En la barra de exploración principal, elija Sitios.
  2. Seleccione el identificador del sitio Web que desea pausar.
  3. En la lista Tareas, elija Pausar.
