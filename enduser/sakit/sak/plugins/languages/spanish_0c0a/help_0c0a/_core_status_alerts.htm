Alertas de estado

Cuando elige Estado: <tipoDeEstado> en el área de estado, se muestra una página Alerta.

La lista siguiente indica el tipo de contenido de alerta que produce cada tipo de estado:

  • Estado Normal: la página Alerta indicará que no hay ningún mensaje.
  • Estado Información: la página Alerta mostrará los detalles como hipervínculos, con un icono que contiene la letra i junto a cada detalle encontrado por el sistema.
  • Estado Advertencia: la página Alerta mostrará los errores como hipervínculos, con un triángulo amarillo que contiene un signo de exclamación (!) junto a cada error encontrado por el sistema.
  • Estado Crítico: la página Alerta mostrará los errores como hipervínculos, con un círculo rojo que contiene una x blanca junto a cada error encontrado por el sistema.

Para borrar una alerta

  1. En la barra de exploración principal, elija Estado.
  2. Elija la alerta que desea borrar.
  3. Elija la acción que desea realizar para las condiciones de alerta o elija Borrar mensaje para borrar el mensaje de alerta sin modificar la configuración del módulo servidor.
