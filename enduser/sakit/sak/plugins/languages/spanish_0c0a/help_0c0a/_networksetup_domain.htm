Dominio

El módulo servidor puede estar en un grupo de trabajo, en un entorno de Active Directory o en un dominio de Windows NT 4. En entornos de Microsoft Windows NT 4 y Microsoft Active Directory, un dominio es una colección de equipos definida por el administrador de una red que comparten una base de datos de directorio común.

Los dominios de Windows, utilizados para obtener información de usuarios y grupos de Windows, tienen un nombre único y proporcionan acceso a las cuentas de usuario y de grupo centralizadas que mantiene el administrador del dominio. Cada dominio tiene sus propias directivas de seguridad y relaciones de seguridad con otros dominios, y cada dominio representa un único límite de seguridad de una red de equipos con Windows. Active Directory está compuesto de uno o varios dominios, cada uno de los cuales puede abarcar más de una ubicación física.

Para DNS, un dominio es cualquier árbol o subárbol del espacio de nombres DNS. Aunque los nombres de los dominios DNS se suelen corresponder con los dominios de Active Directory, no deben confundirse con los dominios de red de Microsoft Windows y Active Directory.

De forma predeterminada, un usuario de un dominio tiene que escribir dominio\usuario para iniciar la sesión en un módulo servidor Web o en un NAS con un explorador Netscape.

Para establecer el dominio predeterminado para iniciar sesión

  1. En la barra de exploración principal, elija Mantenimiento.
  2. Elija Servicios de Terminal Server.
  3. Inicie sesión en Servicios de Terminal Server.
  4. Haga clic con el botón secundario del mouse (ratón) en Mi PC y elija Administrar.
  5. Abra Servicios de Internet Information Server (IIS).
  6. Haga clic con el botón secundario del mouse en Sitio Web de administración y elija Propiedades.
  7. Seleccione Seguridad de directorios.
  8. En el área Acceso anónimo y control de autenticación, elija Modificar.
  9. En el área Autenticación básica, elija Modificar.
  10. Escriba el nombre de dominio predeterminado que desea utilizar para la autenticación.
