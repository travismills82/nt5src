Establecer las propiedades de uso compartido de Web (HTTP)

Utilice esta página para especificar qué clientes HTTP tienen acceso a cada recurso compartido. Se puede conceder o denegar acceso para cada nombre de host del cliente. El acceso también se puede conceder o denegar por grupos de clientes, donde cada grupo de clientes contiene uno o varios nombres de host cliente.

Para conceder permisos a los clientes para un recurso compartido Web

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. En la página Recursos compartidos, elija Recursos compartidos.
  3. Seleccione el recurso compartido para el que desea agregar acceso para clientes HTTP y, a continuación, elija Propiedades.
  4. En la ficha General, active la casilla de verificación Web (HTTP).
  5. Seleccione la ficha Uso compartido HTTP y, a continuación:
    • Active la casilla de verificación Leer para permitir el acceso de lectura.
- O bien -
  • Active la casilla de verificación Escribir para permitir el acceso de escritura.
  1. Haga clic en Aceptar.
