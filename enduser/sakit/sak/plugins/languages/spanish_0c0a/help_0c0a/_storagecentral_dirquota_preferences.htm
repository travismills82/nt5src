Directory Quota Preferences

Use the Preferences page to configure the directory quota settings. You can select the Auto detect disks check box to automatically create a partition-level directory quota for new and existing devices.  You must specify a policy to apply to the directory quota when a new drive is detected. 

When a policy is selected from the Copy policy settings from list and the Auto detect disks check box is selected, only severe and warning alarm values are applied from the policy to the newly-detected disk.
