Modifying File Screening Group Properties

To modify the properties of a file screening group properties

  1. On the primary navigation bar, choose Disks.
  2. Choose File Screening.
  3. Select the directory name you want, and then choose Policies from the Tasks list.
  4. In the Tasks list, choose Groups.
  5. Select the file screening group you want to modify.
  6. In the Tasks list, choose Properties.
  7. Make any changes you want.
  8. Choose OK.
