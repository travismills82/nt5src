Protocolos para uso compartido

La página Protocolos para uso compartido le permite habilitar, deshabilitar, detener o configurar los protocolos de red relevantes. La página Protocolos para uso compartido muestra el Selector de objeto o de tarea, que tiene las columnas siguientes:

  • Nombre: muestra el nombre de los protocolos. Para habilitar, deshabilitar o cambiar las propiedades de un protocolo dado, haga clic en el botón que se encuentra junto al protocolo que desea modificar.
  • Estado: indica que el protocolo está En ejecución, Detenido o En pausa.
  • Tipo de inicio: indica si el protocolo debe iniciarse automáticamente cuando se inicie el módulo servidor, debe invocarse manualmente o está deshabilitado.
  • Descripción: muestra una breve descripción del protocolo.

    La lista Tareas se encuentra junto al Selector de objeto o de tarea. Utilice la columna Nombre del Selector de objeto o de tarea para seleccionar un protocolo. Para realizar una tarea, elija la tarea apropiada en la lista Tareas.
