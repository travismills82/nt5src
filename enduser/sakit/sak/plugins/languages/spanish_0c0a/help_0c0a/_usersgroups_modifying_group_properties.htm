Modificar las propiedades del grupo

La página Propiedades del grupo muestra las fichas General y Miembros. Utilice la ficha General para establecer o modificar el nombre y la descripción del grupo. Utilice la ficha Miembros para agregar o quitar usuarios y grupos.

Para establecer o modificar el nombre o la descripción de grupo

  1. En la barra de exploración principal, elija Usuarios.
  2. Elija Grupos locales.
  3. Seleccione la cuenta de grupo que desea modificar.
  4. En la lista Tareas, elija Propiedades.
  5. En la ficha General, escriba el nombre y la descripción del grupo deseado.
  6. Haga clic en Aceptar.

Para establecer o modificar la pertenencia al grupo

  1. En la barra de exploración principal, elija Usuarios.
  2. Elija Grupos locales.
  3. Seleccione la cuenta de grupo que desea modificar.
  4. En la lista Tarea, elija Propiedades y, a continuación, seleccione la ficha Miembros.
  5. Para agregar un miembro nuevo:
    • En el cuadro Agregar un usuario o un grupo, seleccione un usuario o un grupo local en la lista y, a continuación, haga clic en el botón Agregar.

    - O bien -

    • Escriba el dominio y el nombre (<dominio\nombreDeGrupo>) de un grupo del dominio, o la cuenta de usuario del dominio (<dominio\nombreDeUsuario>) y, a continuación, elija Agregar. Sin embargo, si va a agregar Dominio\Grupo, también debe escribir las credenciales que permitirán agregar desde dicho dominio. 
  6. Para quitar un miembro, seleccione un nombre de usuario en la lista Miembros y, a continuación, elija Quitar.
  7. Haga clic en Aceptar.
