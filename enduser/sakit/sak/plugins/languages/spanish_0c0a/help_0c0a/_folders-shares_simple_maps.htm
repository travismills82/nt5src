Habilitar asignaciones simples

Si están habilitadas, las asignaciones simples crean asignaciones automáticas entre usuarios de UNIX y usuarios de Microsoft Windows que tengan el mismo nombre de usuario. En una asignación de usuario simple, los usuarios de un dominio de Windows quedan implícitamente asignados de forma unívoca con usuarios de UNIX por su nombre de usuario. Cuando se identifican el dominio Windows y los archivos de contraseñas y grupos de UNIX o dominios NIS, la asignación simple asigna los usuarios que tienen el mismo nombre en Windows y en UNIX o el dominio NIS. Si no se encuentra un usuario con el mismo nombre en uno de los dos lados, no se asigna dicho usuario.

Nota   Para tener acceso a esta página debe haber escrito un nombre de servidor NIS válido en la ficha General.

Para habilitar las asignaciones simples

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. Elija Protocolos para uso compartido.
  3. Seleccione Protocolo NFS y, a continuación, elija Propiedades.
  4. Elija Asignaciones de usuarios y grupos.
  5. Seleccione la ficha Asignaciones simples.
  6. Active la casilla de verificación Habilitar asignaciones simples.
  7. En la lista Dominio de Windows, seleccione el nombre del módulo servidor o el dominio al que pertenece el módulo servidor.

    Si selecciona el nombre del módulo servidor, se asignarán los usuarios y los grupos locales.

  8. Haga clic en Aceptar.
