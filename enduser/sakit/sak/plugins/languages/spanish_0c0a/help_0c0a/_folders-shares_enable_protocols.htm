Habilitar protocolos

Microsoft recomienda habilitar únicamente los protocolos de red necesarios. La limitación del número de protocolos de red habilitados aumentará el rendimiento de otros protocolos de red. Además, si se encuentra un problema en una conexión de red o de acceso telefónico, el sistema intentará establecer la conexión mediante todos los protocolos de red instalados y habilitados. Al habilitar sólo los protocolos que el sistema puede utilizar, el módulo servidor puede ahorrar recursos y tener mejor rendimiento.

Para habilitar un protocolo de uso compartido

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. Elija Protocolos para uso compartido.
  3. Seleccione el protocolo que desea habilitar.
  4. En la lista Tareas, elija Habilitar.
