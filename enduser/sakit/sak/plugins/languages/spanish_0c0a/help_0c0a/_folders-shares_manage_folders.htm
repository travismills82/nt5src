Administrar carpetas

La página Volúmenes le permite abrir, o compartir, varios volúmenes de red. La página muestra un selector de objeto o de tarea que tiene las columnas siguientes:

  • Nombre del volumen: muestra los nombres de los volúmenes. Para crear, abrir, eliminar o configurar las propiedades de un volumen dado, active la casilla de verificación que se encuentra junto al nombre del volumen que desea modificar.
  • Tamaño total: muestra el tamaño total del volumen.
  • Espacio libre: muestra la cantidad de espacio libre disponible en el volumen.
  • Tipo de recurso compartido: indica el tipo de uso compartido para la carpeta:

    W = Uso compartido de Windows (CIFS)

    U = Uso compartido de UNIX (NFS)

    F = Uso compartido de FTP

    H = Uso compartido de HTTP

    A = Uso compartido de AppleTalk

    N = Uso compartido de NetWare

El selector de objeto o de tarea muestra las 100 primeras carpetas encontradas. Para recorrer la lista de carpetas utilizando el selector de objeto o de tarea, puede buscar por los campos disponibles en la lista Buscar y, a continuación, escribir los criterios de búsqueda en el cuadro de la izquierda del botón Ir, o bien puede desplazarse por la lista. Además, si hay más de 100 carpetas, puede ver carpetas en grupos de 100 mediante los botones Página anterior y Página siguiente que hay a la derecha del botón Ir.

Para compartir carpetas

  1. En la página Recursos compartidos, elija Carpetas.
  2. Seleccione los volúmenes para los que desea compartir carpetas y, a continuación, elija Compartir en la lista Tareas.
  3. Escriba la información que se le pida. Para obtener más información acerca de cómo completar la página de propiedades para un método de compartir archivos específico, consulte el tema de Ayuda de cada protocolo.

Para administrar carpetas

  1. En la página Recursos compartidos, elija Carpetas.
  2. Seleccione los volúmenes para los que desea ver o administrar carpetas o recursos compartidos y elija Abrir en la lista Tareas.

    La página Carpetas le permite crear, abrir, eliminar o configurar varias carpetas de red. La página muestra un selector de objeto o de tarea que tiene las columnas siguientes:

    Nombre de carpeta: muestra los nombres de las carpetas.

    Fecha de modificación: muestra la fecha que se modificó por última vez la carpeta.

    Atributos: muestra cualquiera de los siguientes atributos de carpeta:

    R = Sólo lectura

    A = Listo para archivar

    H = Oculto

    C = Comprimido

    S = Carpeta del sistema

    Tipo de recurso compartido: indica el tipo de uso compartido para la carpeta.

  3. Seleccione una carpeta y, a continuación, elija la tarea que desea realizar en la lista Tareas.
  4. Para desplazarse a los subdirectorios de una carpeta raíz, seleccione la carpeta raíz y, a continuación, elija Abrir en la lista Tareas.
  5. Para ir desde un subdirectorio a su principal, elija Arriba en la lista Tareas. Si en el selector de objeto o de tarea se muestran las carpetas raíz, no se pueden elegir carpetas principales.
