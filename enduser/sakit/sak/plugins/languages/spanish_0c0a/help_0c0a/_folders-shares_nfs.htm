NFS

Con el protocolo NFS, un módulo servidor puede actuar como servidor de Sistema de archivos de red (NFS). Los usuarios pueden compartir archivos en un entorno mixto de equipos, sistemas operativos y redes.

Los usuarios de equipos que ejecutan software cliente de NFS pueden tener acceso a los recursos compartidos del módulo servidor si conectan, o montan, dichos recursos compartidos a sus equipos.

Los equipos UNIX siguen la recomendación de bloqueo para todas las solicitudes de bloqueo. Esto significa que el sistema operativo no impone la semántica de bloqueo en un archivo y las aplicaciones que comprueban la existencia de bloqueos pueden utilizar dichos bloqueos de forma efectiva. Sin embargo, el Protocolo NFS implementa bloqueos obligatorios incluso para aquellas solicitudes de bloqueo que se reciben a través de NFS. Esto asegura que los bloqueos adquiridos a través de NFS son visibles a través del protocolo Bloque de mensajes del servidor (SMB) y para las aplicaciones que tienen acceso a los archivos de forma local. El SO impone bloqueos obligatorios.
