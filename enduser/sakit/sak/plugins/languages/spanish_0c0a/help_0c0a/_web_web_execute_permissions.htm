Configurar permisos de ejecución en Web

Esta página le permite establecer los permisos predeterminados para los sitios del módulo servidor como secuencias de comandos y ejecutables, sólo secuencias de comandos o ninguno.

Para establecer los permisos de ejecución en Web

  1. En la barra de tareas principal, elija Servidor Web.
  2. Elija Permisos de ejecución en Web.
  3. Seleccione los permisos predeterminados que desee en la lista Permisos de ejecución predeterminados.
  4. Haga clic en el botón apropiado para aplicar esta configuración a todos los sitios Web que utilizan el valor predeterminado.
  5. Haga clic en Aceptar.
