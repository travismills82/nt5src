Configuración de direcciones IP

Cada equipo de la red debe tener una dirección IP única para enviar y recibir datos. Puede utilizar la página Configuración de la dirección IP para que el módulo servidor obtenga automáticamente la configuración de la dirección IP desde el servidor del Protocolo de configuración dinámica de hosts (DHCP). Como alternativa, puede configurar las direcciones IP de forma manual.

Además, puede utilizar la página Configuración de la dirección IP para especificar una o más direcciones de puerta de enlace. La dirección de una puerta de enlace es la dirección de un enrutador IP local que reside en la misma red que el módulo servidor q ue se utiliza para reenviar el tráfico a los destinos externos a la red local. El valor de cada campo debe ser un número entre 0 y 255.

Nota   El cambio de la dirección IP puede hacer que el cliente pierda su conexión con el módulo servidor. Para volver a conectar, el usuario debe utilizar la nueva dirección IP o esperar a que se actualice el servidor DNS.

Para establecer o cambiar la configuración IP en la ficha General

  1. En la barra de exploración principal, elija Red.
  2. Elija Interfaces.
  3. Seleccione la conexión de red que desea modificar.
  4. En la lista Tareas, elija IP.
  5. Seleccione la ficha General.
  6. Seleccione si va a obtener la configuración automáticamente desde el servidor DHCP o si va a configurar las direccione IP de manera estática.
  7. Si elige obtener la configuración desde el servidor DHCP, haga clic en Aceptar.
  8. Si elige utilizar una configuración IP estática, escriba la dirección IP, la máscara de subred y la puerta de enlace predeterminada en los cuadros de texto proporcionados para ello.

Para establecer o cambiar la configuración IP en la ficha Avanzada

  1. En la barra de exploración principal, elija Red.
  2. Elija Interfaces.
  3. Seleccione la conexión de red que desea modificar.
  4. En la lista Tareas, elija IP.
  5. Seleccione la ficha Avanzada.
  6. En el cuadro Dirección IP de la derecha, escriba la dirección IP y, a continuación, haga clic en Agregar.
  7. Si tiene una conexión de área local, escriba la información de máscara apropiada en el cuadro Máscara de subred.

    La máscara de subred es un número de 32 bits que se expresa con cuatro números comprendidos entre 0 y 255, separados por puntos. Normalmente, los números de máscara de subred predeterminados utilizan los valores 0 ó 255, como 255.255.255.0. Sin embargo, pueden aparecer otros valores numéricos, lo que indica que la subred está configurada para una única red TCP/IP. Este número, con un valor distinto de 0 ó 255, se combina con el número de la dirección IP para identificar la red en la que reside un equipo.

  8. Si es necesario, actualice Métrica de la conexión IP.

    La métrica indica el costo que supone usar las rutas asociadas con esta conexión y se convierte en el valor de la columna Métrica para dichas rutas en la tabla de enrutamiento IP. Si hay varias rutas para un destino, se utilizará la ruta cuya métrica sea menor. El valor predeterminado es 1.

  9. Repita los pasos 1 a 3 para todas las direcciones IP que desee agregar.

Para establecer o cambiar la configuración de la dirección de la puerta de enlace

  1. En los cuadros Puerta de enlace y en Métrica, escriba la dirección IP de la puerta de enlace predeterminada y la métrica y, a continuación, haga clic en Agregar.
  2. Repita este paso para cada puerta de enlace predeterminada que desee agregar.
  3. Cuando haya terminado de modificar las configuraciones en esta pantalla, haga clic en Aceptar.
