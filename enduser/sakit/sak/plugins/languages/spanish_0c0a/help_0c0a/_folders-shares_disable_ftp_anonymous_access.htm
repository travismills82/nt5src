Deshabilitar el acceso anónimo a FTP

Para deshabilitar el FTP anónimo

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. Elija Protocolos para uso compartido.
  3. Seleccione Protocolo FTP y, a continuación, elija Propiedades.
  4. Seleccione la ficha Acceso anónimo.
  5. Desactive la casilla de verificación Habilitar acceso anónimo y, a continuación, haga clic en Aceptar.
