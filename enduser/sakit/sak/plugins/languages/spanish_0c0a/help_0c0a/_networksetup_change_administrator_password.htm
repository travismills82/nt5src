Administrador

El módulo servidor viene con un conjunto de cuentas predeterminadas. Sólo la cuenta de administrador tiene privilegios administrativos.

Nota   Si un administrador agrega una cuenta de dominio al grupo de administradores locales, el usuario del dominio puede tener acceso y administrar el módulo servidor. Sin embargo, la cuenta de administrador no puede utilizar la página Cambiar la contraseña de administrador para cambiar la contraseña de su cuenta de dominio. Esta página sólo se puede utilizar para cambiar la contraseña de la cuenta del administrador local.

Cuando cambia la contraseña del administrador, no hay ninguna página de confirmación explícita. Sin embargo, la contraseña se cambia correctamente si no aparece ningún mensaje de error después de hacer el cambio.

Nota   No puede cambiar la contraseña del administrador si ha iniciado sesión como usuario del dominio, ya que está fuera del alcance de la interfaz de usuario Web del módulo servidor hacer modificaciones en las cuentas de usuario del dominio, que están almacenadas en el controlador del dominio, no en el módulo servidor.

En este contexto, el administrador es la cuenta de usuario que es miembro del grupo local administradores y el usuario actual la utiliza para iniciar sesión. No hace referencia a la cuenta de administrador predeterminada, llamada administrador.

Si recibe el mensaje de error No se puede cambiar la contraseña de esta cuenta de dominio o No se puede cambiar el nombre de cuenta de esta cuenta de dominio cuando trata de cambiar la contraseña o el nombre de cuenta del administrador, ha iniciado la sesión como usuario del dominio. Para poder cambiar la contraseña del administrador debe haber iniciado la sesión como administrador del módulo servidor.

Para cambiar el nombre de cuenta del administrador del módulo servidor

  1. Inicie una sesión en el módulo servidor como Administrador.
  2. En la barra de exploración principal, elija Red.
  3. Elija Administrador.
  4. Escriba la contraseña actual del administrador en el cuadro Contraseña actual.
  5. Escriba la nueva contraseña del administrador en el cuadro Nueva contraseña.

    Nota  La nueva contraseña del administrador debe cumplir las reglas de complejidad de contraseñas que estén en vigor en el dominio al que pertenece el módulo servidor.

  6. Vuelva a escribir la nueva contraseña del administrador en el cuadro Confirmar nueva contraseña.
  7. Haga clic en Aceptar.

Para cambiar el nombre de cuenta del administrador del módulo servidor

  1. Inicie una sesión en el módulo servidor como Administrador.
  2. Elija Red.
  3. Elija Administrador.
  4. Escriba el nuevo nombre de la cuenta del administrador en el cuadro Nombre de usuario.
  5. Haga clic en Aceptar.
