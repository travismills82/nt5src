Configuración del maestro FTP

Permitir conexiones FTP anónimas es la manera de habilitar o deshabilitar el acceso anónimo desde los clientes FTP al módulo servidor.

El cuadro Directorio raíz de FTP muestra el directorio raíz de FTP al que se tiene acceso cuando se conecta mediante FTP anónimo.

El cuadro Estilo del listado del directorio define el formato en el que se enumeran los directorios cuando un cliente FTP envía una operación LIST.

  • El estilo de listado MS-DOS muestra el año con formato de dos dígitos de forma predeterminada.
  • El estilo de listado UNIX muestra el año con formato de cuatro dígitos cuando la fecha del archivo es diferente del año del servidor FTP. Si el año del archivo es el mismo que el del servidor, en la fecha no se mostrará la parte del año.

Número máximo de conexiones es el número máximo de conexiones FTP simultáneas permitidas en el módulo servidor.

Tiempo de espera de la conexión, en segundos es el número de segundos de inactividad antes de que el módulo servidor se desconecte automáticamente.
