Modifying a Directory Quota Policy

To modify the properties of a policy 

  1. On the primary navigation bar, choose Disks.
  2. Choose Directory Quota.
  3. In the Tasks list, choose Policies.
  4. Select the policy you want to modify.
  5. In the Tasks list, choose Properties.
  6. Select the tab you want to modify, and then make any changes you want.
  7. Choose OK.
