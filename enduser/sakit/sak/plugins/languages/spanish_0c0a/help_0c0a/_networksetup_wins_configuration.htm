Configuración WINS

Esta página de propiedades le permite escribir las direcciones de los servidores WINS externos. El módulo servidor no contiene un servidor WINS. Para el propósito de este tema, el módulo servidor es un cliente WINS.

Los equipos cliente preparados para WINS pueden configurarse para que utilicen directamente un servidor WINS. La mayoría de los equipos cliente WINS suelen tener varios nombres del Sistema básico de entrada/salida de red (NetBIOS) y tienen que registrarlos en la red. Estos nombres se utilizan para publicar varios tipos de servicios de red, como Messenger o Estación de trabajo, que los equipos pueden utilizar de varias formas para comunicarse con otros equipos de la red.

Los clientes preparados para WINS se comunican con el servidor WINS para poder completar las tareas siguientes:

  • Registrar los nombres de los clientes en la base de datos WINS.
  • Renovar los nombres de los clientes en la base de datos WINS.
  • Liberar los nombres de los clientes en la base de datos WINS.
  • Resolver nombres a través de las asignaciones que la base de datos WINS tiene para nombres de usuario, nombres NetBIOS, nombres DNS y direcciones IP.

Los clientes que no están preparados para WINS pueden participar en estos procesos de forma limitada a través de servidores proxy WINS. Si utiliza un servidor DHCP para asignar las direcciones IP de servidor WINS, no tendrá que agregar direcciones de servidor WINS.

Tenga en cuenta que la interfaz de usuario Web sólo le permite manipular dos direcciones WINS, y sólo si asigna estáticamente la dirección IP del adaptador. Si tiene DHCP habilitado, puede quitar una o dos direcciones existentes y agregar direcciones diferentes, pero no podrá quitar todos los servidores WINS de un adaptador preparado para DHCP. Si quita dos direcciones WINS y no agrega al menos una, DHCP asignará automáticamente las direcciones WINS.

Los clientes WINS registran sus nombres en un servidor WINS al iniciarse o cuando se unen a la red. A partir de ese momento, los clientes WINS consultan al servidor WINS para resolver los nombres remotos cuando lo necesiten.

Para cambiar la configuración WINS del módulo servidor

  1. En la barra de exploración principal, elija Red.
  2. Elija Interfaces.
  3. Seleccione la conexión de red que desea modificar.
  4. En la lista Tareas, elija WINS.
  5. En la lista Servidores WINS, seleccione la dirección IP del servidor WINS que desea eliminar y, a continuación, haga clic en Quitar.
  6. En el cuadro Dirección del servidor WINS, escriba la dirección IP del servidor WINS y, a continuación, haga clic en Agregar.
  7. Repita los pasos 4 y 5 para dirección IP de servidor WINS que desee agregar.
  8. Haga clic en Aceptar.
