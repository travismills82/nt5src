Descargar archivos de registro

En esta página puede descargar archivos de registro específicos en el módulo servidor.

Para descargar registros de aplicación, seguridad o sistema

  1. En la barra de exploración principal, elija Mantenimiento.
  2. Elija Registros.
  3. Elija el tipo de registro que desea descargar.
  4. Elija Descargar.
  5. Seleccione Guardar este archivo en disco.
  6. Haga clic en Aceptar para descargar el archivo.

Para descargar registros de NFS

  1. En la barra de exploración principal, elija Mantenimiento.
  2. Elija Registros.
  3. Elija Registros NFS.
  4. Elija Descargar.
  5. Seleccione Guardar este archivo en disco.
  6. Haga clic en Aceptar.

Para descargar registros de FTP, de administración Web o de recursos compartidos Web (HTTP)

  1. En la barra de exploración principal, elija Mantenimiento.
  2. Elija Registros.
  3. Seleccione el registro que va a descargar.
  4. Seleccione el archivo de registro que va a descargar.
  5. Elija Descargar.
  6. Seleccione Guardar este archivo en disco.
  7. Haga clic en Aceptar.
