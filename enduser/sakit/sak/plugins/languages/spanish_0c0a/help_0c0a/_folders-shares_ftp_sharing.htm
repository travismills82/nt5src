Establecer las propiedades de uso compartido de FTP

Utilice esta página para especificar qué clientes FTP tienen acceso a cada recurso compartido. Se puede conceder o denegar acceso para cada nombre de host del cliente.

Para conceder permisos a los clientes para un recurso compartido de FTP

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. En la página Recursos compartidos, elija Recursos compartidos.
  3. Seleccione el recurso compartido para el que desea agregar acceso para clientes FTP y, a continuación, elija Propiedades.
  4. Seleccione la ficha FTP y, a continuación:
    • Active la casilla de verificación Leer para permitir el acceso de lectura.
- O bien -
  • Active la casilla de verificación Escribir para permitir el acceso de escritura.
Puede elegir entre conceder permisos de sólo lectura, de sólo escritura o de lectura y escritura.
  1. Haga clic en Aceptar.

Para registrar las visitas de los clientes a un recurso compartido de FTP

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. En la página Recursos compartidos, elija Recursos compartidos.
  3. Seleccione el recurso compartido para el que desea agregar acceso para clientes FTP y, a continuación, elija Propiedades.
  4. Seleccione la ficha Uso compartido FTP.
  5. Active la casilla de verificación Registrar visitas.
  6. Haga clic en Aceptar.
