Modificar las propiedades de un recurso compartido

Utilice la página Recursos compartidos para ver y modificar las propiedades de los recursos compartidos.

Para modificar las propiedades de los recursos compartidos

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. En la página Recursos compartidos, elija Recursos compartidos.
  3. Seleccione el recurso compartido cuyas propiedades va a modificar.
  4. En la lista Tareas, elija Propiedades.

    Se mostrará la página de propiedades General. Utilice esta página para cambiar el tipo de cliente desde el que es accesible el recurso compartido.

  5. Haga clic en Aceptar.
