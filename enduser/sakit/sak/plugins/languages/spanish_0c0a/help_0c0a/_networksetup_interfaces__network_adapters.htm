Interfaces: adaptadores de red

Un adaptador de red proporciona la interfaz física, o conector, y el hardware que permite a un equipo tener acceso a una red. También se suele llamar al adaptador de red tarjeta adaptadora, tarjeta de interfaz de red (NIC) o simplemente tarjeta.

En la página Interfaces de la UI Web del módulo servidor puede realizar una de las tareas siguientes:

  • Cambiar el el nombre de la conexión.
  • Establecer o modificar las direcciones del Protocolo de Internet (IP), las direcciones de las puertas de enlace, las máscaras de subred y las métricas.
  • Establecer o modificar cómo el módulo servidor resuelve los nombres DNS.
  • Establecer o cambiar la configuración de los clientes del Servicio de nombres Internet de Windows (WINS).
  • Configurar AppleTalk, si está habilitado por el fabricante del módulo servidor.

Para configurar un adaptador de red

  1. Seleccione el adaptador que desee en la columna Descripción.
  2. Seleccione la tarea que desee en la lista Tareas.
  3. En la página Tarea, utilice las fichas del panel izquierdo para mostrar la configuración del adaptador de red elegido.
