Disaster Recovery Disk

The disaster recovery system will create a disk or folder image containing all the files necessary to boot and recover the system volume. Before you can use the files, you must copy them to a formatted and bootable disk.

To format a disk

  1. Insert a disk into the disk drive.
  2. Open Windows Explorer.
  3. Select My Computer.
  4. Right-click 3 1/2 Floppy (A:), and then choose Format.
  5. Choose Start.

    The system will prepare the disk to hold files.

To copy files to the disk

  1. Open Windows Explorer.
  2. Select the folder containing the image files.
  3. Select all image files and folders in the window.
  4. Select the Copy button on the toolbar.
  5. In the Folders pane, select My Computer.
  6. Insert a formatted disk into the disk drive.
  7. In the Folders pane, choose 3 1/2 Floppy (A:).
  8. Select the Paste button on the toolbar.

To make the disk bootable

  1. Insert the formatted disk in the disk drive.
  2. Open Windows Explorer.
  3. In the Folders pane, select My Computer.
  4. Choose 3 1/2 Floppy (A:).
  5. Open FixBoot.
  6. When prompted, press any key to complete the process.
  7. After the window closes, remove the disk.
