Administrar registros de recursos compartidos Web (HTTP)

El registro de recursos compartidos Web (HTTP) contiene sucesos registrados por el servidor Web relacionados con el acceso a los recursos compartidos HTTP.

Para administrar los registros de recursos compartidos Web (HTTP)

  1. En la barra de exploración principal, elija Mantenimiento.
  2. Elija Registros.
  3. Elija Registro de recursos compartidos Web (HTTP).
