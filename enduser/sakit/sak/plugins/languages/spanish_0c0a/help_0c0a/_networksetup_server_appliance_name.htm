Nombre del módulo servidor

El nombre del módulo servidor es el nombre del módulo servidor en una red. El nombre del módulo servidor debe ser único y cumplir ciertos requisitos. El nuevo nombre del módulo servidor no puede ser el mismo que el de otro equipo o que el nombre de un dominio de Microsoft Windows.

Se recomienda que utilice nombres que no contengan más de 15 caracteres. El nombre del módulo servidor puede tener 63 caracteres como máximo, pero sólo debe contener los números del 0 al 9, las letras mayúsculas de la A a la Z, las letras minúsculas de la a a la z o guiones. Puede utilizar otros caracteres, pero si lo hace puede impedir que otros usuarios encuentren su equipo en la red. Si la red utiliza el servidor DNS de Microsoft, puede utilizar cualquier carácter excepto puntos. Si están instalados otros protocolos de red sin TCP/IP, el nombre del módulo servidor está limitado a 15 caracteres.

Si especifica un nombre de módulo servidor de más de 15 caracteres y desea que el dominio de Microsoft Active Directory reconozca nombres más largos, el administrador del dominio debe habilitar el registro de nombres DNS con 16 caracteres o más.
