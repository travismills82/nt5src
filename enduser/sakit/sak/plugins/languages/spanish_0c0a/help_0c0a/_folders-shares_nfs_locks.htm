Bloqueos NFS

Los bloqueos NFS permiten que un proceso tenga acceso exclusivo a una parte o a todo un archivo. El bloqueo de archivos se implementa tanto en el módulo servidor como en el cliente. Cuando un archivo está bloqueado, la caché de búfer no se utiliza para dicho archivo y todas las solicitudes de escritura se envían inmediatamente al servidor.

Después de producirse errores en el sistema, cuando se reinicia el módulo servidor, éste intenta restaurar el estado de bloqueo de los archivos a su condición anterior. Si el error se produce en el cliente, el módulo servidor liberará el bloqueo del archivo. Sin embargo, después del reinicio del cliente éste dispone de un corto período de tiempo para recuperar el bloqueo del archivo.

Para administrar los bloqueos NFS

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. Elija Protocolos para uso compartido.
  3. Seleccione Protocolo NFS y, a continuación, elija Propiedades.
  4. Elija Bloqueos.
  5. Opcional: En la lista Bloqueos actuales, seleccione el cliente cuyos bloqueos desea liberar.
  6. Opcional: En el cuadro Período de espera, escriba el número de segundos después del reinicio que el módulo servidor espera para volver a establecer un bloqueo de archivo con un cliente.
  7. Haga clic en Aceptar.
