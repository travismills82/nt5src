Configurar el registro de FTP

Esta página se utiliza para configurar el registro de FTP.

Para configurar el registro de FTP

  1. En la barra de tareas principal, elija Servidor Web.
  2. Elija Configuración del registro de FTP.
  3. Active la casilla de verificación Habilitar registro.
  4. Seleccione el formato de registro apropiado en el cuadro Formato de registro activo.
  5. Haga clic en el botón apropiado para establecer el intervalo de tiempo necesario para generar nuevos archivos de registro.

    Por ejemplo, si hace clic en el botón Diario se abrirá un nuevo archivo de registro cada 24 horas.

  6. Active la casilla de verificación Utilizar la hora local del servidor para la nomenclatura de archivos y para deshacer las acciones.

    El módulo servidor utilizará su hora local para la asignación de nombres de archivos y para deshacer las acciones del registro. De lo contrario, el servidor utilizará la hora del meridiano de Greenwich (GMT).

    Nota   Si hace clic en el botón Cuando el tamaño del archivo alcance, la casilla de verificación Utilizar la hora local del servidor para la nomenclatura de archivos y para deshacer las acciones estará desactivada.

  7. Compruebe que el cuadro Directorio del archivo de registro contiene la ruta de acceso al directorio correcto para los archivos de registro.
  8. Haga clic en Aceptar.
