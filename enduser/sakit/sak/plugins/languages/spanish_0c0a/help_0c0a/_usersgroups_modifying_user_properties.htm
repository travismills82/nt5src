Modificar las propiedades del usuario

Las propiedades del usuario incluyen el nombre de usuario, el nombre completo y la descripción. En la página Propiedades del usuario también puede habilitar o deshabilitar una cuenta de usuario. Además, también puede modificar las propiedades de varias cuentas de usuario de una vez si selecciona todas las cuentas que desea modificar antes de abrir la página Propiedades.

Para tener acceso a las propiedades del usuario

  1. En la barra de exploración principal, elija Usuarios.
  2. Elija Usuarios locales.
  3. Seleccione las cuentas de usuario que desea modificar.
  4. En la lista Tareas, elija Propiedades.
  5. Modifique las propiedades del usuario que desee.

    El campo Directorio de inicio especifica el nuevo directorio que se va a crear, y al que el usuario tendrá permiso de acceso exclusivo. El nombre del directorio es el mismo que el nombre de usuario definido anteriormente y se ubicará en la ruta de acceso especificada.

  6. Haga clic en Aceptar.
