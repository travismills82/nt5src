Sitio Web de administración

Esta característica le permite modificar las direcciones IP y el puerto que pueden utilizarse para tener acceso al sitio Web de administración del módulo servidor.

La dirección IP predeterminada a la que responde, o donde escucha, el módulo servidor se suele cambiar cuando el módulo servidor sólo se administra en una subred o en una red de administración diferente.

Los puertos de escucha predeterminados para los accesos cifrado y no cifrado se pueden modificar según sea necesario para que funcionen con software y configuraciones de red existentes, por ejemplo cuando no está permitido el tráfico por encima de un número de puerto dado.

Para cambiar las propiedades del sitio Web de administración

  1. En la barra de exploración principal, elija Red.
  2. Elija Sitio Web de administración.
  3. En la página Propiedades del sitio de administración:
    1. Especifique si va a utilizar Todas las direcciones IP o Sólo esta dirección IP.
    2. Si elige Sólo esta dirección IP, utilice la lista para seleccionar la dirección IP que desea utilizar.
    3. Si va a cambiar el puerto para el acceso no cifrado, escriba el nuevo número de puerto en el cuadro Puerto para el acceso sin cifrar.
    4. Si va a cambiar el puerto para el acceso cifrado, escriba el nuevo número de puerto en el cuadro Puerto para el acceso cifrado (SSL).
  4. Haga clic en Aceptar.
