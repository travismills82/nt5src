Deshabilitar protocolos

Para deshabilitar protocolos de uso compartido

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. Elija Protocolos para uso compartido.
  3. Seleccione el protocolo que desea deshabilitar.
  4. En la lista Tareas, elija Deshabilitar.

Temas relacionados

Para obtener más información acerca de cómo configurar las propiedades de un protocolo, consulte Configurar las propiedades del protocolo. Para obtener más información acerca de cómo habilitar protocolos, consulte Habilitar protocolos.
