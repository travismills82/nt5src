Modificar las propiedades de registro

Puede configurar las propiedades de los registros de aplicación, sistema y seguridad. En esta página puede especificar el tamaño de registro máximo y determinar cómo el sistema controla las entradas de registro cuando se ha alcanzado la capacidad máxima del módulo servidor.

Para modificar las propiedades de un archivo de registro

  1. En la barra de exploración principal, elija Mantenimiento.
  2. Elija Registros.
  3. Elija el registro de Aplicación, Sistema o Seguridad que desea configurar.
  4. Seleccione el registro que desea configurar.
  5. Elija Propiedades.
  6. En el cuadro Tamaño máximo del registro, escriba el tamaño máximo del registro, en kilobytes.
  7. Haga clic en el botón que represente mejor cómo desea controlar las entradas de registro cuando se alcanza el tamaño máximo del registro.
