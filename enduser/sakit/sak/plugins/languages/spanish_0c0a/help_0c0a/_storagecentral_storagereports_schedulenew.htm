Scheduling a Storage Report

To create a scheduled report

  1. On the primary navigation bar, choose Disks.
  2. Choose Storage Reports.
  3. Select the report you want.
  4. In the Tasks list, choose Schedule.
  5. In the Tasks list, choose New.
  6. On the Location tab, type a directory name to report on and select a report to run.

    7. Note If you choose OK from this page, the report will execute immediately.

  7. On the Date and Time tab, select options for Frequency, Start date, Start time, and Day.
  8. Choose OK.
