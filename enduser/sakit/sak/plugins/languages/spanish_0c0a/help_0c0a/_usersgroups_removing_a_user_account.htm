Quitar una cuenta de usuario

Con la excepción de la última cuenta que quede y de su cuenta personal, puede quitar todas las cuentas de usuario que haya creado en el módulo Microsoft Windows Powered. Además, puede quitar varias cuentas de usuario de una vez. Sin embargo, si quita la única cuenta de usuario del módulo servidor, la seguridad queda deshabilitada.

Cuando quite cuentas de usuario, tenga en cuenta lo siguiente:

  • No se pueden eliminar los usuarios integrados.
  • No se puede recuperar un usuario eliminado.

    Si elimina una cuenta de usuario y crea otra con el mismo nombre de usuario, tendrá que establecer nuevos permisos para el usuario nuevo, ya que éste no heredará los permisos concedidos al usuario antiguo.

Para quitar cuentas de usuario

  1. En la barra de exploración principal, elija Usuarios.
  2. Elija Usuarios locales.
  3. Seleccione las cuentas de usuario que desea quitar.
  4. En la lista Tareas, elija Eliminar.
  5. Compruebe que las cuentas de usuario indicadas son las que desea eliminar y, a continuación, haga clic en Aceptar.
