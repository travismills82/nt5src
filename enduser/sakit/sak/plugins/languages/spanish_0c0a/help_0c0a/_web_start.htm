Iniciar un sitio Web

El selector de objeto o de tarea de la página Configuración del sitio Web indica el estado de cada uno de los sitios Web que residen en el módulo servidor. Si es necesario, puede iniciar cualquier sitio Web que esté detenido o pausado; sin embargo, si otro sitio está utilizando el puerto de un sitio Web detenido, no se puede iniciar el sitio Web detenido.

Para iniciar un sitio Web detenido o pausado

  1. En la barra de exploración principal, elija Sitios.
  2. Seleccione el identificador del sitio Web que desea iniciar.
  3. En la lista Tareas, elija Iniciar.
