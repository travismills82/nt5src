Encabezados de host

Los nombres de encabezados de host son direcciones de sitios Web que permiten que una única dirección IP contenga varios sitios Web. Por ejemplo, el servidor de una empresa podría alojar sitios Web para varios departamentos internos, como marketing, contabilidad y recursos humanos. Para distinguir estos sitios Web entre sí, puesto que tienen la misma dirección IP, se les asignan nombres únicos de encabezado de host.

En la tabla siguiente se muestra cómo puede utilizar encabezados de host únicos con una misma dirección IP para distinguir entre varios sitios Web.

Departamento Dirección IP Ejemplo de encabezado de host
Sitio principal de la empresa 172.28.144.10 www.ServidorEmpresa.com
Marketing 172.28.144.10 www.Marketing.ServidorEmpresa.com
Contabilidad 172.28.144.10 www.Contabilidad.ServidorEmpresa.com
Recursos humanos 172.28.144.10 www.RecursosHumanos.ServidorEmpresa.com
