Establecer la contraseña de un usuario

Para establecer la contraseña del usuario

  1. En la barra de exploración principal, elija Usuarios.
  2. Elija Usuarios locales.
  3. Seleccione la cuenta de usuario cuya contraseña desee cambiar.
  4. En la lista Tareas, elija Establecer una contraseña.
  5. Escriba la nueva contraseña y, a continuación, confírmela en los cuadros que se proporcionan a tal efecto.

    La nueva contraseña debe cumplir las reglas de complejidad de contraseñas vigentes para el dominio al que pertenece el módulo servidor.

  6. Haga clic en Aceptar.
