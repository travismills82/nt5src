Establecer las propiedades de uso compartido de Windows (CIFS)

Los equipos cliente de Windows utilizan el protocolo Sistema de archivos común de Internet (CIFS) para compartir archivos. Utilice esta página para cambiar el número de usuarios que tienen acceso a un recurso compartido, modificar las opciones de caché relativas al recurso compartido y establecer o modificar los permisos de usuario.

Configuración del módulo

En la sección Límite de usuarios puede elegir permitir el máximo número de usuarios o puede especificar el número de conexiones simultáneas que pueden existir en un momento dado.

Para establecer el límite de usuarios

  • Haga clic en el botón Máximo permitido para permitir que inicien sesión en el módulo servidor todos los usuarios que éste admita.

    - O bien -

  • Haga clic en el botón Permitir usuarios y, a continuación, especifique el número de usuarios que va a permitir.
  • Si permite almacenar en caché archivos de la carpeta compartida, utilice la lista Configuración para especificar la opción de caché que va a utilizar. Las opciones de caché se describen en Windows (CIFS).

Permisos

También puede establecer permisos para que se conceda o deniegue el acceso al módulo servidor a usuarios y grupos.

Para establecer permisos de usuario

  1. En el cuadro Agregar un usuario o grupo, escriba el nombre del usuario o el grupo que va a agregar a la lista de permisos, o seleccione un usuario de la lista que se encuentra bajo el mismo.
Puede seleccionar en la lista usuarios locales o grupos locales. Para agregar usuarios o grupos de un dominio debe escribir la cuenta en el formato <dominio\usuario> o <dominio\grupo>.
  1. Elija Agregar.
  2. Utilice la lista Permitir para establecer el grado de control que los usuarios seleccionados en la lista Permisos van a tener sobre los archivos del módulo servidor.
Los usuarios pueden no tener ningún control, acceso de sólo lectura, acceso de cambio, acceso de cambio y lectura o control total.
  1. Utilice la lista Denegar para denegar un nivel de control a los usuarios y grupos seleccionados en la lista Permisos.
  2. Para quitar un usuario o un grupo de la lista Permisos, seleccione el usuario o el grupo en la lista y, a continuación, haga clic en Quitar.
  3. Haga clic en Aceptar.
