Running a Storage Report

To run a report

  1. On the primary navigation bar, choose Disks. 
  2. Choose Storage Reports.
  3. Select the report you want to run.
  4. In the Tasks list, choose Run Report.
  5. Type the directory name that you want to report on, and then choose Add.
  6. Choose OK.
