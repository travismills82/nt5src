Detener un sitio Web

El selector de objeto o de tarea de la página Configuración del sitio Web indica el estado de cada uno de los sitios Web que residen en el módulo servidor. Si es necesario, puede detener cualquier sitio Web que esté pausado o en funcionamiento.

La detención de un sitio Web impide que los servicios de Internet acepten conexiones nuevas y detendrá todas las solicitudes que ya se estén procesando. La pausa de un sitio Web impide que los servicios de Internet acepten conexiones nuevas pero no afecta a las solicitudes que ya se están procesando.

Para detener un sitio Web

  1. En la barra de exploración principal, elija Sitios.
  2. Seleccione el identificador del sitio Web que desea detener.
  3. En la lista Tareas, elija Detener.
