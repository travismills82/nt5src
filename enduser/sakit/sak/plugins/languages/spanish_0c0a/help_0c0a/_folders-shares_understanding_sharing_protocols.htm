Comprender los protocolos de uso compartido

Los protocolos de uso compartido permiten que los equipos cliente tengan acceso a los archivos del módulo servidor.

Su módulo servidor puede ser compatible con alguno o con todos los tipos siguientes de protocolos de uso compartido:

  • Windows (CIFS)

    El protocolo Sistema de archivos común de Internet (CIFS) se utiliza en los clientes que ejecutan sistemas operativos Windows.

  • UNIX (NFS)

    El protocolo Sistema de archivos de red se utiliza en clientes que ejecutan sistemas operativos UNIX.

  • FTP

    El Protocolo de transferencia de archivos es una alternativa para tener acceso a recursos compartidos de archivos desde cualquier sistema operativo.

  • Web (HTTP)

    El Protocolo de transferencia de hipertexto es el protocolo para tener acceso a recursos compartidos de archivos desde exploradores Web.

  • Netware

    El protocolo para tener acceso a recursos compartidos de archivos desde clientes NetWare.

  • AppleTalk

    El protocolo para tener acceso a recursos compartidos de archivos desde clientes Macintosh.
