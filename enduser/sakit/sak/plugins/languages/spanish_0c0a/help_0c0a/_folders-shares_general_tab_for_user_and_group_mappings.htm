Configurar asignaciones de usuarios y grupos

Para asignar usuarios y grupos en NFS

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. Elija Protocolos para uso compartido.
  3. Seleccione Protocolo NFS y, a continuación, elija Propiedades.
  4. Elija Asignaciones de usuarios y grupos.

Para configurar el uso de un servidor NIS

  1. En la ficha General, haga clic en el botón Utilizar el servidor NIS.
  2. En el cuadro Dominio NIS, escriba el nombre del dominio desde el que se va a obtener la información de usuarios y grupos de UNIX.
  3. Opcionalmente: En el cuadro Servidor NIS (opcional), escriba el nombre del servidor que desea asignar.
  4. Para especificar el período de tiempo durante el que el módulo servidor espera para actualizar la información de usuarios y grupos, escriba dicho tiempo en los cuadros Horas y Minutos.
  5. Haga clic en Aceptar.

Para configurar el uso de archivos de contraseñas y grupos

  1. Haga clic en el botón Utilizar archivos de contraseña y grupo.
  2. En el cuadro Archivo de contraseña, escriba el nombre del archivo de contraseña que va a utilizar.
Es un archivo en formato passwd de un sistema UNIX que contiene todas las cuentas de usuario de UNIX que se pueden asignar.
  1. En el cuadro Archivo de grupo, escriba el nombre del archivo de grupo que va a utilizar.
  2. Haga clic en Aceptar.
