Protocolo NFS

Puede utilizar la opción Protocolo NFS para configurar el módulo servidor de manera que actúe como servidor NFS. El Protocolo NFS permite que los usuarios compartan archivos en un entorno mixto de equipos, sistemas operativos y redes.

Puede utilizar el Protocolo NFS para administrar Grupos de clientes NFS, Bloqueos NFS y Asignaciones de usuarios y grupos NFS. Sin embargo, los Recursos compartidos NFS se crean desde la sección Recursos compartidos de la interfaz de usuario Web.

Para configurar el protocolo NFS

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. Elija Protocolos para uso compartido.
  3. Seleccione Protocolo NFS y, a continuación, elija Propiedades.
  4. Seleccione la tarea de configuración de NFS que desee.
