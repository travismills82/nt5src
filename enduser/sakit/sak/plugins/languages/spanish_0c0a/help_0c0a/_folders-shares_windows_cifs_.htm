Windows (CIFS)

Compartir archivos con Microsoft Windows

Cuando comparte una carpeta, puede elegir los permisos que permiten o deniegan el acceso a los archivos de dicha carpeta a otros usuarios de la red. Para los equipos cliente que ejecutan Microsoft Windows, también puede especificar si otros usuarios de Windows podrán hacer que la carpeta compartida esté disponible sin conexión.

Para hacer que un archivo compartido de la red esté disponible sin conexión, se almacena una versión del archivo en una parte reservada del disco del equipo cliente llamada caché. El equipo puede tener acceso a esta caché aunque no esté conectado a la red. Al compartir archivos, puede utilizar tres opciones de caché:

Caché manual de documentos

El almacenamiento manual en caché de documentos proporciona acceso sin conexión solamente a aquellos archivos identificados específica o manualmente por quien usa la carpeta compartida en el módulo servidor. Esta opción de caché es ideal para una carpeta compartida del módulo servidor que contiene archivos a los que tendrán acceso y serán modificados por varias personas. Ésta es la opción predeterminada cuando se configura una carpeta compartida para utilizarse sin conexión.

Caché automático de documentos

El almacenamiento automático en caché de documentos hace que todos los archivos de la carpeta compartida del módulo servidor estén disponibles sin conexión para quienes abren los archivos.

El almacenamiento automático en caché hace que el contenido de una carpeta esté disponible sin conexión ya esté o no configurado específicamente por un usuario que esté utilizando la carpeta compartida del módulo servidor. Los usuarios tendrán disponibles documentos, dibujos, archivos de programa y otros archivos.

Sólo aquellos archivos que un usuario haya abierto en la carpeta compartida del módulo servidor seguirán estando disponibles para ese usuario cuando trabaje sin conexión.

Caché automático de programas

El almacenamiento automático en caché de programas proporciona acceso de sólo lectura sin conexión a los archivos de la carpeta compartida. Esta opción de caché es ideal para permitir el acceso sin conexión a archivos cuando se consultan, ejecutan o leen, pero que no deben modificarse durante el proceso. El almacenamiento automático en caché de programas reduce el tráfico en la red porque los archivos sin conexión se abren directamente, sin tener acceso a las versiones de red, y generalmente se inician y se ejecutan más rápido que las versiones de red.

Cuando utilice el almacenamiento automático en caché de programas, asegúrese de restringir los permisos de los archivos de la carpeta compartida a acceso de sólo lectura.
