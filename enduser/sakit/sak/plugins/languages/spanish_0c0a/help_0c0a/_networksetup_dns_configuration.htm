Configuración de DNS

El sistema de nombres de dominio (DNS) es un servicio de nombres estático jerárquico para hosts TCP/IP. El administrador de la red configura DNS con una lista de nombres y direcciones IP de hosts. Esto permite que los usuarios de la red consulten DNS para especificar sistemas remotos por sus nombres de host en vez de por sus direcciones IP.

Nota   El propósito de esta página de propiedades es permitirle escribir las direcciones de los servidores DNS externos. El módulo servidor no contiene un servidor DNS.

Por ejemplo, una estación de trabajo configurada para utilizar la resolución de nombres DNS podría utilizar el comando ping hostremoto en lugar de ping 1.2.3.4 si la asignación para el sistema llamado hostremoto se encontrara en la base de datos de DNS. No se debe confundir los dominios DNS con los dominios de Microsoft Windows.

En el modelo cliente-servidor de DNS, el servidor que contiene información acerca de una parte de la base de datos de DNS (la parte que facilita nombres de equipos a los clientes) consulta la resolución de nombres a través de Internet.

Para establecer el módulo servidor de manera que obtenga automáticamente la información de los servidores DNS desde un servidor DHCP

  1. En la barra de exploración principal, elija Red.
  2. Elija Interfaces.
  3. Seleccione la conexión de red que desea modificar.
  4. En la lista Tareas, elija DNS.
  5. Haga clic en el botón Obtener la configuración de un servidor DHCP.
  6. Haga clic en Aceptar.

Para establecer manualmente los servidores DNS que va a utilizar el módulo servidor

  1. En la página Red, elija Interfaces.
  2. Seleccione la conexión de red que desea modificar.
  3. En la lista Tareas, elija DNS.
  4. Haga clic en el botón Configurar manualmente.
  5. Escriba la dirección IP apropiada en el cuadro que se encuentra junto al botón Agregar y, a continuación, haga clic en Agregar.
  6. Para agregar otro servidor DNS, repita el paso 5.
  7. Cuando haya terminado de agregar servidores DNS, haga clic en Aceptar.

    Nota   Si establece que la dirección IP se obtenga a través de DHCP y establece DNS manualmente, el sistema aceptará la entrada manual y las propiedades del módulo servidor se establecerán automáticamente a Configurar manualmente. Sin embargo, la columna Configuración actual del Selector de objeto o de tarea de la página Interfaces seguirá mostrando DHCP como origen de la dirección IP. Puede volver a la página de propiedades Configuración de DNS para confirmar que se ha guardado la configuración manual.
