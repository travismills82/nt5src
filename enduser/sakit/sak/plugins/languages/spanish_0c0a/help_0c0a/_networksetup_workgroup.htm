Grupo de trabajo

Un grupo de trabajo es una agrupación simple de equipos, destinada únicamente a ayudar a los usuarios a encontrar elementos como impresoras y carpetas compartidas dentro de ese grupo. El módulo servidor puede estar en un grupo de trabajo, un entorno de Active Directory o un dominio de Windows NT 4. Los grupos de trabajo de Microsoft Windows 2000 no ofrecen las cuentas de usuario y la autenticación centralizadas que ofrecen los dominios.

El nombre del grupo de trabajo no puede ser el nombre del equipo. El nombre del grupo de trabajo puede tener hasta 15 caracteres y no puede contener ninguno de los caracteres siguientes: ; : " < > * + = \ | ? , .

Para establecer o cambiar la pertenencia al grupo de trabajo del módulo servidor

  1. En la barra de exploración principal, elija Red.
  2. Elija Identificación.
  3. Haga clic en el botón Grupo de trabajo y escriba el nombre del grupo de trabajo al que desea unirse.
  4. Si el módulo servidor pertenecía a un dominio antes de unirse al grupo de trabajo, se separará del dominio y la cuenta de equipo quedará deshabilitada.
  5. Haga clic en Aceptar.
  6. Se le preguntará si desea reiniciar el módulo servidor.

    Elija si desea reiniciar o no el módulo servidor.

    • Si hace clic en Aceptar, se reiniciará el módulo servidor y aparecerá una página que indica que el módulo servidor se está reiniciando. Cuando el módulo servidor vuelva a estar en funcionamiento, aparecerá la página predeterminada de la interfaz de usuario (UI) Web y las modificaciones surtirán efecto.
    • Si elige Cancelar, las modificaciones realizadas en la identidad del módulo servidor no surtirán efecto hasta el siguiente reinicio.
