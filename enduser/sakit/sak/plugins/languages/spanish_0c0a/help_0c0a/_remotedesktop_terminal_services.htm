Servicios de Terminal Server

Servicios de Terminal Server le ofrece la posibilidad de iniciar sesión y administrar de forma remota el módulo servidor, proporcionándole un método para administrarlo desde cualquier cliente. Cuando se instala para administración remota, sólo se permiten dos sesiones simultáneas de Servicios de Terminal Server. El dejar abierta una sesión hace que se utilice una licencia y puede afectar a otros usuarios. Si hay dos sesiones abiertas, se denegará el acceso a los nuevos usuarios.

Nota El Cliente avanzado de Servicios de Terminal Server, disponible desde la página Mantenimiento, sólo se acepta en sistemas que ejecutan Internet Explorer.

Cerrar la sesión y desconectar

Servicios de Terminal Server proporciona dos opciones para cerrar un cliente: puede desconectar o cerrar la sesión del sistema.

La desconexión deja la sesión ejecutándose en el servidor. Puede volver a conectar con el servidor y reanudar la sesión. Si está realizando una tarea en el servidor, como ejecutar una consulta de una base de datos que requiera mucho tiempo, puede iniciar la tarea y desconectarse de la sesión. Más tarde, puede volver a iniciar sesión en el servidor, volver a entrar en la sesión y reanudar la tarea o comprobar los resultados. Esto resulta especialmente útil cuando se utiliza una conexión de acceso remoto en una línea de pago de larga distancia.

Los administradores de red pueden limitar el período de tiempo durante el cual una sesión desconectada puede permanecer en el servidor. Consulte con el administrador para obtener información acerca de los límites de la duración de las sesiones.

La finalización de una sesión se denomina cierre de sesión. El cierre de sesión termina la ejecución de la sesión en el servidor. Todas las aplicaciones que se ejecuten dentro de la sesión se cerrarán y se perderán los cambios realizados en los archivos abiertos y que no se hayan guardado. La próxima vez que inicie sesión en el servidor se creará una sesión nueva.

Finalmente, el Cliente avanzado de Servicios de Terminal Server requiere que todos los usuarios que se conectan estén autenticados, que es por lo que los usuarios deben iniciar sesión para poder abrir una.

Para tener acceso al Cliente avanzado de Servicios de Terminal Server

  1. En la barra de exploración principal, elija Mantenimiento.
  2. Elija Servicios de Terminal Server.
  3. Inicie sesión.
