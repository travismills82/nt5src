Mostrar mensajes de FTP

Para mostrar los mensajes de bienvenida, salida y conexiones máximas

  1. En la barra de tareas principal, elija Servidor Web.
  2. Elija Mensajes de FTP.
  3. Escriba sus mensajes de bienvenida, salida y conexiones máximas en los cuadros apropiados:
    • Mensaje de bienvenida: los usuarios verán este mensaje cuando inicien una sesión en el módulo servidor mediante FTP.
    • Mensaje de salida: este mensaje se mostrará a los usuarios cuando cierren la sesión del módulo servidor mediante FTP.
    • Mensaje para el número máximo de conexiones: este mensaje informa al usuario de que se ha alcanzado el número máximo de conexiones FTP que el módulo servidor admite.
  4. Haga clic en Aceptar.
