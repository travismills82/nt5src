File Screening Groups

File screening groups consist of a set of files to be filtered and files to exclude from filtering. For example, a group called Graphics File might contain the filters *.jpg; *.bmp but exclude the file important.jpg. File screening includes a set of predefined groups to which you can add.  File screening groups are used within a file screening policy to define the authorized and unauthorized file types.
