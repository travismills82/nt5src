Creating a File Screening Group

To create a file screening group

  1. On the primary navigation bar, choose Disks.
  2. Choose File Screening.   
  3. In the Tasks list, choose Policies.
  4. In the Tasks list, choose Groups.
  5. In the Tasks list, choose New.
  6. Type a file screening group name.
  7. Type the file types that are included and excluded in this screening group. 

    For example, you could include *.txt and exclude myfile.txt.  You can type as many files types as you want.

  8. To add the file, choose the folder icon.
  9. Choose OK.
