Configurar las propiedades de disco y volumen

En esta página puede configurar los discos o los volúmenes del módulo servidor.

Para administrar discos y volúmenes del módulo servidor

  1. En la barra de exploración principal, elija Discos.
  2. Elija Discos y volúmenes.
  3. Inicie sesión en el Cliente avanzado de Servicios de Terminal Server.
  4. Cuando haya terminado, cierre la aplicación.

    Nota   Puede que al cerrar la aplicación se tarde unos momentos en cerrar la sesión del Cliente avanzado de Servicios de Terminal Server.

Cliente avanzado de Servicios de Terminal Server

El Cliente avanzado de Servicios de Terminal Server es una herramienta de propósito general que le ofrece acceso total al módulo servidor. Cuando se tiene acceso al mismo a través de la página Discos y volúmenes, el Cliente avanzado de Servicios de Terminal Server asume un modo dedicado y sólo puede utilizarse para administrar discos y volúmenes del módulo servidor.

El Cliente avanzado de Servicios de Terminal Server sólo acepta dos conexiones simultáneas. Además, si se desplaza a otra página durante una sesión abierta, se desconectará el cliente pero se mantendrá la sesión.
