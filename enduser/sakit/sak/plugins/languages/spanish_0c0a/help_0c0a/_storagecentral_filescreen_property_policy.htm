Modifying a File Screening Policy

 To modify a file screening policy

  1. On the primary navigation bar, choose Disks.
  2. Choose File Screening.
  3. Select the directory you want to modify.
  4. In the Tasks list, choose Policies.
  5. Select the file screening policy you want, and then choose Properties from the Tasks list.
  6. Select the Screening Groups tab to modify the associated screening groups.
  7. Select the Filters tab to modify the policy directory filters.
  8. Select the Alarm tab to modify the policy alarm.
  9. Make any changes you want.
  10. Choose OK.
