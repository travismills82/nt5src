Modifying Directory Quota Properties

To modify the properties of a directory quota 

  1. On the primary navigation bar, choose Disks.
  2. Choose  Directory Quota.
  3. Select the directory quota you want to modify.
  4. In the Tasks list, choose Properties.
  5. On the Copy Policy Setting  tab, select the policy you want from the list or choose appropriate tab to modify.
  6. Choose OK.

Note  Do not use directory quotas in place of file security. Use directory quotas only to manage disk space usage, not to prevent a user from writing to a partition or directory. If you want to prevent a user from writing to a directory, set the appropriate protection on that directory.
