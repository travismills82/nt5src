Desplazarse entre las carpetas

Utilice el selector de objeto o de tarea para desplazarse entre las carpetas. Por cada carpeta que tenga subcarpetas, habrá una tarea Abrir en la lista Tareas. Por cada carpeta que tenga una carpeta principal, habrá una tarea Arriba en la lista Tareas.

Para desplazarse entre las carpetas

  1. En la barra de exploración principal, elija Recursos compartidos.
  2. Elija Carpetas.
  3. Seleccione el volumen con el que desea trabajar y, a continuación, elija Abrir.
  4. Seleccione la carpeta que desee.
  5. En la lista Tareas, elija Abrir.
  6. Seleccione la carpeta dentro de la que va a desplazarse y, a continuación, elija Abrir en la lista Tareas.

    - O bien -

    • En la lista Tareas, elija Arriba para volver al directorio raíz del volumen.
