Protocolo FTP

El Protocolo de transferencia de archivos (FTP) está integrado en el modelo de seguridad de Windows. Los usuarios que se conectan mediante FTP se autentican con las cuentas de usuario del módulo servidor Windows Powered. Reciben el acceso de acuerdo con sus perfiles de usuario. Sin embargo, recuerde que el servidor FTP está basado en la posibilidad de enviar contraseñas de usuario sin cifrar a través de la red. Como resultado, un usuario que tenga acceso físico a la red podría examinar las contraseñas de los usuarios durante el proceso de validación de FTP.

FTP acepta todos los comandos del cliente FTP de Microsoft Windows; cuando un módulo servidor ejecuta FTP, otros equipos que empleen la utilidad FTP pueden conectar con el servidor y transferir archivos. Por otra parte, otros clientes FTP que no sean de Microsoft podrían contener comandos no compatibles con el protocolo del servidor FTP.
