Administrar los registros de aplicación

El registro de aplicación contiene los sucesos registrados por las aplicaciones o los programas. Por ejemplo, un programa de proceso de textos podría registrar un error de archivo en el registro de aplicación. Los sucesos que se registran dependen de cada aplicación.

Para administrar los registros de aplicación

  1. En la barra de exploración principal, elija Mantenimiento.
  2. Elija Registros.
  3. Elija Registro de aplicación.
