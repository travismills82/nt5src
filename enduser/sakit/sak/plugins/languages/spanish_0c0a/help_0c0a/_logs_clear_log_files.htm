Borrar archivos de registro

En esta página puede borrar archivos de registro específicos. Cuando borra registros de aplicación, NFS, de seguridad o de sistema, se borra todo su contenido; sin embargo, cuando borra registros de FTP, administración Web o recursos compartidos Web (HTTP), puede seleccionar los archivos de registro específicos que desea borrar.

Para borrar registros de aplicación, NFS, seguridad o sistema

  1. En la barra de exploración principal, elija Mantenimiento.
  2. Elija Registros.
  3. Seleccione el tipo de registro que desea borrar.
  4. Seleccione Borrar registro.
  5. Haga clic en Aceptar.

Para borrar registros de FTP, de administración Web o de recursos compartidos Web (HTTP)

  1. En la barra de exploración principal, elija Mantenimiento.
  2. Elija Registros.
  3. Seleccione el tipo de registro que desea borrar.
  4. Seleccione los archivos de registro que va a borrar.
  5. Elija Borrar registro.
  6. Haga clic en Aceptar.
