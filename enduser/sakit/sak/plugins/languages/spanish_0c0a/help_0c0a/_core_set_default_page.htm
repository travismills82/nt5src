Establecer la página predeterminada

Puede establecer la página predeterminada del módulo servidor. Será la página que se muestre cada vez que tenga acceso al módulo servidor. Puede elegir que la página predeterminada del módulo servidor sea la página Bienvenida o la página Estado.

Para establecer la página predeterminada del módulo servidor

  1. En la barra de exploración principal, elija Bienvenida.
  2. En la página Bienvenida, elija Establecer la página predeterminada.
  3. En Establecer la página predeterminada, elija qué página desea establecer como predeterminada.
  4. Haga clic en Aceptar.
