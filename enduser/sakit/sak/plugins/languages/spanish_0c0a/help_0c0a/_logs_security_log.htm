Administrar los registros de seguridad

El registro de seguridad puede contener sucesos de seguridad como los intentos válidos o no válidos de inicio de sesión, así como los sucesos relacionados con el uso de recursos, como crear, abrir o eliminar archivos. Un administrador puede especificar los sucesos que se registrarán en el registro de seguridad.

Para administrar los registros de seguridad

  1. En la barra de exploración principal, elija Mantenimiento.
  2. Elija Registros.
  3. Elija Registro de seguridad.
