Administrar recursos compartidos

La página Administración de recursos compartidos le permite crear, abrir, eliminar o configurar diversos recursos compartidos de archivo. La página Recursos compartidos muestra un selector de objeto o de tarea que tiene las columnas siguientes:

  • Nombre del recurso compartido: muestra los nombres de las carpetas compartidas. Para crear, abrir, eliminar o configurar las propiedades de un recurso compartido dado, active la casilla de verificación que se encuentra junto al nombre del recurso compartido que desea modificar o eliminar.
  • Ruta de acceso al recurso compartido: muestra la ruta de acceso al recurso compartido.
  • Tipo: indica uno de los siguientes tipos de recurso compartido:

    W = Windows (CIFS)

    U = UNIX (NFS)

    F = FTP

    H = HTTP

  • Comentario: muestra una breve descripción del recurso compartido, si ha escrito antes alguna.

Para seleccionar un recurso compartido, utilice el Selector de objeto o de tarea y, a continuación, elija la tarea que desea realizar en la lista Tareas.
