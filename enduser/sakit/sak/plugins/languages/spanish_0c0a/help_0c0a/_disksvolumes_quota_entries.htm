Configurar entradas de cuota

La página Entradas de cuota le permite agregar, eliminar o configurar cuotas de disco para cualquier usuario del módulo servidor. Las cuotas se administran mediante el Selector de objeto o tarea, que tiene las columnas siguientes:

  • Nombre de inicio de sesión: esta columna muestra el nombre de inicio de sesión de todos los usuarios que tienen acceso registrado al módulo servidor.
  • Estado: esta columna indica si el usuario ha sobrepasado o no el límite de cuota asignado.
  • Cantidad utilizada: esta columna indica la cantidad de espacio en disco utilizado actualmente por un usuario dado.
  • Límite de cuota: esta columna indica la cantidad máxima de espacio en disco que un usuario puede ocupar en un volumen.

El comportamiento del módulo servidor cuando se sobrepasa el límite de cuota depende de la configuración de la página de propiedades Cuota, a la que se puede tener acceso a través de la ficha Cuota de disco. Si la casilla de verificación Denegar el espacio en disco a los usuarios que superen el límite de cuota está activada, el usuario no podrá sobrepasar este límite. Si la casilla de verificación Registrar el suceso de límite de cuota cuando el usuario supera su límite de cuota está activada, se registrará un mensaje en el registro de sucesos. Si ninguna de las opciones está activada, no se realizará ninguna acción.

  • Nivel de advertencia: esta columna indica la cantidad máxima de espacio en disco que un usuario determinado puede utilizar antes de que aparezca una advertencia indicándole que casi ha alcanzado el límite de cuota.

    Nota   Sólo se generará una advertencia si el usuario sobrepasa el límite de advertencia especificado en la página Entradas de cuota y si Registrar suceso está activada en la página Cuota predeterminada. Si la casilla de verificación Registrar suceso no está activada, no se generará ninguna advertencia y esta columna estará vacía. Normalmente, el valor de Límite de advertencia se establece ligeramente por debajo del valor de Límite de cuota.

Para establecer o modificar entradas de cuota en el módulo servidor

  1. En la barra de exploración principal, elija Discos.
  2. Elija Cuota de disco.
  3. Seleccione el volumen que desea administrar.
  4. En la lista Tareas, elija Entradas de cuota.
