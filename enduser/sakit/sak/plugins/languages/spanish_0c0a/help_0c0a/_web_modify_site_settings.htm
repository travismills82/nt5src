Modificar la configuración del sitio Web

Después de crear un sitio Web puede modificar las opciones generales, de identidad del sitio y de la aplicación, si es necesario.

Para modificar la configuración de un sitio Web existente

  1. En la barra de exploración principal, elija Sitios.
  2. Seleccione el sitio Web que desea modificar.
  3. En la lista Tareas, elija Modificar.

    Se abrirá la página Modificar sitio Web y la ficha General será la ficha activa. Puede modificar las opciones de esta ficha o puede hacer cambios en cualquiera de las opciones disponibles en las fichas Identidades del sitio o Configuración de la aplicación.
