Storage Reports Preferences

Use the Preferences page to change the storage report options. 

The Settings tab allows you to define the following filter options and output settings:

Filter Options

Exclude OS files Excludes all files with a .sys extension, all files in the %SystemRoot% directories, and all files in the temp directory.

Exclude DLL files Excludes all files with a .dll extension.

Output Settings

Report output directory Use this setting to specify the default output directory for your reports. If you specify an output directory in the report set definition, it will override this value.

Maximum memory usage

Maximum virtual memory used Use this option to limit the amount of virtual memory used to the percentage of virtual memory displayed. This will limit the number of files that can be read from the selected directory.

The E-Mail tab allows you to define the the mail server and mail sender's address.

Mail server Use this option to specify the address of the e-mail server. This may be a name or an IP address.
Mail sender Use this option to specify the sender's mail address that appears in the From field of e-mail messages sent by storage reports.
