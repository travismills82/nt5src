Mantenimiento

En la página principal de Mantenimiento los usuarios pueden realizar las siguientes tareas generales de mantenimiento del módulo servidor:

  • Actualización de software: instalar una actualización de software.
  • Fecha y hora: establecer la fecha y la hora en el módulo servidor.
  • Apagar: apagar o reiniciar el módulo servidor.
  • Registros: ver, descargar, configurar y borrar registros de sucesos.
  • Copia de seguridad: hacer copia de seguridad o restaurar el sistema operativo del módulo servidor.
  • Servicios de Terminal Server: administrar todos los aspectos del módulo servidor conectando con él mediante el Cliente avanzado de Servicios de Terminal Server.
  • Correo electrónico de alerta: establecer correo electrónico de alerta en el módulo servidor.
  • Idioma: cambiar el idioma utilizado por el módulo servidor.
