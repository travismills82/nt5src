Modificar las identidades de los sitios

En la página Identidades del sitio puede asignar la dirección IP y el número de puerto TCP del módulo servidor, así como agregar o cambiar el nombre de encabezado de host del sitio Web que va a crear o modificar.

La Dirección IP está establecida como Todos sin asignar de forma predeterminada. En la lista aparecerán las direcciones IP de todas las tarjetas de interfaz de red (NIC) instaladas en el módulo servidor.

El Puerto TCP está establecido como 80 de forma predeterminada. Este número de puerto predeterminado es el estándar y se sobreentiende cuando escribe una dirección Web en un explorador, por ejemplo www.suSitio.com. Si modifica el número de puerto por otro distinto de 80, por ejemplo 8055, cuando los usuarios escriban la dirección del sitio Web tienen que incluir también un signo de dos puntos y el número del puerto, como en www.suSitio.com:8055.

Para modificar los parámetros de Identidades del sitio

  1. En la barra de exploración principal, elija Sitios.
  2. Seleccione el sitio Web que desea modificar.
  3. En la lista Tareas, elija Modificar.
  4. Seleccione la ficha Identidades del sitio.
  5. En la lista Dirección IP, seleccione la dirección IP que desea asignar al sitio Web.
  6. En el cuadro Puerto TCP, escriba el número de puerto TCP que desee.
  7. En el cuadro Encabezados del host, escriba el nombre del encabezado de host.
  8. Haga clic en Aceptar o seleccione otra ficha para hacer otras modificaciones en las propiedades del sitio Web.
