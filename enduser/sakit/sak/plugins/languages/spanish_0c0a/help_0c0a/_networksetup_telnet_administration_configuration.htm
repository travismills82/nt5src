Telnet

La característica Configuración de administración Telnet le permite administrar el módulo servidor Windows® Powered desde un sistema remoto mediante el protocolo Telnet. Puede iniciar sesión en el sistema desde un cliente Telnet remoto y ejecutar aplicaciones en modo de caracteres en el módulo servidor. El servidor Telnet incluido en el módulo servidor acepta un máximo de dos clientes Telnet simultáneos, a menos que el fabricante de hardware del módulo servidor especifique otra cantidad.

Para configurar el módulo servidor para la administración Telnet

  1. En la barra de exploración principal, elija Red.
  2. Elija Telnet.
  3. Active la casilla de verificación Habilitar el acceso mediante Telnet a este módulo servidor.
  4. Haga clic en Aceptar.
