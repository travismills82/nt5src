ログ ファイルのダウンロード

このページでは、特定のログ ファイルをサーバー アプライアンスへダウンロードできます。

アプリケーション ログ、セキュリティ ログ、システム ログをダウンロードするには

  1. プライマリ ナビゲーション バーで、[メンテナンス] をクリックします。
  2. [ログ] をクリックします。
  3. ダウンロードするログの種類を選択します。
  4. [ダウンロード] をクリックします。
  5. [このファイルをディスクに保存する] をクリックします。
  6. [OK] をクリックして、ファイルをダウンロードします。

NFS ログをダウンロードするには

  1. プライマリ ナビゲーション バーで、[メンテナンス] をクリックします。
  2. [ログ] をクリックします。
  3. [NFS ログ] をクリックします。
  4. [ダウンロード] をクリックします。
  5. [このファイルをディスクに保存する] をクリックします。
  6. [OK] をクリックします。

FTP ログ、Web 管理ログ、Web (HTTP) 共有ログをダウンロードするには

  1. プライマリ ナビゲーション バーで、[メンテナンス] をクリックします。
  2. [ログ] をクリックします。
  3. ダウンロードするログを選択します。
  4. ダウンロードするログ ファイルを選択します。
  5. [ダウンロード] をクリックします。
  6. [このファイルをディスクに保存する] をクリックします。
  7. [OK] をクリックします。
