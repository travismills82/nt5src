Web サイト設定の変更

Web サイトを作成した後に、一般設定、サイトの ID、およびアプリケーション設定を必要に応じて変更することができます。

既存の Web サイトの設定を変更するには

  1. プライマリ ナビゲーション バーで、[サイト] をクリックします。
  2. 設定を変更する Web サイトを選択します。
  3. [タスク] ボックスの一覧で、[変更] をクリックします。

    [Web サイトの変更] ページが表示され、[全般] タブが有効になります。このタブの設定を変更したり、[サイトの ID] タブや [アプリケーション設定] タブの設定を変更することができます。
