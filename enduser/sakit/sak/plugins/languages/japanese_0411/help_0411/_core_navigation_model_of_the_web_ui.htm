Web UI のナビゲーション モデル

Web UI の各ページの上部は、ステータス領域、プライマリ ナビゲーション バー、セカンダリ ナビゲーション バーで構成され、本体部分はコンテンツ領域で構成されます。

次に、これらの構成部分について説明します。

ステータス領域

ウィンドウ最上部のステータス領域には、左から順に次の項目が表示されます。

  • Windows Powered サーバー アプライアンスの製造元のロゴ (オプション)
  • サーバー アプライアンスのホスト名と状態
  • Microsoft Windows Powered のロゴ

表示される状態には次の 4 種類があります。

  • 通常 (緑のテキスト)
  • 情報 (グレーのテキスト)
  • 警告 (黄のテキスト)
  • 重大 (赤のテキスト)

[状態 : <情報>] をクリックすると、サーバー アプライアンスの状態に関する詳細情報が表示されます。

ナビゲーション バー

ナビゲーション バーはステータス領域の直下に表示されます。次の図は、標準的な Web サーバー アプライアンスのプライマリ ナビゲーション バー (青) とセカンダリ ナビゲーション バー (白) を示しています。

プライマリ ナビゲーション バーには、使用可能なサーバー アプライアンスのタスクが種類別に一覧表示されます。セカンダリ ナビゲーション バーには、選択したプライマリ タスクのサブタスクが一覧表示されます。セカンダリ ナビゲーション バーは動的で、選択されるタスク グループに応じて、表示される使用可能なタスクの種類が変動します。

ナビゲーション バーの各項目には、タスクのより詳細な情報を提供するロールオーバー テキストが用意されています。各項目にマウス カーソルを合わせると、ロールオーバー テキストが表示されます。

コンテンツ領域

メニュー バーの下にあるコンテンツ領域には、ページ情報が表示されます。Web UI のコンテンツ領域には、そのページで実行可能な管理動作が示されます。選択したタスクを実行するために使用するツールについての説明が表示される場合もあります。

多くのタスク ページには、[オブジェクト/タスク セレクタ] が含まれています。[オブジェクト/タスク セレクタ] には、ユーザーによる管理や構成が可能なオブジェクト、それらのオブジェクトの説明、および実行可能なタスクが表示されます。[オブジェクト/タスク セレクタ] の左端の列には、オブジェクトを選択するためのオプション ボタンが表示されます。[オブジェクト/タスク セレクタ] で複数のオブジェクトが選択された場合は、左端の列にはオプション ボタンの代わりにチェック ボックスが表示されます。右端の列には、実行可能なタスクが一覧表示されます。

Web UI 内を移動するには

  1. プライマリ ナビゲーション バーで、実行する一般的なタスクの種類をクリックします。
  2. セカンダリ ナビゲーション バー、またはコンテンツ領域のタスク一覧で、実行する具体的なタスクの種類をクリックします。
  3. 続いて表示されるページのコンテンツ領域で次の操作を行います。
    1. [オブジェクト/タスク セレクタ] が使用できる場合は、該当するオプション ボタンをクリックして、管理または構成するオブジェクトを選択します。次に、右側の [タスク] ボックスの一覧で、実行するタスクをクリックします。ページによっては、[オブジェクト/タスク セレクタ] に検索機能および表内の情報をページングする機能が備わっています。
    2. [オブジェクト/タスク セレクタ] を使用できない場合は、選択したタスクを完成させるためのフィールドに必要なデータを入力します。
  4. 各タスクの処理が終了したら、[OK] をクリックして変更を確定します。変更前の設定を維持するには [キャンセル] をクリックします。
変更またはキャンセルを実行すると、前のページが表示されます。

プロパティ ページにいるときに別のナビゲーション バーの項目を選択すると、"[OK] をクリックすると、変更が無効となります。" というメッセージと共にポップアップ ウィンドウが表示されます。ここでユーザーは、次に選択したページへ移動する前に、変更を確定するか、あるいはキャンセルするかを選択することができます。
