Web マスタ設定

このページでは、サーバー アプライアンス上のすべての Web サイトに適用する Web サーバーの汎用設定を構成することができます。

既定のディレクトリ パスが、[Web サイト ルート ディレクトリ] ボックスに表示されます。 このディレクトリ パスは変更することができます。このディレクトリ パスを変更すると、その後の Web サイトの既定のディレクトリも変更されます。この変更の前に作成された Web サイトは、元のディレクトリのままです。

既定のルート ディレクトリを変更するには、[Web サイト ルート ディレクトリ] ボックスに新しいパスを入力します。

注   Web サイト ルート ディレクトリを変更しても、FTP ディレクトリには影響ありません。

次に示すオプションも使用できます。

  • ASP スクリプト タイムアウト (秒) - ASP がスクリプトの実行を許可する時間の長さを指定します。タイムアウト期間が終了するまでにスクリプトの実行が完了しない場合、ASP はスクリプトを停止し、Windows イベント ログにイベントを書き込みます。タイムアウト期間には、1 〜 2147483647 までの値を設定できます。ASP ページのこのオプションは Server.ScriptTimeout メソッドを使用して上書きできます。
  • Web サイトにインデックスをつける - Microsoft インデックス サービスを使ったファイルのフル テキスト検索を可能にします。

注   どの場合でも、機能を有効にするにはチェック ボックスをオンにする必要があります。

最後に、サーバー アプライアンス上でホストされる Web サイトの最大接続数を設定できます。
