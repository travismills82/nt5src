AppleTalk

Microsoft Windows 2000 Server の AppleTalk ネットワーク統合は、ネットワークに接続されているすべての Apple Macintosh クライアントとサーバー アプライアンスとの間でファイルやプリンタを共有できるようにするための機能です。

AppleTalk ネットワーク統合に必要なのは、クライアントとして機能する Macintosh OS ソフトウェアだけです。その他のソフトウェアは必要ありません。

AppleTalk ネットワーク統合を使用すれば、すべてのユーザー アカウントを一元的に管理できるので、管理作業が容易になります。たとえば、Macintosh サーバーと、Windows 2000 Server を実行しているコンピュータで、ユーザー アカウントを別々に管理する必要はありません。
