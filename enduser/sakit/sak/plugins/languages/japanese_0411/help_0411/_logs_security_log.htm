セキュリティ ログの管理

セキュリティ ログには、有効なログオンや無効なログオンなど、セキュリティ イベントを記録できます。また、作成されたファイル、開いているファイル、削除されたファイルなど、リソースの利用状況に関するイベントも記録できます。管理者は、セキュリティ ログにどのイベントを記録するかを指定できます。

セキュリティ ログを管理するには

  1. プライマリ ナビゲーション バーで、[メンテナンス] をクリックします。
  2. [ログ] をクリックします。
  3. [セキュリティ ログ] をクリックします。
