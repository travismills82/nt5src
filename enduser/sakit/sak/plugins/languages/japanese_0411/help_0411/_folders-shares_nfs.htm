NFS

NFS プロトコルを使用すると、サーバー アプライアンスは NFS (Network File System) サーバーとして動作することができます。ユーザーは、さまざまなコンピュータ、オペレーティング システム、およびネットワークが混在する環境でファイルを共有することができます。

NFS クライアント ソフトウェアを実行しているコンピュータ上のユーザーは、サーバー アプライアンス上の共有をそれぞれのコンピュータへ接続する、つまりマウントすることによって、共有にアクセスすることができます。

UNIX コンピュータは、アドバイザリ ロックに従ってすべてのロック要求を処理します。つまり、OS によりファイルへのロック セマンティクスが強制されるのではなく、ロックの存在を確認するアプリケーションによってこれらのロックが効果的に使用されます。ただし NFS プロトコル では、NFS を介して受信するロック要求に対しても必須ロックが実行されます。これにより、NFS を介して実行されるロックが、SMB (Server Message Block) プロトコル、およびファイルへローカルにアクセスするアプリケーションを使って見えるようになります。OS では必須ロックが強制されます。
