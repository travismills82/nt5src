はじめに

Microsoft® Windows® Powered サーバー アプライアンスは、コンピュータ ネットワークに直接接続されます。サーバー アプライアンスは、Web UI を使って管理および監視されます。また、ネットワークに接続された遠隔地のクライアント コンピュータからも管理することができます。この装置には、モニタ、あるいはキーボードやマウスなどの入力装置がない場合があります。サーバー アプライアンスは Microsoft® Windows® 2000 と同じコードに基づいているため、Windows 2000 プラットフォームで使用可能なリモート管理方法の多くが、Windows Powered サーバー アプライアンスでも使用できます。
