FTP プロトコル

FTP (ファイル転送プロトコル) は、Windows セキュリティ モデルに統合されています。FTP を使用して接続するユーザーは、Windows Powered サーバー アプライアンス上のユーザー アカウントに基づいて認証されます。FTP を使用して接続するユーザーは、それぞれのユーザー プロファイルに基づいてアクセスが許可されます。ただし、FTP サーバーが認証に使用するユーザー パスワードは、暗号化されない状態でネットワーク上を送信されます。したがって、ネットワークへの物理アクセスが可能なユーザーは、FTP 検証プロセス時にユーザー パスワードを調べることができます。

FTP は、すべての Microsoft Windows FTP クライアント コマンドをサポートしています。サーバー アプライアンスが FTP を実行している場合、ほかのコンピュータは FTP ユーティリティを使用してサーバーに接続し、ファイルを転送することができます。一方、Microsoft 以外の FTP クライアントでは、FTP サーバー プロトコルがサポートしていないコマンドを使用できます。
