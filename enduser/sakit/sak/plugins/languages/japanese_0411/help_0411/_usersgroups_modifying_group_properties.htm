グループ プロパティの変更

[グループ プロパティ] ページには、[全般] タブと [メンバ] タブが表示されます。[全般] タブでは、グループの名前と説明を設定または変更します。[メンバ] タブでは、ユーザーおよびグループの追加や削除を行います。

グループの名前と説明を設定または変更するには

  1. プライマリ ナビゲーション バーで、[ユーザー] をクリックします。
  2. [ローカル グループ] をクリックします。
  3. 変更するグループ アカウントを選択します。
  4. [タスク] ボックスの一覧で、[プロパティ] をクリックします。
  5. [全般] タブで、目的のグループの名前と説明を入力します。
  6. [OK] をクリックします。

グループのメンバシップを設定または変更するには

  1. プライマリ ナビゲーション バーで、[ユーザー] をクリックします。
  2. [ローカル グループ] をクリックします。
  3. 変更するグループ アカウントを選択します。
  4. [タスク] ボックスの一覧で [プロパティ] をクリックし、[メンバ] タブを選択します。
  5. 新しいメンバを追加します。
    • [ユーザーまたはグループを追加] ボックスの一覧でローカル ユーザーまたはグループを選択し、[追加] ボタンをクリックします。

    または

    • ドメイン グループのドメインとグループ名 (<ドメイン\グループ名>) を入力するか、ドメイン ユーザー アカウント (<ドメイン\ユーザー名>) を入力して、[追加] をクリックします。ただし、<ドメイン\グループ> を追加する場合は、そのドメインからの追加を許可する資格情報も入力しなければなりません。 
  6. メンバを削除するには、[メンバ] ボックスの一覧でユーザー名を選択し、[削除] をクリックします。
  7. [OK] をクリックします。

関連項目
