Web 実行アクセス許可の設定

このページで、サーバー アプライアンス上のサイトの既定のアクセス許可を、"スクリプトおよび実行形式"、"スクリプトのみ"、または "なし" に設定できます。

Web 実行アクセス許可を設定するには

  1. プライマリ タスク バーで、[Web サーバー] をクリックします。
  2. [Web 実行アクセス許可] をクリックします。
  3. [既定の実行アクセス許可] ボックスの一覧で、既定のアクセス許可を選択します。
  4. ボタンをクリックし、既定の値を使用するすべての Web サイトにこの設定を適用します。
  5. [OK] をクリックします。
