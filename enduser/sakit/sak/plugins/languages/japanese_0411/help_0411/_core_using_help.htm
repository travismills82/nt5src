ヘルプの使用

ヘルプでは次の方法で情報を見つけることができます。

カテゴリ別にトピックを参照するには

  • トピックを参照するには、[目次] の一覧でトピック タイトルをクリックします。

ポップ ヒントを表示するには

  • ヒントが必要なページで、プライマリ ナビゲーション バーの右端にある ? アイコンをクリックします。

    注   各ヘルプ ページの右上隅にある ［前のトピック] をクリックすると、ヘルプを逆に移動することができます。

サーバー アプライアンス固有のオンライン ヘルプ以外に、[メンテナンス] ページの [ターミナル サービス] から Microsort Windows 2000 のヘルプにアクセスできます。

Windows 2000 のヘルプを開くには

  1. プライマリ ナビゲーション バーで、[メンテナンス] をクリックします。
  2. [ターミナル サービス] をクリックします。
  3. ユーザー名とパスワードを入力してログオンします。
  4. [スタート] ボタンをクリックし、[ヘルプ] をクリックします。
