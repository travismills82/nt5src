FTP メッセージの表示

開始メッセージ、終了メッセージ、および最大接続数メッセージを表示するには

  1. プライマリ タスク バーで、[Web サーバー] をクリックします。
  2. [FTP メッセージ] をクリックします。
  3. 該当するボックスに、開始メッセージ、終了メッセージ、および最大接続数メッセージを入力します。
    • 開始メッセージ - FTP を介してサーバー アプライアンスにログオンする際に表示されます。
    • 終了メッセージ - FTP を介してサーバー アプライアンスからログオフする際に表示されます。
    • 最大接続数メッセージ - サーバー アプライアンスが処理できる FTP 接続数が最大に達したことをユーザーに通知します。
  4. [OK] をクリックします。
