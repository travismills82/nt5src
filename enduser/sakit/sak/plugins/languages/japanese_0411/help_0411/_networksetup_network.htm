ネットワーク

[ネットワーク] ページでは、次に示すサーバー アプライアンスのネットワーク関連プロパティから、構成する対象を選択できます。

  • 識別 ― サーバー アプライアンスの名前およびドメイン メンバシップを設定します。
  • グローバル設定 ― サーバー アプライアンス上のすべてのネットワーク アダプタに適用されるネットワーク設定を構成します。
  • インターフェイス ― サーバー アプライアンス上の各ネットワーク アダプタのプロパティを構成します。
  • Administrator ― サーバー アプライアンス管理 Web サイトへのアクセスに使用するユーザー アカウントのパスワードを変更します。
  • 管理 Web サイト ― 管理 Web サイトへのアクセスに使用される IP アドレスとポートを指定します。
  • Telnet ― サーバー アプライアンスを管理するために Telnet を構成します。
