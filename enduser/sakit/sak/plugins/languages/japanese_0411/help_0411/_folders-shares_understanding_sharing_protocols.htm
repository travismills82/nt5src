共有プロトコルについて

共有プロトコルを使用すると、クライアント コンピュータはサーバー アプライアンス上のファイルにアクセスできます。

サーバー アプライアンスでは、以下の共有プロトコルのいずれかまたはすべてをサポートします。

  • Windows (CIFS)

    Common Internet File System (CIFS) プロトコルは、Windows OS を稼動しているクライアントによって使用されます。

  • UNIX (NFS)

    Network File System プロトコルは、UNIX を稼動しているクライアントによって使用されます。

  • FTP

    ファイル転送プロトコルは、あらゆる OS からファイル共有にアクセスする代替方法です。

  • Web (HTTP)

    ハイパーテキスト転送プロトコルは Web ブラウザからファイル共有にアクセスするためのプロトコルです。

  • NetWare

    NetWare を稼動しているクライアントからファイル共有にアクセスするためのプロトコルです。

  • AppleTalk

    Macintosh クライアントからファイル共有にアクセスするためのプロトコルです。
