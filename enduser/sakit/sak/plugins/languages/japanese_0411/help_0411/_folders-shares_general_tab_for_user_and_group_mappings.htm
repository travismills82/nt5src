ユーザーとグループのマッピングの構成

NFS ユーザーとグループを関連付けるには

  1. プライマリ ナビゲーション バーで、[共有] をクリックします。
  2. [共有プロトコル] をクリックします。
  3. [NFS プロトコル] をクリックし、[プロパティ] をクリックします。
  4. [ユーザーとグループのマッピング] をクリックします。

NIS サーバーを使用できるように構成するには

  1. [全般] タブで、[NIS サーバーの使用] ボタンをクリックします。
  2. [NIS ドメイン] ボックスに、UNIX ユーザー情報とグループ情報の取得元となるドメインの名前を入力します。
  3. オプション : [NIS サーバー (オプション)] ボックスに、関連付けるサーバーの名前を入力します。
  4. [時間] ボックスと [分] ボックスに、サーバー アプライアンスがユーザー情報とグループ情報を更新する時間間隔を入力します。
  5. [OK] をクリックします。

パスワード ファイルとグループ ファイルを使用できるように構成するには

  1. [パスワード ファイルとグループ ファイルを使用します。] ボタンをクリックします。
  2. [パスワード ファイル] ボックスに、使用するパスワード ファイルの名前を入力します。
UNIX システムで使用するこの passwd 形式ファイルには、関連付けの可能なすべての UNIX ユーザー アカウントが保存されています。
  1. [グループ ファイル] ボックスに、使用するグループ ファイルの名前を入力します。
  2. [OK] をクリックします。
