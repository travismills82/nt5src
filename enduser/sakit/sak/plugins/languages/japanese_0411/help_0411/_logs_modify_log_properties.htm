ログ プロパティの変更

アプリケーション ログ、システム ログ、セキュリティ ログのプロパティを構成することができます。このページでは、ログの最大サイズを指定します。また、ログがサーバー アプライアンスの最大サイズに達したとき、ログ エントリをどのように処理するかを指定することもできます。

ログ ファイルのプロパティを変更するには

  1. プライマリ ナビゲーション バーで、[メンテナンス] をクリックします。
  2. [ログ] をクリックします。
  3. 構成するログの種類 ([アプリケーション]、[システム]、または [セキュリティ]) をクリックします。
  4. 構成するログを選択します。
  5. [プロパティ] をクリックします。
  6. [最大のログ サイズ] ボックスに、ログの最大サイズを KB 単位で入力します。
  7. 該当するボタンをクリックして、ログの最大サイズに達したとき、ログ エントリをどのように処理するかを指定します。
