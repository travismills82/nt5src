NFS クライアント グループの編集

NFS クライアント グループにメンバを追加するには

  1. プライマリ ナビゲーション バーで、[共有] をクリックします。
  2. [共有プロトコル] をクリックします。
  3. [NFS プロトコル] をクリックし、[プロパティ] をクリックします。
  4. [クライアント グループ] をクリックします。
  5. 編集するグループを選択します。
  6. [タスク] ボックスの一覧で、[編集] をクリックします。
  7. [NFS クライアント グループの編集] ページで、グループに追加するメンバの IP アドレスまたはコンピュータ名を入力します。
  8. [追加] をクリックします。
  9. [OK] をクリックします。

NFS クライアント グループからメンバを削除するには

  1. プライマリ ナビゲーション バーで、[共有] をクリックします。
  2. [共有プロトコル] をクリックします。
  3. [NFS プロトコル] をクリックし、[プロパティ] をクリックします。
  4. [クライアント グループ] をクリックします。
  5. 編集するグループを選択します。
  6. [タスク] ボックスの一覧で、[編集] をクリックします。
  7. [NFS クライアント グループの編集] ページで、グループから削除するメンバの IP アドレスまたはコンピュータ名を選択します。
  8. [削除] をクリックします。
  9. [OK] をクリックします。

関連項目

クライアント グループを追加する方法については、「NFS クライアント グループの追加」を参照してください。クライアント グループを削除する方法については、「NFS クライアント グループの削除」を参照してください。
