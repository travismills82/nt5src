Web (HTTP) プロトコル

ハイパーテキスト転送プロトコル (HTTP) は、コンピュータ間で Web を介してハイパーテキスト ドキュメントを転送するための通信プロトコルです。HTTP は、さまざまなコマンドに対して Web サーバーおよびブラウザが実行するアクションを定義します。

World Wide Web サーバー

Web で使用されるコマンドは HTTP で定義されます。

リソースの場所を指定するために、HTTP は URL (Uniform Resource Locator) を使用します。URL は、インターネット上のコンピュータ、ディレクトリ、またはファイルの場所を一意に識別する名前付け規則に従って作成されます。URL は、リソースを取得するために必要な、FTP や HTTP などのインターネット プロトコルも指定します。リソースの URL がわかっている場合は、URL を指定したり、Web ユーザーが使用する文書内からその URL にリンクすることができます。

HTTP プロトコルは、匿名アクセスだけでなく、基本認証および Windows 認証もサポートします。

Web (HTTP) 共有プロパティを構成するには

  1. プライマリ ナビゲーション バーで、[共有] をクリックします。
  2. [共有プロトコル] をクリックします。
  3. [HTTP プロトコル] をクリックし、[プロパティ] をクリックします。
  4. サーバー アプライアンス上のデータ共有へのアクセスをすべての IP アドレスに許可するか、一覧から 1 つの IP アドレスを選択します。
  5. アプライアンス上のデータ共有へのアクセスに使用するポート番号を入力します。既定のポート番号は 80 です。
