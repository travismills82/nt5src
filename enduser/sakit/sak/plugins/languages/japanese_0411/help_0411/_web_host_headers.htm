ホスト ヘッダ

ホスト ヘッダ名は Web サイト アドレスです。この名前を使用すれば、単一の IP アドレスで複数の Web サイトをサポートできます。たとえば会社のサーバーが、マーケティング、経理、人事などの内部の部署ごとに Web サイトをホスティングする場合があります。同じ IP アドレスを使用するこれらの Web サイトを区別するために、それぞれのサイトに一意のホスト ヘッダ名を割り当てます。

次の表は、同じ IP アドレスを使用する一意のホスト ヘッダを使用して、異なる Web サイトを区別する例を示しています。

部署 IP アドレス ホスト ヘッダの例
会社のメイン サイト 172.28.144.10 www.CompanyServer.com
マーケティング 172.28.144.10 www.Marketing.CompanyServer.com
経理 172.28.144.10 www.Accounting.CompanyServer.com
人事 172.28.144.10 www.HumanResources.CompanyServer.com
