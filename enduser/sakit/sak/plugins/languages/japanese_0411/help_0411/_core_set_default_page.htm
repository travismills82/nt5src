既定のページの設定

サーバー アプライアンスの既定のページを設定することができます。設定された既定のページは、サーバー アプライアンスへアクセスするたびに表示されます。開始ページまたは状態ページを、サーバー アプライアンスの既定のページに設定することができます。

サーバー アプライアンスの既定のページを設定するには

  1. プライマリ ナビゲーション バーで、[開始] をクリックします。
  2. [開始] ページで [既定のページの設定] をクリックします。
  3. [既定のページの設定] で、既定として設定するページを選択します。
  4. [OK] をクリックします。
