Web (HTTP)

HTTP は、World Wide Web ブラウザおよびサーバーが情報を交換するために使用するインターネット プロトコルです。プロトコルには、さまざまなコマンドに対して Web サーバーおよびブラウザが実行するアクションが定義されます。ユーザーは、クライアント プログラムを使用して URL を入力したりハイパーリンクを選択することにより、Web サーバーからテキスト、グラフィック、サウンド、およびその他のデジタル情報を取得することができます。Web サーバー上のファイルの URL はすべて http:// で始まります。
