FTP 共有プロパティの設定

このページを使用して、各共有へのアクセスを許可する FTP クライアントを指定します。クライアント ホスト名に基づいてアクセスの許可または拒否を設定できます。

FTP 共有へのクライアント アクセスを許可するには

  1. プライマリ ナビゲーション バーで、[共有] をクリックします。
  2. [共有] ページで、[共有] をクリックします。
  3. FTP クライアント アクセスを追加する共有を選択し、[プロパティ] をクリックします。
  4. [FTP] タブをクリックします。
    • [読み取り] チェック ボックスをオンにして、読み取りアクセスを許可します。
または
  • [書き込み] チェック ボックスをオンにして、書き込みアクセスを許可します。
読み取り専用、書き込み専用、または読み取り/書き込みアクセス許可を有効にできます。
  1. [OK] をクリックします。

FTP 共有へのクライアント アクセスをログに記録するには

  1. プライマリ ナビゲーション バーで、[共有] をクリックします。
  2. [共有] ページで、[共有] をクリックします。
  3. FTP クライアント アクセスを追加する共有を選択し、[プロパティ] をクリックします。
  4. [ＦTP 共有] タブをクリックします。
  5. [アクセスのログ] チェック ボックスをオンにします。
  6. [OK] をクリックします。
