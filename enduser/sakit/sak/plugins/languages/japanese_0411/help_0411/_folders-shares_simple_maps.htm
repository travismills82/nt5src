簡略マップの有効化

簡略マップを有効にすると、同じユーザー名を共有する UNIX ユーザーと Microsoft Windows ユーザーの間に自動マッピングが作成されます。簡略ユーザー マップでは、ユーザー名に基づいて、Windows ドメインのユーザーが UNIX ユーザーに 1 対 1 で暗黙的にマップされます。簡略マップ機能によって Windows ドメイン、UNIX passwd、およびグループ ファイルまたは NIS ドメインが識別され、Windows および UNIX または NIS ドメインで同じ名前を持つユーザーがマップされます。一致するユーザー名がない場合、そのユーザーはマップされません。

注   このページにアクセスするには、[全般] タブで有効な NIS サーバー名を入力しておく必要があります。

簡略マップを有効化するには

  1. プライマリ ナビゲーション バーで、[共有] をクリックします。
  2. [共有プロトコル] をクリックします。
  3. [NFS プロトコル] をクリックし、[プロパティ] をクリックします。
  4. [ユーザーとグループのマッピング] をクリックします。
  5. [簡略マップ] タブをクリックします。
  6. [簡略マップを有効にする] チェック ボックスをオンにします。
  7. [Windows ドメイン] ボックスの一覧で、サーバー アプライアンス名、またはサーバー アプライアンスが属するドメインを選択します。

    サーバー アプライアンス名を選択した場合は、ローカル ユーザーおよびグループがマップされます。

  8. [OK] をクリックします。
