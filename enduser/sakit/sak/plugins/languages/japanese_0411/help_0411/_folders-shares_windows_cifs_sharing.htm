Windows (CIFS) 共有プロパティの設定

Windows クライアント コンピュータは、Common Internet File System (CIFS) プロトコルを使用してファイルを共有します。このページを使用して、共有にアクセスできるユーザー数や関連するキャッシュ オプションを変更したり、ユーザーのアクセス許可を設定または変更します。

アプライアンス設定

[ユーザー制限] セクションで、最大数のユーザー接続を許可するか、同時に実行できる接続数を指定することができます。

ユーザー制限を設定するには

  • [無制限] ボタンをクリックして、処理可能な最大数のユーザーにサーバー アプライアンスへのログオンを許可します。

    または

  • [最大] ボタンをクリックし、接続を許可するユーザー数を指定します。
  • 共有フォルダでのファイルのキャッシュを許可する場合は、[設定] ボックスの一覧で、使用するキャッシュ オプションを指定します。キャッシュ オプションについては、「Windows (CIFS)」で説明しています。

アクセス許可

サーバー アプライアンスへのアクセスを許可または拒否するユーザーまたはグループに対して、アクセス許可を設定することもできます。

ユーザーのアクセス許可を設定するには

  1. [ユーザーまたはグループの追加] ボックスに、アクセス許可の一覧に追加するユーザーまたはグループの名前を入力するか、その下の一覧からユーザーを選択します。
一覧からローカル ユーザーまたはローカル グループを選択できます。ドメイン ユーザーまたはドメイン グループを追加するには、アカウントを、<ドメイン名\ユーザー名> または <ドメイン名\グループ名> のように入力します。
  1. [追加] をクリックします。
  2. [許可] ボックスの一覧を使用して、[アクセス許可] ボックスの一覧で選択されたユーザーがサーバー アプライアンス上のファイルに対して持つ制御のレベルを設定します。
ユーザーに設定できる制御は、制御なし、読み取り専用アクセス、変更アクセス、変更および読み取りアクセス、フル コントロールのいずれかです。
  1. [拒否] ボックスの一覧を使用して、[アクセス許可] ボックスの一覧で選択したユーザーおよびグループに対する制御のレベルを拒否します。
  2. [アクセス許可] の一覧からユーザーまたはグループを削除するには、ユーザーまたはグループを選択して、[削除] をクリックします。
  3. [OK] をクリックします。
