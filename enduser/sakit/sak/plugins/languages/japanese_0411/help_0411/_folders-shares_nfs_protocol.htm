NFS プロトコル

NFS プロトコル オプションを使用すると、NFS サーバーとして動作するようにサーバー アプライアンスを構成できます。ユーザーは、さまざまなコンピュータ、オペレーティング システム、およびネットワークが混在する環境でファイルを共有することができます。

NFS プロトコルを使用すると、NFS クライアント グループ、NFS ロック、および NFS ユーザーとグループのマッピングを管理することができます。ただし、NFS 共有は、Web UI の [共有] セクションで作成されます。

NFS プロトコルを構成するには

  1. プライマリ ナビゲーション バーで、[共有] をクリックします。
  2. [共有プロトコル] をクリックします。
  3. [NFS プロトコル] をクリックし、[プロパティ] をクリックします。
  4. 目的の NFS 構成タスクを選択します。
