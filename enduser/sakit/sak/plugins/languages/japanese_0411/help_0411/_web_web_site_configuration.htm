Web サイト構成

[Web サイトの構成] ページでは、以下のいずれかのタスクを実行できます。

  • Web サイトの作成
  • Web サイト設定の変更
  • Web サイトの削除
  • Web サイトの一時停止
  • Web サイトの停止
  • Web サイトの開始

またこのページでは、特定の値から始まるサイトを検索することもできます。たとえば、"www" で始まるすべてのホスト ヘッダを検索したり、[状態] が [停止] になっているすべてのサイトを検索することができます。

状態が停止になっているすべてのサイトを表示するには

  1. [検索] プロンプトの横にあるドロップ ダウン リストで、[状態] を選択します。
  2. 検索条件として「停止」を入力します。
  3. [開始] をクリックします。
