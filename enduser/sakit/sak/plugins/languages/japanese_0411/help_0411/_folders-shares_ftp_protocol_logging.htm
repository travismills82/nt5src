FTP ログの有効化

[FTP ログ] を有効にすれば、外部からの FTP 接続を FTP ログに記録することができます。既定では、%windir%\system32\logfiles\msftpsvc1 に FTP ログが格納されます。

管理者が自分のワークステーションからこれらのファイルにアクセスするには、管理共有 (\\appliancename\admin$\winnt\system32 など) にアクセスするか、このフォルダに新しい共有を作成します。

FTP ログを有効にするには

  1. プライマリ ナビゲーション バーで、[共有] をクリックします。
  2. [共有プロトコル] をクリックします。
  3. [FTP プロトコル] をクリックし、[プロパティ] をクリックします。
  4. [ログ] タブをクリックします。
  5. [ログを有効] チェック ボックスをオンにして、[OK] をクリックします。
