Web サイトの開始

[Web サイトの構成] ページの [オブジェクト/タスク セレクタ] には、サーバー アプライアンス上の各 Web サイトの状態が表示されます。必要に応じて、停止または一時停止した Web サイトを開始できますが、停止した Web サイトのポートが別のサイトによって使用されている場合は、その Web サイトを開始することはできません。

停止または一時停止した Web サイトを開始するには

  1. プライマリ ナビゲーション バーで、[サイト] をクリックします。
  2. 開始するサイトの Web サイト識別子を選択します。
  3. [タスク] ボックスの一覧で、[開始] をクリックします。
