カスタム FTP メッセージの追加

ユーザーがサーバー アプライアンスに接続したとき、またはサーバー アプライアンスから切断するときに送信される独自の開始メッセージと終了メッセージを作成できます。

カスタム メッセージを追加するには

  1. プライマリ ナビゲーション バーで、[共有] をクリックします。
  2. [共有プロトコル] をクリックします。
  3. [FTP プロトコル] をクリックし、[プロパティ] をクリックします。
  4. [メッセージ] タブをクリックします。
  5. [開始メッセージ] ボックスに、ユーザーがサーバー アプライアンスへ接続したときに表示するメッセージを入力します。
  6. [終了メッセージ] ボックスに、ユーザーがサーバー アプライアンスから切断するときに表示するメッセージを入力します。
  7. [OK] をクリックします。
