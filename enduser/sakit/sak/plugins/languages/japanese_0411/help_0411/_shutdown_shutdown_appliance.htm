サーバー アプライアンスのシャットダウン

このページを使用して、サーバー アプライアンスのシャットダウン、再起動、およびシャットダウンや再起動のスケジュールを行います。

サーバー アプライアンスをシャットダウンまたは再起動するには

  1. プライマリ ナビゲーション バーで、[メンテナンス] をクリックします。
  2. [シャットダウン] をクリックします。
  3. 実行するタスクを選択します。
  4. 選択した内容を確認して、[OK] をクリックします。

    サーバー アプライアンスの再起動を選択した場合、[再起動中] ページが表示されます。

注   [再起動中] ページでは、サーバー アプライアンスがオンラインかどうか定期的にチェックされます。サーバー アプライアンスがオンラインの場合は、既定のページへ自動的に戻ります。

サーバー アプライアンスのシャットダウンまたは再起動をスケジュールするには

  1. プライマリ ナビゲーション バーで、[メンテナンス] をクリックします。
  2. [シャットダウン] をクリックします。
  3. [スケジュールされたシャットダウン] をクリックします。
  4. スケジュールされたシャットダウンの設定を選択します。
  5. [OK] をクリックします。

注   現在スケジュールされているイベントを取り消すには、[スケジュールされたシャットダウンまたは再起動はありません。] ボタンをクリックします。
