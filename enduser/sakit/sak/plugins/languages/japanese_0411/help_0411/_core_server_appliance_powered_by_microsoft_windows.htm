Microsoft® Windows® Powered サーバー アプライアンス

Microsoft® Windows® Powered サーバー アプライアンスのオンライン ヘルプへようこそ。このオンライン ヘルプでは、サーバー アプライアンスを初めて使用する場合の事前の構成方法、サーバー アプライアンスの Web ベース ユーザー インターフェイス (Web UI) の操作方法、およびエンタープライズでのサーバー アプライアンスの使い方について説明します。さらに、Web UI のあらゆる画面でポップ ヒントを利用することができます。ポップ ヒントを開くには、プライマリ ナビゲーション バーの右端に表示される疑問符 (?) をクリックします。
