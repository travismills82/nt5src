Web のログ設定の構成

このページでは、Web ログ設定または IIS ログ設定を構成します。

目的の設定を構成するには

  1. プライマリ タスク バーで、[Web サーバー] をクリックします。
  2. [Web のログ設定] を選択します。
  3. [ログを有効] チェック ボックスをオンにします。
  4. [アクティブなログ形式] ボックスで、適切なログ ファイル形式を選択します。
  5. 適切なボタンをクリックして、新しいログを生成するまでの時間の間隔を設定します。

    たとえば、[毎日] ボタンを選択すると、24 時間ごとに新しいログが開始されます。

  6. [ファイルの名前付けと新しいログ ファイル作成時にサーバーのローカル時間を使います] チェック ボックスをオンにします。

    サーバー アプライアンスでは、ファイルの名前付けと新規ログ ファイル作成時にローカル時間が使用されるようになります。選択をオフにすると、グリニッジ標準時 (GMT) が使用されます。

    注   [ファイルサイズが次のサイズに達したとき (MB)] ボタンをクリックすると、[ファイルの名前付けと新しいログ ファイル作成時にローカル時間を使います] チェックボックスは無効になります。

  7. [ログ ファイルのディレクトリ] ボックスに、ログ ファイルの正しいディレクトリ パスが表示されていることを確認します。
  8. 適切なボタンをクリックし、既定の値を使用するすべての Web サイトに設定を適用します。
  9. [OK] をクリックします。
