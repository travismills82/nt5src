共有の管理

[共有] 管理ページでは、さまざまなファイル共有を作成、削除、または構成したり、開くことができます。[共有] ページには、次の列で構成された [オブジェクト/タスク セレクタ] が表示されます。

  • [共有名] - すべての共有フォルダの名前が一覧表示されます。特定の共有を開いたり、共有を作成または削除したり、共有のプロパティを構成するには、目的の共有名の横にあるチェック ボックスをオンにします。
  • [共有パス] - 共有パスが表示されます。
  • [種類] - 次のいずれかの共有の種類が表示されます。

    W = Windows (CIFS)

    U = UNIX (NFS)

    F = FTP

    H = HTTP

  • [コメント] - 共有についての簡単な説明が指定されている場合、その説明が表示されます。

[オブジェクト/タスク セレクタ] を使用して共有を選択し、[タスク] ボックスの一覧から実行するタスクを選択します。
