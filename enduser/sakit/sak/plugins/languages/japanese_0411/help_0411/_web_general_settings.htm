一般サイトの ID のプロパティ設定

  1. [サイト識別子] ボックスに、Web サイトの一意の名前を入力します。

    Web サイトの識別子は一意でなければなりません。12 文字以内で指定してください。大文字と小文字を使用できますが、" / \ [ ] : ; | = , + * ? < > は使用できません。さらに、ピリオド (.) またはスペースだけの識別子も使用できません。

  2. サーバー アプライアンスでは、既定のディレクトリ パスが入力された [ディレクトリ] ボックスが自動的に表示されます。このパスは変更することができます。
  3. Web サイトの管理アクセスに使用するユーザー アカウントの種類を選択します。ドメイン ユーザー アカウントまたはローカル ユーザー アカウントを作成することも、既存のユーザー アカウントを指定することもできます。

    注   ドメイン アカウントを作成できるのは、サーバー アプライアンスがそのドメインに属している場合のみです。

  4. オプション : 管理者特権を既存のユーザー アカウントに割り当てる場合は、[Web サイト管理者] ボックスに対象となるユーザー名を入力します。

    注   サーバー アプライアンスがドメインに属していて、管理者特権を既存のユーザー アカウントに割り当てる場合、ドメイン名とユーザー名を <ドメイン名>\<ユーザー名> の形式で入力する必要があります。

  5. オプション : ドメイン ユーザー アカウントまたはローカル ユーザー アカウントを作成する場合は、[管理者のパスワード] および [パスワードの確認入力] ボックスに管理者パスワードを入力して確認する必要があります。パスワードは、サーバー アプライアンスが属しているドメインのパスワード作成ルールに準拠していなければなりません。

関連項目

サイトの ID の詳細については、「サイトの ID の変更」を参照してください。アプリケーション設定の詳細については、「アプリケーション設定の作成」を参照してください。
