サーバー アプライアンス名

サーバー アプライアンス名は、ネットワークにおけるサーバー アプライアンスの名前です。サーバー アプライアンス名は一意であること、および一定の条件を満たしていることが必要です。サーバー アプライアンス名には、別のコンピュータまたは Microsoft Windows ドメインと同じ名前を指定できません。

名前は、15 文字以内で指定することをお勧めします。サーバー アプライアンス名には最大 63 文字を使用できます。ただし、使用する文字は数字の 0 〜 9、大文字の A 〜 Z、小文字の a 〜 z、およびハイフンに限定する必要があります。これ以外の文字も使用できますが、それらを使用するとほかのユーザーがネットワーク上でコンピュータを検出できなくなる場合があります。ネットワークが Microsoft DNS サーバーを使用している場合は、ピリオド以外のすべての文字を使用できます。TCP/IP なしでほかのネットワーク プロトコルがインストールされている場合は、サーバー アプライアンス名が 15 文字に制限されます。

15 文字より長いサーバー アプライアンス名を指定する場合に、Microsoft Active Directory ドメインで長い名前が認識されるようにするには、ドメイン管理者が 16 文字以上の DNS 名の登録を可能にする必要があります。
