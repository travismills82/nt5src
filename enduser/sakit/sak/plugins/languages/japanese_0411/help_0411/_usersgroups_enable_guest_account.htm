Guest アカウントの有効化

既定では、Guest アカウントが無効になっています。ワークグループに Windows 95 および Windows 98 クライアント コンピュータが含まれている場合、サーバー アプライアンス上のリソースへアクセスするための最も簡単な方法は Guest アカウントを有効にすることです。ただし、Guest アカウントを有効にすると、ネットワークに接続するすべてのユーザーがサーバー アプライアンス上のリソースへアクセスできるようになります。ほかの方法として、ネットワーク上の各ユーザーに対してユーザー アカウントを作成する方法があります。

Guest アカウントを有効にするには

  1. プライマリ ナビゲーション バーで、[ユーザー] をクリックします。
  2. [ローカル ユーザー] をクリックします。
  3. [名前] 列で、[Guest] をクリックします。
  4. [タスク] ボックスの一覧で、[プロパティ] をクリックします。
  5. [全般] タブで、[このユーザー アカウントを無効にする] チェック ボックスをオフにします。
  6. [OK] をクリックします。
