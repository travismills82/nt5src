ログ

ログ ファイルには、アプリケーション、サービス、または Microsoft® Windows® によって生成されたメッセージやイベント ログが保存されます。これらのメッセージは、サーバー アプライアンスで実行された操作を調査するときに使用します。

[ログ] 機能を使用すれば、システムによって生成される次の種類のイベント ログを表示、消去、ダウンロード、および構成することができます。

  • アプリケーション ログの管理
  • FTP ログの管理
  • NFS ログの管理
  • セキュリティ ログの管理
  • システム ログの管理
  • Web (HTTP) 共有ログの管理
  • Web 管理ログの管理
  • ログ ファイルの消去
  • ログ ファイルのダウンロード
  • ログ プロパティの変更
  • ログの詳細の表示
