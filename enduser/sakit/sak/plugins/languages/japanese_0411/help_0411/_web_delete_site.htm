Web サイトの削除

サーバー アプライアンス上に作成した Web サイトを削除することができます。

Web サイトを削除するには

  1. プライマリ ナビゲーション バーで、[サイト] をクリックします。
  2. 削除する Web サイトを選択します。
  3. [タスク] ボックスの一覧で、[削除] をクリックします。
  4. 選択されている Web サイトが、削除するサイトであることを確認します。
  5. [OK] をクリックします。
