ユーザー アカウントの追加

ユーザー アカウントを追加するには、ユーザー名、ユーザーの氏名、アカウントの簡単な説明、アカウント パスワードを指定する必要があります。

ユーザー名は一意でなければなりません。また、既存のグループ名と重複する名前も使用できません。

ユーザー名は、管理対象のコンピュータで使用されているほかのすべてのユーザー名およびグループ名と同じであってはなりません。ユーザー名は 20 文字以内で指定してください。大文字と小文字を使用できますが、" / \ [ ] : ; | = , + * ? < > は使用できません。さらに、ピリオド (.) またはスペースだけのユーザー名も使用できません。

[パスワード] ボックスと [パスワードの確認入力] ボックスには、最大 127 文字のパスワードを入力できます。ただし、Microsoft Windows 95 や Microsoft Windows 98 を使用しているコンピュータが接続されているネットワーク上で Microsoft® Windows® 2000 を使用している場合は、14 文字以内のパスワードを使用してください。Windows 95 および Windows 98 では、14 文字までのパスワードしか使用できません。パスワードがそれよりも長いと、これらのコンピュータからネットワークにログオンできなくなります。

管理者グループに新しく追加されたユーザーだけが管理タスクを実行できます。

ユーザー アカウントを追加するには

  1. プライマリ ナビゲーション バーで、[ユーザー] をクリックします。
  2. [ユーザー] ページで、[ローカル ユーザー] をクリックします。
  3. [タスク] ボックスの一覧で、[新規...] をクリックします。
  4. 新しいユーザー アカウントの情報を入力します。

    [ホーム ディレクトリ] では、新しく作成するディレクトリを指定します。ユーザーには、このディレクトリへの排他アクセス権が与えられます。このディレクトリには、上記で指定したユーザー名と同じ名前が付けられ、指定したパスに格納されます。

  5. [OK] をクリックします。
