FTP マスタ設定

[匿名 FTP 接続を許可する] を使うと、FTP クライアントからサーバー アプライアンスへの匿名アクセスを有効または無効にすることができます。

[FTP ルート ディレクトリ] ボックスには、匿名 FTP を使用して接続する際にアクセスする FTP ルート ディレクトリが表示されます。

[ディレクトリの一覧表示のスタイル] ボックスでは、FTP クライアントが LIST 操作を送信したときにディレクトリを一覧表示する形式を定義します。

  • MS-DOS の既定の一覧表示のスタイルでは、西暦が 2 桁で表示されます。
  • UNIX の一覧表示のスタイルでは、ファイルの日付が FTP サーバー上の西暦と異なる場合、西暦は 4 桁で表示されます。ファイルとサーバーの西暦が同じ場合、西暦は返されません。

[最大接続数] は、サーバー アプライアンスへの同時 FTP 接続の最大許容数です。

[接続タイムアウト (秒)] は、サーバー アプライアンスが自動的に切断されるまでの、アクティビティがない状態の秒数です。
