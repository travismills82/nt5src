ログ ファイルの消去

このページでは、特定のログ ファイルを消去できます。アプリケーション ログ、NFS ログ、セキュリティ ログ、またはシステム ログを消去すると、指定した種類のログ ファイルがすべて消去されます。ただし、FTP ログ、Web 管理ログ、Web (HTTP) 共有ログを消去する場合は、消去する特定のログ ファイルを選択することができます。

アプリケーション ログ、NFS ログ、セキュリティ ログ、システム ログを消去するには

  1. プライマリ ナビゲーション バーで、[メンテナンス] をクリックします。
  2. [ログ] をクリックします。
  3. 消去するログの種類を選択します。
  4. [ログの消去] をクリックします。
  5. [OK] をクリックします。

FTP ログ、Web 管理ログ、Web (HTTP) 共有ログを消去するには

  1. プライマリ ナビゲーション バーで、[メンテナンス] をクリックします。
  2. [ログ] をクリックします。
  3. 消去するログの種類を選択します。
  4. 消去する 1 つまたは複数のログ ファイルを選択します。
  5. [ログの消去] をクリックします。
  6. [OK] をクリックします。
