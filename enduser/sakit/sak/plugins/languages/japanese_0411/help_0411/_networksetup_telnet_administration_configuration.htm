Telnet

Telnet 管理構成機能を使用すれば、リモート システムから Telnet プロトコルを使って Windows® Powered サーバー アプライアンスを管理することができます。リモートの Telnet クライアントからシステムにログオンし、サーバー アプライアンス上で文字モードのアプリケーションを実行できます。サーバー アプライアンス付属の Telnet サーバーは、サーバー アプライアンス ハードウェアの製造元が特に指定しない限り、一度に最大 2 つの Telnet クライアントをサポートします。

サーバー アプライアンスで Telnet 管理を構成するには

  1. プライマリ ナビゲーション バーで、[ネットワーク] をクリックします。
  2. [Telnet] をクリックします。
  3. [このアプライアンスへの Telnet によるアクセスを有効にします] チェック ボックスをオンにします。
  4. [OK] をクリックします。
