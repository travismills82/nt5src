アプリケーション ログの管理

アプリケーション ログには、アプリケーションまたはプログラムによって記録されたイベントが保存されます。たとえば、ワードプロセッシング プログラムは、ファイル エラーなどをアプリケーション ログに記録します。記録されるイベントはアプリケーションによって異なります。

アプリケーション ログを管理するには

  1. プライマリ ナビゲーション バーで、[メンテナンス] をクリックします。
  2. [ログ] をクリックします。
  3. [アプリケーション ログ] をクリックします。
