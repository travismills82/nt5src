グローバル設定

このページでは、IPX 設定を構成し、使用する DNS サフィックス、LMHOSTS ファイル、および HOSTS ファイルを指定することで、サーバー アプライアンスの全体的なネットワーク設定を変更できます。LMHOSTS および HOSTS ファイルは、あらゆるコンピュータまたはデバイスの名前を解決するために使用されます。ここで使用する DNS サフィックスは、サーバー アプライアンスがホストまたはドメインの名前を解決する際に適用されます。

DNS サフィックスを自動的に設定または変更するには

  1. プライマリ ナビゲーション バーで、[ネットワーク] をクリックします。
  2. [グローバル設定] をクリックします。
  3. [DNS 解決] タブをクリックします。
  4. [プライマリ DNS サフィックスの追加] をクリックします。
  5. オプションとして、[プライマリ DNS サフィックスの親サフィックスを追加] チェック ボックスをオンにすることができます。
  6. [OK] をクリックします。

特定の DNS サフィックスを手動で追加するには

  1. [ネットワーク] ページで、[グローバル設定] をクリックします。
  2. [DNS 解決] タブをクリックします。
  3. [使用にあたって、次の DNS サフィックスを追加してください] をクリックします。
  4. 追加する DNS サフィックスを [ドメイン サフィックス] ボックスに入力し、[追加] をクリックします。
  5. [OK] をクリックします。

特定の DNS サフィックスを手動で削除するには

  1. [ネットワーク] ページで、[グローバル設定] をクリックします。
  2. [DNS 解決] タブをクリックします。
  3. [使用にあたって、次の DNS サフィックスを追加してください] をクリックします。
  4. [上へ] および [下へ] ボタンを使って、ドメイン サフィックスの一覧をスクロールします。削除するサフィックスを選択し、[削除] をクリックします。
  5. [OK] をクリックします。
