Web サイトの作成

[Web サイトの作成] ページでは、新しい Web サイトを作成してプロパティを構成することができます。

Web サイトを作成するには

  1. プライマリ ナビゲーション バーで、[サイト] をクリックします。
  2. [タスク] ボックスの一覧で、[作成...] をクリックします。
  3. 必要な情報を入力し、新しい Web サイトを作成します。
  4. [OK] をクリックします。
