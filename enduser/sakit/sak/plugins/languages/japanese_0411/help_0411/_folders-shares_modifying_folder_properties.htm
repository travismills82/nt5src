フォルダのプロパティの変更

[フォルダのプロパティ] ページでは、フォルダ名の設定または変更、フォルダの種類、サイズ、格納場所に関する詳細情報の取得、フォルダ内のデータの圧縮を実行できます。

フォルダの名前を変更するには

  1. プライマリ ナビゲーション バーで、[共有] をクリックします。
  2. [共有] ページで、[フォルダ] をクリックします。
  3. [オブジェクト/タスク セレクタ] を使用して、変更するフォルダが格納されているディレクトリに移動します。
  4. 目的のフォルダのチェック ボックスをオンにします。
  5. [タスク] ボックスの一覧で、[プロパティ] をクリックします。
  6. [全般] タブをクリックし、[名前] ボックスに新しいフォルダ名を入力します。
  7. [OK] をクリックします。

フォルダを圧縮するには

  1. プライマリ ナビゲーション バーで、[共有] をクリックします。
  2. [共有] ページで、[フォルダ] をクリックします。
  3. [オブジェクト/タスク セレクタ] を使用して、新しいフォルダを追加するディレクトリに移動します。
  4. [タスク] ボックスの一覧で、[プロパティ] をクリックします。
  5. [圧縮] タブをクリックし、[このフォルダの内容を圧縮してディスク領域を節約する] チェック ボックスをオンにします。
  6. [このフォルダだけに変更を適用] または [このフォルダ、サブフォルダおよびファイルに変更を適用] を選択します。
  7. [OK] をクリックします。
