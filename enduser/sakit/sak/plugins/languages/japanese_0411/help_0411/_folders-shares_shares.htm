共有

コンピュータが Microsoft Windows オペレーティング システム (OS) を稼動しているか UNIX OS を稼動しているかに関係なく、サーバー アプライアンス上のフォルダは、ネットワーク上のほかのユーザーと共有することができます。

使用可能なファイル共有プロトコルはサーバー アプライアンスの製造元によって異なることもありますが、サーバー アプライアンスでは次のプロトコルをサポートできます。

  • Windows (CIFS)- Common Internet File System (CIFS) プロトコルは、Windows OS を稼動しているクライアントによって使用されます。
  • NFS- Network File System プロトコルは、UNIX を稼動しているクライアントによって使用されます。
  • FTP- ファイル転送プロトコルは、あらゆる OS からファイル共有にアクセスする代替方法です。
  • HTTP- ハイパーテキスト転送プロトコルは Web ブラウザからファイル共有にアクセスするためのプロトコルです。
  • NetWare- NetWare を稼動しているクライアントからファイル共有にアクセスするためのプロトコルです。
  • AppleTalk- Macintosh クライアントからファイル共有にアクセスするためのプロトコルです。

サーバー アプライアンスに共有を作成する場合は、一覧表示され、サーバー アプライアンス上で現在有効になっている、あるいはインストールされているプロトコルの、いずれかまたはすべてを有効にすることができます。
