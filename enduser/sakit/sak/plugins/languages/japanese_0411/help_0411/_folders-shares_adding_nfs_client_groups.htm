NFS クライアント グループの追加

NFS クライアント グループを追加するには

  1. プライマリ ナビゲーション バーで、[共有] をクリックします。
  2. [共有プロトコル] をクリックします。
  3. [NFS プロトコル] をクリックし、[プロパティ] をクリックします。
  4. [クライアント グループ] をクリックします。
  5. [タスク] ボックスの一覧で、[新規...] をクリックします。
  6. [新しい NFS クライアント グループ] ページの [グループ名] ボックスに、追加するグループの名前を入力します。
  7. [クライアント名または IP アドレス] ボックスに、そのグループに追加するコンピュータの名前または IP アドレスを入力します。
  8. [追加] をクリックします。
  9. [OK] をクリックします。

関連項目

クライアント グループを編集する方法については、「NFS クライアント グループの編集」を参照してください。クライアント グループを削除する方法については、「NFS クライアント グループの削除」を参照してください。
