サイトの ID の変更

[サイトの ID] ページでは、サーバー アプライアンス の IP アドレスや TCP ポート番号を割り当てるだけでなく、作成または修正する Web サイトにホスト ヘッダ名を追加したり、既存のホスト ヘッダ名を変更することができます。

[IP アドレス] は既定で [未割り当てすべて] に設定されています。サーバー アプライアンスにインストールされたすべてのネットワーク インターフェイス カード (NIC) の IP アドレスが一覧表示されます。

[TCP ポート] は既定で 80 に設定されています。この既定のポート番号は業界標準であり、たとえば www.yoursite.com のようにブラウザに Web アドレスを入力する際に認識されます。ポート番号を 8055 など 80 以外に変更する場合は、www.yoursite.com:8055 のように、Web サイト アドレスにコロンとそのポート番号を含める必要があります。

サイトの ID のパラメータを変更するには

  1. プライマリ ナビゲーション バーで、[サイト] をクリックします。
  2. 設定を変更する Web サイトを選択します。
  3. [タスク] ボックスの一覧で、[変更] をクリックします。
  4. [サイトの ID] タブをクリックします。
  5. [IP アドレス] ボックスの一覧で、Web サイトに割り当てる IP アドレスを選択します。
  6. [TCP ポート] ボックスに、使用する TCP ポート番号を入力します。
  7. [ホスト ヘッダ] ボックスに、ホスト ヘッダ名を入力します。
  8. [OK] を選択するか、その他のタブをクリックして Web サイト プロパティをさらに変更します。
