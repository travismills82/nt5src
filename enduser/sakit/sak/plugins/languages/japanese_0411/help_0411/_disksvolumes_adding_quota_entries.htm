クォータ エントリの追加

新しいクォータ エントリを追加するには

  1. プライマリ ナビゲーション バーで、[ディスク] をクリックします。
  2. [ディスク クォータ] をクリックします。
  3. 管理するボリュームを選択します。
  4. [タスク] ボックスの一覧で、[クォータのエントリ] をクリックします。
  5. [タスク] ボックスの一覧で、[新規...] をクリックします。
  6. 一覧からローカル ユーザーを選択するか、またはボックスに <ドメイン名\ユーザー名> の形式でドメイン アカウントの名前を入力します。

無制限のディスク使用を許可するには

  • [ディスクの使用を制限しない] ボタンをクリックします。

ディスク領域を制限するには

  1. [ディスク領域を制限する] ボタンをクリックします。
  2. ボックスに、特定のユーザーまたはグループに割り当てるディスク領域の大きさを指定する数値を入力します。一覧から、KB (キロバイト)、MB (メガバイト)、GB (ギガバイト)、TB (テラバイト)、PB (ペタバイト)、または EB (エクサバイト) を指定します。
  3. ディスク領域の使用量が上限に近づいていることをユーザーまたはグループに警告する、基準となるディスク使用量を入力します。一覧から、KB、MB、GB、TB、PB、または EB を指定します。
  4. [OK] をクリックします。
