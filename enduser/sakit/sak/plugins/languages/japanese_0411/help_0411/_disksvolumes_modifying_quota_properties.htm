クォータ プロパティの更新

クォータ エントリのプロパティを更新するには

  1. プライマリ ナビゲーション バーで、[ディスク] をクリックします。
  2. [ディスク クォータ] をクリックします。
  3. 管理するボリュームを選択します。
  4. [タスク] ボックスの一覧で、[クォータのエントリ] をクリックします。
  5. 選択したボリュームの [クォータのエントリ] ページで、ユーザー アカウントを選択します。
  6. [タスク] ボックスの一覧で、[プロパティ] をクリックします。
  7. [ユーザーのクォータ エントリ] ページで、次のタスクのいずれかを実行します。

    無制限のディスク使用を許可するには

    • [ディスクの使用を制限しない] ボタンをクリックします。

    または

    ディスク領域を制限するには

    1. [ディスク領域を制限する] ボタンをクリックします。
    2. ボックスに、特定のユーザーまたはグループに割り当てるディスク領域の大きさを指定する数値を入力します。一覧から、KB、MB、GB、TB、PB、または EB を指定します。
    3. ディスク領域の使用量が上限に近づいていることをユーザーまたはグループに警告する、基準となるディスク使用量を入力します。一覧から、KB、MB、GB、TB、PB、または EB を指定します。
    4. [OK] をクリックします。

  2. [ディスク クォータ] をクリックします。
  3. 管理するボリュームを選択します。

  5. 選択したボリュームの [クォータのエントリ] ページで、複数のユーザー アカウントを選択します。
  6. [タスク] ボックスの一覧で、[プロパティ] をクリックします。
  7. [ユーザーのクォータ エントリ] ページで、次のタスクのいずれかを実行します。

    • [ディスクの使用を制限しない] ボタンをクリックします。

    2. ボックスに、特定のユーザーまたはグループに割り当てるディスク領域の大きさを指定する数値を入力します。一覧から、KB、MB、GB、TB、PB、または EB を指定します。
    3. ディスク領域の使用量が上限に近づいていることをユーザーまたはグループに警告する、基準となるディスク使用量を入力します。一覧から、KB、MB、GB、TB、PB、または EB を指定します。
