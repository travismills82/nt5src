Setting the Date and Time

Using the Date/Time page, you can set the date, time, and time zone used by the server.

To set the date, time, and time zone of the server

  1. On the primary navigation bar, choose Maintenance.
  2. Choose Date/Time.
  3. Type the date and time in the format defined by the default language selected in your browser. 
  4. Select the appropriate time zone from the list.

    You can also automatically adjust the server for daylight saving changes, which is recommended.

  5. Choose OK.
