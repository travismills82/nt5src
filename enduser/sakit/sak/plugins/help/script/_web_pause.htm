Pausing a Web Site

The Object/Task Selector on the Web Site Configuration page indicates the status of each Web site residing on your server. If needed, you can pause any Web site that is running.

Pausing a Web site prevents Internet services from accepting new connections but does not affect requests that are already being processed. Stopping a Web site prevents Internet services from accepting new connections and will stop all requests that are already being processed.

To pause a running Web site

  1. From the primary navigation bar, choose Sites.
  2. Select the Web site identifier for the site you want to pause.
  3. In the Tasks list, choose Pause.
