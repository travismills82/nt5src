Starting a Web Site

The Object/Task Selector on the Web Site Configuration page indicates the status of each Web site residing on your server. If needed, you can start any Web site that is stopped or paused, however, if the port for a stopped Web site is in use by another site, the stopped Web site cannot be started.

To start a stopped or paused Web site

  1. From the primary navigation bar, choose Sites.
  2. Select the Web site identifier for the site you want to start.
  3. In the Tasks list, choose Start.
