Maintenance

From the main Maintenance page, users can perform the following general server maintenance tasks:

  • Date and Time—Set the date and time on the server.
  • Shutdown—Shut down or restart the server.
  • Logs—View, download, configure, and clear event logs.
  • Remote Desktop—Manage all aspects of the server by connecting to it using Remote Desktop.
  • Alert E-Mail—Set alert e-mail on the server.
  • Language—Change the language used by the server.
