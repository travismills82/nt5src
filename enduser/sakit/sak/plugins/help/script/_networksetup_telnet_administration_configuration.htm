Telnet

The Telnet Administration Configuration feature allows you to administer your server from a remote system using the Telnet protocol. You can log onto the system from a remote Telnet client and run character-mode applications on the server. The Telnet server included with your Windows server software supports a maximum of two Telnet clients at any time, unless otherwise specified by your server hardware manufacturer.

To configure your server for Telnet administration

  1. On the primary navigation bar, choose Network.
  2. Choose Telnet.
  3. Select the Enable Telnet access to this server check box.
  4. Choose OK.
