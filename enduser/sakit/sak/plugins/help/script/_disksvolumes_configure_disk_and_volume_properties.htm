Configuring Disk and Volume Properties

From this page, you can choose to configure the disks or volumes on the server.

To manage disks and volumes on the server

  1. On the primary navigation bar, choose Disks.
  2. Choose Disks and Volumes.
  3. Log on to Remote Desktop Connection.
  4. When you are finished, close the application.
    Note   It may take a few moments for the Remote Desktop Connection session to log off when closing the application.

Remote Desktop Connection

Remote Desktop Connection is a general purpose tool that gives you full access to your server. When accessed using the Disks and Volumes page, Remote Desktop Connection assumes a dedicated mode and can only be used to manage disks and volumes on the server. If you navigate to another page during an open session, the session will be closed. Remote Desktop supports only two concurrent connections.
