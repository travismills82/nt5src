Setting the Default Page

You can set the default page of the Web UI for Windows Server Administration. This will be the page that displays each time you access the server. You may choose to have the default page of the Web UI be the Welcome page or the Status page.

To set the default page of the server

  1. From the primary navigation bar, choose Welcome.
  2. From the Welcome page, choose Set Default Page.
  3. On the Set Default Page, choose which page to set as the default.
  4. Choose OK.
