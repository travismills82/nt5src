Deleting a Web Site

You can remove any Web site you have created on the server.

To delete a Web site

  1. From the primary navigation bar, choose Sites.
  2. Select the Web site you wish to remove.
  3. In the Tasks list, choose Delete.
  4. Verify the Web site indicated is the one you want to remove.
  5. Choose OK.
