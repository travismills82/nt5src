Web Site Configuration

From the Web Site Configuration page you can complete one of the following tasks:

  • Creating a Web site
  • Modifying Web site settings
  • Deleting a Web site
  • Pausing a Web site
  • Stopping a Web site
  • Starting a Web site

The one caveat to modifying Web sites via the Web UI for Server Administration, however, is that although all available Web sites are listed in the Object/Task Selector, you can only modify those sites that were originally created via the Web UI. Your option for modifying other sites is limited to pausing, starting, and stopping them.

This page also allows you to search for sites that begin with a certain value. For example you could search for all host headers that begin "www," or all sites that have a Status of Stopped.

To display all sites that have a status of Stopped

  1. Select the Status value from the drop down list next to the Search prompt.
  2. Enter Stopped as the search criteria.
  3. Choose Go.
