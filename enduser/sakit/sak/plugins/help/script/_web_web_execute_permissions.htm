Setting Web Execute Permissions

This page allows you to set the default permissions for the sites on the server to scripts and executables, scripts only, or none.

To set the Web execute permissions

  1. On the primary taskbar, choose Web Server.
  2. Choose Web Execute Permissions.
  3. Select the default permission you want from the Default Execute Permissions list.
  4. Select the button to apply this setting to all Web sites that use the default value.
  5. Choose OK.
