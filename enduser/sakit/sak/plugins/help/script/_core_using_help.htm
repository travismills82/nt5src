Using Help

You can locate information in Help by using any of the following procedures:

To browse through topics by category

  • To browse through the topics, click the topic title in the Table of Contents list.

To invoke context-sensitive Help

  • From the page for which you want assistance, click the ? icon at the right end of the primary navigation bar.
    Note   You can move backwards in Help by clicking the Previous Topic link in the upper right corner of each Help page.

In addition to the online Help specific to the Web UI, you can access general Microsoft Windows server Help using Remote Desktop from the Maintenance page.

To open Windows Help

  1. On the primary navigation bar, choose Maintenance.
  2. Choose Remote Desktop.
  3. Log on using your user name and password.
  4. Click the Start button, and then choose Help.
