Setting Web (HTTP) Sharing Properties

Use this page to specify which Web clients are granted access to each Web share. Access can be granted or denied on the basis of client host name. Access can also be granted or denied on the basis of client groups, where a client group contains one or more client host names.

To allow clients permission to a Web share

  1. On the primary navigation bar, choose Shares.
  2. On the Shares page, choose Shares.
  3. Select the share for which you want to add Web client access, and then choose Properties.
  4. From the General tab, select the Web (HTTP) check box.
  5. Select the Web Sharing tab, and then:
    • Select the Read check box to allow read access.

    - or -

    • Select the Write check box to allow write access.
  6. Choose OK.
