Configuring FTP Log Settings

This page is used to configure the FTP log settings.

To configure FTP log settings

  1. On the primary taskbar, choose Web Server.
  2. Choose FTP Log Settings.
  3. Select the Enable Logging check box.
  4. Select the appropriate log file format in the Active Log Format box.
  5. Select the appropriate button to set the time interval before each new log is generated.

    For example, if you select the Daily button, a new log will begin every 24 hours.

  6. Select the Use server’s local time for file naming and rollover check box.

    The server will use its local time for file naming and log rollover. Otherwise, the server will use Greenwich Mean Time (GMT).

    Note   If you select the When file size reaches button, the Use local time for file naming and rollover check box will be disabled.
  7. Verify that the Log File Directory box contains the correct directory path for your log files.
  8. Choose OK.
