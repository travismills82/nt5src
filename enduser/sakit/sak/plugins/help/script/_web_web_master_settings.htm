Web Master Settings

Using this page, you can configure generic Web server settings that apply to all Web sites on your server.

The default directory path is displayed in the Web Site Root Directory text box. You can change the default directory path by typing a new path in the text box, but doing so changes the default directory for any new Web sites that you create. Web sites created before you changed the default directory path remain in their original directories.

The following options are also available:

  • ASP script time-out, in seconds—Specifies the length of time ASP will allow a script to run. If the script does not finish running by the end of the timeout period, ASP stops the script and writes an event to the Windows event log. You can set the timeout period to a value between 1 and 2147483647. You can override this option in an ASP page by using the Server.ScriptTimeout method.
  • Enable FrontPage Extensions by default—Specifies that Microsoft FrontPage extensions are enabled by default whenever a user creates a new Web site using the Web UI.
Note   In all cases, the check boxes must be selected to enable the functionality.

Finally, you can set the maximum number of connections to Web sites hosted on your server.
