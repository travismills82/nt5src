Understanding Sharing Protocols

Sharing protocols allow client computers to access files on your server.

Your server may support any or all of the following types of sharing protocols:

  • Windows (CIFS)

    The Common Internet File System (CIFS) protocol is used by clients running a Windows OS.

  • UNIX (NFS)

    The Network File System protocol is used by clients running UNIX.

  • FTP

    The File Transfer Protocol is an alternative way of accessing a file share from any OS.

  • Web (HTTP)

    The Hypertext Transfer Protocol is the protocol for accessing a file share from Web browsers.

  • AppleTalk

    The protocol for accessing a file share from Macintosh clients.
