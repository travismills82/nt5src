Web User Interface for Windows Server Administration

Welcome to the online Help for your Windows server administration Web user interface. You can use the online Help to learn how to configure your server before first use, how to navigate through the Windows server Web-based user interface (Web UI), and how to use the server in your enterprise. Additionally, context-sensitive Help is available from any screen in the Web UI. To open context-sensitive Help, choose the question mark (?) at the far right of the primary navigation bar.
