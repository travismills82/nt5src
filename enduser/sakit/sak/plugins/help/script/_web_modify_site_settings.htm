Modifying Web Site Settings

Once your Web site has been created, you can change the general, site identity, and application settings as necessary.

To modify the settings of an existing Web site

  1. From the primary navigation bar, choose Sites.
  2. Select the Web site you want to modify.
  3. In the Tasks list, choose Modify.

    The Modify Web Site page opens and the General tab is active. You can change the settings on this tab, or you can make changes to any of the settings available on the Site Identities or Application Settings tabs.
