FTP Master Settings

The Enable FTP for updating content check box, when selected, creates a separate FTP site for uploading content that has read, write, and log visit properties enabled. This FTP site also prevents anonymous access. Selecting this check box also stops the Default FTP Site and starts the FTP service for this server and the Web Site Content FTP Site.

Directory Listing Style box defines the format in the directories that are listed when an FTP client sends a LIST operation.

  • The MS-DOS listing style displays the year in two-digit format by default.
  • The UNIX listing style displays the year in four-digit format when the date of the file is different from the year on the FTP server. If the file year is the same as the server year, no year date is returned.

Maximum connections is the maximum number of allowable, simultaneous FTP connections to the server.

Connection time-out, in seconds is the number of seconds of no activity before the server automatically disconnects.
