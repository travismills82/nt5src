Displaying FTP Messages

To display Welcome, Exit, and Max Connections messages

  1. On the primary taskbar, choose Web Server.
  2. Choose FTP Messages.
  3. Type your Welcome, Exit and Max Connections messages in the appropriate boxes:
    • Welcome Message—Users will see this message when they log onto the server through FTP.
    • Exit Message—This message will display to users as they log off of the server through FTP.
    • Max Connections Message—This message informs a user that the maximum number of FTP connections the server can handle has been reached.
  4. Choose OK.
