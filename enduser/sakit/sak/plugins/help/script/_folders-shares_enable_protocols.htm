Enabling Protocols

Microsoft recommends that you enable only the necessary network protocols. Limiting the number of enabled network protocols will enhance the performance of other network protocols. Additionally, if a problem is encountered with a network or dial-up connection, the system will attempt to establish connectivity by using every network protocol that is installed and enabled. By only enabling the protocols that your system can use, the server can conserve resources and perform better.

To enable a sharing protocol

  1. On the primary navigation bar, choose Shares.
  2. Choose Sharing Protocols.
  3. Select the protocol you want to enable. Only protocols available on this server are displayed.
  4. In the Tasks list, choose Enable.
