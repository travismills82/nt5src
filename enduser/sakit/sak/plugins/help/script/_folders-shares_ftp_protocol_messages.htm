Adding Custom FTP Messages

You can create customized welcome and exit messages that are sent to users when they connect or disconnect from the server.

To add custom messages

  1. On the primary navigation bar, choose Shares.
  2. Choose Sharing Protocols.
  3. Select FTP Protocol, and then choose Properties.
  4. Select the Messages tab.
  5. In the Welcome message box, type the message that will greet users when they connect to the server.
  6. In the Exit message box, type the message that will appear when users disconnect from the server.
  7. Choose OK.
