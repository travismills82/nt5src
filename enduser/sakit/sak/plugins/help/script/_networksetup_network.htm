Network

From the Network page, you can choose which of the following network-related properties of the server to configure:

  • Identification—Set the name and domain membership of the server.
  • Global Settings—Configure network settings that apply to all network adapters on the server.
  • Interfaces—Configure the properties of each network adapter on the server.
  • Administrator—Change the password of the user account you are using to access the server administration Web site.
  • Administration Web Site—specify which IP address(es) and port are used to access the administration Web site.
  • Telnet—Configure Telnet to administer the server.
