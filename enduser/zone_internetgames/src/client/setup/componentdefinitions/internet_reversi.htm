Internet Reversi

The Internet Reversi component provides the necessary files for the Internet Reversi game client portion of the Internet Games.

This component includes the following files and applications:

  • Rvsezm.exe – An executable that launches zclientm.exe with the necessary settings to run Internet Reversi.
  • Rvse.chm – Help file for Internet Reversi.
  • Rvse.dll – Game logic for Internet Reversi.
  • Rvseres.dll – Resources for Internet Reversi (art, sounds, strings, etc.).

The executable rvsezm.exe launches the Internet Games Core client application, instructing it to use the other three files as the specific game help, logic, and resources.

Component Configuration

There are no configuration requirements for this component.

Special Notes

The Internet Reversi component requires the Internet Games Core component.
