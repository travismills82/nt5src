<*-1-*>

Tasks

View Permissions

This allows you to view the permissions in this permission set and to view and modify individual permissions by right-clicking them in the permission list.

<*-2-*>

Change Permissions

The Configure an Assembly wizard allows you to add or remove permissions from this permission set. This wizard also allows you to modify permissions in this permission set by selecting a permission and clicking Properties.

Rename Permission Set

This option allows you to give this permission set another name. Be careful when renaming permission sets, as code groups depending on the old name might also need to be updated.
